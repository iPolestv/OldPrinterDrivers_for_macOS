Problemy związane z transmisją

W przypadku problemów dotyczących transmisji należy przeprowadzić następujące czynności rozwiązywania problemów.
Problem
Co należy zrobić
FAX XXXXXX (twoja drukarka) nie wyświetla się na liście drukarek.
Upewnij się, że jest zainstalowany sterownik drukarki i sterownik PC-FAX oraz, że sterownik PC-FAX został dodany do listy drukarek.
Przycisk Fax (Faks) jest wyszarzony (niedostępny).
Ten przycisk będzie niedostępny, aż do dodania odbiorcy do Recipient List (Lista adresatów). Określ co najmniej jednego odbiorcę.
Nie można wysłać faksów z drukarki.
  • Po wyświetleniu “No dial tone detected (Nie wykryto sygnału)”, wybierz Fax Settings (Ustawienia faksu) - Dial Tone Detection Settings (Ustawienia wykrywania sygnału wybierania) i usuń okno opcji dla Detect Dial Tone (Wykrywaj sygnał wybierania).
  • Upewnij się, że ustawiony został Your Phone Number (Twój numer telefonu) w Header Information Settings (Ustawienia informacji o nagłówku). Faks odbiorcy mógł zostać skonfigurowany na blokowanie wszelkich faksów, które nie zawierają informacji o numerze telefonicznym/faksu nadawcy.
Stan transmitowanych danych nie zmienia się od stanu Waiting for printer response (Oczekiwanie na odpowiedź drukarki).
Waiting for printer response (Oczekiwanie na odpowiedź drukarki) jest wyświetlane, gdy komputer nie może się skomunikować z drukarką (FAX) i pozostaje, do czasu, gdy komunikacja będzie możliwa. Sprawdź następujące elementy.
  • Czy drukarka jest włączona.
  • Czy drukarka i komputer są prawidłowo podłączone.
  • Jeśli drukarka to drukarka sieciowa, sprawdź, czy został prawidłowo określony adres sieciowy drukarki.
  • Czy drukarka może drukować normalnie.
Jeśli wszystkie wymienione wyżej elementy są prawidłowe wyłącz, a następnie włącz drukarkę.
Aplikacja przestaje działać po wyświetleniu “Cannot print other fax data because a transmission is being processed.”.
Niektóre aplikacje mogą przestać działać w trakcie transmisji faksu, przy jednoczesnym wyświetleniu ostrzeżenia przy uruchomieniu. (Po pewnym czasie mogą wznowić działanie)

Problemy z odbieraniem (Tylko dla modeli z odbiorem PC-FAX)

Po wystąpieniu problemu należy przeprowadzić następujące czynności rozwiązywania problemów.
Problem
Co należy zrobić
Ikona Fax Receive Monitor (Monitor odbioru faksu) nie jest wyświetlana w FAX Utility (główne okno).
Ikona jest wyświetlana po wybraniu drukarki z obsługą odbioru PC-FAX.
Podłącz drukarkę (faks) z obsługą tej funkcji i włącz drukarkę, a następnie wybierz drukarkę.
Received Fax Output Settings (Ustawienia drukowania odebranego faksu) zostały ustawione na zapisywanie faksów w komputerze, ale faksy nie są zapisywane.
Po uruchomieniu Fax Receive Monitor (Monitor odbioru faksu) odebrane faksy są zapisywane wyłącznie w określonym folderze. Zalecamy, aby utrzymać działanie Fax Receive Monitor (Monitor odbioru faksu).
Nie można odbierać faksów.
  • Wybierz okno opcji Auto Answer (Automatyczna odpowiedź) w Fax Settings (Ustawienia faksu) - Auto Answer Settings (Ustawienia automatycznej odpowiedzi). Przy braku zewnętrznego telefonu podłączonego do drukarki, drukarka nie może otrzymywać faksów, bez ustawienia Auto Answer (Automatyczna odpowiedź).
  • Sprawdź, czy drukarka nie jest w stanie błędu.
Nie można zmienić ustawień, dlatego odebrane faksy są drukowane.
Nie można zmienić ustawień, aż do zapisania odebranych faksów w komputerze. W komputerze, ustawionym na zapisywanie odebranych faksów, zmień ustawienia po sprawdzeniu nowych faksów lub zmień ustawienia wykorzystując panel sterowania drukarki.
"Gdy folder udostępniany w sieci jest skonfigurowany jako folder odebranych faksów, wyświetlony zostanie komunikat "Cannot find the received fax folder (Nie znaleziono folderu odebranych faksów)".
Aby zapisywać w folderze udostępnionym w sieci, należy połączyć się z nim wcześniej. Wybierz pasek menu na pulpicie Go (Przejdź do) - Connect to Server (Połącz z serwerem) i połącz z udostępnionym folderem w sieci.
Okno Fax Receive Monitor (Monitor odbioru faksu) nie jest wyświetlane, po automatycznym uruchomieniu podczas logowania Fax Receive Monitor (Monitor odbioru faksu).
Okno Fax Receive Monitor (Monitor odbioru faksu) nie jest wyświetlane automatycznie po zalogowaniu komputera. Kliknij ikonę Fax Receive Monitor (Monitor odbioru faksu) w doku, aby otworzyć okno. Nawet, jeśli okno Fax Receive Monitor (Monitor odbioru faksu) nie jest wyświetlane automatycznie, wykonywane są operacje w tle, takie jak sprawdzanie nowych faksów, itp.
Po każdym odebraniu faksu w komputerze wyświetlany jest ekran wprowadzania uwierzytelnienia użytkownika.
Ten ekran jest wyświetlany po odebraniu faksu w komputerze, podczas używania drukarki podłączonej do sieci, po ustawieniu w drukarce uwierzytelniania użytkownika odbioru PC-FAX (nazwa i hasło do udostępniania plików). Ekran ten jest także wyświetlany, gdy informacje uwierzytelniania ustawione w drukarce różnią się od informacji zapisanych w kluczu.
Ponownie wykonaj Received Fax Output Settings (Ustawienia drukowania odebranego faksu), a następnie po wyświetleniu ekranu wprowadzania, wprowadź prawidłowe informacje uwierzytelniania. Pamiętaj, aby wybrać Remember this password in my keychain (Zapamiętaj to hasło w moim kluczu).

Komunikaty o błędach

Po wyświetleniu komunikatu o błędzie, rozwiąż problem poprzez wykonanie następujących instrukcji rozwiązywania problemów.
Komunikat
Co należy zrobić
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (Nie można uzyskać informacji o drukarce (faksie). Należy wybrać opcję "Preferencje drukowania i faksowania" i dodać sterownik PC-FAX.)
Dodaj sterownik PC-FAX do listy drukarek.
Kliknij OK w oknie dialogowym komunikatu. Wyświetlane jest okno dialogowe dodawania drukarki.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (Nie można otworzyć rejestru transmisji faksu, ponieważ działa inna funkcja programu FAX Utility. Należy zaczekać do jej ukończenia, a następnie spróbować ponownie.)
Jest już uruchomiona inna funkcja FAX Utility. Zamknij inną funkcję, a następnie spróbuj ponownie.
Cannot open the transmission record file. You have no access rights or the file does not exist. (Nie można otworzyć pliku rejestru transmisji. Przyczyną może być brak uprawnień dostępu lub plik nie istnieje.)
Sprawdź, czy posiadasz prawa dostępu do zapisu transmisji.
Failed to save the log file. Change the path and file name, and save it again. (Nie można zapisać pliku dziennika. Należy zmienić ścieżkę i nazwę pliku, a następnie zapisać plik dziennika ponownie.)
Sprawdź miejsce docelowe zapisu pliku dziennika.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (Nie można usunąć pliku dziennika. Przyczyną może być używanie pliku przez inną aplikację, brak uprawnień dostępu lub plik został już usunięty.)
Jeśli plik dziennika jest obecny, sprawdź ustawienia praw dostępu lub zamknij inne aplikacje.
A printer communication error occurred. (Wystąpił błąd komunikacji drukarki.)
Wyłącz drukarkę, a następnie włącz ponownie. Jeśli nie działa, zaczekaj chwilę i spróbuj ponownie.
An error was detected while communicating with the printer. (Podczas komunikacji z drukarką wykryto błąd. )
Zaczekaj chwilę, a następnie spróbuj ponownie.
Failed to write the backup file. There is not enough free space or it may be write protected. (Nie udało się zapisać pliku kopii zapasowej. Przyczyną może być niewystarczająca ilość wolnego miejsca lub zabezpieczenie przed zapisywaniem.)
Sprawdź wolne miejsce na dysku twardym. Lub usuń zabezpieczenie przed zapisem pliku kopii zapasowej.
Failed to write the backup file. The file specified is being used by another application. (Nie udało się zapisać pliku kopii zapasowej. Wybrany plik jest używany przez inną aplikację.)
Plik kopii zapasowej jest używany przez inny program. Wyjdź z innego programu.
Cannot recognize the file format. The file may be corrupt or created by another application. (Nie rozpoznano formatu pliku. Plik może być uszkodzony lub został utworzony przez inną aplikację.)
Obsługiwane rozszerzenie dla pliku kopii zapasowej, to tylko *.fpb. Sprawdź plik kopii zapasowej.
Failed to read the backup file. The specified file is being used by another application. (Nie udało się odczytać pliku kopii zapasowej. Wybrany plik jest używany przez inną aplikację.)
Plik kopii zapasowej jest używany przez inny program. Wyjdź z innego programu.
Processing canceled because an error occurred. (Przetwarzanie zostało anulowane, ponieważ wystąpił błąd.)
Po ponownym uruchomieniu drukarki, gdy komputer nie rozwiązuje problemu, skontaktuj się z pomocą techniczną Epson. Informacje kontaktowe należy sprawdzić w podręczniku drukarki.
