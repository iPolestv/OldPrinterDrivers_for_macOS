Lista błędów transmisji

Następująca lista zawiera komunikaty błędu wyświetlane w oknie Communication Error (Błąd komunikacji ), z krótkim opisem.
Komunikat błędu
Opis
Unsupported paper size (Nieobsługiwany format papieru)
Nie można wykonać transmisji, ponieważ wybrano nieobsługiwany rozmiar papieru. Sprawdź, czy wybrany rozmiar papieru jest prawidłowy.
No recipient specified (Nie określono adresata)
Nie można wykonać transmisji ponieważ nie określono żadnego odbiorcy. Określ odbiorcę w menu Recipient Settings (Ustawienia adresata).
Transmisja dokumentu utworzonego z użyciem aplikacji przetwarzania tekstów lub arkusza kalkulacyjnego
Out of disk space (Brak miejsca na dysku)
Nie można wykonać transmisji z powodu za małej ilości miejsca na dysku.
Fax line is busy (Linia faksu jest zajęta)
Nie można wykonać transmisji, ponieważ jest zajęta linia odbiorcy lub linia jest innego typu.
Dialed number invalid (Wybrano nieprawidłowy numer)
Nie można wykonać transmisji, ponieważ nie odpowiada faks odbiorcy ze względu na nieprawidłowy numer.
Color transmission failure (Błąd transmisji koloru)
Transmisja w kolorze nie jest możliwa, ponieważ faks w kolorze nie jest obsługiwany przez faks odbiorcy. Zmień ustawienie Color (Kolor) w sterowniku PC-FAX na Black & White (Czarno-biały) i ponownie wykonaj transmisję.
Transmisja dokumentu utworzonego z użyciem aplikacji przetwarzania tekstów lub arkusza kalkulacyjnego
No dial tone detected (Nie wykryto sygnału)
Nie powiodło się wykrycie sygnału wybierania. Sprawdź podręcznik drukarki w celu uzyskania ustawień Ustawienia Dial Tone Detection (Wykrywanie sygnału wybierania).
No response (Brak odpowiedzi)
W określonym okresie czasu wykryto brak odpowiedzi od odbiorcy. Sprawdź czy faksy można transmitować bezpośrednio z drukarki (bez użycia komputera). Jeśli błąd nadal pozostaje sprawdź, czy jest prawidłowo podłączona i czy działa normalnie linia telefoniczna.
Page limit exceeded (Przekroczono limit stron)
Transmisja nie jest możliwa ponieważ istnieje problem z danymi lub dane przekraczają 100 stron.
PC to printer comm. error (Błąd kom. (komputer-drukarka))
Rozłączenie komunikacji z powodu błędu w połączeniu drukarki lub drukarka została wyłączona.
Printer to recipient comm. error (Błąd kom. (drukarka-adresat))
Transmisja nie jest możliwa, ponieważ przerwana została komunikacja lub wystąpił problem z linią telefoniczną. Sprawdź połączenie linii telefonicznej, a następnie spróbuj ponownie.
PC-FAX is off (PC-FAX jest wył.)
Wyłączona w drukarce transmisja faksu z komputera.
Communication Error (Błąd komunikacji ) (450)
Drukarka nie może przetwarzać danych faksu wysłanych z komputera lub komputer nie może rozpoznać drukarki.
Communication Error (Błąd komunikacji ) (454)
Communication Error (Błąd komunikacji ) (459)
Incorrect phone line connection (Niepoprawne połączenie telefoniczne)
Kabel telefoniczny nie jest prawidłowo podłączony do drukarki (faks). Upewnij się, że podłączony został kabel telefoniczny od gniazda ściennego do gniazda LINE.
