> Strona główna > Transmisja faksu

Transmisja dokumentu utworzonego z użyciem aplikacji przetwarzania tekstów lub arkusza kalkulacyjnego

Dla Mac OS X 10.5.x lub wersji nowszych
Dla Mac OS X 10.4.x
Poprzez wybranie drukarki faksu w menu Print (Drukuj) niektórych aplikacji, można bezpośrednio przesyłać takie dane, jak utworzone dokumenty, rysunki i tabele.
Uwaga:
Następujące wyjaśnienie wykorzystuje jako przykład TextEdit, standardową aplikację Mac OS X. Rzeczywiste działanie zależy od używanej aplikacji. Szczegółowe informacje znajdują się w pomocy aplikacji.

Dla Mac OS X 10.5.x lub wersji nowszych

Używając aplikacji, utwórz dokument do transmisji przez faks.
Kliknij Print (Drukuj) w menu File (Plik) aplikacji.
Wyświetlone zostanie okno dialogowe Print (Drukuj) aplikacji.
Wybierz FAX XXXXXX (twoja drukarka) z opcji Printer (Drukarki), a następnie kliknij w celu wyświetlenia szczegółowych ustawień.
Wykonaj odpowiednie ustawienia.
    • Określ 1 w opcji Copies (Kopie). Nawet po określeniu 2 lub więcej, przesyłana jest tylko jedna kopia.
    • W zależności od drukarki (faksu), faksowanie w kolorze może nie być dostępne dla papieru o rozmiarze większym niż A4, Letter lub Legal.
    • W jednej transmisji faksowej można wysłać do 100 stron.
Wybierz Fax Settings (Ustawienia faksu) i wykonaj odpowiednie ustawienia.
Objaśnienie każdego z ustawień, znajduje się w pomocy sterownika PC-FAX. Pomoc sterownika można wyświetlić klikając ? w dolnej, lewej części ekranu, po wybraniu Fax Settings (Ustawienia faksu).
Wybierz Recipient Settings (Ustawienia adresata) i określ odbiorcę.
Bezpośrednie określenie odbiorcy (nazwa, numer faksu, itp.):
Wprowadź niezbędne informacje w części Add (Dodaj), a następnie kliknij +. Odbiorca zostanie dodany do listy Recipient List (Lista adresatów), wyświetlonej w górnej części okna dialogowego.
Uwaga:
Po wybraniu Enter fax number twice (Wprowadź numer faksu dwa razy) w ustawieniach sterownika PC-FAX, należy ponownie wprowadzić ten sam numer.
Jeśli podłączona linia faksu wymaga prefiksu zewnętrznego dostępu, wprowadź prefiks w polu External Access Prefix (Prefiks zewnętrznego dostępu).
Uwaga:
Jeśli drukarka ma ustawienie typu linii PBX i została ustawiona na używanie # (krzyżyk) zamiast wprowadzania dokładnego kodu prefiksu, wprowadź # (krzyżyk).
Wybieranie odbiorcy (nazwa, numer faksu, itp.) z Address book (Książka adresowa) lub Contacts (Kontakty):
Jeśli odbiorca został zapisany w książce adresowej, kliknij . Wybierz odbiorcę z listy, kliknij Add (Dodaj), a następnie OK.
Jeśli podłączona linia faksu wymaga prefiksu zewnętrznego dostępu, wprowadź prefiks w polu External Access Prefix (Prefiks zewnętrznego dostępu).
Uwaga:
Jeśli drukarka ma ustawienie typu linii PBX i została ustawiona na używanie # (krzyżyk) zamiast wprowadzania dokładnego kodu prefiksu, wprowadź # (krzyżyk).
Sprawdź ustawienia odbiorcy i kliknij Fax (Faks).
Kliknij Fax (Faks), aby rozpocząć transmisję. Przed transmisją upewnij się, że nazwa i numer faksu odbiorcy są prawidłowe.
Uwaga:
  • Kliknij ikonę drukarki w oknie Dock (Dok), aby wyświetlić stan transmisji. Aby zatrzymać transmisję, wybierz wysłane dane, a następnie kliknij Delete (Usuń).
  • Po wystąpieniu błędu podczas transmisji, wyświetlony zostanie komunikat błędu. Sprawdź zapis transmisji w oknie Fax Transmission Record (Rejestr transmisji faksu).
    Sprawdzanie/Usuwanie zapisów transmisji
    Lista błędów transmisji
  • Dokumenty, które zawierają kilka różnych rozmiarów papieru, mogą zostać przesłane nieprawidłowo.
