Przywracanie kopii zapasowej listy szybkiego wybierania

Kopię zapasową książki adresowej telefonu drukarki można przywrócić (listy szybkiego wybierania/wybieranie grupowego), a następnie przywrócić je w drukarce.
Uwaga:
  • Nie można przywrócić danych, których kopia zapasowa została wykonana w narzędziu Fax Utility w systemie Windows.
  • Jeśli wykonana kopia zapasowa listy szybkiego wybierania/wybieranie grupowego zawiera pozycje, które nie są obsługiwane przez drukarkę, w której ma zostać przywrócona lista, pozycje te nie zostaną przywrócone.
  • Jeśli wykonana kopia zapasowa listy szybkiego wybierania/wybieranie grupowego nie zawiera pozycji, które nie są obsługiwane przez drukarkę, w której ma zostać przywrócona lista, zostaną one przywrócone, jak pokazano poniżej.
    Index Word (Słowo indeksowe): puste
    Fax Speed (Szybkość faksu): wstępnie ustawiona wartość
  • Wszelkie pozycje, takie jak Name/Group Name (Nazwa/Nazwa grupy), które zawierają znaki nieobsługiwane przez drukarkę w której mają zostać przywrócone dane, podczas przywracania są puste.
  • Wszelkie pozycje wskazane jako In use by another function (Używana przez inną funkcję) w FAX Utility, są zastępowane przywróconymi danymi, jeśli tylko pasują pozycje numerów.
  • Wszystkie używane i zabezpieczone pozycje nie są zastępowane przywracanymi danymi.
Uruchom FAX Utility.
Uruchomienie FAX Utility
Wybierz FAX XXXXXX (twoja drukarka) z opcji Printer (Drukarki), a następnie kliknij Speed Dial/Group Dial List (Lista szybkiego wybierania/grupowego wybierania).
Uwaga:
Jeśli drukarka ma funkcję zabezpieczenia, która pozwala administratorom na ograniczanie użytkownikom uprawnień do zmiany ustawień faksu drukarki, aby kontynuować może być konieczne wprowadzenie hasła administratora.
Kliknij File (Plik) - Back up (Kopia zapasowa) - Restore Backup (Przywróć kopię zapasową).
Wybierz plik do przywrócenia.
Lista przywracania jest wyświetlana w oknie Speed Dial/Group Dial List (Lista szybkiego wybierania/grupowego wybierania).
Kliknij w oknie .
Przywracane dane książki adresowej są zapisywane w drukarce.
