Uruchomienie FAX Utility

Dla Mac OS X 10.6.x lub wersji nowszych
Kliknij Apple - System Preferences (Preferencje systemu) - Print & Scan (Drukowanie i skanowanie) lub Print & Fax (Drukowanie i faksowanie).
Wybierz FAX XXXXXX (twoja drukarka) z Printer (Drukarki), a następnie kliknij Options & Supplies (Opcje i zaopatrzenie) - Utility (Narzędzie) - Open Printer Utility (Otwórz program drukarki).
Dla Mac OS X 10.5.x
Kliknij Apple - System Preferences (Preferencje systemu) - Print & Fax (Drukowanie i faksowanie).
Wybierz FAX XXXXXX (twoja drukarka) z opcji Printer (Drukarki), a następnie kliknij Open Print Queue (Otwórz kolejkę drukowania) - Utility (Narzędzie).
Dla Mac OS X 10.4.x
Kliknij dwukrotnie dysk twardy - Applications (Aplikacje) - Utilities (Narzędzia) - Printer Setup Utility (Narzędzie konfiguracji drukarki).
Wybierz FAX XXXXXX (twoja drukarka) z opcji Printer (Drukarki), a następnie kliknij Utility (Narzędzie).
