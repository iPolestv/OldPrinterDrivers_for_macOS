> Strona główna > Rozwiązywanie problemów

Komunikaty błędu odbioru (Tylko dla modeli z odbiorem PC-FAX)

Poniżej zawarto listę komunikatów o błędach, które mogą zostać wyświetlone po odebraniu faksu oraz sposobów rozwiązania problemu.
Komunikat
Co należy zrobić
A printer communication error occurred. (Wystąpił błąd komunikacji drukarki.)
Wyłącz drukarkę, a następnie włącz ponownie. Zaczekaj chwilę, a następnie spróbuj ponownie.
The printer (fax driver) you are setting has been deleted. Add the printer (fax driver) in [System Preferences] - [Print & Fax]. (Drukarka (sterownik faksu), dla której konfigurowane są ustawienia, została usunięta. Drukarkę (sterownik faksu) należy dodać, klikając kolejno opcje [Preferencje systemowe] - [Drukowanie i faksowanie].)
Dodaj sterownik PC-FAX do listy drukarek. Kliknij OK w oknie dialogowym komunikatu. Wyświetlane jest okno dialogowe dodawania drukarki.
Change not accepted. Some printer settings have been changed by another computer, print queue, or the printer. Please try again. (Nie zaakceptowano zmiany. Niektóre ustawienia drukarki zostały zmienione przez inny komputer, kolejkę drukarki lub drukarkę. Proszę spróbować ponownie.)
Received Fax Output Settings (Ustawienia drukowania odebranego faksu) zostały zmienione z innego komputera w sieci, z kolejki drukowania lub z drukarki. Zaczekaj chwilę, a następnie spróbuj ponownie.
Odebrane faksy można zapisać wyłącznie w komputerze.
Received faxes not saved due to an error. (Z powodu błędu nie zapisano odebranych faksów.)
Nie można zapisać odebranych faksów w określonym folderze. Sprawdź połączenie z określonym folderem, aby się upewnić, że jest dostępny oraz, aby sprawdzić, że dysponujesz prawami dostępu do tego folderu.
Processing canceled because an error occurred. (Przetwarzanie zostało anulowane, ponieważ wystąpił błąd.)
Po ponownym uruchomieniu drukarki, gdy komputer nie rozwiązuje problemu, skontaktuj się z pomocą techniczną Epson. Informacje kontaktowe można znaleźć w podręczniku drukarki.
Cannot save faxes. User authentication failed. (Nie można zapisać faksów. Uwierzytelnianie użytkownika nie powiodło się.)
Nie można zapisać odebranych faksów, ponieważ uwierzytelnienia użytkownika odbiorcy PC-FAX (nazwa i hasło do udostępniania plików) nie zostały prawidłowo ustawione w kluczu komputera.
Ponownie wykonaj Fax Output Setting (Ustawienia drukowania odebranego faksu), a następnie po wyświetleniu ekranu wprowadzania, wprowadź prawidłowe informacje uwierzytelniania. Pamiętaj, aby wybrać Remember this password in my keychain (Zapamiętaj to hasło w moim kluczu).
