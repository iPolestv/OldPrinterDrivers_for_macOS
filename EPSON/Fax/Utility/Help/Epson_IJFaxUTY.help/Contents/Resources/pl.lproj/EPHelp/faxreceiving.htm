Fax Receive Monitor (Monitor odbioru faksu) (Tylko dla modeli z odbiorem PC-FAX)

Pasek narzędzi
Ikona i komunikat na liście drukarek
To okno umożliwia sprawdzenie stanu odbioru faksu i wykonanie ustawień wyjścia dla odebranych faksów.
Uwaga:
To okno jest wyświetlane po kliknięciu Fax Receive Monitor (Monitor odbioru faksu) w FAX Utility (główne okno).

Pasek narzędzi

Pozycja
Objaśnienie
Open the received fax folder (Otwórz folder odebranych faksów)
Otwieranie folderu w którym są zapisywane odebrane faksy.
Check new faxes (Sprawdź nowe faksy)
Sprawdzanie nowych faksów.
Received Fax Output Settings (Ustawienia drukowania odebranego faksu)
Ustawienia wyboru zapisywania w komputerze odebranych faksów.
Preferences (Preferencje)
Konfiguracja preferencji monitora odbioru faksu.

Ikona i komunikat na liście drukarek

Ikona
Komunikat
Objaśnienie
Waiting (Oczekiwanie) / Checking for new faxes (Sprawdzanie nowych faksów)
Sprawdzanie nowych faksów lub oczekiwanie.
Received faxes saved successfully in the specified folder (Odebrane faksy zostały pomyślnie zapisane w wybranym folderze).
Nowe faksy odebrane przez drukarkę (faks) zostały zapisane w określonym folderze.
The printer is not responding (Drukarka nie odpowiada)
Sprawdź połączenie między komputerem i drukarką lub sprawdź, czy drukarka jest włączona.
The printer is processing data (Drukarka przetwarza dane)
Drukarka przetwarza inne dane. Zaczekaj.
Cannot find the received fax folder (Nie znaleziono folderu odebranych faksów).
Sprawdź, czy nazwa określonego folderu jest prawidłowa. Jeśli jest używany folder współdzielony w sieci, upewnij się, że masz prawa dostępu do folderu.
Failed to save faxes in the specified folder (Nie można zapisać faksów w wybranym folderze).
Sprawdź, czy określony folder jest dostępny lub, czy masz uprawnienia dostępu do folderu.
A printer communication error occurred (Wystąpił błąd komunikacji drukarki)
Sprawdź połączenie między komputerem i drukarką lub sprawdź, czy drukarka jest włączona. Jeśli jest używane połączenie sieciowe, sprawdź czy można uzyskać dostęp do drukarki przez sieć.
The "FAXRECV" disk is being used by another application. (Dysk "FAXRECV" jest używany przez inną aplikację.)
Zamknij program, który używa dysk "FAXRECV".
Cannot save faxes. User authentication failed. (Nie można zapisać faksów. Uwierzytelnianie użytkownika nie powiodło się.)
Sprawdź uprawnienia użytkownika do odbioru PC-FAX (nazwa i hasło do udostępniania pliku) ustawione w drukarce, ponownie wykonaj Received Fax Output Settings (Ustawienia drukowania odebranego faksu), a następnie wprowadź prawidłowo informacje uwierzytelnienia użytkownika. Pamiętaj, aby wybrać Remember this password in my keychain (Zapamiętaj to hasło w moim kluczu).
