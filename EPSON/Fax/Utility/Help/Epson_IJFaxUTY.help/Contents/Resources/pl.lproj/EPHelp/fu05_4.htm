Zapisywanie kopii zapasowej szybkiego wybierania

Można zapisać kopię zapasową skonfigurowanego szybkiego wybierania i wybierania grupowego. Dane listy szybkiego wybierania i wybierania grupowego, mogą zostać utracone z powodu nieprawidłowego działania drukarki. Zalecamy wykonanie kopii zapasowej danych, przy każdej aktualizacji listy wybierania. Firma Epson nie odpowiada za utratę jakichkolwiek danych, za wykonywanie kopii zapasowej lub przywracanie danych i/lub ustawień nawet w okresie obowiązywania gwarancji.
Uwaga:
Dane oznaczone jako "In use by another function (Używana przez inną funkcję)", nie są zapisywane.
Uruchom FAX Utility.
Uruchomienie FAX Utility
Wybierz FAX XXXXXX (twoja drukarka) z opcji Printer (Drukarki), a następnie kliknij Speed Dial/Group Dial List (Lista szybkiego wybierania/grupowego wybierania).
Wyświetlone zostaną zapisane w drukarce listy szybkiego wybierania/wybierania grupowego.
Uwaga:
Jeśli drukarka ma funkcję zabezpieczenia, która pozwala administratorom na ograniczanie użytkownikom uprawnień do zmiany ustawień faksu drukarki, aby kontynuować może być konieczne wprowadzenie hasła administratora.
Kliknij File (Plik) - Back up (Kopia zapasowa) - Save Backup (Zapisz kopię zapasową).
Określ nazwę pliku i katalog pliku, a następnie kliknij Save (Zachowaj).
