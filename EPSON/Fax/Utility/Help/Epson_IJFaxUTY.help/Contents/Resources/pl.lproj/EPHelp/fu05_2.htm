Rejestracja lub edycja szybkiego wybierania i wybierania grupowego

Można zarejestrować lub edytować szybkie wybieranie lub wybieranie grupowe faksu dla drukarki.
Po zarejestrowaniu szybkiego wybierania, można łatwo wprowadzić numer faksu podczas wysyłania faksu z drukarki. Po zarejestrowaniu wybierania grupowego, można określić w jednej operacji kilku odbiorców.
Uwaga:
Przed rejestracją szybkiego wybierania, nie można skonfigurować wybierania grupowego.
Uruchom FAX Utility.
Uruchomienie FAX Utility
Wybierz FAX XXXXXX (twoja drukarka) z opcji Printer (Drukarki), a następnie kliknij Speed Dial/Group Dial List (Lista szybkiego wybierania/grupowego wybierania).
Wyświetlone zostaną zapisane w drukarce listy szybkiego wybierania/wybierania grupowego.
Uwaga:
Jeśli drukarka ma funkcję zabezpieczenia, która pozwala administratorom na ograniczanie użytkownikom uprawnień do zmiany ustawień faksu drukarki, aby kontynuować może być konieczne wprowadzenie hasła administratora.
Aby utworzyć nową pozycję, wybierz pustą linię. W celu edycji istniejącej pozycji, wybierz pozycję do edycji.
Następnie kliknij dwukrotnie linię lub naciśnij przycisk Enter, aby otworzyć okno dialogowe edycji.
Uwaga:
  • Aby usunąć istniejącą pozycję, wybierz pozycję do usunięcia, a następnie naciśnij przycisk usuwania lub kliknij menu Edit (Zmiany), po czym kliknij Delete (Usuń). Jednakże, usunięcie pozycji wybierania grupowego nie usuwa pozycji szybkiego wybierania, wykonanych dla grupy.
  • Zabezpieczonej pozycji szybkiego wybierania lub wybierania grupowego, nie można edytować lub usunąć.
Konfiguracja szybkiego wybierania lub wybierania grupowego.
Konfiguracja szybkiego wybierania:
Wybierz Speed Dial (Szybkie wybieranie), wprowadź Name/Group Name (Nazwa/Nazwa grupy), Index Word (Słowo indeksowe), Fax Number (Numer faksu) i Fax Speed (Szybkość faksu), a następnie kliknij OK. Skonfigurowane szybkie wybieranie jest dodawane do Speed Dial/Group Dial List (Lista szybkiego wybierania/grupowego wybierania).
Uwaga:
  • Jeśli posiadany system telefoniczny wymaga prefiksu zewnętrznego dostępu, takiego jak 0 lub 9, aby uzyskać linię zewnętrzną, wprowadź prefiks na początku numeru faksu. Jeśli drukarka została ustawiona na używanie # (krzyżyk) zamiast wprowadzania dokładnego kodu prefiksu, wprowadź # (krzyżyk).
  • Szybkość faksu ustawiona dla każdej pozycji wybierania, jest używana wyłącznie podczas wysyłania faksów bezpośrednio z drukarki. FAX Utility ignoruje ustawienie i wykorzystuje szybkość faksu ustawioną w drukarce.
Konfiguracja wybierania grupowego:
Wybierz Group Dial (Grupowe wybieranie), wprowadź Name/Group Name (Nazwa/Nazwa grupy) i Index Word (Słowo indeksowe), wybierz okno opcji szybkiego wybierania, które ma zostać dodane do wybierania grupowego, a następnie kliknij OK. Skonfigurowane wybieranie grupowe jest dodawane do Speed Dial/Group Dial List (Lista szybkiego wybierania/grupowego wybierania).
Uwaga:
Index Word (Słowo indeksowe) i Fax Speed (Szybkość faksu) w zależności od drukarki, mogą nie być dostępne.
Po wykonaniu wszystkich ustawień, kliknij .
Wyświetlone na liście szybkie wybieranie i wybieranie grupowe, są zapisywane w drukarce.
