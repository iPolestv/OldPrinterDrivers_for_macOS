Speed Dial/Group Dial List (Hurtigopkalds-/gruppeopkaldsliste)

Menu
Værktøjslinje
Med Speed Dial/Group Dial List (Hurtigopkalds-/gruppeopkaldsliste) kan du registrere eller redigere hurtigopkald og gruppeopkald for faxer for din printer.
Bemærk:
  • Dette vindue vises, når du klikker på Speed Dial/Group Dial List (Hurtigopkalds-/gruppeopkaldsliste) i FAX Utility (hovedvinduet).
  • Det maksimale antal af poster, som er tilgængelige for den forbundne printer (fax) vises på linjen nederst i vinduet.

Menu

Menuen File (Filer)

Emne
Forklaring
Get Speed Dials (Hent hurtigopkald...)
Indlæser hurtigopkald og gruppeopkald for faxer for din printer.
Send Speed Dials (Send hurtigopkald...)
Gemmer hurtigopkaldene og gruppeopkaldene, som vises i listen på printeren.
Import (Importer)
From Address Book (Fra Adressebog)
Importerer data fra Address Book (Adressebog) til hurtigopkald for din printer.
Back up (Backup)
Save Backup (Gemmer backup)
Gemmer en sikkerhedskopi af de konfigurerede hurtigopkald og gruppeopkald.
Restore Backup (Gendanner backup)
Indlæser de sikkerhedskopierede hurtigopkald eller gruppeopkald.
Close (Luk)
Lukker vinduet.

Menuen Edit (Rediger)

Emne
Forklaring
Edit (Rediger)
Redigerer hurtigopkald eller gruppeopkald, som er valgt i listen.
Delete (Slet)
Sletter hurtigopkald eller gruppeopkald, som er valgt i listen.
Delete All (Slet alle)
Sletter alle hurtigopkald eller gruppeopkald i listen.
