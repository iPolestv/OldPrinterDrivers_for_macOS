> Start > Speed Dial/Group Dial List (Hurtigopkalds-/gruppeopkaldsliste)

Gendanne en sikkerhedskopieret hurtigopkaldsliste

Du kan gendanne sikkerhedskopierede telefonbogsdata for printeren (hastigheds-/gruppeopkaldslister) og derefter gendanne dem til printeren.
Bemærk:
  • Du kan ikke gendanne data, som er sikkerhedskopieret i Windows Fax Utility.
  • Hvis de sikkerhedskopierede hurtigopkalds-/gruppeopkaldslister indeholder emner, som ikke er understøttet af printeren, som du vil gendanne listen til, gendannes disse emner ikke.
  • Hvis de sikkerhedskopierede hurtigopkalds-/gruppeopkaldslister ikke indeholder emner, som ikke understøttes af printeren, som du vil gendanne listen til, gendannes disse emner som vist nedenfor.
    Index Word (Indeksord): tom
    Fax Speed (Faxhastighed): forudindstillet værdi
  • Alle emner, som f.eks. Name/Group Name (Navn/Gruppenavn), som indeholder tegn, som ikke understøttes af printeren, som du vil gendanne dataene til, er tomme, når de gendannes.
  • Alle poster, som er indikeret som In use by another function (I brug af en anden funktion) på FAX Utility overskrives med de gendannede data hvis nummeret stemmer overens.
  • Alle poster. som er i brug og som er beskyttet overskrives ikke med de gendannede data.
Start FAX Utility.
Start af programmet FAX Utility
Vælg FAX XXXXXX (din printer) fra Printer, og klik derefter på Speed Dial/Group Dial List (Hurtigopkalds-/gruppeopkaldsliste).
Bemærk:
Hvis din printer har en sikkerhedsfunktion, som gør det muligt for administratorer at hindre brugere i at ændre printerens faxindstillinger, skal du muligvis indtaste en administratoradgangskode for at kunne fortsætte.
Klik på File (Filer) - Back up (Backup) - Restore Backup (Gendanner backup).
Vælg den fil, du vil gendanne.
Den gendannede liste vises i vinduet Speed Dial/Group Dial List (Hurtigopkalds-/gruppeopkaldsliste).
Klik på i vinduet.
De gendannede telefonbogsdata gemmes på printeren.
