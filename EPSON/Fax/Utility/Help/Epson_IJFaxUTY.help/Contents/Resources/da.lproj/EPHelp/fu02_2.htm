Kontrollere/slette overførselsrapporter

Du kan kontrollere og slette listen i faxoverførselsrapporten.
Start FAX Utility.
Start af programmet FAX Utility
Vælg FAX XXXXXX (din printer) fra Printer, og klik derefter på Fax Transmission Record (Faxtransmissions registrering).
Kontroller overførselsrapporterne.
Listen gemmer op til 1000 overførselsrapporter.
For at slette faxoverførselsrapporter skal du vælge Delete (Slet) i menuen Edit (Rediger). Alle data slettes. (Du kan ikke slette hver registrering individuelt.)
