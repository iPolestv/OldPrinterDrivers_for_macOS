Start af programmet FAX Utility

For Mac OS X 10,6.x eller senere
Klik på Apple - System Preferences (Systemindstillinger) - Print & Scan (Udskriv & scan) eller Print & Fax (Udskriv & fax).
Vælg FAX XXXXXX (din printer) fra Printer, og klik derefter på Options & Supplies (Indstillinger & forsyninger) - Utility (Hjælpefunktioner) - Open Printer Utility (Åbn printer hjælpeprogram).
For Mac OS X 10,5.x
Klik på Apple - System Preferences (Systemindstillinger) - Print & Fax (Udskriv & fax).
Vælg FAX XXXXXX (din printer) fra Printer, og klik derefter på Open Print Queue (Åbn printerkø) - Utility (Hjælpefunktioner).
For Mac OS X 10.4.x
Dobbeltklik på Harddisk - Applications (Programmer) - Utilities (Hjælpeprogrammer) - Printer Setup Utility (Hjælpeprogram til printerkonfigurering).
Vælg FAX XXXXXX (din printer) fra Printer, og klik derefter på Utility (Hjælpefunktioner).
