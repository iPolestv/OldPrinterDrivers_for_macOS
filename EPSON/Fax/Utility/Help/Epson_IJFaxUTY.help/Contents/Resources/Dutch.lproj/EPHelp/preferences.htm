Preferences (Voorkeuren) (alleen voor modellen met de functie voor het ontvangen van faxen via de pc)

In dit venster kunt u faxontvangstinstellingen configureren, zoals hoe u wordt gewaarschuwd wanneer u een nieuwe fax ontvangt.
Opmerking:
  • Dit venster wordt weergegeven wanneer u in het venster Fax Receive Monitor Preferences (Faxontvangstmonitor-voorkeuren) op Fax Receive Monitor (Faxontvangstmonitor) klikt.
  • Deze instellingen worden toegepast op alle printers en afdrukwachtrijen die worden weergegeven in Fax Receive Monitor (Faxontvangstmonitor).

Specify an interval to check for new faxes (Geef een interval op voor controle op nieuwe faxen)
Geef een interval op voor controle op nieuwe faxen (tussen 1 - 30 minuten).
Notify me of new faxes via a dock icon (Graag melding bij nieuwe faxen via een Dock-pictogram)
Selecteer deze optie om het pictogram van Fax Receive Monitor (Faxontvangstmonitor) op het Dock te laten springen wanneer een nieuwe fax wordt ontvangen.
Notify me of errors via a dock icon (Graag melding bij fouten via een Dock-pictogram)
Selecteer deze optie om het pictogram van Fax Receive Monitor (Faxontvangstmonitor) op het Dock te laten springen wanneer er een fout optreedt. Klik op het pictogram om het venster Fax Receive Monitor (Faxontvangstmonitor) weer te geven waarin u de foutgegevens kunt zien.
Start at login (Starten bij aanmelding)
Het is raadzaam deze optie op Automatic (Automatisch) in te stellen. Wanneer "Save" (save faxes on this computer) ("Opslaan" (faxen opslaan op deze computer)) is ingesteld in Received Fax Output Settings (Instellingen uitvoer ontvangen faxen), wordt Fax Receive Monitor (Faxontvangstmonitor) automatisch gestart wanneer u zich aanmeldt. Als u On (Aan) selecteert, wordt Fax Receive Monitor (Faxontvangstmonitor) gestart via aanmelding, ongeacht de uitvoerinstellingen. Als u Off (Uit) selecteert, wordt Fax Receive Monitor (Faxontvangstmonitor) niet gestart wanneer u zich aanmeldt.
Wait time before checking new faxes (Wachttijd voor controle nieuwe faxen)
Deze optie is beschikbaar wanneer Automatic (Automatisch) of On (Aan) is ingesteld in Start at login (Starten bij aanmelding). Geef op hoelang (0 tot 10 minuten) moet worden gewacht met het controleren op nieuwe faxen nadat Fax Receive Monitor (Faxontvangstmonitor) is gestart wanneer u zich aanmeldt. Dit is nuttig wanneer de map waarin faxen worden opgeslagen zich op een netwerk bevindt, en u tijd nodig hebt om een verbinding met de map tot stand te brengen nadat u zich hebt aangemeld.
Opmerking:
Als de map waarin ontvangen faxen worden opgeslagen, een gedeelde map op een netwerk is, kunt u verbinding maken met de map wanneer u zich aanmeldt door de map te registreren in de lijst met aanmeldingsopties. Klik op Apple - System Preferences (Systeemvoorkeuren) - Account - Login to register (Log in om te registreren).
