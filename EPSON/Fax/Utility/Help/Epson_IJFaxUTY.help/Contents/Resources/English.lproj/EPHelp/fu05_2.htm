Registering or Editing Speed Dial and Group Dial

You can register or edit the speed dial or group dial of the fax for your printer.
You can easily enter the fax number when you send a fax from printer after registering the speed dial. You can specify several recipients in one operation after registering the group dial.
Note:
You cannot configure the group dial before registering the speed dials.
Start FAX Utility.
Starting FAX Utility
Select FAX XXXXXX (your printer) from Printer, and then click Speed Dial/Group Dial List.
The speed dial/group dial lists saved in your printer are displayed.
Note:
If your printer has a security function that allows administrators to restrict users from changing the printer's fax settings, you may need to enter an administrator password to continue.
To create a new entry, select an empty line. To edit an existing entry, select the entry you want to edit.
Then double-click the line or press the Enter key to open the edit dialog.
Note:
  • To delete an existing entry, select the entry you want to delete and then press the delete key, or click the Edit menu and then click Delete. However, deleting a group dial does not delete the speed dials that made up the group.
  • You cannot edit or delete a protected speed dial entry or a group dial entry.
Configure the speed dial or group dial.
Configuring speed dial:
Select Speed Dial, enter Name/Group Name, Index Word, Fax Number and Fax Speed, and then click OK. The configured speed dial is added to Speed Dial/Group Dial List.
Note:
  • If your phone system requires an external access prefix such as 0 or 9 to get an outside line, enter the prefix at the beginning of the fax number. If your printer has been set to use # (hash) instead of entering the exact prefix code, enter # (hash).
  • The fax speed set for each speed dial entry is only used when sending a fax directly from a printer. FAX Utility ignores the setting and uses the fax speed set on the printer.
Configuring group dial:
Select Group Dial, enter Name/Group Name and Index Word, select the check box of the speed dial you want to add to the group dial, and then click OK. The configured group dial is added to Speed Dial/Group Dial List.
Note:
Index Word and Fax Speed may not be available depending on the printer.
After completing all settings, click .
The speed dials and group dials displayed in the list are saved to the printer.
