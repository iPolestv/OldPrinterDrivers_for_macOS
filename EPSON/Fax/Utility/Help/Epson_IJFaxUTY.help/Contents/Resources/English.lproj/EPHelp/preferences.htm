Preferences (Only for models with PC-FAX reception)

This window allows you to configure fax reception settings, such as how notifications are made when a new fax is received.
Note:
  • This window is displayed when you click Fax Receive Monitor Preferences on the Fax Receive Monitor.
  • These settings are applied to all printers and print queues displayed on the Fax Receive Monitor.
Item
Explanation
Specify an interval to check for new faxes
Specify the interval at which to check for new faxes from 1 to 30 minutes.
Notify me of new faxes via a dock icon
Select this to make the Fax Receive Monitor icon on the Dock jump when a new fax arrives.
Notify me of errors via a dock icon
Select this to make the Fax Receive Monitor icon on the Dock jump when an error occurs. Click the icon to display the Fax Receive Monitor window where you can check the error information.
Start at login
We recommend setting this to Automatic. When "Save" (save faxes on this computer) is set in Received Fax Output Settings, Fax Receive Monitor starts automatically when you log in. If you select On, the Fax Receive Monitor starts by logging in, regardless of the output settings. If you select Off, Fax Receive Monitor does not start when you log in.
Wait time before checking new faxes
This item is available when Automatic or On is set in Start at login. Specify how long (0 to 10 minutes) to wait before checking for new faxes after Fax Receive Monitor starts when you login. This is useful when the folder in which to save faxes is on a network, and you need time to establish a connection to the folder after logging in.
Note:
If the folder in which to save received faxes is a shared folder on a network, you can connect to the folder when you login by registering the folder to the list of login items. Click Apple - System Preferences - Account - Login to register.
