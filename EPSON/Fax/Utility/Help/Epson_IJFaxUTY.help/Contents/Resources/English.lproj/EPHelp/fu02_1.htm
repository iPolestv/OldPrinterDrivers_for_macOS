Transmitting a Document Created Using a Word Processing or Spreadsheet Application

For Mac OS X 10.5.x or later
For Mac OS X 10.4.x
By selecting a printer fax from the Print menu of some applications, you can transmit data directly such as documents, drawings, and tables you have created.
Note:
The following explanation uses TextEdit, the standard Mac OS X application as an example. The actual operation may differ depending on the application you use. For details, see the application’s help.

For Mac OS X 10.5.x or later

Using an application, create a document to transmit by fax.
Click Print from your application’s File menu.
The application’s Print dialog is displayed.
Select FAX XXXXXX (your printer) from Printer, and then click to display detailed settings.
Make appropriate settings.
    • Specify 1 in Copies. Even if you specify 2 or more, only one copy is transmitted.
    • Depending on the printer (fax), color faxing may not be available for paper sizes larger than A4, Letter, or Legal.
    • You can transmit up to 100 pages in one fax transmission.
Select Fax Settings and make appropriate settings.
For an explanation on each setting, see PC-FAX driver help. You can display the driver help by clicking ? at the bottom left of the screen after selecting Fax Settings.
Select Recipient Settings and specify a recipient.
Specifying a recipient (name, fax number and so on) directly:
Enter the necessary information in the Add section, and then click +. The recipient is added to the Recipient List displayed in the upper part of the dialog.
Note:
If you have selected Enter fax number twice in the PC-FAX driver settings, you need to enter the same number again.
If the connected fax line requires an external access prefix, enter the prefix in the External Access Prefix box.
Note:
If your printer has a PBX line type setting and has been set to use # (hash) instead of entering the exact prefix code, enter # (hash).
Selecting a recipient (name, fax number and so on) from the Address book or Contacts:
If the recipient is saved in the address book, click . Select the recipient from the list, click Add, and then OK.
If the connected fax line requires an external access prefix, enter the prefix in the External Access Prefix box.
Note:
If your printer has a PBX line type setting and has been set to use # (hash) instead of entering the exact prefix code, enter # (hash).
Check the recipient settings and click Fax.
Click Fax to start transmitting. Make sure the name and fax number of the recipient are correct before transmitting.
Note:
  • Click the printer icon in Dock to display the transmission status. To stop transmitting, select the data you sent and then click Delete.
  • If an error occurs during transmission, an error message is displayed. Check the transmission record from the Fax Transmission Record window.
    Checking / Deleting Transmission Records
    Transmission Error List
  • Documents that contain several different paper sizes may not be transmitted correctly.

For Mac OS X 10.4.x

Using the application, create a document to transmit by fax.
Click Page Setup from your application’s File menu.
The application’s Page Setup dialog is displayed.
Select FAX XXXXXX (your printer) from Printer, make appropriate settings, and then click OK.
Note:
Depending on the printer, color faxing may not be available for paper sizes larger than A4, Letter, or Legal.
Click Print from your application’s File menu. The application’s Print dialog is displayed.
Select FAX XXXXXX (your printer) from Printer, and then make appropriate settings.
    • Specify 1 in Copies. Even if you specify 2 or more, only one copy is transmitted.
    • You can transmit up to 100 pages in one fax transmission.
Select Fax Settings and then make appropriate settings.
For an explanation on each setting, see PC-FAX driver help. You can display the driver help by clicking ? at the bottom left of the screen after selecting Fax Settings.
Select Recipient Settings and then specify a recipient.
Specifying a recipient (name, fax number and so on) directly:
Enter the necessary information in the Add section, and then click +. The recipient is added to the Recipient List displayed in the upper part of the dialog.
If the connected fax line requires an external access prefix, enter the prefix in the External Access Prefix box.
Note:
If your printer has a PBX line type setting and has been set to use # (hash) instead of entering the exact prefix code, enter # (hash).
Selecting a recipient (name, fax number and so on) from the address book:
If the recipient is saved in the address book, click . Select the recipient from the list, click Add, and then click OK to close the list window.
If the connected fax line requires an external access prefix, enter the prefix in the External Access Prefix box.
Note:
  • If your printer has a PBX line type setting and has been set to use # (hash) instead of entering the exact prefix code, enter # (hash).
  • For details on how to save a recipient in the address book, see the address book help.
Check the recipient settings and click Print.
Click Print to start transmitting. Make sure the name and fax number of the recipient are correct before transmitting.
Note:
  • Click the printer icon in Dock to display the transmission status. To stop transmitting, select the data you sent and click Delete.
  • If an error occurs during transmission, an error message is displayed. Check the transmission record from the Fax Transmission Record window.
    Checking / Deleting Transmission Records
    Transmission Error List
  • Documents that contain several different paper sizes may not be transmitted correctly.
