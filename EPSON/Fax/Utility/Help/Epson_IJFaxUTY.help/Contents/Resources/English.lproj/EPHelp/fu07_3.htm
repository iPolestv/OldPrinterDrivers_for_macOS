Reception Error Messages (Only for models with PC-FAX reception)

The following is a list of error messages that may be displayed when a fax is received, and how to handle the problem.
Message
What to do
A printer communication error occurred.
Turn the printer off and then back on. Wait for a while and then try again.
The printer (fax driver) you are setting has been deleted. Add the printer (fax driver) in [System Preferences] - [Print & Scan] or [Print & Fax].
Add the PC-FAX driver to the printer list. Click OK in the message dialog. The dialog to add the printer is displayed.
Change not accepted. Some printer settings have been changed by another computer, print queue, or the printer. Please try again.
The Received Fax Output Settings has been changed from another computer on the network, from the print queue, or from the printer. Wait for a while and then try again.
Received faxes can only be saved on one computer.
Received faxes not saved due to an error.
Failed to save the received fax in the specified folder. Check the connection to the specified folder to make sure that it is available, and check that you have access rights to the folder.
Processing canceled because an error occurred.
If restarting your printer and computer does not solve the problem, contact Epson support. For contact information, see your printer's manual.
Cannot save faxes. User authentication failed.
Cannot save received faxes because the PC-FAX reception's user authentication (name and password for file sharing) is not set in the computer's keychain correctly.
Make the Fax Output Setting again, and then enter the correct user authentication information when the entry screen is displayed. Remember to select Remember this password in my keychain.
