Speed Dial/Group Dial List

Menu
Toolbar
Speed Dial/Group Dial List allows you to register or edit the speed dial and group dial of the fax for your printer.
Note:
  • This window is displayed when you click Speed Dial/Group Dial List in the FAX Utility (main window).
  • The maximum number of entries available for the connected printer (fax) is displayed on the bar at the bottom of the window.

Menu

File menu

Item
Explanation
Get Speed Dials
Reads the speed dial or group dial of the fax for your printer.
Send Speed Dials
Saves the speed dials and group dials displayed in the list to the printer.
Import
From Address Book
Imports the Address Book data to the speed dials of your printer.
Back up
Save Backup
Saves backup of the configured speed dial and group dial.
Restore Backup
Reads the backed up speed dial or group dial.
Close
Closes the window.

Edit menu

Item
Explanation
Edit
Edits the speed dial or group dial selected in the list.
Delete
Deletes the speed dial or group dial selected in the list.
Delete All
Deletes all the speed dial/group dial in the list.
