Received Fax Output Settings (Only for models with PC-FAX reception)

This window allows you to set whether or not to save received faxes on a computer.
Note:
This window is displayed when you click Received Fax Output Settings in the Fax Receive Monitor.
Item
Explanation
Setting on the printer
Displays the current received fax output setting.
"Print" *
Prints received faxes on the printer.
Not save faxes *
Select this to stop saving faxes on a computer. Received faxes are printed or stored in the printer's memory depending on the printer's settings.
"Save" (save faxes on this computer)
Saves received faxes to the specified folder. The faxes are saved as PDF files.
Save to
Specifies the folder in which to save received faxes.
Send the settings to the printer
Transfers the received fax output settings to the printer.
* These items may not be displayed depending on the printer.
When the PC-FAX reception's user authentication (name and password for file sharing) is set in the printer, you need to register that information to the computer's keychain.
If the information is not registered in the keychain, the registration entry screen is displayed. Enter your name and password, select Remember this password in my keychain, and then click Connect.
If you do not save this authentication information in the keychain, you will be asked to enter it every time you receive a fax and save it to the computer.
