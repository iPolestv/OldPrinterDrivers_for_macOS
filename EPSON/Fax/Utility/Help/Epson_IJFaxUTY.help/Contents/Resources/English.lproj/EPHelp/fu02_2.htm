Checking / Deleting Transmission Records

You can check and delete the fax transmission records list.
Start FAX Utility.
Starting FAX Utility
Select FAX XXXXXX (your printer) from Printer, and then click Fax Transmission Record.
Check the transmission records.
The list saves up to 1000 transmission records.
To delete fax transmission records, select Delete from the Edit menu. All data is deleted. (You cannot delete each record individually.)
