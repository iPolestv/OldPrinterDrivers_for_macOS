Restoring a Backed up Speed Dial List

You can restore backed up printer phone book data (speed/group dial lists) and then restore them on the printer.
Note:
  • You cannot restore data backed up in Windows Fax Utility.
  • If the backed up speed dial/group dial list contains items that are not supported by the printer to which you want to restore the list, those items are not restored.
  • If the backed up speed dial/group dial list does not contain items that are supported by the printer to which you want to restore the data, they are restored as shown below.
    Index Word: blank
    Fax Speed: preset value
  • Any items, such as Name/Group Name, that include characters unsupported by the printer to which you want to restore the data, are blank when they are restored.
  • Any entries indicated as In use by another function on FAX Utility are overwritten with the restored data as long as the entry number matches.
  • Any entries that are being used and protected are not overwritten with the restored data.
Start FAX Utility.
Starting FAX Utility
Select FAX XXXXXX (your printer) from Printer, and then click Speed Dial/Group Dial List.
Note:
If your printer has a security function that allows administrators to restrict users from changing the printer's fax settings, you may need to enter an administrator password to continue.
Click File - Back up - Restore Backup.
Select the file you want to restore.
Restored list is displayed in the Speed Dial/Group Dial List window.
Click in the window.
The restored phone book data is saved to the printer.
