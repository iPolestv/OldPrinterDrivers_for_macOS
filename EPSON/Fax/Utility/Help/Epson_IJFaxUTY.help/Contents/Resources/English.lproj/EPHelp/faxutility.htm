FAX Utility (Main Window)

This is the window where you select the FAX Utility function to use.
Note:
This window appears when FAX Utility starts.
Starting FAX Utility
Item
Explanation
Fax Transmission Record
Displays the transmission records.
Fax Transmission Record
Fax Receive Monitor
Only displayed when a printer (fax) that supports PC-FAX reception is selected. Allows you to select whether or not to save received faxes on a computer.
Check Fax Connection
Checks if the fax function for the connected printer is working normally.
Fax Settings
Sets your fax header and various fax communication settings saved in the printer.
Speed Dial/Group Dial List
Saves or edits the fax speed dial or group dial saved in the printer.
Synchronize Address Book or Synchronize Contacts
Synchronizes data for Address Book or Contacts with FAX Utility so that you can select recipients from the contact list when sending a fax using FAX Utility. If you update Address Book or Contacts after synchronizing, you need to run this synchronization function again to use the updated data with FAX Utility.
