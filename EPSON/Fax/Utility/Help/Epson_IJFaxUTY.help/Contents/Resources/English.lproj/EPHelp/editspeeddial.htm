Speed Dial/Group Dial List - Edit Speed Dial/Group Dial-

This window allows you to register or edit a speed dial or group dial entry of the fax for your printer.
Item
Explanation
Entry Number
Displays the entry number of the speed dial.
Speed Dial
Select to create or edit a speed dial entry.
Group Dial
Select to create or edit a group dial entry. You cannot create a group dial unless you have created speed dial entries.
Name/Group Name
Enter a speed dial or group dial name you want to register.
Index Word *3
Enter an index word for the recipient so that you can use it when performing searches.
Fax Number *1
Enter the recipient fax number. If you want to separate numbers, such as separating city codes from personal numbers, use a space instead of a hyphen (-) because a hyphen (-) acts as a pause when dialing. This item is displayed when you edit the speed dial.
Member *2
Displays a list of registered speed dials. Select the speed dial you want to add to the group dial, and then click OK. This item is displayed when you edit the group dial.
Fax Speed *3
Select the fax speed used for sending a fax to the recipient. The fax speed set for each speed dial entry is only used when sending a fax directly from a printer. FAX Utility ignores the setting and uses the fax speed set on the printer.
OK
Saves the settings and closes the window.
Cancel
Cancels the registration or edit and closes the window.
*1 Required item for registering the speed dial.
*2 When registering a group dial, make sure there is always one or more speed dials selected.
*3 These items may not be displayed depending on the printer.
Note:
  • For information on the characters and number of characters that can be entered, see the following section.
    Acceptable Characters
  • The maximum number of speed dial entries per group is displayed on the bar at the bottom of the group dial edit window.
