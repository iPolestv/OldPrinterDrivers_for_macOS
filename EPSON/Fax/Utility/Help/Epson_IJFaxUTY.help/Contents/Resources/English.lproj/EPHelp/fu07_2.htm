Transmission Error List

The following is a list of error messages displayed in the Communication Error window, with a brief overview.
Error message
Overview
Unsupported paper size
Cannot transmit because an unsupported paper size is selected. Check that the selected paper size is correct.
No recipient specified
Cannot transmit because no recipient is specified. Specify the recipient from the Recipient Settings menu.
Transmitting a Document Created Using a Word Processing or Spreadsheet Application
Out of disk space
Cannot transmit because there is not enough disk space.
Fax line is busy
Transmission is not possible because the recipient’s line is busy, or the type of line differs.
Dialed number invalid
Cannot transmit because the recipient's fax is not responding as the fax number is wrong.
Color transmission failure
Color transmission is not possible because color fax is not supported by the recipient’s fax machine. Change the Color setting on the PC-FAX driver to Black & White and retransmit.
Transmitting a Document Created Using a Word Processing or Spreadsheet Application
No dial tone detected
Failed to detect dial tone. See the printer's manual for the printer’s Dial Tone Detection settings.
No response
No response from the recipient has been detected within a given time period. Check whether faxes can be transmitted directly from the printer (without using a computer). If the error still occurs, check whether the phone line is connected properly and is working normally.
Page limit exceeded
Transmission is not possible because there is a problem with the data, or because the data exceeds 100 pages.
PC to printer comm. error
Communication has been disconnected due to a mistake in the printer connection, or the printer has been turned off.
Printer to recipient comm. error
Transmission is not possible because communication was interrupted or there is a problem with the phone line. Check the phone line connection, and then try again.
PC-FAX is off
Fax transmission from a computer is disabled in your printer.
Communication Error (450)
The printer could not process the fax data sent from the computer, or the computer could not recognize the printer.
Communication Error (454)
Communication Error (459)
Incorrect phone line connection
The phone cable is not correctly connected to the printer (fax). Make sure that you connect a phone cable leading from a telephone wall jack to the LINE port.
