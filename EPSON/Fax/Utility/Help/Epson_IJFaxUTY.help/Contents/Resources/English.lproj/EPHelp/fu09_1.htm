Setting to save a received fax on a computer

You can change the received fax output settings to save received faxes as PDF files on a computer or in a shared folder on a server (PC-FAX reception).
Note:
  • If your printer supports PC-FAX reception, Fax Receive Monitor is displayed on the FAX Utility main window.
  • When using PC-FAX reception, set auto answer.
  • Depending on the printer, received fax data saved to the computer is removed from the printer's memory.
  • Because received faxes are saved as PDF files, you need Cover Flow (10.5 or later), Quick Look (10.5 or later), or Preview.app that are pre-installed with Mac OS X.
Start FAX Utility.
Starting FAX Utility
Click Fax Receive Monitor.
Note:
  • You cannot start multiple instances of Fax Receive Monitor under a multiuser environment.
  • You can only import and save a received fax when Fax Receive Monitor is running.
  • When Fax Receive Monitor is running, an icon appears on the Dock. Adding Fax Receive Monitor to the Dock allows you to start Fax Receive Monitor easily. To add Fax Receive Monitor to the Dock, click the Fax Receive Monitor icon while holding down the Control key, and then click Add to Dock on the menu.
Select your printer, and then Receive Settings.
Note:
  • Because you can only save faxes to one print (fax) queue at a time, if there are multiple print (fax) queues on the same printer, such as "FAX XXXXXX (USB)" for USB connections and "FAX XXXXXX(IP)" for network connections, select which one you want to use to save received faxes.
  • If your printer has a security function that allows administrators to restrict users from changing the printer's fax settings, you may need to enter an administrator password to continue.
Select “Save” (save faxes on this computer), and then click Save to to specify a folder to save received faxes.
Note:
  • The following folder is specified by default.
    \User\Login name\PCFax\Printer name
  • When the printer is shared on a network, you can change settings from any computer on the network. However, faxes can only be saved to one account at a time. If “Save” (save faxes on this computer) is set and Received Fax Output Settings is selected on another computer, the Received Fax Output Settings window displays the message "Save faxes on another computer or print queue".
  • On the window, you can change to "Save" (save faxes on this computer) while no faxes remain on the printer. If any faxes remain on the printer, an error message is displayed informing you that you cannot change the setting.
    When sharing the printer (fax), we recommend that you create a folder on the network for saving faxes.
  • You can change the setting of the Fax Receive Monitor when you click Preferences on the Fax Receive Monitor.
    Preferences (Only for models with PC-FAX reception)
Click Send to send the settings to the printer.
Note:
When the PC-FAX reception's user authentication (name and password for file sharing) is set in the printer, you need to register that information to the computer's keychain.
If the information is not registered in the keychain, the registration entry screen is displayed. Enter your name and password, select Remember this password in my keychain, and then click Connect.
If you do not save this authentication information in the keychain, you will be asked to enter it every time you receive a fax and save it to the computer.
