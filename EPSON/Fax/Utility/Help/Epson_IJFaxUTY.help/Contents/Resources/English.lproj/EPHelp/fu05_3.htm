Registering Speed Dials by Importing from Address Book or Contacts

You can register the information in the Address Book or Contacts to the speed dial.
Start FAX Utility.
Starting FAX Utility
Select FAX XXXXXX (your printer) from Printer, and then click Speed Dial/Group Dial List.
The speed dial/group dial lists saved in your printer are displayed.
Note:
If your printer has a security function that allows administrators to restrict users from changing the printer's fax settings, you may need to enter an administrator password to continue.
To add a new speed dial entry by importing data from Address Book or Contacts, select an empty line and then click . To overwrite an existing entry with Address Book data or Contacts data, select the target entry and click .
Select the data you want to register to speed dial, and then click Import.
The selected data is added to the Speed Dial/Group Dial List.
After completing all settings, click in the Speed Dial/Group Dial List window.
The speed dials and group dials displayed in the list are saved to the printer.
Note
  • If the imported Address Book data or Contacts data does not have name information, the data is imported without a name.
  • If the imported Address Book data or Contacts data includes any characters that are not supported by the Speed Dial/Group Dial List, the data is imported without a name.
    For information on the supported characters, see the following section.
    Acceptable Characters
  • If the name of the imported Address Book data or Contacts data has 31 characters or more, the first 30 characters are imported and the rest are dropped.
