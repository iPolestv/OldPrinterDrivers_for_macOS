Saving Backup of Speed Dial

You can save a backup of the configured speed dial and group dial. Speed/group dial list data may be lost due to a printer malfunction. We recommend that you make a backup of the data whenever you update the dial list. Epson shall not be responsible for the loss of any data, for backing up or recovering data and/or settings even during a warranty period.
Note:
Data indicated as "In use by another function" is not saved.
Start FAX Utility.
Starting FAX Utility
Select FAX XXXXXX (your printer) from Printer, and then click Speed Dial/Group Dial List.
The speed dial/group dial lists saved in your printer are displayed.
Note:
If your printer has a security function that allows administrators to restrict users from changing the printer's fax settings, you may need to enter an administrator password to continue.
Click File - Back up - Save Backup.
Specify the file name and file directory, and then click Save.
