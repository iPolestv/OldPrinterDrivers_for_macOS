Fax Receive Monitor (Only for models with PC-FAX reception)

Toolbar
Icon and Message on the Printer List
This window allows you to check the fax reception status and make output settings for received faxes.
Note:
This window is displayed when you click Fax Receive Monitor in the FAX Utility (main window).

Toolbar

Item
Explanation
Open the received fax folder
Opens the folder in which received faxes are saved.
Check new faxes
Checks for new faxes.
Received Fax Output Settings
Sets whether or not to save received faxes on a computer.
Preferences
Configures the Fax Receive Monitor preferences.

Icon and Message on the Printer List

Icon
Message
Explanation
Waiting / Checking for new faxes
Checking for new faxes or standing by.
Received faxes saved successfully in the specified folder.
New faxes received by a printer (fax) have been saved to the specified folder.
The printer is not responding
Check the connection between your computer and printer, or check that the printer is turned on.
The printer is processing data
The printer is processing other data. Please wait.
Cannot find the received fax folder.
Check that the name of the specified folder is correct. If you are using a shared folder on the network, make sure that you have access rights to the folder.
Failed to save faxes in the specified folder.
Check that the specified folder is available or that you have access rights to the folder.
A printer communication error occurred
Check the connection between your computer and printer, or check that the printer is turned on. If you are using a network connection, check that you can access the printer via the network.
The "FAXRECV" disk is being used by another application.
Close the program that is using the "FAXRECV" disk.
Cannot save faxes. User authentication failed.
Check the PC-FAX reception's user authentication (name and password for file sharing) that is set in the printer, make the Received Fax Output Settings again, and then enter the user authentication information correctly. Remember to select Remember this password in my keychain.
