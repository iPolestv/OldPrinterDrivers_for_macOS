Checking New Faxes

Although you can check new faxes by opening the folder specified in the Received Fax Output Settings, the procedure explained here is for checking new faxes by using the Fax Receive Monitor. This is only available on the computer with “Save” (save faxes on this computer) selected.
Note:
If Notify me of new faxes via a dock icon is selected in the Fax Receive Monitor preferences, the Fax Receive Monitor icon "jumps" to notify you that a new fax is available.
Follow the steps below to open the folder for received faxes.
1.
Click the Fax Receive Monitor icon on the Dock to open the Fax Receive Monitor.
2.
Select a printer and click Open the received fax folder, or double-click the printer.
Check the date and the sender in the file name, and then open the PDF file.
Note:
  • Received faxes are automatically renamed using the following naming format:
    YYYYMMDDHHMMSS_xxxxxxxxxx_nnnnn (Year/Month/Day/Hour/Minute/Second_sender’s number)
    "Sender's number" displays information transmitted from the sender. This number may not be displayed depending on the sender's settings.
  • You can check new faxes instantly by selecting Check new faxes on the Fax Receive Monitor.
