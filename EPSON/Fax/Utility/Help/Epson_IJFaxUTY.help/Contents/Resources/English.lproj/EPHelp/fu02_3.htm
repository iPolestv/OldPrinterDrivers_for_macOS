Synchronizing Address Book or Contacts with FAX Utility (for Mac OS X 10.7.x or later)

If you want to select a recipient from Address Book or Contacts to send a fax using FAX Utility, you need to synchronize it in advance. If you update Address Book or Contacts after synchronizing, you need to run this synchronization function again to use the updated data with FAX Utility.
Start FAX Utility.
Starting FAX Utility
Select Synchronize Address Book or Synchronize Contacts.
Follow the on-screen instructions to synchronize.
