Kontrollere/slette sendingsrapporter

Du kan sjekke og slette fakssendingsrapportlisten.
Start FAX Utility.
Starte FAX Utility
Velg FAX XXXXXX (skriveren) fra Printer (Skriver), og klikk på Fax Transmission Record (Fakssendings rapport).
Sjekk sendingsrapportene.
Listen lagrer opptil 1000 sendingsrapporter.
Hvis du vil slette fakssendingsrapport, velger du Delete (Slett) fra Edit (Rediger)-menyen. All data blir slettet. (Du kan ikke slette hver post individuelt.)
