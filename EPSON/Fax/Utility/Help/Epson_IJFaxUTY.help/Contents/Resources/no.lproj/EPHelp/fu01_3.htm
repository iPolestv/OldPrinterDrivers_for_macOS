Starte FAX Utility

For Mac OS X 10.6.x eller nyere
Klikk Apple - System Preferences (Systemvalg) – Print & Scan (Skrivere og skannere) eller Print & Fax (Utskrift og faks).
Velg FAX XXXXXX (skriveren) fra Printer (Skriver), og klikk Options & Supplies (Valg og forbruksmateriell) - Utility (Verktøy) - Open Printer Utility (Åpne Skriververktøy).
For Mac OS X 10.5.x
Klikk Apple - System Preferences (Systemvalg) - Print & Fax (Utskrift og faks).
Velg FAX XXXXXX (skriveren) fra Printer (Skriver), og klikk deretter på Open Print Queue (Åpne utskriftskø) - Utility (Verktøy).
For Mac OS X 10.4.x
Dobbeltklikk på harddisken - Applications (Programmer) - Utilities (Verktøy) - Printer Setup Utility (Skriveroppsettverktøy).
Velg FAX XXXXXX (skriveren) fra Printer (Skriver), og klikk på Utility (Verktøy).
