Sette opp faksfunksjonen på skriveren

Du kan konfigurere Header Information Settings (Innstillinger for topptekstinformasjon), Auto Answer Settings (Innstillinger for automatisk svar) og Dial Tone Detection Settings (Innstillinger for registrering av summetone) i faksfunksjonen på skriveren.
Start FAX Utility.
Starte FAX Utility
Velg FAX XXXXXX (skriveren) fra Printer (Skriver), og klikk på Fax Settings (Faksinnstillinger).
Følg instruksjonene på skjermen for å konfigurere Header Information Settings (Innstillinger for topptekstinformasjon), Auto Answer Settings (Innstillinger for automatisk svar) og Dial Tone Detection Settings (Innstillinger for registrering av summetone).
Merk:
Hvis skriveren har en sikkerhetsfunksjon som gjør det mulig for administratorer å begrense brukere som kan endre faksinnstillingene for skriveren, kan det hende du må oppgi et administratorpassord for å fortsette.
