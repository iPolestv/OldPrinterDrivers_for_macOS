Sikkerhetskopiere direktevalg

Du kan lagre en sikkerhetskopi av konfigurerte direktevalg og gruppevalg. Data for direktevalg/gruppevalg kan gå tapt dersom det er feil med skriveren. Vi anbefaler at du tar en sikkerhetskopi av dataen når du oppdaterer listen. Epson skal ikke holdes ansvarlig for eventuelle tap av data, for sikkerhetskopiering eller gjenoppretting av data og/eller innstillinger, selv i løpet av en garantiperiode.
Merk:
Data angitt som "In use by another function (Brukes av en annen funksjon)" blir ikke lagret.
Start FAX Utility.
Starte FAX Utility
Velg FAX XXXXXX (skriveren) fra Printer (Skriver), og klikk på Speed Dial/Group Dial List (Liste over direktevalg/gruppevalg).
Liste over direktevalg/gruppevalg som er lagret i skriveren vises.
Merk:
Hvis skriveren har en sikkerhetsfunksjon som gjør det mulig for administratorer å begrense brukere som kan endre faksinnstillingene for skriveren, kan det hende du må oppgi et administratorpassord for å fortsette.
Klikk på File (Fil) - Back up (Sikkerhetskopier) - Save Backup (Lagre sikkerhetskopi).
Angi filnavn og mappe, og klikk Save (Lagre).
