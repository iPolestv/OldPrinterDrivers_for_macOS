Feilmeldinger for mottak (kun for modeller med PC-FAX-mottak)

Følgende er en liste over feilmeldinger som kan vises når en faks mottas og hvordan man problemet kan løses.
Melding
Prøv dette
A printer communication error occurred. (Det oppstod en kommunikasjonsfeil for skriveren.)
Slå av skriveren, og slå den på igjen. Vent en stund, og prøv på nytt.
The printer (fax driver) you are setting has been deleted. Add the printer (fax driver) in [System Preferences] - [Print & Fax]. (Skriveren (faksdriver) du angir, er blitt slettet. Legg til skriveren (faksdriver) i [Systemvalg] og deretter [Utskrift og faks].)
Legg til PC-FAX-driveren i skriverlisten. Klikk OK i dialogboksen. Dialogboksen for å legge til en skriver vises.
Change not accepted. Some printer settings have been changed by another computer, print queue, or the printer. Please try again. (Endring ble ikke godtatt. Enkelte skriverinnstillinger er blitt endret fra en annen datamaskin, utskriftskø eller skriveren. Prøv på nytt.)
Received Fax Output Settings (Utskriftsinnstillinger for mottatt faks) er endret fra en annen datamaskin på nettverket, fra utskriftskøen eller fra skriveren. Vent en stund, og prøv på nytt.
Mottatte fakser kan bare lagres på én datamaskin.
Received faxes not saved due to an error. (Mottatte fakser ble ikke lagret pga. en feil.)
Kunne ikke lagre den mottatte faksen i den angitte mappen. Kontroller tilkoblingen til den angitte mappen for å forsikre deg om at den er tilgjengelig, og kontroller at du har tilgangsrettigheter til mappen.
Processing canceled because an error occurred. (Behandling ble avbrutt fordi det oppstod en feil.)
Hvis du ikke kan løse problemet ved å starte skriveren og datamaskinen på nytt, kan du kontakte Epsons kundestøtte. Du finner kontaktinformasjon i bruksanvisningen til skriveren.
Cannot save faxes. User authentication failed. (Kan ikke lagre fakser. Brukergodkjenning mislyktes.)
Kan ikke lagre mottatte fakser fordi PC-FAX-mottaks brukerautentisering (navn og passord for fildeling) ikke er riktig innstilt i datamaskinens nøkkelring.
Still inn Fax Output Setting (Utskriftsinnstillinger for mottatt faks) på nytt, og skriv inn riktig brukerautentiseringsinformasjon når påloggingsskjermen vises. Husk å velge Remember this password in my keychain (Legg til dette passordet på nøkkelringen min).
