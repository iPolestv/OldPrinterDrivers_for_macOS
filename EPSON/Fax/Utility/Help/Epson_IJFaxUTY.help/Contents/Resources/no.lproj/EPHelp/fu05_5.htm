Gjenopprette en sikkerhetskopiert direktevalgliste

Du kan sikkerhetskopiere telefonbokdata (direktevalg/gruppevalg) og deretter gjenopprette dem på skriveren.
Merk:
  • Du kan ikke gjenopprette data sikkerhetskopiert i Windows Fax Utility.
  • Hvis den sikkerhetskopierte direktevalg-/gruppevalglisten inneholder elementer som ikke støttes av skriveren som du vil gjenopprette listen til, blir ikke disse elementene gjenopprettet.
  • Hvis den sikkerhetskopierte direktevalg-/gruppevalglisten ikke inneholder elementer som støttes av skriveren som du vil gjenopprette dataene til, blir de gjenopprettet på følgende måte.
    Index Word (Indeksord): tom
    Fax Speed (Fakshastighet): forhåndsinnstilt verdi
  • Alle elementer, for Name/Group Name (Navn/gruppenavn) som inneholder tegn som ikke støttes av skriveren som du vil gjenopprette dataene til, blir tomme når de gjenopprettes.
  • Oppføringer angitt som In use by another function (Brukes av en annen funksjon) i FAX Utility overskrives med de gjenopprettede dataene så lenge oppføringsnummeret samsvarer.
  • Oppføringer som er i bruk og beskyttet blir ikke overskrevet med gjenopprettede data.
Start FAX Utility.
Starte FAX Utility
Velg FAX XXXXXX (skriveren) fra Printer (Skriver), og klikk på Speed Dial/Group Dial List (Liste over direktevalg/gruppevalg).
Merk:
Hvis skriveren har en sikkerhetsfunksjon som gjør det mulig for administratorer å begrense brukere som kan endre faksinnstillingene for skriveren, kan det hende du må oppgi et administratorpassord for å fortsette.
Klikk på File (Fil) - Back up (Sikkerhetskopier) - Restore Backup (Gjenopprett sikkerhetskopi).
Velg den første filen du vil gjenopprette.
Den gjenopprettede listen vises i vinduet til Speed Dial/Group Dial List (Liste over direktevalg/gruppevalg).
Klikk i vinduet.
Gjenopprettede telefonbokdata lagres på skriveren.
