受信ファクスをコンピューターで保存する設定に切り替える

受信ファクスの出力設定で設定を変更すると、受信したファクスを指定フォルダにPDF形式で保存できます。
参考
  • PC-FAX受信対応プリンターでは、FAX Utility（メイン画面）に［ファクス受信モニター］が表示されます。
  • PC-FAX受信のときは、プリンター（ファクス）を自動受信設定にしてください。
  • お使いのプリンターによっては、コンピューターに受信ファクスデータが保存されるとプリンターの内部メモリーからは削除されます。詳しくはプリンターのマニュアルで確認してください。
  • 受信ファクスデータ（PDF）は、Mac OS X標準のCover Flow（Mac OS X v10.5.x以上）、クイックルック（Mac OS X v10.5.x以上）、プレビュー.appなどで見ることができます。
FAX Utilityを起動します。
FAX Utilityの起動方法
［ファクス受信モニター］をクリックします。
参考
  • マルチユーザー環境ではファクス受信モニターの複数起動はできません。
  • ファクス受信モニターを起動しているときのみ、受信ファクスの取り込み/保存ができます。
  • ファクス受信モニターを起動すると、Dockにファクス受信モニターのアイコンが表示されます。ファクス受信モニターをDockに登録しておくと、ファクス受信モニターを簡単に起動できます。登録するには、［Control］キーを押したままファクス受信モニターのアイコンをクリックします。表示されたメニューの［Dockに追加］をクリックすると、アイコンがDockに追加されます。
ご使用のプリンターをクリックし、［受信ファクスの出力設定］をクリックします。
参考
  • 受信ファクスの出力設定はプリントキューごとにします。1台のプリンターをUSBとネットワークの両方で接続しているときは、USB接続のプリントキュー（FAX XX-XXXX （USB））とネットワーク接続のプリントキュー（FAX XX-XXXX （IP））の2つがプリンターのリストに表示され、それぞれで受信ファクスの出力設定ができます。ただし、片方のプリントキューで次の手順4の設定をすると、もう片方のプリントキューで受信ファクスの取り込み/保存はできません。
  • プリンターにセキュリティー設定機能があり、パネルがロックされている場合は、管理者のパスワードを入力する画面が表示されます。
［ "ファクス保存"（このコンピューターで保存する）］を選択し、［保存先...］をクリックして受信ファクスデータを保存するフォルダーを指定します。
参考
  • 初期設定では以下のフォルダーが指定されています。
    ￥ユーザ￥ログイン名￥PCFax￥機種名
  • 複数のコンピューターで1台のプリンター（ファクス）を共有しているとき、FAX Utilityがインストールされていればどのコンピューターからも［受信ファクスの出力設定］ができます。ただし、受信ファクスデータを取り込めるのは1台のコンピューターのみです。［"ファクス保存"（このコンピューターで保存する）］が設定されているとき、他のコンピューターで［受信ファクスの出力設定］を開くと、プリンターの設定が他のコンピューターなどでファクスを保存する設定になっていることが分かります。
    ここで［"ファクス保存"（このコンピューターで保存する）］を選択すると受信ファクスの保存先を変更できます。ただし、まだ取り込んでいない受信ファクスがプリンターに残っているときは、エラーメッセージが表示され、設定は変更できません。また、複数のコンピューターで1台のプリンター（ファクス）を共有するときは、保存フォルダーをネットワーク上の共有フォルダーにすることをお勧めします。
  • ［ファクス受信モニター］画面の［環境設定］をクリックすると、［環境設定］画面が表示され、ファクス受信モニターの設定を変更できます。
    環境設定（PC-FAX受信対応プリンターのみ）
［送信］をクリックしてプリンターに設定情報を送信します。
参考
PC-FAX受信のユーザー認証（ファイル共有の名前/パスワード）がプリンターに設定されている場合は、プリンターに設定されたユーザー認証情報をコンピューターのキーチェーンに登録する必要があります。
ユーザー認証情報がコンピューターのキーチェーンに登録されていない場合は、登録のための入力画面が表示されるので、名前とパスワードを入力して、 ［このパスワードをキーチェーンに保存］をチェックしてから［接続］をクリックしてください。
認証情報をキーチェーンに保存しなかった場合は、ファクスを受信してデータをコンピューターに保存する際に、毎回認証情報を求められます。
以上で終了です。
