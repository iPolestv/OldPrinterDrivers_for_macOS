Received Fax Output Settings (Ρυθμίσεις εξόδου εισερχόμενων φαξ) (Μόνο για μοντέλα με λήψη PC-FAX)

Το παράθυρο αυτό σας επιτρέπει να ορίσετε εάν ή όχι θα αποθηκεύσετε ληφθέντα φαξ σε υπολογιστή.
Σημείωση:
Το παράθυρο αυτό προβάλλεται όταν κάνετε κλικ στο Received Fax Output Settings (Ρυθμίσεις εξόδου εισερχόμενων φαξ) στην Fax Receive Monitor (Παρακολούθηση λήψης φαξ).
Στοιχείο
Επεξήγηση
Setting on the printer (Ρύθμιση στον εκτυπωτή)
Προβάλλει την τρέχουσα ρύθμιση εξόδου εισερχόμενων φαξ.
"Print" ("Εκτύπωση") *
Εκτυπώνει ληφθέντα φαξ στον εκτυπωτή.
Not save faxes (Μη αποθήκευση φαξ) *
Επιλέξτε το για να σταματήσετε την αποθήκευση φαξ σε έναν υπολογιστή. Τα ληφθέντα φαξ εκτυπώνονται ή αποθηκεύονται στη μνήμη του εκτυπωτή ανάλογα με τις ρυθμίσεις του εκτυπωτή.
"Save" (save faxes on this computer) ("Αποθήκευση" (αποθήκευση φαξ σε αυτόν τον υπολογιστή))
Αποθηκεύει ληφθέντα φαξ στον καθορισμένο φάκελο. Τα φαξ αποθηκεύονται ως αρχεία PDF.
Save to (Αποθήκ. σε)
Καθορίζει το φάκελο στον οποίο θα αποθηκευτούν τα ληφθέντα φαξ.
Send the settings to the printer (Αποστολή των ρυθμίσεων στον εκτυπωτή)
Μεταφέρει τις ρυθμίσεις εξόδου εισερχόμενων φαξ στον εκτυπωτή.
* Τα στοιχεία αυτά ενδέχεται να μην προβάλλονται, ανάλογα με τον εκτυπωτή.
Όταν η πιστοποίηση ταυτότητας χρήστη για τη λήψη PC-FAX (όνομα και κωδικός πρόσβασης για την κοινή χρήση αρχείων) έχει οριστεί στον εκτυπωτή, πρέπει να καταχωρήσετε τις πληροφορίες αυτές στην αλυσίδα κλειδιών του υπολογιστή.
Εάν οι πληροφορίες δεν είναι καταχωρημένες στην αλυσίδα κλειδιών, προβάλλεται η οθόνη καταχώρησης εγγραφής. Εισαγάγετε το όνομα και τον κωδικό πρόσβασής σας, επιλέξτε Remember this password in my keychain (Απομνημόνευση αυτού του κωδικού πρόσβασης στην αλυσίδα κλειδιών μου) και κατόπιν κάντε κλικ στο Connect (Σύνδεση).
Εάν δεν αποθηκεύσετε αυτές τις πληροφορίες επαλήθευσης στην αλυσίδα κλειδιών, θα σας ζητηθεί να τις εισαγάγετε κάθε φορά που λαμβάνετε ένα φαξ και το αποθηκεύετε στον υπολογιστή.
