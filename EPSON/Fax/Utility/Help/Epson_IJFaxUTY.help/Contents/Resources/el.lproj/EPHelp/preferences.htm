Preferences (Προτιμήσεις) (Μόνο για μοντέλα με λήψη PC-FAX)

Το παράθυρο αυτό σας επιτρέπει να διαμορφώσετε τις ρυθμίσεις λήψης φαξ, όπως πώς γίνονται οι ειδοποιήσεις όταν λαμβάνεται ένα νέο φαξ.
Σημείωση:
  • Το παράθυρο αυτό προβάλλεται όταν κάνετε κλικ στις Fax Receive Monitor Preferences (Προτιμήσεις παρακολούθησης λήψης φαξ) στην Fax Receive Monitor (Παρακολούθηση λήψης φαξ).
  • Οι ρυθμίσεις αυτές εφαρμόζονται σε όλους τους εκτυπωτές και ουρές εκτύπωσης που προβάλλονται στην Fax Receive Monitor (Παρακολούθηση λήψης φαξ).
Στοιχείο
Επεξήγηση
Specify an interval to check for new faxes (Καθορίστε ένα χρονικό διάστημα μεταξύ των ελέγχων για νέα φαξ )
Καθορίζει το χρονικό διάστημα στο οποίο θα γίνεται έλεγχος για νέα φαξ, από 1 έως 30 λεπτά.
Notify me of new faxes via a dock icon (Να ειδοποιούμαι για νέα φαξ μέσω εικονιδίου στο dock)
Επιλέξτε το για αναπήδηση του εικονιδίου Fax Receive Monitor (Παρακολούθηση λήψης φαξ) στο Σταθμό υποδοχής όταν φτάνει νέο φαξ.
Notify me of errors via a dock icon (Να ειδοποιούμαι για σφάλματα μέσω εικονιδίου στο dock)
Επιλέξτε το για αναπήδηση του εικονιδίου Fax Receive Monitor (Παρακολούθηση λήψης φαξ) στο Σταθμό υποδοχής όταν προκύπτει σφάλμα. Κάντε κλικ στο εικονίδιο για προβολή του παραθύρου Παρακολούθηση λήψης φαξ, όπου μπορείτε να ελέγξετε τις πληροφορίες σφάλματος.
Start at login (Έναρξη κατά τη σύνδεση)
Προτείνουμε ρύθμιση αυτού σε Automatic (Αυτόματο). Όταν έχει οριστεί "Save" (save faxes on this computer) ("Αποθήκευση" (αποθήκευση φαξ σε αυτόν τον υπολογιστή)) στις Received Fax Output Settings (Ρυθμίσεις εξόδου εισερχόμενων φαξ), η Fax Receive Monitor (Παρακολούθηση λήψης φαξ) ξεκινάει αυτόματα όταν συνδέεστε. Εάν επιλέξετε On (Ενεργοποίηση), η Fax Receive Monitor (Παρακολούθηση λήψης φαξ) ξεκινάει κατά τη σύνδεση, ανεξαρτήτως των ρυθμίσεων αποστολής. Εάν επιλέξετε Off (Απενεργοποίηση), η Fax Receive Monitor (Παρακολούθηση λήψης φαξ) δεν εκκινεί όταν συνδέεστε.
Wait time before checking new faxes (Χρόνος αναμονής πριν από τον έλεγχο για νέα φαξ)
Το στοιχείο αυτό είναι διαθέσιμο όταν έχει οριστεί Automatic (Αυτόματο) ή On (Ενεργοποίηση) στην Start at login (Έναρξη κατά τη σύνδεση). Καθορίστε το χρόνο αναμονής (0 έως 10 λεπτά) πριν τον έλεγχο για νέα φαξ από την έναρξη της Fax Receive Monitor (Παρακολούθηση λήψης φαξ) όταν συνδέεστε. Αυτό είναι χρήσιμο όταν ο φάκελος στον οποίο αποθηκεύετε φαξ βρίσκεται σε δίκτυο και χρειάζεστε χρόνο για να δημιουργήσετε σύνδεση με το φάκελο μετά τη σύνδεσή σας.
Σημείωση:
Εάν ο φάκελος στον οποίο θα αποθηκευτούν ληφθέντα φαξ είναι κοινόχρηστος φάκελος σε δίκτυο, μπορείτε να συνδεθείτε στο φάκελο όταν συνδέεστε, καταχωρώντας το φάκελο στον κατάλογο στοιχείων σύνδεσης. Κάντε κλικ στα Apple - System Preferences (Προτιμήσεις συστήματος) - Account (Λογαριασμός) - Login to register (Σύνδεση για καταχώρηση).
