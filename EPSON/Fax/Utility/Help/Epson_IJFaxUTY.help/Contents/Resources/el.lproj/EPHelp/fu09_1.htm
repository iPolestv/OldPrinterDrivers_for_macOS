Ρύθμιση για αποθήκευση ενός ληφθέντος φαξ σε υπολογιστή

Μπορείτε να αλλάξετε τις ρυθμίσεις εξόδου ληφθέντος φαξ για αποθήκευση ληφθέντων φαξ ως αρχεία PDF σε υπολογιστή ή σε κοινόχρηστο φάκελο σε διακομιστή (λήψη PC-FAX).
Σημείωση:
  • Εάν ο εκτυπωτής σας υποστηρίζει λήψη PC-FAX, το Fax Receive Monitor (Παρακολούθηση λήψης φαξ) εμφανίζεται στο κεντρικό παράθυρο του FAX Utility.
  • Όταν χρησιμοποιείτε τη λήψη PC-FAX, ορίστε την αυτόματη απάντηση.
  • Ανάλογα με τον εκτυπωτή, τα δεδομένα ληφθέντων φαξ που αποθηκεύονται στον υπολογιστή αφαιρούνται από τη μνήμη του εκτυπωτή.
  • Καθώς τα ληφθέντα φαξ αποθηκεύονται ως αρχεία PDF, χρειάζεστε Cover Flow (10.5 ή νεότερη έκδοση), Quick Look (10.5 ή νεότερη έκδοση), ή Preview.app τα οποία είναι προεγκατεστημένα με τοMac OS X.
Εκκινήστε το FAX Utility.
Εκκίνηση FAX Utility
Κάντε κλικ στο Fax Receive Monitor (Παρακολούθηση λήψης φαξ).
Σημείωση:
  • Δεν μπορείτε να ξεκινήσετε πολλαπλές περιπτώσεις του Fax Receive Monitor (Παρακολούθηση λήψης φαξ) σε περιβάλλον πολλαπλών χρηστών.
  • Μπορείτε να εισάγετε και να αποθηκεύσετε ληφθέν φαξ μόνο όταν λειτουργεί το Fax Receive Monitor (Παρακολούθηση λήψης φαξ).
  • Όταν εκτελείται το Fax Receive Monitor (Παρακολούθηση λήψης φαξ), εμφανίζεται εικονίδιο στο Σταθμό υποδοχής. Προσθέτοντας το Fax Receive Monitor (Παρακολούθηση λήψης φαξ) στο Σταθμό υποδοχής μπορείτε να ξεκινήσετε εύκολα το Fax Receive Monitor (Παρακολούθηση λήψης φαξ). Για να προσθέσετε το Fax Receive Monitor (Παρακολούθηση λήψης φαξ) στο Σταθμό υποδοχής, κάντε κλικ στο εικονίδιο Fax Receive Monitor (Παρακολούθηση λήψης φαξ) ενόσω κρατάτε πατημένο το πλήκτρο Control (Έλεγχος) και έπειτα κάντε κλικ στο Add to Dock (Προσθήκη στο σταθμό υποδοχής) στο μενού.
Επιλέξτε τον εκτυπωτή σας και έπειτα Receive Settings (Ρυθμίσεις λήψης).
Σημείωση:
  • Καθώς μπορείτε να αποθηκεύσετε φαξ μόνο σε μία ουρά εργασιών εκτύπωσης (φαξ) κάθε φορά, εάν υπάρχουν πολλαπλές ουρές εκτύπωσης (φαξ) στον ίδιο εκτυπωτή, όπως το "ΦΑΞ XXXXXX (USB)" για συνδέσεις USB και "ΦΑΞ XXXXXX (IP)" για συνδέσεις δικτύου, επιλέξτε ποια επιθυμείτε να χρησιμοποιήσετε για να αποθηκεύσετε τα ληφθέντα φαξ.
  • Εάν ο εκτυπωτής σας διαθέτει λειτουργία ασφαλείας η οποία επιτρέπει σε διαχειριστές να περιορίζουν χρήστες να αλλάζουν τις ρυθμίσεις φαξ του εκτυπωτή, ίσως χρειαστεί να εισαγάγετε κωδικό πρόσβασης διαχειριστή για να συνεχίσετε.
Επιλέξτε "Save" (save faxes on this computer) ("Αποθήκευση" (αποθήκευση φαξ σε αυτόν τον υπολογιστή)) και έπειτα κάντε κλικ στο Save to (Αποθήκ. σε) για να καθορίσετε φάκελο προς αποθήκευση των ληφθέντων φαξ.
Σημείωση:
  • Ο ακόλουθος φάκελος καθορίζεται βάσει προεπιλογής.
    \Χρήστης\Όνομα σύνδεσης\ΗΥΦαξ\Όνομα εκτυπωτή
  • Όταν ο εκτυπωτής είναι κοινόχρηστος σε ένα δίκτυο, μπορείτε να αλλάξετε ρυθμίσεις από οποιονδήποτε υπολογιστή στο δίκτυο. Ωστόσο, τα φαξ μπορούν να αποθηκευτούν μόνο σε ένα λογαριασμό κάθε φορά. Εάν έχει οριστεί η επιλογή "Save" (save faxes on this computer) ("Αποθήκευση" (αποθήκευση φαξ σε αυτόν τον υπολογιστή)) και έχουν επιλεγεί Received Fax Output Settings (Ρυθμίσεις εξόδου εισερχόμενων φαξ) σε άλλο υπολογιστή, στο παράθυρο Received Fax Output Settings (Ρυθμίσεις εξόδου εισερχόμενων φαξ) εμφανίζεται το μήνυμα "Save faxes on another computer or print queue (Αποθ. φαξ σε άλλον υπολ. ή στην ουρά αναμ. εκτύπ.)".
  • Στο παράθυρο, μπορείτε να αλλάξετε σε "Save" (save faxes on this computer) ("Αποθήκευση" (αποθήκευση φαξ σε αυτόν τον υπολογιστή)) όταν δεν απομένουν άλλα φαξ στον εκτυπωτή. Εάν παραμένουν φαξ στον εκτυπωτή, εμφανίζεται μήνυμα σφάλματος ενημερώνοντάς σας ότι δεν μπορείτε να αλλάξετε τη ρύθμιση.
    Κατά την κοινή χρήση του εκτυπωτή (φαξ), σας προτείνουμε να δημιουργήσετε φάκελο στο δίκτυο για την αποθήκευση φαξ.
  • Μπορείτε να αλλάξετε τη ρύθμιση Fax Receive Monitor (Παρακολούθηση λήψης φαξ) όταν κάνετε κλικ στο Preferences (Προτιμήσεις) στοFax Receive Monitor (Παρακολούθηση λήψης φαξ).
    Preferences (Προτιμήσεις) (Μόνο για μοντέλα με λήψη PC-FAX)
Κάντε κλικ στο Send (Αποστολή) για να στείλετε τις ρυθμίσεις στον εκτυπωτή.
Σημείωση:
Όταν η πιστοποίηση ταυτότητας χρήστη για τη λήψη PC-FAX (όνομα και κωδικός πρόσβασης για την κοινή χρήση αρχείων) έχει οριστεί στον εκτυπωτή, πρέπει να καταχωρήσετε τις πληροφορίες αυτές στην αλυσίδα κλειδιών του υπολογιστή.
Εάν οι πληροφορίες δεν είναι καταχωρημένες στην αλυσίδα κλειδιών, προβάλλεται η οθόνη καταχώρησης εγγραφής. Εισαγάγετε το όνομα και τον κωδικό πρόσβασής σας, επιλέξτε Remember this password in my keychain (Απομνημόνευση αυτού του κωδικού πρόσβασης στην αλυσίδα κλειδιών μου) και κατόπιν κάντε κλικ στο Connect (Σύνδεση).
Εάν δεν αποθηκεύσετε αυτές τις πληροφορίες επαλήθευσης στην αλυσίδα κλειδιών, θα σας ζητηθεί να τις εισαγάγετε κάθε φορά που λαμβάνετε ένα φαξ και το αποθηκεύετε στον υπολογιστή.
