Καταχώρηση ταχέων κλήσεων με εισαγωγή από Address Book (Βιβλίο διευθύνσεων) ή Contacts (Επαφές)

Μπορείτε να καταχωρήσετε τις πληροφορίες από Address Book (Βιβλίο διευθύνσεων) ή Contacts (Επαφές) στην ταχεία κλήση.
Εκκινήστε το FAX Utility.
Εκκίνηση FAX Utility
Επιλέξτε ΦΑΞ XXXXXX (ο εκτυπωτής σας) από Printer (Εκτυπωτής) και κατόπιν κάντε κλικ στο Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων).
Προβάλλονται οι λίστες ταχείας κλήσης/ομαδικής κλήσης που είναι αποθηκευμένες στον εκτυπωτή σας.
Σημείωση:
Εάν ο εκτυπωτής σας διαθέτει λειτουργία ασφαλείας η οποία επιτρέπει σε διαχειριστές να περιορίζουν χρήστες να αλλάζουν τις ρυθμίσεις φαξ του εκτυπωτή, ίσως χρειαστεί να εισαγάγετε κωδικό πρόσβασης διαχειριστή για να συνεχίσετε.
Για να προσθέσετε νέα καταχώρηση ταχείας κλήσης με εισαγωγή δεδομένων από Address Book (Βιβλίο διευθύνσεων) ή Contacts (Επαφές), επιλέξτε κενή γραμμή και κατόπιν κάντε κλικ στο . Για να αντικαταστήσετε μία υπάρχουσα καταχώρηση με δεδομένα από Address Book (Βιβλίο διευθύνσεων) ή Contacts (Επαφές), επιλέξτε την καταχώρηση στόχο και κάντε κλικ στο .
Επιλέξτε τα δεδομένα που επιθυμείτε να καταχωρήσετε στην ταχεία κλήση και κατόπιν κάντε κλικ στο Import (Εισαγωγή).
Τα επιλεγμένα δεδομένα προστίθενται στο Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων).
Αφού ολοκληρώσετε όλες τις ρυθμίσεις, κάντε κλικ στο στο παράθυρο Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων).
Οι ταχείες κλήσεις και οι ομαδικές κλήσεις που προβάλλονται στον κατάλογο αποθηκεύονται στον εκτυπωτή.
Σημείωση
  • Εάν τα εισηγμένα δεδομένα στο Address Book (Βιβλίο διευθύνσεων) ή στις Contacts (Επαφές) δεν έχουν πληροφορίες ονόματος, τα δεδομένα εισάγονται χωρίς όνομα.
  • Εάν τα εισηγμένα δεδομένα στο Address Book (Βιβλίο διευθύνσεων) ή στις Contacts (Επαφές) περιλαμβάνουν χαρακτήρες που δεν υποστηρίζονται από το Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων), τα δεδομένα εισάγονται χωρίς όνομα.
    Για πληροφορίες σχετικά με τους υποστηριζόμενους χαρακτήρες δείτε την ακόλουθη ενότητα.
    Αποδεκτοί χαρακτήρες
  • Εάν το όνομα των εισηγμένων δεδομένων στο Address Book (Βιβλίο διευθύνσεων) ή στις Contacts (Επαφές) έχει 31 χαρακτήρες ή περισσότερους, οι πρώτοι 30 χαρακτήρες εισάγονται και οι υπόλοιποι παραλείπονται.
