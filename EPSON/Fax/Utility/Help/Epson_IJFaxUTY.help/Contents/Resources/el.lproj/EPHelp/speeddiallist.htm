Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων)

Μενού
Γραμμή εργαλείων
Το Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων) σας επιτρέπει να καταχωρήσετε ή επεξεργαστείτε την ταχεία κλήση και την ομαδική κλήση του φαξ για τον εκτυπωτή σας.
Σημείωση:
  • Το παράθυρο αυτό προβάλλεται όταν κάνετε κλικ στο Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων) στο FAX Utility (κεντρικό παράθυρο).
  • Ο μέγιστος αριθμός καταχωρήσεων που είναι διαθέσιμες για το συνδεδεμένο εκτυπωτή (φαξ) προβάλλεται στη γραμμή στο κάτω μέρος του παραθύρου.

Μενού

Μενού File (Αρχείο)

Στοιχείο
Επεξήγηση
Get Speed Dials (Λήψη ταχέων κλήσεων)
Διαβάζει την ταχεία κλήση ή την ομαδική κλήση του φαξ για τον εκτυπωτή σας.
Send Speed Dials (Αποστολή ταχέων κλήσεων)
Αποθηκεύει τις ταχείες κλήσεις και τις ομαδικές κλήσεις που προβάλλονται στον κατάλογο στον εκτυπωτή.
Import (Εισαγωγή)
From Address Book (Από βιβλίο διευθύνσεων)
Εισαγάγει τα δεδομένα του Address Book (Βιβλίο διευθύνσεων) στις ταχείες κλήσεις του εκτυπωτή σας.
Back up (Αντίγραφο ασφαλείας)
Save Backup (Αποθήκευση αντιγράφου ασφαλείας)
Αποθηκεύει το αντίγραφο ασφαλείας των διαμορφωμένων ταχέων κλήσεων ή ομαδικών κλήσεων.
Restore Backup (Ανάκτηση αντιγράφου ασφαλείας)
Διαβάζει το αντίγραφο ασφαλείας για ταχείες κλήσεις ή ομαδικές κλήσεις.
Close (Κλείσιμο)
Κλείνει το παράθυρο.

Στοιχείο
Επεξήγηση
Edit (Επεξεργασία)
Επεξεργάζεται την ταχεία κλήση ή ομαδική κλήση που έχει επιλεγεί στον κατάλογο.
Delete (Διαγραφή)
Διαγράφει την ταχεία κλήση ή ομαδική κλήση που έχει επιλεγεί στον κατάλογο.
Delete All (Διαγραφή όλων)
Διαγράφει όλες τις ταχείες/ομαδικές κλήσεις στον κατάλογο.
