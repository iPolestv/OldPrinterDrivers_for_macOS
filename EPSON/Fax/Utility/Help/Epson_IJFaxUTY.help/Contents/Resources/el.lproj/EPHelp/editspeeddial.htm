Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων) - Επεξεργασία ταχείας κλήσης/ομαδικής κλήσης-

Το παράθυρο αυτό σας επιτρέπει να καταχωρήσετε ή επεξεργαστείτε μία καταχώρηση ταχείας κλήσης ή ομαδικής κλήσης του φαξ για τον εκτυπωτή σας.
Στοιχείο
Επεξήγηση
Entry Number (Αριθμός καταχώρισης)
Προβάλλει τον αριθμό καταχώρησης της ταχείας κλήσης.
Speed Dial (Ταχεία κλήση)
Επιλέξτε το για να δημιουργήσετε ή επεξεργαστείτε μία καταχώρηση ταχείας κλήσης.
Group Dial (Ομαδική κλήση)
Επιλέξτε το για να δημιουργήσετε ή επεξεργαστείτε μία καταχώρηση ομαδικής κλήσης. Δεν μπορείτε να δημιουργήσετε ομαδική κλήση, εκτός και έχετε δημιουργήσει καταχωρήσεις ταχείας κλήσης.
Name/Group Name (Όνομα/Όνομα ομάδας)
Εισαγάγετε όνομα ταχείας κλήσης ή ομαδικής κλήσης που επιθυμείτε να καταχωρήσετε.
Index Word (Λέξη ευρετηρίου) *3
Εισαγάγετε μια λέξη ευρετηρίου για τον παραλήπτη, ώστε να μπορείτε να τη χρησιμοποιήσετε όταν πραγματοποιείτε αναζητήσεις.
Fax Number (Αριθμός φαξ) *1
Εισαγάγετε τον αριθμό φαξ παραλήπτη. Εάν επιθυμείτε να διαχωρίσετε αριθμούς, όπως διαχωρισμό κωδικών πόλεων από προσωπικούς αριθμούς, χρησιμοποιήστε διάστημα αντί για παύλα (-), καθώς μία παύλα (-) ενεργεί ως παύση κατά την κλήση. Το στοιχείο αυτό προβάλλεται όταν επεξεργάζεστε την ταχεία κλήση.
Member (Μέλος) *2
Προβάλλεται κατάλογος καταχωρημένων ταχέων κλήσεων. Επιλέξτε την ταχεία κλήση που επιθυμείτε να προσθέσετε στην ομαδική κλήση και κατόπιν κάντε κλικ στο OK. Το στοιχείο αυτό προβάλλεται όταν επεξεργάζεστε την ομαδική κλήση.
Fax Speed (Ταχύτητα φαξ) *3
Επιλέξτε την ταχύτητα φαξ που χρησιμοποιείται για αποστολή φαξ στον παραλήπτη. Η ταχύτητα φαξ που ορίζεται για κάθε καταχώρηση ταχείας κλήσης χρησιμοποιείται μόνο κατά την αποστολή φαξ απευθείας από εκτυπωτή. FAX Utility αγνοεί τη ρύθμιση και χρησιμοποιεί την ταχύτητα φαξ που έχει οριστεί στον εκτυπωτή.
OK
Αποθηκεύει τις ρυθμίσεις και κλείνει το παράθυρο.
Cancel (Άκυρο)
Ακυρώνει την καταχώρηση ή επεξεργασία και κλείνει το παράθυρο.
*1 Απαιτούμενο στοιχείο για καταχώρηση της ταχείας κλήσης.
*2 Κατά την καταχώρηση ομαδικής κλήσης, βεβαιωθείτε ότι υπάρχει πάντα μία ή περισσότερες ταχείες κλήσεις επιλεγμένες.
*3 Τα στοιχεία αυτά ενδέχεται να μην προβάλλονται, ανάλογα με τον εκτυπωτή.
Σημείωση:
  • Για πληροφορίες σχετικά με τους χαρακτήρες και τον αριθμό χαρακτήρων που μπορούν να εισαχθούν, δείτε την ακόλουθη ενότητα.
    Αποδεκτοί χαρακτήρες
  • Ο μέγιστος αριθμός καταχωρήσεων ταχείας κλήσης ανά ομάδα προβάλλεται στη γραμμή στο κάτω μέρος του παραθύρου επεξεργασίας ομαδικών κλήσεων.
