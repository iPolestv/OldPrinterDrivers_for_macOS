> Κεντρική σελίδα > Μετάδοση φαξ

Μετάδοση εγγράφου που δημιουργήθηκε χρησιμοποιώντας εφαρμογή επεξεργασίας κειμένου ή λογιστικό φύλλο.

Για Mac OS X 10.5.x ή νεότερη έκδοση
Για Mac OS X 10.4.x
Επιλέγοντας εκτυπωτή φαξ από το μενού Print (Εκτύπωση) κάποιων εφαρμογών, μπορείτε να μεταδίδετε δεδομένα απευθείας όπως έγγραφα, σχέδια και πίνακες που έχετε δημιουργήσει.
Σημείωση:
Η ακόλουθη επεξήγηση χρησιμοποιεί TextEdit, την τυπική εφαρμογή του Mac OS X ως παράδειγμα. Η πραγματική λειτουργία ενδέχεται να διαφέρει, ανάλογα με την εφαρμογή που χρησιμοποιείτε. Για λεπτομέρειες, δείτε τη βοήθεια της εφαρμογής.

Για Mac OS X 10.5.x ή νεότερη έκδοση

Χρησιμοποιώντας μία εφαρμογή, δημιουργήστε έγγραφο για μετάδοση μέσω φαξ.
Κάντε κλικ στο Print (Εκτύπωση) από το μενού File (Αρχείο) της εφαρμογής σας.
Προβάλλεται ο διάλογος Print (Εκτύπωση) της εφαρμογής σας.
Επιλέξτε ΦΑΞ XXXXXX (ο εκτυπωτής σας) από Printer (Εκτυπωτής) και κατόπιν κάντε κλικ στο για προβολή λεπτομερών ρυθμίσεων.
Προβείτε στις κατάλληλες ρυθμίσεις.
    • Ορίστε 1 στην επιλογή Copies (Αντίγραφα). Ακόμα και αν ορίσετε 2 ή περισσότερα, μεταδίδεται μόνο ένα αντίγραφο.
    • Ανάλογα με τον εκτυπωτή (φαξ), η αποστολή έγχρωμου φαξ ενδέχεται να μην είναι διαθέσιμη για μεγέθη χαρτιού μεγαλύτερα από A4, Letter ή Legal.
    • Μπορείτε να μεταδώσετε έως και 100 σελίδες σε μία μετάδοση φαξ.
Επιλέξτε Fax Settings (Ρυθμίσεις φαξ) και προβείτε στις κατάλληλες ρυθμίσεις.
Για επεξήγηση κάθε μίας ρύθμισης, δείτε τον οδηγό βοήθειας του προγράμματος οδήγησης PC-FAX. Μπορείτε να προβάλλετε τον οδηγό βοήθειας κάνοντας κλικ στο ? κάτω αριστερά της οθόνης αφού επιλέξετε Fax Settings (Ρυθμίσεις φαξ).
Επιλέξτε Recipient Settings (Ρυθμίσεις παραλήπτη) και καθορίστε έναν παραλήπτη.
Καθορίστε έναν παραλήπτη (όνομα, αριθμό φαξ και τα λοιπά) απευθείας:
Εισαγάγετε τις απαραίτητες πληροφορίες στην ενότητα Add (Προσθήκη) και κατόπιν κάντε κλικ στο +. Ο παραλήπτης προστίθεται στο Recipient List (Λίστα παραληπτών) που προβάλλεται στο άνω τμήμα του διαλόγου.
Σημείωση:
Εάν έχετε επιλέξει Enter fax number twice (Εισαγωγή αριθμού φαξ δύο φορές) στις ρυθμίσεις προγράμματος οδήγησης PC-FAX, πρέπει να εισαγάγετε ξανά τον ίδιο αριθμό.
Εάν η συνδεδεμένη γραμμή φαξ απαιτεί πρόθεμα εξωτερικής πρόσβασης, εισαγάγετε το πρόθεμα στο πλαίσιο External Access Prefix (Πρόθεμα εξωτερικής πρόσβασης).
Σημείωση:
Εάν ο εκτυπωτής σας διαθέτει ρύθμιση γραμμής τύπου PBX και έχει ρυθμιστεί για χρήση του # (δίεση) αντί του ακριβούς κωδικού προθέματος, εισαγάγετε # (δίεση).
Επιλογή παραλήπτη (όνομα, αριθμός φαξ και τα λοιπά) από το Address book (Βιβλίο διευθύνσεων) ή από Contacts (Επαφές):
Εάν ο παραλήπτης έχει αποθηκευτεί στο βιβλίο διευθύνσεων, κάντε κλικ στο . Επιλέξτε παραλήπτη από τον κατάλογο, κάντε κλικ στο Add (Προσθήκη) και έπειτα στο OK.
Εάν η συνδεδεμένη γραμμή φαξ απαιτεί πρόθεμα εξωτερικής πρόσβασης, εισαγάγετε το πρόθεμα στο πλαίσιο External Access Prefix (Πρόθεμα εξωτερικής πρόσβασης).
Σημείωση:
Εάν ο εκτυπωτής σας διαθέτει ρύθμιση γραμμής τύπου PBX και έχει ρυθμιστεί για χρήση του # (δίεση) αντί του ακριβούς κωδικού προθέματος, εισαγάγετε # (δίεση).
Ελέγξτε τις ρυθμίσεις παραλήπτη και κάντε κλικ στο Fax (Φαξ).
Κάντε κλικ στο Fax (Φαξ) για να ξεκινήσει η μετάδοση. Βεβαιωθείτε ότι το όνομα και ο αριθμός φαξ του παραλήπτη είναι σωστά πριν τη μετάδοση.
Σημείωση:
  • Κάντε κλικ στο εικονίδιο του εκτυπωτή στο Dock (Σταθμό υποδοχής) για να εμφανιστεί η κατάσταση μετάδοσης. Για διακοπή της μετάδοσης, επιλέξτε τα δεδομένα που στείλατε και κατόπιν κάντε κλικ στο Delete (Διαγραφή).
  • Αν προκύψει σφάλμα κατά τη διάρκεια της μετάδοσης, εμφανίζεται μήνυμα σφάλματος. Ελέγξτε το αρχείο μετάδοσης από το παράθυρο Fax Transmission Record (Αρχείο μεταδόσεων φαξ).
    Έλεγχος / διαγραφή αρχείων μετάδοσης
    Λίστα σφαλμάτων μετάδοσης
  • Έγγραφα τα οποία περιέχουν αρκετά διαφορετικά μεγέθη χαρτιού ενδέχεται να μην μεταδοθούν σωστά.
