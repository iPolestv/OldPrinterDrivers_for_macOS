Εγγραφή ή επεξεργασία ταχείας κλήσης και ομαδικής κλήσης

Μπορείτε να καταχωρήσετε ή επεξεργαστείτε την ταχεία κλήση ή την ομαδική κλήση του φαξ για τον εκτυπωτή σας.
Μπορείτε να εισαγάγετε εύκολα τον αριθμό φαξ όταν στέλνετε φαξ από εκτυπωτή, αφού καταχωρήσετε την ταχεία κλήση. Μπορείτε να ορίσετε πολλαπλούς παραλήπτες σε μία λειτουργία αφού καταχωρήσετε την ομαδική κλήση.
Σημείωση:
Δεν μπορείτε να διαμορφώσετε την ομαδική κλήση προτού καταχωρήσετε τις ταχείες κλήσεις.
Εκκινήστε το FAX Utility.
Εκκίνηση FAX Utility
Επιλέξτε ΦΑΞ XXXXXX (ο εκτυπωτής σας) από Printer (Εκτυπωτής) και κατόπιν κάντε κλικ στο Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων).
Προβάλλονται οι λίστες ταχείας κλήσης/ομαδικής κλήσης που είναι αποθηκευμένες στον εκτυπωτή σας.
Σημείωση:
Εάν ο εκτυπωτής σας διαθέτει λειτουργία ασφαλείας η οποία επιτρέπει σε διαχειριστές να περιορίζουν χρήστες να αλλάζουν τις ρυθμίσεις φαξ του εκτυπωτή, ίσως χρειαστεί να εισαγάγετε κωδικό πρόσβασης διαχειριστή για να συνεχίσετε.
Για να δημιουργήσετε νέα καταχώρηση, επιλέξτε μια κενή γραμμή. Για να επεξεργαστείτε υπάρχουσα καταχώρηση, επιλέξτε την καταχώρηση που επιθυμείτε να επεξεργαστείτε.
Κατόπιν, κάντε διπλό κλικ στη γραμμή ή πατήστε το πλήκτρο Enter για να ανοίξετε το διάλογο επεξεργασίας.
Σημείωση:
  • Για να διαγράψετε μια υπάρχουσα καταχώρηση, επιλέξτε την καταχώρηση που επιθυμείτε να διαγράψετε και κατόπιν πατήστε το πλήκτρο διαγραφής, ή κάντε κλικ στο μενού Edit (Επεξεργασία) και έπειτα κάντε κλικ στο Delete (Διαγραφή). Ωστόσο, η διαγραφή μίας ομαδικής κλήσης δεν διαγράφει τις ταχείες κλήσεις που αποτελούσαν την ομάδα.
  • Δεν μπορείτε να επεξεργαστείτε ή διαγράψετε μία προστατευόμενη καταχώρηση ταχείας κλήσης ή καταχώρηση ομαδικών κλήσεων.
Διαμορφώστε την ταχεία κλήση ή την ομαδική κλήση.
Διαμόρφωση ταχείας κλήσης:
Επιλέξτε Speed Dial (Ταχεία κλήση), εισαγάγετε Name/Group Name (Όνομα/Όνομα ομάδας), Index Word (Λέξη ευρετηρίου), Fax Number (Αριθμός φαξ) και Fax Speed (Ταχύτητα φαξ) και κατόπιν κάντε κλικ στο OK. Η διαμορφωμένη ταχεία κλήση προστίθεται στον Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων).
Σημείωση:
  • Σε περίπτωση που το τηλεφωνικό σύστημά σας απαιτεί πρόθεμα εξωτερικής πρόσβασης όπως 0 ή 9 για να έχετε εξωτερική γραμμή, εισαγάγετε το πρόθεμα στην αρχή του αριθμού φαξ. Εάν ο εκτυπωτής σας έχει ρυθμιστεί για χρήση του # (δίεση) αντί ακριβούς κωδικού προθέματος, εισαγάγετε # (δίεση).
  • Η ταχύτητα φαξ που ορίζεται για κάθε καταχώρηση ταχείας κλήσης χρησιμοποιείται μόνο κατά την αποστολή φαξ απευθείας από εκτυπωτή. FAX Utility αγνοεί τη ρύθμιση και χρησιμοποιεί την ταχύτητα φαξ που έχει οριστεί στον εκτυπωτή.
Διαμόρφωση ομαδικής κλήσης:
Επιλέξτε Group Dial (Ομαδική κλήση), εισαγάγετε Name/Group Name (Όνομα/Όνομα ομάδας) και Index Word (Λέξη ευρετηρίου), επιλέξτε το πλαίσιο επιλογής της ταχείας κλήσης που επιθυμείτε να προσθέσετε στην ομαδική κλήση και κατόπιν κάντε κλικ στο OK. Η διαμορφωμένη ομαδική κλήση προστίθεται στο Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων).
Σημείωση:
Τα Index Word (Λέξη ευρετηρίου) και Fax Speed (Ταχύτητα φαξ) ενδέχεται να μην είναι διαθέσιμα, ανάλογα με τον εκτυπωτή.
Αφού ολοκληρώσετε όλες τις ρυθμίσεις, κάντε κλικ στο .
Οι ταχείες κλήσεις και οι ομαδικές κλήσεις που προβάλλονται στον κατάλογο αποθηκεύονται στον εκτυπωτή.
