Fax Receive Monitor (Παρακολούθηση λήψης φαξ) (Μόνο για μοντέλα με λήψη PC-FAX)

Γραμμή εργαλείων
Εικονίδιο και μήνυμα στην Λίστα εκτυπωτών
Το παράθυρο αυτό σας επιτρέπει να ελέγχετε την κατάσταση λήψης φαξ και να πραγματοποιείτε ρυθμίσεις αποστολής για ληφθέντα φαξ.
Σημείωση:
Το παράθυρο αυτό προβάλλεται όταν κάνετε κλικ στο Fax Receive Monitor (Παρακολούθηση λήψης φαξ) στο FAX Utility (κεντρικό παράθυρο).

Γραμμή εργαλείων

Στοιχείο
Επεξήγηση
Open the received fax folder (Άνοιγμα του φακέλου εισερχόμενων φαξ)
Ανοίγει το φάκελο στον οποίο έχουν αποθηκευτεί τα ληφθέντα φαξ.
Check new faxes (Έλεγχος για νέα φαξ)
Ελέγχει για νέα φαξ.
Received Fax Output Settings (Ρυθμίσεις εξόδου εισερχόμενων φαξ)
Ορίζει εάν ή όχι θα αποθηκεύσετε ληφθέντα φαξ σε υπολογιστή.
Preferences (Προτιμήσεις)
Διαμορφώνει τις προτιμήσεις της Προτιμήσεις παρακολούθησης λήψης φαξ.
Επιστροφή στην αρχή 

Εικονίδιο και μήνυμα στην Λίστα εκτυπωτών

Εικονίδιο
Μήνυμα
Επεξήγηση
Waiting (Αναμονή) / Checking for new faxes (Εκτελείται έλεγχος για νέα φαξ)
Έλεγχος για νέα φαξ ή λειτουργία αναμονής.
Received faxes saved successfully in the specified folder (Τα εισερχόμενα φαξ αποθηκεύτηκαν επιτυχώς στον καθορισμένο φάκελο).
Νέα φαξ που λήφθηκαν από εκτυπωτή (φαξ) έχουν αποθηκευτεί στον καθορισμένο φάκελο.
The printer is not responding (Ο εκτυπωτής δεν ανταποκρίνεται)
Ελέγχει τη σύνδεση μεταξύ υπολογιστή και εκτυπωτή, ή ότι ο εκτυπωτής είναι ενεργοποιημένος.
The printer is processing data (Ο εκτυπωτής εκτελεί επεξεργασία δεδομένων)
Ο εκτυπωτής επεξεργάζεται άλλα δεδομένα. Περιμένετε.
Cannot find the received fax folder (Δεν είναι δυνατός ο εντοπισμός του φακέλου εισερχόμενων φαξ).
Ελέγχει ότι το όνομα του καθορισμένου φακέλου είναι σωστό. Εάν χρησιμοποιείτε κοινόχρηστο φάκελο στο δίκτυο, βεβαιωθείτε ότι έχετε δικαιώματα πρόσβασης στο φάκελο.
Failed to save faxes in the specified folder (Η αποθήκευση φαξ στον καθορισμένο φάκελο απέτυχε).
Ελέγχει ότι ο καθορισμένος φάκελος είναι διαθέσιμος ή ότι έχετε δικαιώματα πρόσβασης στο φάκελο.
A printer communication error occurred (Σημειώθηκε σφάλμα επικοινωνίας με τον εκτυπωτή)
Ελέγχει τη σύνδεση μεταξύ υπολογιστή και εκτυπωτή, ή ότι ο εκτυπωτής είναι ενεργοποιημένος. Εάν χρησιμοποιείτε σύνδεση δικτύου, βεβαιωθείτε ότι μπορείτε να έχετε πρόσβαση στον εκτυπωτή μέσω του δικτύου.
The "FAXRECV" disk is being used by another application. (Ο δίσκος "FAXRECV" χρησιμοποιείται από άλλη εφαρμογή.)
Κλείνει το πρόγραμμα που χρησιμοποιεί το δίσκο "FAXRECV".
Cannot save faxes. User authentication failed. (Δεν είναι δυνατή η αποθήκευση φαξ. Η πιστοποίηση ταυτότητας χρήστη απέτυχε.)
Ελέγξτε την πιστοποίηση ταυτότητας χρήστη λήψης PC-FAX (όνομα και κωδικός πρόσβασης για κοινή χρήση) που έχει οριστεί στον εκτυπωτή, πραγματοποιήστε ξανά τις Received Fax Output Settings (Ρυθμίσεις εξόδου εισερχόμενων φαξ) και κατόπιν εισαγάγετε σωστά τις πληροφορίες πιστοποίησης ταυτότητας χρήστη. Να θυμάστε να επιλέξετε το Remember this password in my keychain (Απομνημόνευση αυτού του κωδικού πρόσβασης στην αλυσίδα κλειδιών μου).
