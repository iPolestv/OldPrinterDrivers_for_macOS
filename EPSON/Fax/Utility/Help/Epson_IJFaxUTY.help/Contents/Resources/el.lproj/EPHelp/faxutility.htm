FAX Utility (Main (κύριο) παράθυρο)

Αυτό είναι το παράθυρο όπου επιλέγετε τη λειτουργία FAX Utility προς χρήση.
Σημείωση:
Το παράθυρο αυτό εμφανίζεται όταν ξεκινάει το FAX Utility.
Εκκίνηση FAX Utility
Στοιχείο
Επεξήγηση
Fax Transmission Record (Αρχείο μεταδόσεων φαξ)
Προβάλλει τα αρχεία μετάδοσης.
Fax Transmission Record (Αρχείο μεταδόσεων φαξ)
Fax Receive Monitor (Παρακολούθηση λήψης φαξ)
Προβάλλεται μόνο όταν έχει επιλεγεί εκτυπωτής (φαξ) που υποστηρίζει λήψη PC-FAX. Σας επιτρέπει να επιλέξετε εάν ή όχι θα αποθηκεύσετε ληφθέντα φαξ σε υπολογιστή.
Check Fax Connection (Έλεγχος σύνδεσης φαξ)
Ελέγχει εάν η λειτουργία φαξ για το συνδεδεμένο εκτυπωτή λειτουργεί κανονικά.
Fax Settings (Ρυθμίσεις φαξ)
Ορίζει την κεφαλίδα φαξ και τις διάφορες ρυθμίσεις επικοινωνίας με φαξ που είναι αποθηκευμένες στον εκτυπωτή.
Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων)
Αποθηκεύει ή επεξεργάζεται την ταχεία κλήση ή ομαδική κλήση φαξ που είναι αποθηκευμένη στον εκτυπωτή.
Συγχρονισμός Address Book (Βιβλίο διευθύνσεων) ή Συγχρονισμός Contacts (Επαφές)
Συγχρονίζει δεδομένα για το Address Book (Βιβλίο διευθύνσεων) ή τις Contacts (Επαφές) με το FAX Utility ώστε να μπορείτε να επιλέξετε παραλήπτες από τον κατάλογο επαφών κατά την αποστολή φαξ χρησιμοποιώντας το FAX Utility. Εάν ενημερώσετε το Address Book (Βιβλίο διευθύνσεων) ή τις Contacts (Επαφές) έπειτα από συγχρονισμό, θα πρέπει να εκτελέσετε ξανά τη συγκεκριμένη λειτουργία συγχρονισμού για να χρησιμοποιήσετε τα ενημερωμένα δεδομένα με το FAX Utility.
