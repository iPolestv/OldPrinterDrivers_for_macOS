Σφάλματα προβλημάτων λήψης (Μόνο για μοντέλα με λήψη PC-FAX)

Ακολουθεί κατάλογος των μηνυμάτων σφάλματος που ενδέχεται να προβληθούν κατά τη λήψη φαξ και τον τρόπο διαχείρισης του προβλήματος.
Μήνυμα
Τι πρέπει να κάνετε
A printer communication error occurred. (Σημειώθηκε σφάλμα επικοινωνίας με τον εκτυπωτή.)
Απενεργοποιήσετε τον εκτυπωτή και ενεργοποιήστε τον ξανά. Περιμένετε λίγο και κατόπιν προσπαθήστε ξανά.
The printer (fax driver) you are setting has been deleted. Add the printer (fax driver) in [System Preferences] - [Print & Fax]. (Ο εκτυπωτής (πρόγραμμα οδήγησης φαξ) που ρυθμίζετε έχει διαγραφεί. Προσθέστε τον εκτυπωτή (πρόγραμμα οδήγησης φαξ) στη ρύθμιση [Προτιμήσεις συστήματος] - [Εκτύπωση και φαξ].)
Προσθέστε το πρόγραμμα οδήγησης PC-FAX στη λίστα εκτυπωτών. Κάντε κλικ στο OK στο διάλογο μηνυμάτων. Προβάλλεται ο διάλογος για προσθήκη του εκτυπωτή.
Change not accepted. Some printer settings have been changed by another computer, print queue, or the printer. Please try again. (Η αλλαγή δεν είναι αποδεκτή. Κάποιες ρυθμίσεις εκτυπωτή έχουν αλλάξει από άλλον υπολογιστή, άλλη ουρά αναμονής ή από τον εκτυπωτή. Δοκιμάστε ξανά.)
Οι Received Fax Output Settings (Ρυθμίσεις εξόδου εισερχόμενων φαξ) έχουν αλλάξει από άλλο υπολογιστή στο δίκτυο, από ουρά εκτύπωσης ή από τον εκτυπωτή. Περιμένετε λίγο και κατόπιν προσπαθήστε ξανά.
Τα ληφθέντα φαξ μπορούν να αποθηκευτούν μόνο σε έναν υπολογιστή.
Received faxes not saved due to an error. (Τα εισερχόμενα φαξ δεν αποθηκεύτηκαν λόγω σφάλματος.)
Η αποθήκευση ληφθέντος φαξ στον καθορισμένο φάκελο απέτυχε. Ελέγξτε τη σύνδεση στον καθορισμένο φάκελο για να βεβαιωθείτε ότι είναι διαθέσιμη και ελέγξτε ότι έχετε δικαιώματα πρόσβασης στο φάκελο.
Processing canceled because an error occurred. (Η επεξεργασία ακυρώθηκε επειδή σημειώθηκε σφάλμα.)
Εάν η επανεκκίνηση του εκτυπωτή και του υπολογιστή σας δεν επιλύουν το πρόβλημα, επικοινωνήστε με την υποστήριξη Epson. Για πληροφορίες επικοινωνίας, ανατρέξτε στο εγχειρίδιο του εκτυπωτή.
Cannot save faxes. User authentication failed. (Δεν είναι δυνατή η αποθήκευση φαξ. Η πιστοποίηση ταυτότητας χρήστη απέτυχε.)
Αδυναμία αποθήκευσης ληφθέντων φαξ επειδή η πιστοποίηση της ταυτότητας χρήστη λήψης PC-FAX (όνομα και κωδικός πρόσβασης για κοινή χρήση αρχείων) δεν ορίζεται ορθά στην αλυσίδα κλειδιών του υπολογιστή.
Πραγματοποιήστε ξανά τις Fax Output Setting (Ρυθμίσεις εξόδου εισερχόμενων φαξ.) και κατόπιν εισαγάγετε τις ορθές πληροφορίες επαλήθευσης χρήστη όταν προβάλλεται η οθόνη καταχώρησης. Να θυμάστε να επιλέξετε το Remember this password in my keychain (Απομνημόνευση αυτού του κωδικού πρόσβασης στην αλυσίδα κλειδιών μου).
