Ανάκτηση αντιγράφου ασφαλείας λίστας ταχέων κλήσεων

Μπορείτε να ανακτήσετε αντίγραφα ασφαλείας δεδομένων τηλεφωνικού καταλόγου εκτυπωτή (λίστες ταχέων/ομαδικών κλήσεων) και κατόπιν να τα ανακτήσετε στον εκτυπωτή.
Σημείωση:
  • Δεν μπορείτε να ανακτήσετε δεδομένα για τα οποία έχουν δημιουργηθεί αντίγραφα ασφαλείας στο Windows Fax Utility.
  • Εάν το αντίγραφο ασφαλείας της λίστας ταχείας κλήσης/ομαδικής κλήσης περιέχει στοιχεία τα οποία δεν υποστηρίζονται από τον εκτυπωτή στον οποίο επιθυμείτε να ανακτήσετε τον κατάλογο, τα στοιχεία αυτά δεν ανακτώνται.
  • Εάν το αντίγραφο ασφαλείας της λίστας ταχείας κλήσης/ομαδικής κλήσης δεν περιέχει στοιχεία τα οποία υποστηρίζονται από τον εκτυπωτή στον οποίο επιθυμείτε να ανακτήσετε τα δεδομένα, τα στοιχεία αυτά ανακτώνται όπως φαίνεται πιο κάτω.
    Index Word (Λέξη ευρετηρίου): κενό
    Fax Speed (Ταχύτητα φαξ): προκαθορισμένη τιμή
  • Τυχόν στοιχεία, όπως Name/Group Name (Όνομα/Όνομα ομάδας), που περιλαμβάνουν χαρακτήρες οι οποίοι δεν υποστηρίζονται από τον εκτυπωτή στον οποίο επιθυμείτε να ανακτήσετε τα δεδομένα, είναι κενά όταν ανακτώνται.
  • Τυχόν καταχωρήσεις που υποδηλώνονται ως In use by another function (Χρησιμοποιείται από άλλη λειτουργία) στο FAX Utility αντικαθίστανται από τα ανακτηθέντα δεδομένα, δεδομένου ότι ταιριάζουν οι αριθμοί καταχώρησης.
  • Τυχόν καταχωρήσεις που χρησιμοποιούνται και προστατεύονται δεν αντικαθίστανται με τα ανακτηθέντα δεδομένα.
Εκκινήστε το FAX Utility.
Εκκίνηση FAX Utility
Επιλέξτε FAX XXXXXX (ο εκτυπωτής σας) από Printer (Εκτυπωτής) και κατόπιν κάντε κλικ στο Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων).
Σημείωση:
Εάν ο εκτυπωτής σας διαθέτει λειτουργία ασφαλείας η οποία επιτρέπει σε διαχειριστές να περιορίζουν χρήστες να αλλάζουν τις ρυθμίσεις φαξ του εκτυπωτή, ίσως χρειαστεί να εισαγάγετε κωδικό πρόσβασης διαχειριστή για να συνεχίσετε.
Κάντε κλικ στα File (Αρχείο) - Back up (Αντίγραφο ασφαλείας) - Restore Backup (Ανάκτηση αντιγράφου ασφαλείας).
Επιλέξτε το αρχείο το οποίο θέλετε να ανακτήσετε.
Η λίστα που έχει ανακτηθεί προβάλλεται στο παράθυρο Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων).
Κάντε κλικ στο στο παράθυρο.
Ο τηλεφωνικός κατάλογος που έχει ανακτηθεί αποθηκεύεται στον εκτυπωτή.
