Αποθήκευση αντιγράφου ασφαλείας ταχέων κλήσεων

Μπορείτε να αποθηκεύσετε αντίγραφο ασφαλείας των διαμορφωμένων ταχέων κλήσεων ή ομαδικών κλήσεων. Τα δεδομένα της λίστας ταχέων/ομαδικών κλήσεων ενδέχεται να χαθούν λόγω δυσλειτουργίας του εκτυπωτή. Προτείνουμε να δημιουργείτε αντίγραφο ασφαλείας των δεδομένων κάθε φορά που ενημερώνετε τη λίστα κλήσεων. Η Epson δεν φέρει ευθύνη για την απώλεια δεδομένων, για τη δημιουργία αντιγράφων ασφαλείας ή την ανάκτηση δεδομένων και/ή ρυθμίσεων ακόμα και κατά την περίοδο εγγύησης.
Σημείωση:
Δεδομένα με την ένδειξη "In use by another function (Χρησιμοποιείται από άλλη λειτουργία)" δεν αποθηκεύονται.
Εκκινήστε το FAX Utility.
Εκκίνηση FAX Utility
Επιλέξτε ΦΑΞ XXXXXX (ο εκτυπωτής σας) από Printer (Εκτυπωτής) και κατόπιν κάντε κλικ στο Speed Dial/Group Dial List (Λίστα ταχέων κλήσεων/ομαδικών κλήσεων).
Προβάλλονται οι λίστες ταχείας κλήσης/ομαδικής κλήσης που είναι αποθηκευμένες στον εκτυπωτή σας.
Σημείωση:
Εάν ο εκτυπωτής σας διαθέτει λειτουργία ασφαλείας η οποία επιτρέπει σε διαχειριστές να περιορίζουν χρήστες να αλλάζουν τις ρυθμίσεις φαξ του εκτυπωτή, ίσως χρειαστεί να εισαγάγετε κωδικό πρόσβασης διαχειριστή για να συνεχίσετε.
Κάντε κλικ στα File (Αρχείο) - Back up (Αντίγραφο ασφαλείας) - Save Backup (Αποθήκευση αντιγράφου ασφαλείας).
Καθορίστε το όνομα αρχείου και τον κατάλογο αρχείου και κατόπιν κάντε κλικ στο Save (Αποθήκευση).
