Επισκόπηση προϊόντος

FAX Utility είναι μια εφαρμογή η οποία παρέχει διάφορες βολικές λειτουργίες για την αποστολή φαξ.
FAX Utility λειτουργεί μόνο με πολυλειτουργικούς εκτυπωτές Epson με δυνατότητα αποστολής φαξ.
Κατά τη χρήση εκτυπωτή ο οποίος υποστηρίζει λήψη PC-FAX, μπορείτε να αποθηκεύσετε φαξ που έχετε λάβει ως αρχεία PDF στον υπολογιστή σας. Αυτό βοηθάει στο να μειώσετε τις περιττές εκτυπώσεις.
Σημείωση:
Εάν ο εκτυπωτής σας υποστηρίζει λήψη PC-FAX, το Fax Receive Monitor (Παρακολούθηση λήψης φαξ) εμφανίζεται στο κεντρικό παράθυρο του FAX Utility.
Περιήγηση σε αρχεία μετάδοσης
Μπορείτε να περιηγηθείτε στα αρχεία μετάδοσης φαξ που διαθέτετε.
Έλεγχος / διαγραφή αρχείων μετάδοσης
Fax Receive Monitor (Παρακολούθηση λήψης φαξ) (Μόνο για μοντέλα με λήψη PC-FAX)
Το Fax Receive Monitor (Παρακολούθηση λήψης φαξ) εμφανίζεται όταν ένας εκτυπωτής με λήψη PC-FAX επιλέγεται στο κεντρικό παράθυρο του FAX Utility. Στο παράθυρο Fax Receive Monitor (Παρακολούθηση λήψης φαξ), μπορείτε να προβείτε σε ρυθμίσεις για να αποθηκεύσετε φαξ που έχετε λάβει στον υπολογιστή σας και να ορίσετε μήνυμα ειδοποίησης όταν λαμβάνονται νέα φαξ.
Ρύθμιση για αποθήκευση ενός ληφθέντος φαξ σε υπολογιστή
Έλεγχος νέων φαξ
Πραγματοποίηση ρυθμίσεων ώστε τα ληφθέντα φαξ να μην αποθηκεύονται σε υπολογιστή
Ρυθμίσεις ΦΑΞ για τη λειτουργία εκτυπωτή
  • Μπορείτε να ελέγξετε εάν ο συνδεδεμένος εκτυπωτής (φαξ) είναι έτοιμος για αποστολή φαξ. Το αποτέλεσμα της δοκιμής εκτυπώνεται ως αναφορά. Σε περίπτωση που εντοπιστεί πρόβλημα, λύστε το ελέγχοντας τις πληροφορίες στην αναφορά.
    Έλεγχος της κατάστασης λειτουργίας ΦΑΞ του εκτυπωτή σας
  • Μπορείτε να διαμορφώσετε ή να αλλάξετε τις ρυθμίσεις φαξ που είναι αποθηκευμένες στον εκτυπωτή, όπως τις ρυθμίσεις Auto Answer (Αυτόματη απάντηση) και Fax Header (Κεφαλίδα φαξ).
    Ρύθμιση της λειτουργίας φαξ του εκτυπωτή σας
  • Μπορείτε να δημιουργήσετε ή επεξεργαστείτε μία ταχεία κλήση και καταχωρήσεις ομαδικών κλήσεων που έχουν αποθηκευτεί στον εκτυπωτή.
