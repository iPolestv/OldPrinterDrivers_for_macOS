Εκκίνηση FAX Utility

Για Mac OS X 10.6.x ή νεότερη έκδοση
Κάντε κλικ στο Apple - System Preferences (Προτιμήσεις συστήματος) - Print & Scan (Εκτύπωση & σάρωση) ή Print & Fax (Εκτύπωση & φαξ).
Επιλέξτε ΦΑΞ XXXXXX (ο εκτυπωτής σας) από την επιλογή Printer (Εκτυπωτής) και κατόπιν κάντε κλικ στο Options & Supplies (Επιλογές & τροφοδοσία) - Utility (Βοηθητικό πρόγραμμα) - Open Printer Utility (Άνοιγμα βοηθητικού προγράμματος εκτυπωτή).
Για Mac OS X 10.5.x
Κάντε κλικ στο Apple - System Preferences (Προτιμήσεις συστήματος) - Print & Fax (Εκτύπωση & φαξ).
Επιλέξτε ΦΑΞ XXXXXX (ο εκτυπωτής σας) από τον Εκτυπωτή και κατόπιν κάντε κλικ στο Open Print Queue (Άνοιγμα εργασιών εκτύπωσης σε ουρά) - Utility (Βοηθητικό πρόγραμμα).
Για Mac OS X 10.4.x
Κάντε διπλό κλικ σε Σκληρός δίσκος - Applications (Εφαρμογές) - Utilities (Βοηθητικά προγράμματα) - Printer Setup Utility (Βοηθητικό πρόγραμμα ρύθμισης εκτυπωτή).
Επιλέξτε ΦΑΞ XXXXXX (ο εκτυπωτής σας) από τον Εκτυπωτή και κατόπιν κάντε κλικ στο Utility (Βοηθητικό πρόγραμμα).
