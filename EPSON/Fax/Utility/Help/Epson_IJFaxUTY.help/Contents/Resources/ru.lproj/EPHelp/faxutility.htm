FAX Utility (окно Main (Главное) (главное окно)

В этом окне можно выбрать одну из функций приложения FAX Utility.
Примечание:
Это окно отображается при запуске приложения FAX Utility.
Запуск FAX Utility
Пункт
Описание
Fax Transmission Record (Запись передачи факса)
Отображаются записи передач.
Fax Transmission Record (Запись передачи факса)
Fax Receive Monitor (Монитор получения факсов)
Отображается только при выборе принтера (факса) с поддержкой функции ПК-ФАКС-приема. Определяет необходимость сохранения факсимильных сообщений на компьютере.
Check Fax Connection (Проверьте соед. Факса)
Проверяет, правильно ли работает ли подключенный принтер с функцией факса.
Fax Settings (Настройки факса)
Можно выполнить настройки заголовка факсимильного сообщения, а также различные настройки соединения, сохраняемые в принтере.
Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров)
Можно сохранять или редактировать списки быстрого или группового набора, сохраненные на принтере.
Synchronize (Синхронизировать) Address Book или Synchronize (Синхронизировать) Contacts
Синхронизация данных для приложения Address Book или Contacts с FAX Utility для выбора получателей из списка контактов при отправке факсимильного сообщения с использованием FAX Utility. В случае обновления приложения Address Book или Contacts после синхронизации для использования обновленных данных с FAX Utility потребуется повторная синхронизация.
