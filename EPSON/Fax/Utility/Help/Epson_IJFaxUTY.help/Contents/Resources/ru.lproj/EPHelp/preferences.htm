Preferences (Настройки) (только для моделей с ПК-ФАКС-приемом)

В этом окне можно выполнить настройку приема факсимильных сообщений, например, уведомления о получении новых сообщений.
Примечание:
  • Чтобы открыть это окно, в Fax Receive Monitor (Монитор получения факсов) выберите Fax Receive Monitor Preferences (Настройки Монитор получения факсов).
  • Настройки применяются ко всем принтерам и очередям печати, отображаемым в окне Fax Receive Monitor (Монитор получения факсов).
Пункт
Описание
Specify an interval to check for new faxes (Укажите интервал для проверки новых факсов)
Укажите временной интервал от 1 до 30 минут для проверки новых факсимильных сообщений.
Notify me of new faxes via a dock icon (Уведомлять меня о новых факсах с помощью значка дока)
Выберите, чтобы значок Fax Receive Monitor (Монитор получения факсов) в Dock "подпрыгивал" при получении новых сообщений.
Notify me of errors via a dock icon (Уведомлять меня об ошибках с помощью значка дока)
Выберите, чтобы значок монитора в Fax Receive Monitor (Монитор получения факсов) в Dock "подпрыгивал" в случае ошибки. Просмотреть информацию об ошибке можно в окне Fax Receive Monitor (Монитор получения факсимильных сообщений) с помощью щелчка по этому значку.
Start at login (Запуск при входе)
Рекомендуется выбрать значение Automatic (Автоматический). Если в меню Received Fax Output Settings (Настройки вывода полученных факсов) выбрана опция "Save" (save faxes on this computer) ("Сохранить" (сохранить факсы на этом компьютере)), Fax Receive Monitor (Монитор получения факсов) запустится автоматически при входе в систему. Если выбрана опция On (Вкл.), Fax Receive Monitor (Монитор получения факсов) будет запускаться автоматически при входе в систему независимо от настроек вывода. При выборе опции Off (Выкл) Fax Receive Monitor (Монитор получения факсов) не будет запускаться при входе в систему.
Wait time before checking new faxes (Время ожидания перед проверкой новых факсов)
Этот пункт доступен, если для параметра Automatic (Автоматический) выбрана опция On (Вкл.) или Start at login (Запуск при входе). Укажите периодичность (от 0 до 10 минут) проверки новых факсимильных сообщений после запуска Fax Receive Monitor (Монитор получения факсов) при входе в систему. Эта функция полезна, когда выбранная для сохранения папка является сетевой, и необходимо время для установления соединения с ней после входа в систему.
Примечание:
Если выбранная для сохранения папка является сетевой, можно установить соединение с ней, зарегистрировав ее в списке. Выберите Apple - System Preferences - Account - Login to register.
