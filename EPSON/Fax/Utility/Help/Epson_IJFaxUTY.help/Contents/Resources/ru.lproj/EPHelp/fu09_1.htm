Настройки сохранения полученных факсимильных сообщений на компьютере

Настройки вывода полученных факсимильных сообщений можно изменить для сохранения полученных сообщений в формате PDF на компьютере или в сетевой папке на сервере (ПК-ФАКС-прием).
Примечание:
  • Если принтер поддерживает ПК-ФАКС-прием, в главном окне FAX Utility отобразится Fax Receive Monitor (Монитор получения факсов).
  • При использовании ПК-ФАКС-приема выбран автоответ.
  • В зависимости от принтера, полученные факсимильные данные, сохраненные на компьютере, могут удаляться из памяти принтера.
  • Так как полученные факсимильные сообщения сохраняются в файлах формата PDF, понадобятся приложения Cover Flow (10.5 или более поздняя версия), Quick Look (10.5 или более поздняя версия) или Preview.app, заранее установленные в Mac OS X.
Запустите FAX Utility.
Запуск FAX Utility
Выберите Fax Receive Monitor (Монитор получения факсов).
Примечание:
  • В многопользовательской среде невозможно запустить несколько экземпляров Fax Receive Monitor (Монитор получения факсов).
  • Импортировать и сохранять полученные факсимильные сообщения можно только во время работы Fax Receive Monitor (Монитор получения факсов).
  • Когда Fax Receive Monitor (Монитор получения факсов) запущен, в Dock появляется соответствующий значок. Добавление Fax Receive Monitor (Монитор получения факсов) в Dock облегчает запуск Fax Receive Monitor (Монитор получения факсов). Чтобы добавить Fax Receive Monitor (Монитор получения факсов) в Dock, удерживая клавишу Control, щелкните значок Fax Receive Monitor (Монитор получения факсов), а затем в меню нажмите Add to Dock.
Выберите принтер, а затем нажмите Receive Settings (Получить настройки).
Примечание:
  • Так как факсимильные сообщения можно сохранять только в одной очереди за один сеанс, если на одном принтере запущены две очереди факсимильных сообщений, например "FAX XXXXXX (USB)" для соединения USB и "FAX XXXXXX(IP)" для сетевого соединения, выберите одну из них для сохранения полученных факсимильных сообщений.
  • Если на принтере доступна функция безопасности, позволяющая администраторам ограничивать изменение настроек принтера с функцией факса для пользователей, для продолжения работы может потребоваться ввод пароля администратора.
Выберите "Save" (save faxes on this computer) ("Сохранить" (сохранить факсы на этом компьютере)), а затем нажмите Save to (Сохранить на), чтобы указать папку для сохранения файлов.
Примечание:
  • По умолчанию выбрана следующая папка.
    \User\Имя пользователя\PCFax\Имя принтера
  • Если принтер настроен как сетевой, настройки могут быть изменены с любого компьютера в этой сети. Однако сохранение факсимильных сообщений можно настроить только для одной учетной записи. При выборе опции “Save” (save faxes on this computer) ("Сохранить" (сохранить факсимильные сообщения на этом компьютере)), и выборе настроек Received Fax Output Settings (Настройки вывода полученных факсов) на другом компьютере в окне Received Fax Output Settings (Настройки вывода полученных факсов) отображается сообщение Save faxes on another computer or print queue (Сохранить факсы на другом компьютере или в очереди на печать).
  • В этом окне можно выбрать "Save" (save faxes on this computer) ("Сохранить" (сохранить факсы на этом компьютере)), если в принтере не осталось факсимильных сообщений. Если в принтере остаются факсимильные сообщения, появится сообщение об ошибке, информирующее о невозможности изменения этой настройки.
    Если принтер (факс) используется как сетевой, рекомендуется создать сетевую папку для хранения факсимильных сообщений.
  • Настройки Fax Receive Monitor (Монитор получения факсов) можно изменить путем выбора Preferences (Настройки) в Fax Receive Monitor (Монитор получения факсов).
    Preferences (Настройки) (только для моделей с ПК-ФАКС-приемом)
Выберите Send (Отправить), чтобы отправить настройки на принтер.
Note:
После настройки идентификации пользователя для получения PC-FAX (имя и пароль для общего доступа к файлам) на принтере необходимо зарегистрировать эти данные в связке ключей компьютера.
Если данные еще не зарегистрированы в связке ключей, отобразится начальный экран регистрации. Введите имя пользователя и пароль, выберите пункт Remember this password in my keychain (Запомнить этот пароль в связке ключей) и нажмите кнопку Connect (Подключить).
Если вы не сохраните информацию для идентификации в связке ключей, то при каждом получении факса и его сохранении на компьютере вам будет необходимо вводить эту информацию.
