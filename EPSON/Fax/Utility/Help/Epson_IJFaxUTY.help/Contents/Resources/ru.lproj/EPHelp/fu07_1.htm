Проблемы, связанные с передачей

В случае возникновения проблемы выполните соответствующие инструкции.
Проблема
Решение
FAX XXXXXX (ваш принтер) не отображается в списке Printer.
Убедитесь, что драйвер принтера и драйвер ПК-ФАКСА установлены и драйвер ПК-ФАКСА добавлен в список принтеров.
Кнопка Fax (Факс) затемнена (отключена).
Эта кнопка недоступна, пока в окне Recipient List (Список получателей) не добавлен ни один адресат. Укажите как минимум одного получателя.
Невозможно отправить факсимильные сообщения с принтера.
  • При выводе сообщения No dial tone detected (Нет сигнала) выберите Fax Settings (Настройки факса) - Dial Tone Detection Settings (Настр. распознавания сигнала) и снимите флажок Detect Dial Tone (Распознавание тонального сигнала).
  • Убедитесь, что в окне Header Information Settings (Настройки информации о заголовке) задан параметр Your Phone Number (Ваш телефонный номер). Факс получателя может быть настроен для блокировки тех факсимильных сообщений, в которых отсутствует информация об отправителе.
Во время передачи данных сообщение Waiting for printer response (Ожидание ответа принтера) не исчезает.
Сообщение Waiting for printer response (Ожидание ответа принтера) отображается, если соединение между компьютером и принтером (факсом) не установлено, и остается на экране до установления соединения. Проверьте следующее:
  • включено ли питание принтера;
  • правильно ли соединены между собой компьютер и принтер;
  • в случае использования сетевого принтера - правильно ли указан сетевой адрес принтера;
  • правильно ли работает принтер.
Если все указанные выше условия соблюдены, выключите и снова включите принтер.
После появления сообщения Cannot print other fax data because a transmission is being processed. (Не удается напечатать другие данные факса, так как выполняется передача.) приложение приостанавливает работу.
Некоторые приложения во время отправки факсимильного сообщения могут приостанавливать работу, одновременно открывая окно с предупреждением. (Позже они могут возобновить работу).

Проблемы с приемом (только для моделей с ПК-ФАКС-приемом)

В случае возникновения проблем с приемом выполните соответствующие инструкции.
Проблема
Решение
Значок Fax Receive Monitor (Монитор получения факсов) не отображается в главном окне приложения FAX Utility.
Значок отображается при выборе принтера с поддержкой ПК-ФАКС-приема.
Подключите принтер (факс), поддерживающий эту функцию, и включите принтер, а затем выберите его.
В меню Received Fax Output Settings (Настройки вывода полученных факсов) выбрано сохранение факсимильных сообщений на компьютере, но они не сохраняются.
Полученные факсимильные сообщения сохраняются в указанной папке, только когда запущен Fax Receive Monitor (Монитор получения факсов). Рекомендуется всегда запускать Fax Receive Monitor (Монитор получения факсов).
Не возможно принять факсимильные сообщения.
  • Установите флажок Auto Answer (Автоответ) в окне Fax Settings (Настройки факса) - Auto Answer Settings (Настройки Автоответа). Если к принтеру не подключен внешний телефон, принтер не сможет получать факсимильные сообщения без включения функции Auto Answer (Автоответ).
  • Проверьте, не находится ли принтер в состоянии ошибки.
Невозможно изменить настройки для печати полученных факсимильных сообщений.
Эти настройки невозможно изменить, пока все полученные факсимильные сообщения не будут переданы на компьютер. На компьютере, выбранном для сохранения факсимильных сообщений, измените настройки после проверки новых факсимильных сообщений либо измените настройки на панели управления принтера.
"При настройке общей сетевой папки для получения новых факсимильных сообщений выводится сообщение Cannot find the received fax folder (Не удалось найти папку полученных факсов).
Чтобы сохранять факсимильные сообщения в сетевой папке, необходимо заранее подключиться к ней. В меню на рабочем столе выберите Go - Connect to Server... и подключите сетевую папку.
При автоматическом запуске Fax Receive Monitor (Монитор получения факсов) окно Fax Receive Monitor (Монитор получения факсов) не отображается.
При входе в систему окно Fax Receive Monitor (Монитор получения факсов) не отображается автоматически. Чтобы открыть это окно, в Dock щелкните по значку Fax Receive Monitor (Монитор получения факсов). Даже если окно Fax Receive Monitor (Монитор получения факсов) не отображается автоматически, такие операции, как проверка новых сообщений, выполняются в фоновом режиме.
При каждом получении факса на компьютере будет отображаться начальный экран для идентификации пользователя.
Если информация для идентификации пользователя для получения PC-FAX (имя и пароль для общего доступа к файлам) соответствует информации, указанной на принтере, то при использовании принтера, подключенного к сети, будет отображаться этот экран при сохранении полученного факса на компьютере. Экран будет отображаться и в том случае, если информация для идентификации пользователя, указанная на принтере, отличается от информации, сохраненной в связке ключей.
Выберите Received Fax Output Settings (Настройки вывода полученных факсов) еще раз и введите правильную информацию для идентификации на начальном экране. Не забудьте выбрать пункт Remember this password in my keychain (Запомнить этот пароль в связке ключей).

Сообщения об ошибках

Если появляется сообщение об ошибке, выполните соответствующие инструкции по устранению ошибки.
Сообщение
Действия
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (Не удалось получить информацию о принтере (факсе). Откройте ;"Настройки печати и факса" и добавьте драйвер ПК-ФАКС.)
Добавьте драйвер ПК-ФАКСА в список принтеров.
В диалоговом окне сообщения нажмите OK. Отобразится окно для добавления принтера.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (Не удалось открыть Запись передачи факса, поскольку уже выполняется другая функция FAX Utility. Пожалуйста, ждите, пока ее выполнение завершится, и повторите попытку.)
Выполняется другая функция FAX Utility. Завершите выполнение этой функции и затем попробуйте повторить попытку.
Cannot open the transmission record file. You have no access rights or the file does not exist. (Не удалось открыть файл записи передачи. У вас нет прав доступа, или файл не существует.)
Проверьте, имеются ли права доступа к записям передач.
Failed to save the log file. Change the path and file name, and save it again. (Не удалось сохранить файл журнала. Измените путь и имя файла и сохраните файл снова.)
Проверьте, где сохраняется файл журнала.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (Не удалось удалить файл журнала. Он может использоваться другим приложением, или у вас нет прав доступа, или он уже удален.)
Если файл журнала не удаляется, проверьте права доступа к файлу либо закройте другие приложения.
A printer communication error occurred. (Произошла ошибка подключения принтера.)
Выключите принтер и снова включите его. Если по-прежнему возникает ошибка, подождите немного и попробуйте еще раз.
An error was detected while communicating with the printer. (При подключении к принтеру обнаружена ошибка.)
Подождите и повторите попытку.
Failed to write the backup file. There is not enough free space or it may be write protected. (Не удалось записать файл резервной копии. Недостаточно места, либо запись защищена.)
Проверьте, сколько свободного места осталось на жестком диске. Если свободного места достаточно, разблокируйте носитель для записи
Failed to write the backup file. The file specified is being used by another application. (Не удалось записать файл резервной копии. Указанный файл используется другой программой.)
Указанный файл используется другой программой. Закройте эту программу.
Cannot recognize the file format. The file may be corrupt or created by another application. (Невозможно распознать формат файла. Возможно, файл создан другим приложением или поврежден.)
Поддерживаемое расширение файла резервных данных - *.fpb. Проверьте файл резервных данных.
Failed to read the backup file. The specified file is being used by another application. (Не удалось прочитать файл резервной копии. Указанный файл используется другой программой.)
Указанный файл используется другой программой. Закройте эту программу.
Processing canceled because an error occurred. (Отмена обработки в связи с ошибкой.)
Если перезапуск принтера и компьютера не решили эту проблему, обратитесь в службу поддержки Epson. Контактную информацию см. в руководстве по работе с принтером.
