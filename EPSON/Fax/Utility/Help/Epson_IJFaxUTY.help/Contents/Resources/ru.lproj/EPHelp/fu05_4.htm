Сохранение резервных данных быстрого набора

Можно сохранить резервные данные быстрого и группового набора. В случае сбоя принтера списки быстрого и группового набора могут быть потеряны. Рекомендуется выполнять резервное копирование данных при каждом обновлении списков номеров. Epson не несет ответственности за потерю любых данных, резервное копирование или восстановление данных и/или настроек даже в течение гарантийного периода.
Примечание:
Данные с пометкой In use by another function (Используется другой функцией).
Запустите FAX Utility.
Запуск FAX Utility
Из списка Printer (Принтер) выберите FAX XXXXXX (ваш принтер) и нажмите Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров).
Отобразятся записи быстрого и группового набора, сохраненные в памяти принтера.
Примечание:
Если на принтере доступна функция безопасности, позволяющая администраторам ограничивать изменение настроек принтера с функцией факса для пользователей, для продолжения работы может потребоваться ввод пароля администратора.
Выберите File (Файл) - Back up (Резервное копирование) - Save Backup (Сохранить резервную копию).
Укажите имя файла и директорию, а затем нажмите кнопку Save (Сохранить).
