Восстановление резервного списка быстрого набора

Можно восстановить резервную копию данных телефонной книги принтера (списки быстрого/группового набора), а затем восстановить их на принтере.
Примечание:
  • Восстановление данных, резервная копия которых сохранена в Windows Fax Utility, невозможно.
  • Если зарезервированный список быстрого/группового набора содержит элементы, не поддерживаемые принтером, для которого требуется восстановить список, эти элементы не восстанавливаются.
  • Если зарезервированный список быстрого/группового набора не содержит элементы, поддерживаемые принтером, для которого требуется восстановить данные, эти элементы восстанавливаются по схеме, описанной ниже.
    Index Word (Слово указателя): не заполнено
    Fax Speed (Скорость факса): предварительно заданное значение
  • Любые элементы, например Name/Group Name (Имя/Имя группы), включающие символы, не поддерживаемые принтером, для которого требуется восстановить данные, остаются пустыми при восстановлении.
  • Любые записи с пометкой In use by another function (Используется другой функцией) в FAX Utility перезаписываются восстановленными данными в случае совпадения номера записи.
  • Любые используемые и защищенные записи не перезаписываются восстановленными данными.
Запустите FAX Utility.
Запуск FAX Utility
Из списка Printer (Принтер) выберите FAX XXXXXX (ваш принтер) и нажмите Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров).
Примечание:
Если на принтере доступна функция безопасности, позволяющая администраторам ограничивать изменение настроек принтера с функцией факса для пользователей, для продолжения работы может потребоваться ввод пароля администратора.
Выберите File (Файл) - Back up (Резервное копирование) - Restore Backup (Восстановить резервную копию).
Выберите файл, который требуется восстановить.
Восстановленный список отображается в окне Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров).
Нажмите в окне.
Восстановленные данные телефонной книги будут сохранены на принтере.
