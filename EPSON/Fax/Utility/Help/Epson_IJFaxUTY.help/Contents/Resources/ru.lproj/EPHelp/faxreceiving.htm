Fax Receive Monitor (Монитор получения факсов) (только для моделей с ПК-ФАКС-приемом)

Toolbar (Панель инструментов)
Значок и сообщение в Printer List (Список принтеров)
В этом окне можно проверить состояние приема факсимильных сообщений и выполнить настройку вывода полученных факсимильных сообщений.
Примечание:
Чтобы открыть это окно, в основном окне FAX Utility выберите Fax Receive Monitor (Монитор получения факсов).

Toolbar (Панель инструментов)

Пункт
Описание
Open the received fax folder (Открыть папку полученных факсов)
Открывает папку, выбранную для сохранения файлов.
Check new faxes (Проверить новые факсы)
Проверяет новые факсимильные сообщения.
Received Fax Output Settings (Настройки вывода полученных факсов)
Определяет необходимость сохранения факсимильных сообщений на компьютере.
Preferences (Настройки)
Можно выполнить настройки монитора приема факсимильных сообщений.

Значок и сообщение в Printer List (Список принтеров)

Значок
Сообщение
Описание
Waiting (Ожидание) / Checking for new faxes (Проверка новых факсов)
Режим ожидания либо проверка новых факсимильных сообщений.
Received faxes saved successfully in the specified folder. (Полученные факсы успешно сохранены в указанной папке.)
Новые факсимильные сообщения, полученные принтером (факсом), сохранены в указанной папке.
The printer is not responding (Принтер не отвечает)
Проверьте соединение между принтером и компьютером, а также состояние принтера (включен ли он).
The printer is processing data (Выполняется обработка данных принтером)
Принтер обрабатывает другие данные. Пожалуйста, подождите.
Cannot find the received fax folder. (Не удалось найти папку полученных факсов.)
Проверьте корректность имени выбранной папки. Если используется общая сетевая папка, убедитесь в наличии прав доступа к ней.
Failed to save faxes in the specified folder. (Не удалось сохранить факсы в указанную папку.)
Проверьте, доступна ли выбранная папка и имеются ли права доступа к ней.
A printer communication error occurred. (Произошла ошибка подключения принтера.)
Проверьте соединение между принтером и компьютером, а также состояние принтера (включен ли он). Если используется сетевое соединение, проверьте, установлено ли соединение с принтером по сети.
The "FAXRECV" disk is being used by another application. (Диск "FAXRECV" используется другим приложением.)
Закройте приложение, использующее диск "FAXRECV".
Cannot save faxes. User authentication failed. (Невозможно сохранить факсы. Не удалось выполнить аутентификацию пользователя.)
Убедитесь, что идентификация пользователя для получения PC-FAX (имя и пароль для общего доступа к файлам) соответствует информации, указанной на принтере, выберите Received Fax Output Settings (Настройки вывода полученных факсов) еще раз и введите правильную информацию для идентификации пользователя. Не забудьте выбрать пункт Remember this password in my keychain (Запомнить этот пароль в связке ключей).
