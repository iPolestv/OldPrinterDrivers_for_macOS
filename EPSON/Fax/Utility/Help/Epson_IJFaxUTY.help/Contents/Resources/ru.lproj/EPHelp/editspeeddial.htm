Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров) - Edit Speed Dial/Group Dial (Редактирование номеров быстрого/группового набора) -

В этом окне можно регистрировать или редактировать записи быстрого и группового набора для принтера.
Пункт
Описание
Entry Number (Номер записи)
Отображает номер быстрого набора.
Speed Dial (Быстрый набор)
Выберите, требуется ли создать или редактировать запись быстрого набора.
Group Dial (Групповой набор)
Выберите, требуется ли создать или редактировать запись группового набора. Создать групповой набор невозможно без предварительного создания записей быстрого набора.
Name/Group Name (Имя/Имя группы)
Для регистрации введите имя записи быстрого или группового набора.
Index Word (Слово указателя) *3
Введите ключевое слова для получателя, используемое при выполнении поиска.
Fax Number (Номер факса) *1
Введите номер факса получателя. Если необходимо разделить номера, например отделить код города от телефона, в качестве разделителя используйте пробел вместо дефиса (-) в номере. В противном случае введенный дефис приведет к паузе (временной остановке) во время передачи. Этот пункт доступен в режиме редактирования записей быстрого набора.
Member (Участник) *2
Отображает список зарегистрированных записей быстрого набора. Выберите для добавления в группу нужную запись и нажмите кнопку OK. Этот пункт доступен в режиме редактирования групповых записей.
Fax Speed (Скорость факса) *3
Выберите скорость факса, используемую для отправки факса получателю. Скорость факса, заданная для каждой записи скоростного набора, применяется только при отправке факсимильного сообщения непосредственно с принтера. FAX Utility игнорирует эту настройку и использует скорость факса, заданную на принтере.
OK
Сохраняет настройки и закрывает окно.
Cancel (Отменить)
Отменяет сохранение или редактирование и закрывает окно.
*1 Обязательный для заполнения пункт при регистрации номера быстрого набора.
*2 При регистрации группы убедитесь, что выбран как минимум один номер быстрого набора.
3* В зависимости от принтера, эти элементы могут не отображаться.
Примечание:
  • Для получения информации о символах и их количестве, разрешенных для ввода, см. следующий раздел.
    ‎Допустимые символы
  • Максимальное количество записей быстрого набора на одну группу отображается на полосе, расположенной в нижней части окна редактирования записей группового набора.
