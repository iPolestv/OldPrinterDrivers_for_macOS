Fax Transmission Record (Запись передачи факса)

Меню
В этом окне можно проверить журнал передачи факсимильных сообщений.
Примечание:
  • Чтобы открыть это окно, в основном окне FAX Utility выберите Fax Transmission Record (Запись передачи факса).
  • В FAX Utility сохраняются только журналы передачи. Сами передаваемые данные не сохраняются, поэтому просмотреть отправленные данные невозможно.

Меню

Меню File (Файл)

Пункт
Описание

Закрывает окно.
Save (Сохранить)
Сохраняет записи передач. Сохраненные данные можно использовать при анализе проблем, возникающих при передаче факсимильных сообщений.
