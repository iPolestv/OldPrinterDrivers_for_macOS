Регистрация быстрого набора путем импорта из приложений Address Book или Contacts

Для быстрого набора можно зарегистрировать информацию в приложении Address Book (Адресная книга) или Contacts.
Запустите FAX Utility.
Запуск FAX Utility
Из списка Printer (Принтер) выберите FAX XXXXXX (ваш принтер) и нажмите Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров).
Отобразятся записи быстрого и группового набора, сохраненные в памяти принтера.
Примечание:
Если на принтере доступна функция безопасности, позволяющая администраторам ограничивать изменение настроек принтера с функцией факса для пользователей, для продолжения работы может потребоваться ввод пароля администратора.
Для добавления новой записи быстрого набора путем импорта из приложения Address Book или Contacts выберите пустую строку и нажмите . Чтобы переписать существующую запись данными приложения Address Book или Contacts, выберите требуемую запись и нажмите .
Выберите данные, которые требуется зарегистрировать для быстрого набора, и нажмите Import (Импорт).
Выбранная запись будет добавлена в Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров).
После завершения всех настроек в окне Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров) нажмите .
Записи быстрого и группового набора, отображающиеся в списке, будут сохранены в памяти принтера.
Примечание:
  • Если импортируемые данные из приложения Address Book или Contacts не содержат имен, данные будут импортированы без имени.
  • Если импортированные данные приложения Address Book или Contacts содержат символы, не поддерживаемые списком Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров), данные импортируются без имени.
    Для получения дополнительной информации о поддерживаемых символах см. следующий раздел.
    ‎Допустимые символы
  • Если имя импортируемых данных Address Book или Contacts содержит 31 символ или более, будут импортированы только первые 30 символов.
