Регистрация или редактирование номеров быстрого и группового набора

Можно регистрировать или редактировать номера быстрого и группового набора для принтера с функцией факса.
Ввести номер факса можно при отправке факсимильного сообщения с принтера после регистрации номера быстрого набора. Зарегистрировав группу набора, можно указать сразу несколько получателей одного факсимильного сообщения.
Примечание:
Настроить групповой набор, без предварительной регистрации записей быстрого набора невозможно.
Запустите FAX Utility.
Запуск FAX Utility
Из списка Printer (Принтер) выберите FAX XXXXXX (ваш принтер) и нажмите Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров).
Отобразятся записи быстрого и группового набора, сохраненные в памяти принтера.
Примечание:
Если на принтере доступна функция безопасности, позволяющая администраторам ограничивать изменение настроек принтера с функцией факса для пользователей, для продолжения работы может потребоваться ввод пароля администратора.
Для создания новой записи выберите пустую строку. Чтобы редактировать существующую запись, выберите ее из списка.
Затем дважды щелкните по линии или нажмите клавишу Enter для открытия диалогового окна редактирования.
Примечание:
  • Чтобы удалить существующую запись, выберите ее и нажмите клавишу удаления, либо откройте меню Edit (Редактировать), а затем нажмите кнопку Delete (Удалить). Однако при удалении группы записи быстрого набора, включенные в группу, не удаляются.
  • Редактирование или удаление защищенной записи быстрого или группового набора невозможно.
Настройте быстрый или групповой набор.
Настройка быстрого набора:
Выберите Speed Dial (Быстрый набор), заполните поля Name/Group Name (Имя/Имя группы), Index Word (Слово указателя), Fax Number (Номер факса) и Fax Speed (Скорость факса) и нажмите кнопку OK. Настроенная запись быстрого набора будет добавлена в Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров).
Примечание:
  • Если для подключения к внешней линии требуется префикс доступа, например 0 или 9, введите его перед номером факса. Если вместо ввода точного префикса для принтера необходимо вводить символ решетки (#), введите этот символ.
  • Скорость факса, заданная для каждой записи скоростного набора, применяется только при отправке факсимильного сообщения непосредственно с принтера. FAX Utility игнорирует эту настройку и использует скорость факса, заданную на принтере.
Настройка группового набора:
Выберите Group Dial (Групповой набор), заполните поля Name/Group Name (Имя/Имя группы) и Index Word (Слово указателя), установите флажки у тех записей быстрого набора, которые требуется добавить в групповой набор, и нажмите кнопку OK. Запись группового набора будет добавлена в Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров).
Примечание:
Параметры Index Word (Слово указателя) и Fax Speed (Скорость факса) могут быть недоступны в зависимости от принтера.
После выполнения настроек нажмите .
Записи быстрого и группового набора, отображающиеся в списке, будут сохранены в памяти принтера.
