> Основная страница > Fax Transmission (Передача факса)

Передача документа, созданного в текстовом или табличном редакторе

Для Mac OS X 10.5 или более поздней версии
Для Mac OS X 10.4.x
Путем выбора принтера с функцией факса в меню Print (Печать) какого-либо приложения можно напрямую пересылать созданные документы, графику и таблицы.
Примечание:
Далее приведены инструкции на примере стандартной программы TextEdit для Mac OS X. Следующие действия могут отличаться в зависимости от используемого приложения. Для получения дополнительной информации см. справку по приложению.

Для Mac OS X 10.5 или более поздней версии

С помощью приложения создайте документ, который необходимо передать по факсу.
В меню приложения File (Файл) выберите Print (Печать).
Откроется диалоговое окно Print (Печать).
Из списка Printer (Принтер) выберите FAX XXXXXX (ваш принтер) и нажмите для просмотра подробных настроек.
Измените соответствующие настройки.
    • Укажите 1 в поле Copies (Копии). Даже в случае ввода значения 2 или более будет передана только одна копия.
    • В зависимости от принтера (факса), передача цветных факсимильных сообщений может быть недоступна для форматов бумаги крупнее, чем A4, Letter (Письмо) или Legal (Стандарт).
    • В течение одного сеанса связи можно передать до 100 страниц.
Выберите Fax Settings (Настройки факса) и выполните необходимые настройки.
Описание каждой настройки см. в справке по драйверу ПК-ФАКС. Открыть справку по драйверу можно нажатием кнопки ? в нижнем левом углу экрана после выбора Fax Settings (Настройки факса).
Выберите Recipient Settings (Настройки получателя) и укажите получателя.
Определение получателя (имени, номера факса и т.д.) напрямую:
Введите необходимую информацию в разделе Add (Добавить) и нажмите кнопку +. Получатель будет добавлен в Recipient List (Список получателей) в верхней части диалогового окна.
Примечание:
При выборе опции Enter fax number twice (Введите дважды номер факса) в настройках драйвера PC-FAX необходимо ввести номер повторно.
Если подключенная телефонная линия требует префикса внешнего доступа, введите его в поле External Access Prefix (Префикс внешнего доступа).
Примечание:
Если принтер подключен к частной телефонной станции, и вместо ввода точного префикса необходимо вводить символ решетки (#), введите этот символ.
Выбор получателя (имя, номер факса и т.д.) из приложения Address book или Contacts:
Если получатель сохранен в адресной книге, нажмите . В списке выберите получателя, нажмите Add (Добавить), а затем OK.
Если подключенная телефонная линия требует префикса внешнего доступа, введите его в поле External Access Prefix (Префикс внешнего доступа).
Примечание:
Если принтер подключен к частной телефонной станции, и вместо ввода точного префикса необходимо вводить символ решетки (#), введите этот символ.
Проверьте настройки получателя и нажмите Fax (Факс).
Для начала передачи нажмите Fax (Факс). Перед отправкой проверьте правильность имени и номера факса получателя.
Примечание:
  • Для просмотра статуса передачи щелкните по значку принтера в Dock. Чтобы остановить передачу, выберите отправляемые данные и нажмите кнопку Delete (Удалить).
  • Если во время отправки произошла ошибка, появится сообщение об ошибке. Проверьте запись передачи в окне Fax Transmission Record (Запись передачи факса).
    Проверка/удаление записей передачи
    Список ошибок передачи
  • Документ, содержащий страницы разного размера, может быть передан некорректно.
