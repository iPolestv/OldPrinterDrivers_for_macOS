Проверка новых факсимильных сообщений

Хотя новые факсимильные сообщения можно проверить путем открытия папки, заданной в меню Received Fax Output Settings (Настройки вывода полученных факсов), в этом разделе описан способ проверки новых факсимильных сообщений с помощью Fax Receive Monitor (Монитор получения факсов). Эта возможность доступна, если выбран пункт "Save" (save faxes on this computer) ("Сохранить" (сохранить факсы на этом компьютере)).
Примечание:
Если в окне Fax Receive Monitor (Монитор получения факсов) выбрана опция Notify me of new faxes via a dock icon (Уведомлять меня о новых факсах с помощью значка дока), значок Fax Receive Monitor (Монитор получения факсов) "подпрыгивает" при получении новых сообщений.
Чтобы открыть папку с сохраненными факсимильными сообщениями, выполните следующее.
1.
Для открытия Fax Receive Monitor (Монитор получения факсов) в Dock щелкните значок Fax Receive Monitor (Монитор получения факсов).
2.
Выберите принтер и нажмите Open the received fax folder (Открыть папку полученных факсов) либо дважды щелкните по значку принтера.
Просмотрите данные и информацию об отправителе в имени файла, а затем откройте файл PDF.
Примечание:
  • Полученные факсимильные сообщения автоматически получают имена согласно следующему формату:
    ГГГГММДДЧЧММСС_xxxxxxxxxx_nnnnn(Год/Месяц/День/Час/Минуты/Секунды_номер отправителя)
    В качестве номера отправителя подставляются данные, переданные отправителем. В зависимости от настроек отправителя этот номер может быть недоступен.
  • Новые факсимильные сообщения можно проверять сразу же путем выбора опции Check new faxes (Проверить новые факсы) на экране Fax Receive Monitor (Монитор получения факсов).
