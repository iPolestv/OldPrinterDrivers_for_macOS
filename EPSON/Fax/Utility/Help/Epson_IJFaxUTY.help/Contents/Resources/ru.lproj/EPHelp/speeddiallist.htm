Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров)

Меню
Toolbar (Панель инструментов)
В списке Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров) можно регистрировать или редактировать номера быстрого и группового набора для принтера с функцией факса.
Примечание:
  • Чтобы открыть это окно, в основном окне FAX Utility выберите Speed Dial/Group Dial List (Список быстрого набора/набора группы номеров).
  • Максимальное количество записей, доступное для подключенного принтера (факса), отображается на полосе, расположенной в нижней части окна.

Меню

Меню File (Файл)

Пункт
Описание
Get Speed Dials (Получение информации о быстром наборе номера)
Предоставляет информацию о номерах быстрого и группового набора для принтера с функцией факса.
Send Speed Dials (Запись информации о быстром наборе)
Сохраняет записи быстрого и группового набора, отображающиеся в списке, в памяти принтера.
Import (Импорт)
From Address Book (Из Адресная книга)
Импортирует данные Address Book (Адресная книга) в список быстрого набора на принтере.
Back up (Резервное копирование)
Save Backup (Сохранить резервную копию)
Сохраняет резервные данные быстрого и группового набора номеров.
Restore Backup (Восстановить резервную копию)
Восстанавливает резервные данные быстрого и группового набора номеров.
Close (Закрыть)
Закрывает окно.

Меню Edit (Редактировать)

Пункт
Описание
Edit (Редактировать)
Редактирует записи быстрого и группового набора номеров, выбранные в списке.
Delete (Удалить)
Удаляет записи быстрого и группового набора номеров, выбранные в списке.
Delete All (Удалить все)
Удаляет все записи быстрого и группового набора номеров в списке.
