Produktöversikt

FAX Utility är ett program med diverse praktiska funktioner för att skicka fax.
FAX Utility fungerar endast med fax-kapabla multifunktionsskrivare från Epson.
Vid användning av en skrivare som stöder PC-FAX-mottagning kan du spara mottagna fax som PDF-filer på datorn. Detta hjälper dig att minska onödiga utskrifter.
Obs!
Om din skrivare stöder PC-FAX-mottagning visas Fax Receive Monitor (Övervakare för faxmottagning) i huvudfönstret FAX Utility.

Fax Receive Monitor (Övervakare för faxmottagning) (Endast för modeller med PC-FAX-mottagning)
Fax Receive Monitor (Övervakare för faxmottagning) visas när en skrivare med PC-FAX-mottagning har valts i huvudfönstret FAX Utility. I fönstret Fax Receive Monitor (Övervakare för faxmottagning) kan du utföra inställningar för att spara mottagna fax på datorn, samt ställa in ett meddelande för nyinkommna fax.
