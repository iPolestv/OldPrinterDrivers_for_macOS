Sända ett dokument som skapats via ett ordbehandlings- eller kalkylprogram

För Mac OS X 10.5.x eller senare
För Mac OS X 10.4.x
Genom att välja ett fax i Print (Skriv ut)-menyn för vissa program kan du direkt överföra data som t.ex. dokument, ritningar och tabeller som skapats.
Obs!
Följande förklaring använder TextEdit, standardprogrammet för Mac OS X som ett exempel. Själva åtgärden kan variera beroende på vilket program som används. För mer information, se programvarans hjälp.

För Mac OS X 10.5.x eller senare

Med hjälp av ett program skapa ett dokument att sända via fax.
Klicka på Print (Skriv ut) från programmenyn File (Fil).
Programmets dialogruta Print (Skriv ut) visas.
Välj FAX XXXXXX (din skrivare) från Printer (Skrivare) och klicka sedan på för att visa detaljerade inställningar.
Utför önskade inställningar.
    • Ange 1 i Copies (Kopior). Även om du anger 2 eller mer så sänds endast en kopia.
    • Beroende på skrivare (fax) kan färgfaxning inte vara tillgängligt för pappersstorlekar större än A4, Letter och Legal.
    • Du kan skicka upp 100 sidor i en enda faxsändning.
Välj Fax Settings (Faxinställningar) och utför önskade inställningar.
För en beskrivning av varje inställning, se PC-FAX drivrutinens hjälp. Du kan visa drivrutinens hjälp genom att klicka på ? nere till vänster på skärmen efter att Fax Settings (Faxinställningar) har valts.
Välj Recipient Settings (Mottagarinställningar) och ange en mottagare.
Ange en mottagare (namn, faxnummer osv.) direkt:
Ange nödvändig information i Add (Lägg till) och klicka sedan på +. Mottagaren har lagts till i Recipient List (Mottagarlista) som visas i den övre delen av dialogrutan.
Obs!
Om du har valt Enter fax number twice (Ange faxnumret två gånger) i PC-FAX-drivrutinsinställningarna måste du ange samma nummer igen.
Om den anslutna faxlinjen kräver ett externt åtkomstprefix, ange prefixet i rutan External Access Prefix (Externt åtkomstprefix).
Obs!
Om skrivaren har en inställning för PBX-linje och har ställts in för # (hash) istället för att ange exakt prefixkod, ange # (hash).
Välja en mottagare (namn, faxnummer osv.) från Address book (Adressbok) eller Contacts (Kontakter):
Om mottagaren har sparats i adressboken, klicka på . Välj mottagare från listan, klicka på Add (Lägg till) och sedan på OK.
Om den anslutna faxlinjen kräver ett externt åtkomstprefix, ange prefixet i rutan External Access Prefix (Externt åtkomstprefix).
Obs!
Om skrivaren har en inställning för PBX-linje och har ställts in för # (hash) istället för att ange exakt prefixkod, ange # (hash).
Kontrollera mottagarinställningarna och klicka på Fax.
Klicka på Fax för att börja skicka. Se till att mottagarens namn och faxnummer är korrekt innan du skickar.
Obs!
  • Klicka på skrivarikonen i Dock för att visa sändningsstatus. För att stoppa sändningen, välj data som skickats och klicka sedan på Delete (Ta bort).
  • Om ett fel inträffar vid sändningen visas ett felmeddelande. Kontrollera sändningsposten från fönstret Fax Transmission Record (Faxsändningspost).
    Kontrollera/ta bort sändningslistor
    Lista för sändningsfel
  • Dokument som innehåller flera olika pappersstorlekar kanske inte kan sändas korrekt.
