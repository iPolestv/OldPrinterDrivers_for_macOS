Återställa en säkerhetskopierad lista för snabbval

Du kan återställa skrivarens säkerhetskopieringsdata för telefonboken (lista för snabbval/gruppnummer) och återställ dem sedan på skrivaren.
Obs!
  • Du kan inte återställa data som säkerhetskopierats i Windows Fax Utility.
  • Om den säkerhetskopierade snabbvals-/gruppnummerlista innehåller objekt som inte stöds av skrivaren som du vill återställa listan på, kommer dessa objekt inte att återställas.
  • Om den säkerhetskopierade snabbvals-/gruppnummerlista inte innehåller objekt som stöds av skrivaren som du vill återställa listan på, kommer dessa objekt att återställas som visas nedan.
    Index Word (Indexord): Tom
    Fax Speed (Hastighet): Förinställt värde
  • Alla objekt, t.ex. Name/Group Name (Namn/gruppnamn), som inkluderar tecken som inte stöds av skrivaren som du vill återställa data på, är tomma när de återställs.
  • Objekt som indikeras med In use by another function (Används av en annan funktion) på FAX Utility skrivs över med återställd data så länge som postens nummer matchas.
  • Poster som användas och skyddas skrivs inte över med återställd data.
Starta FAX Utility.
Startar FAX Utility
Välj FAX XXXXXX (din skrivare) från Printer (Skrivare) och klicka sedan på Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista).
Obs!
Om skrivaren har en säkerhetsfunktion som tillåter administratörer att begränsa användare från att ändra skrivarens faxinställningar måste du kanske ange ett administratörslösenord för att fortsätta.
Klicka på File (Fil) - Back up (Säkerhetskopiera) - Restore Backup (Återställ säkerhetskopia).
Välj den fil som du vill återställa.
Den återställda listan visas i fönstret Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista).
Klicka på i fönstret.
Återställd telefonboksdata sparas i skrivaren.
