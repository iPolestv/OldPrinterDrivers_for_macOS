Inställning för att spara ett mottaget fax på en dator

Du kan ändra det mottagna faxets utdatainställningar för att spara mottagna fax som PDF-filer på en dator eller i en delad mapp på en server (PC-FAX-mottagning).
Obs!
  • Om din skrivare stöder PC-FAX-mottagning visas Fax Receive Monitor (Övervakare för faxmottagning) i huvudfönstret FAX Utility.
  • Ställ in Auto-svar när du använder PC-FAX-mottagning.
  • Beroende på skrivare kan mottagen faxdata som sparats på datorn tas bort från skrivarens minne.
  • Eftersom mottagna fax sparas som PDF-filer behöver du Cover Flow (10.5 eller senare), Quick Look (10.5 eller senare) eller Preview.app som är förinstallerat i Mac OS X.
Starta FAX Utility.
Startar FAX Utility
Klicka på Fax Receive Monitor (Övervakare för faxmottagning).
Obs!
  • Du kan inte starta flera instanser av Fax Receive Monitor (Övervakare för faxmottagning) i en fleranvändarmiljö.
  • Du kan endast importera och spara ett mottaget fax när Fax Receive Monitor (Övervakare för faxmottagning) körs.
  • När Fax Receive Monitor (Övervakare för faxmottagning) körs visas en ikon i Dock. När Fax Receive Monitor (Övervakare för faxmottagning) läggs till i Dock är det enkelt att starta Fax Receive Monitor (Övervakare för faxmottagning). För att lägga till Fax Receive Monitor (Övervakare för faxmottagning) i Dock, klicka på ikonen Fax Receive Monitor (Övervakare för faxmottagning) och håll in Control (Kontroll)-tangenten, klicka sedan på Add to Dock (Lägg till i Dock) i menyn.
Välj skrivare och sedan Receive Settings (Mottagningsinställningar).
Obs!
  • Eftersom du endast kan spara faxmeddelanden till en utskriftskö (fax) åt gången, om det finns flera utskriftsköer (fax) på samma skrivare som t.ex. "FAX XXXXXX (USB)" för USB-anslutningar och "FAX XXXXXX(IP)" för nätverksanslutningar, välj vilken du vill använda för att spara mottagna fax.
  • Om skrivaren har en säkerhetsfunktion som tillåter administratörer att begränsa användare från att ändra skrivarens faxinställningar måste du kanske ange ett administratörslösenord för att fortsätta.
Välj "Save" (save faxes on this computer) ("Spara" (fax sparas på den här datorn)) och klicka sedan på Save to (Spara till) för att ange en mapp för att spara mottagna fax.
Obs!
  • Följande mapp är vald som standard.
    \Användare\Inloggningsnamn\PCFax\Skrivarnamn
  • När skrivaren delas i ett nätverk kan du ändra inställningarna från valfri dator i nätverket. Däremot kan faxmeddelanden endast sparas på ett konto åt gången. Om "Save" (save faxes on this computer) ("Spara" (fax sparas på den här datorn)) har ställts in och Received Fax Output Settings (Utdatainställningar för mottagna fax) har valts på en annan dator visar fönstret Received Fax Output Settings (Utdatainställningar för mottagna fax) meddelandet "Save faxes on another computer or print queue (Spara fax på en annan dator eller i en annan utskriftskö)".
  • I fönstret kan du ändra till "Save" (save faxes on this computer) ("Spara" (fax sparas på den här datorn)) när det inte finns några fax i skrivaren. Om faxmeddelanden finns kvar i skrivaren visas ett felmeddelande som informerar dig om att du inte kan ändra inställningen.
    När skrivaren (fax) delas rekommenderar vi att du skapar en mapp i nätverket för att spara faxmeddelanden.
  • Du kan ändra inställningen för Fax Receive Monitor (Övervakare för faxmottagning) när du klickar på Preferences (Inställningar) i Fax Receive Monitor (Övervakare för faxmottagning).
    Preferences (Inställningar) (Endast för modeller med PC-FAX-mottagning)
Klicka på Send (Skicka) för att skicka inställningarna till skrivaren.
Obs!
När PC-FAX-mottagarens användarautentisering (namn och lösenord för fildelning) ställs in på skrivaren måste du registrera denna information i datorns nyckelring.
Om informationen inte registreras i nyckelringen visas registreringspostens skärm. Ange ditt namn och lösenord, välj Remember this password in my keychain (Kom ihåg lösenordet i min nyckelring) och klicka sedan på Connect (Anslut).
Om du inte sparar denna autentiseringsinformation i nyckelringen måste du ange den varje gång du tar emot ett fax och sparar den på datorn.
