Felmeddelanden för mottagning (endast för modeller med PC-FAX-mottagning)

Följande är en lista över felmeddelanden som kan visas när ett fax tas emot och hur man hanterar problemet.
Meddelande
Lösning
A printer communication error occurred. (Ett skrivarkommunikationsfel har inträffat.)
Stäng och slå sedan på skrivaren igen. Vänta ett litet tag och försök sedan igen.
The printer (fax driver) you are setting has been deleted. Add the printer (fax driver) in [System Preferences] - [Print & Fax]. (Skrivaren (faxdrivrutinen) som du ställer in har tagits bort. Lägg till skrivaren (faxdrivrutinen) i [Systeminställningar] - [Skrivare och fax].)
Lägg till PC-FAX-drivrutinen i skrivarlistan. Klicka på OK i meddelandets dialogruta. Dialogrutan för att lägga till skrivare visas.
Change not accepted. Some printer settings have been changed by another computer, print queue, or the printer. Please try again. (Ändringen tillåts inte. Vissa skrivarinställningar har ändrats med en annan dator, utskriftskö eller skrivaren. Försök igen.)
Received Fax Output Settings (Utdatainställningar för mottagna fax) har ändrats med en annan dator i nätverket, från utskriftskön eller från skrivaren. Vänta ett litet tag och försök sedan igen.
Mottagna fax kan endast sparas på en enskild dator.
Received faxes not saved due to an error. (Mottagna fax har inte sparats på grund av ett fel.)
Kunde inte spara mottaget fax till den angivna mappen. Kontrollera anslutningen till den angivna mappen för att säkerställa att den är tillgänglig, samt kontrollera att du har åtkomsträttigheter till mappen.
Processing canceled because an error occurred. (Bearbetningen avbröts eftersom ett fel inträffade.)
Om problemet kvarstår efter att skrivaren och datorn har startats om, kontakta Epson support. För kontaktinformation, se skrivarens bruksanvisning.
Cannot save faxes. User authentication failed. (Det gick inte att spara fax. Användarautentiseringen misslyckades.)
Kan inte spara mottagna fax eftersom PC-FAX-mottagningens användarautentisering (namn och lösenord för fildelning) inte har ställts in korrekt i datorns nyckelring.
Utför Fax Output Setting (Utdatainställningar för mottagna fax) igen och ange sedan korrekt information för användarautentisering när postskärmen visas. Kom ihåg att välja Remember this password in my keychain (Kom ihåg lösenordet i min nyckelring).
