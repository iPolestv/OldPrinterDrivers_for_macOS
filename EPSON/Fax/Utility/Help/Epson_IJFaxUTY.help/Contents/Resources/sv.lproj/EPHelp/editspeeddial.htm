Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista) - Redigera snabbval/gruppnummer-

Detta fönster låter dig registrera eller redigera en post i ett snabbval eller ett gruppnummer på faxen för skrivaren.
Objekt
Förklaring
Entry Number (Postnummer)
Visar postnumret för snabbvalet.
Speed Dial (Snabbval)
Välj för att skapa eller redigera en post i snabbvalet.
Group Dial (Gruppnummer)
Välj för att skapa eller redigera en post i gruppnumret. Du kan inte skapa ett gruppnummer om du inte har skapat poster för snabbval.
Name/Group Name (Namn/gruppnamn)
Ange ett namn för det snabbval eller gruppnummer som du vill registrera.
Index Word (Indexord) *3
Ange ett indexord för mottagaren för att använda det vid sökningar.
Fax Number (Faxnummer) *1
Ange mottagarens faxnummer. Om du vill separera siffror, t.ex. separera riktnummer från personliga nummer, använd ett mellanslag istället för ett bindestreck (-), eftersom ett bindestreck (-) fungerar som en paus vid uppringning. Detta alternativ visas när du redigerar ett snabbval.
Member (Medlem) *2
Visar en lista över registrerade snabbval. Välj det snabbval som du vill lägga till i gruppnumret och klicka sedan på OK. Detta alternativ visas när du redigerar ett gruppnummer.
Fax Speed (Hastighet) *3
Välj den hastighet som ska användas för att skicka ett fax till mottagaren. Hastigheten som ställts in för varje post i snabbvalen används endast för att skicka ett fax direkt från en skrivare. FAX Utility ignorerar inställningen och använder den hastighet som ställts in på skrivaren.
OK
Sparar inställningarna och stänger fönstret.
Cancel (Avbryt)
Avbryter registreringen eller redigeringen och stänger fönstret.
*1 Alternativ som krävs för att registrera ett snabbval.
*2 Vid registrering av ett gruppnummer, se till att ett eller fler snabbval alltid är valda.
*3 Dessa objekt kanske inte visas beroende på skrivare.
