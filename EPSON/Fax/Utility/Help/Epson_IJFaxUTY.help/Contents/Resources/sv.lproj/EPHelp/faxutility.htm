FAX Utility (Main (huvudnr.) fönster)

I detta fönster väljer du FAX Utility-funktion som ska användas.
Obs!
Detta fönster visas när FAX Utility startar.
Startar FAX Utility
Objekt
Förklaring
Fax Transmission Record (Faxsändningspost)
Visar sändningslistor.
Fax Transmission Record (Faxsändningspost)
Fax Receive Monitor (Övervakare för faxmottagning)
Visas endast när en skrivare (fax) som stöder PC-FAX-mottagning har valts. Låter dig välja om mottagna fax ska sparas på en dator eller inte.
Check Fax Connection (Kontrollera faxanslutning)
Kontrollerar om faxfunktionen för den anslutna skrivaren fungerar normalt.
Fax Settings (Faxinställningar)
Ställer in din faxrubrik och olika kommunikationsinställningar som sparats på skrivaren.
Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista)
Sparar eller redigerar faxens snabbval eller gruppnummer som sparats på skrivaren.
Synkronisera Address Book (Adressbok) eller Synkronisera Contacts (Kontakter)
Synkroniserar data för Address Book (Adressbok) eller Contacts (Kontakter) med FAX Utility så att du kan välja mottagare från kontaktlistan när du skickar ett fax via FAX Utility. Om du uppdaterar Address Book (Adressbok) eller Contacts (Kontakter) efter synkronisering måste du köra synkroniseringen igen för att använda uppdaterad data med FAX Utility.
