Restriktioner för hantering av skrivarens telefonbok (kontaktlista)

Beroende på skrivare kan följande restriktioner tillämpas vid hantering av skrivarens telefonbok (kontaktlista) via FAX Utility.
  • Du kan inte redigera, flytta eller ta bort varje e-postadress och mappnamn som används för vidarebefordran av mottagna fax eller skannad data.
    De ersätts av följande grå meddelandetext; "In use by another function (Används av en annan funktion)".
  • Eventuella poster som används för faxjobb, t.ex. schemalagda fax, skyddas så att de inte kan redigeras, flyttas eller tas bort tills jobbet avbryts eller slutförs. FAX Utility visar de skyddade posterna med grå text.
    • Obs!
    Du kan lägga till ett skyddat snabbval i listan för gruppnummer.
