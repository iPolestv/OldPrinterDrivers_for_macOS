Synkronisera Address Book (Adressbok) eller Contacts (Kontakter) med FAX Utility (för Mac OS X 10.7.x eller senare)

Om du vill välja en mottagare från Address Book (Adressbok) eller Contacts (Kontakter) för att skicka ett fax via FAX Utility, måste du synkronisera den i förväg. Om du uppdaterar Address Book (Adressbok) eller Contacts (Kontakter) efter synkronisering måste du köra synkroniseringen igen för att använda uppdaterad data med FAX Utility.
