Söker efter nya fax

Även om du kan söka efter nya fax genom att öppna mappen som anges i Received Fax Output Settings (Utdatainställningar för mottagna fax), förklaras proceduren för hur man kontrollerar efter nya fax via Fax Receive Monitor (Övervakare för faxmottagning). Detta är endast tillgängligt på datorn när "Save" (save faxes on this computer) ("Spara" (fax sparas på den här datorn)) har valts.
Obs!
Om Notify me of new faxes via a dock icon (Meddela mig om nya fax med en ikon i Dock) har valts i inställningarna för Fax Receive Monitor (Övervakare för faxmottagning), "hoppar" Fax Receive Monitor (Övervakare för faxmottagning)-ikonen för att meddela dig att ett nytt faxmeddelande är tillgängligt.
Följ stegen nedan för att öppna mappen för mottagna fax.
1.
Klicka på Fax Receive Monitor (Övervakare för faxmottagning)-ikonen i Dock för att öppna Fax Receive Monitor (Övervakare för faxmottagning).
2.
Välj en skrivare och klicka på Open the received fax folder (Öppna mappen för mottagna fax) eller dubbelklicka på skrivaren.
Kontrollera datum och avsändare i filnamnet och öppna sedan PDF-filen.
Obs!
  • Mottagna fax byter automatiskt namn till följande namnformat:
    YYYYMMDDHHMMSS_xxxxxxxxxx_nnnnn (År/Månad/Dag/Timme/Minut/Sekund_avsändarens nummer)
    "Avsändarens nummer" visar information som skickats från avsändaren. Detta nummer kanske inte visas beroende på avsändarens inställningar.
  • Du kan kontrollera efter nya faxmeddelanden direkt genom att välja Check new faxes (Kontrollera efter nya fax) i Fax Receive Monitor (Övervakare för faxmottagning).
