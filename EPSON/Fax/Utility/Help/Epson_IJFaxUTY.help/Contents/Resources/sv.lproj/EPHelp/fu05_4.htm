Sparar säkerhetskopia av snabbval

Du kan spara en säkerhetskopia av det konfigurerade snabbvalet och gruppnumret. Listan för snabbval/gruppnummer kan tappas vid skrivarfel. Vi rekommenderar att du skapar en säkerhetskopia av datan när du uppdaterar listan för snabbval/gruppnummer. Epson skall inte hållas ansvariga för dataförlust av något slag, för säkerhetskopiering eller återställning av data och/eller inställningar även under en garantiperiod.
Obs!
Data som indikeras som "In use by another function (Används av en annan funktion)" sparas inte.
Starta FAX Utility.
Startar FAX Utility
Välj FAX XXXXXX (din skrivare) från Printer (Skrivare) och klicka sedan på Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista).
Listorna för snabbvals/gruppnummer som sparats på skrivaren visas.
Obs!
Om skrivaren har en säkerhetsfunktion som tillåter administratörer att begränsa användare från att ändra skrivarens faxinställningar måste du kanske ange ett administratörslösenord för att fortsätta.
Klicka på File (Fil) - Back up (Säkerhetskopiera) - Save Backup (Spara säkerhetskopia).
Ange filnamn och filkatalog och klicka sedan på Save (Spara).
