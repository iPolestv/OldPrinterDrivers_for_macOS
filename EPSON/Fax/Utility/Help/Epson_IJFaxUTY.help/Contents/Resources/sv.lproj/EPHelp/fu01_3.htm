Startar FAX Utility

För Mac OS X 10.6.x eller senare
Klicka på Apple - Systeminställningar - Print & Scan (Skriv ut och skanna) eller Print & Scan (Skriv ut och faxa).
Välj FAX XXXXXX (din skrivare) från Printer (Skrivare) och klicka sedan på Options & Supplies (Alternativ och förbrukningsmaterial) - Verktyg - Open Printer Utility (Öppna skrivarverktyg).
För Mac OS X 10.5.x
Klicka på Apple - System Preferences (Systeminställningar) - Print & Fax (Skriv ut och faxa).
Välj FAX XXXXXX (din skrivare) från skrivaren och klicka sedan på Open Print Queue (Öppna utskriftskö) - Utility (Verktyg).
För Mac OS X 10.4.x
Dubbelklicka på hårddisken - Applications (Program) - Verktyg - Printer Setup Utility (Verktyg för skrivarinställning).
Välj FAX XXXXXX (din skrivare) från skrivaren och klicka sedan på Utility (Verktyg).
