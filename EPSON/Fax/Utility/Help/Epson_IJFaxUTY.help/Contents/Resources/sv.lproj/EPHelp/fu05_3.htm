> Hem > Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista)

Registrera snabbval genom att importera från Address Book (Adressbok) eller Contacts (Kontakter)

Du kan registrera informationen i Address Book (Adressbok) eller Contacts (Kontakter) till snabbvalet.
Starta FAX Utility.
Startar FAX Utility
Välj FAX XXXXXX (din skrivare) från Printer (Skrivare) och klicka sedan på Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista).
Listorna för snabbvals/gruppnummer som sparats på skrivaren visas.
Obs!
Om skrivaren har en säkerhetsfunktion som tillåter administratörer att begränsa användare från att ändra skrivarens faxinställningar måste du kanske ange ett administratörslösenord för att fortsätta.
För att lägga till en ny post för snabbvalet genom att importera data från Address Book (Adressbok) eller Contacts (Kontakter), välj en tom linje och klicka sedan på . För att skriva över en befintlig post med data från Address Book (Adressbok) eller Contacts (Kontakter), välj målpost och klicka på .
Välj den data som du vill registrera till snabbval och klicka sedan på Import (Importera).
Vald data har lagts till i Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista).
När alla inställningar är klara, klicka på i fönstret Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista).
De snabbval och gruppnummer som visas i listan har sparats i skrivaren.
Obs
  • Om data från importerad Address Book (Adressbok) eller Contacts (Kontakter) saknar namninformation så importeras data utan ett namn.
  • Om data från importerad Address Book (Adressbok) eller Contacts (Kontakter) inkluderar något tecken som inte stöds av data Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista), importeras data utan ett namn.
    For information om tecken som stöds, se följande avsnitt.
    Godkända tecken
  • Om namnet för importerad data i Address Book (Adressbok) eller Contacts (Kontakter) har 31 tecken eller mer så importeras endast 30 första tecken.
