Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista)

Meny
Verktygsfält
Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista) låter dig registrera eller redigera faxens snabbval och gruppnummer för skrivaren.
Obs!
  • Detta fönster visas när du klickar på Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista) i FAX Utility (huvudfönster).
  • Det maximala antalet poster som finns tillgängligt för den anslutna skrivaren (fax) visas längst ner i fönstret.

File (Fil)-meny

Objekt
Förklaring
Get Speed Dials (Hämta snabbval...)
Läser in faxens snabbval eller gruppnummer för skrivaren.
Send Speed Dials (Sänd snabbval...)
Sparar de snabbval och gruppnummer som visas i listan på skrivaren.
Import (Importera)
From Address Book (Från Adressbok)
Importerar data för Address Book (Adressbok) till snabbvalen på skrivaren.
Back up (Säkerhetskopiera)
Save Backup (Spara säkerhetskopia)
Sparar säkerhetskopia för konfigurerade snabbval och gruppnummer.
Restore Backup (Återställ säkerhetskopia)
Läser in säkerhetskopierade snabbval eller gruppnummer.
Close (Stäng)
Stänger fönstret.

Objekt

Tar bort snabbval eller gruppnummer som valts i listan.

Tar bort alla snabbval eller gruppnummer som valts i listan.
