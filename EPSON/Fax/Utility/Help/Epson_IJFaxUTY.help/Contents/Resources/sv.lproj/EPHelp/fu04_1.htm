Kontrollera FAX-funktionens status på skrivaren

Du kan kontrollera om FAX-funktionen på skrivaren fungerar normalt. Testresultatet skrivs ut som en rapport. Om ett problem hittas, felsök genom att kontrollera informationen i rapporten.
Se till att skrivaren är ansluten.
  • Skrivaren måste vara ansluten till datorn via USB eller nätverk.
  • Skrivaren måste vara ansluten till en telefonlinje och klara av att skicka/ta emot faxmeddelanden utan anslutning till en dator. För mer information, se skrivarens bruksanvisning.
Sätt in vanligt A4- eller Letter-papper (beroende på land/region) i skrivaren.
Starta FAX Utility.
Startar FAX Utility
Välj FAX XXXXXX (din skrivare) från Printer (Skrivare) och klicka sedan på Check Fax Connection (Kontrollera faxanslutning).
Kontrollera informationen i dialogrutan och klicka sedan på Start (Starta).
Kontrollen startar och resultatet visas i ett fönster. Rapporten skrivs också ut. Om ett problem hittas, felsök genom att kontrollera informationen i rapporten.
Obs!
Om skrivaren har en säkerhetsfunktion som tillåter administratörer att begränsa användare från att ändra skrivarens faxinställningar måste du kanske ange ett administratörslösenord för att fortsätta.
