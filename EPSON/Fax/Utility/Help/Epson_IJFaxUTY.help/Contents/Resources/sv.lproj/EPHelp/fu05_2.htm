Registrera eller redigera snabbval och gruppnummer

Du kan registrera eller redigera faxens snabbval eller gruppnummer för skrivaren.
Du kan enkelt ange faxnumret när du skickar ett fax från skrivaren efter att snabbvalet har registrerats. Du kan ange flera mottagare i ett enda steg efter att gruppnumret har registrerats.
Obs!
Du kan inte konfigurera gruppnummer efter att snabbvalen har registrerats.
Starta FAX Utility.
Startar FAX Utility
Välj FAX XXXXXX (din skrivare) från Printer (Skrivare) och klicka sedan på Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista).
Listorna för snabbvals/gruppnummer som sparats på skrivaren visas.
Obs!
Om skrivaren har en säkerhetsfunktion som tillåter administratörer att begränsa användare från att ändra skrivarens faxinställningar måste du kanske ange ett administratörslösenord för att fortsätta.
För att skapa en ny post, välj en tom linje. För att redigera en befintlig post, välj den post som du vill redigera.
Dubbelklicka sedan på linjen eller tryck på Enter-tangenten för att redigera dialogrutan.
Obs!
  • För att ta bort en befintlig post, välj den post som du vill ta bort och tryck sedan på delete-tangenten, eller klicka på menyn Edit (Redigera) och sedan på Delete (Ta bort). Om du tar bort ett gruppnummer raderas däremot inte snabbvalen i gruppen.
  • Du kan inte redigera eller ta bort en post i ett snabbval eller ett gruppnummer.
Konfigurera snabbval eller gruppnummer.
Konfigurera snabbval:
Välj Speed Dial (Snabbval), ange Name/Group Name (Namn/gruppnamn), Index Word (Indexord), Fax Number (Faxnummer) och Fax Speed (Hastighet), klicka sedan på OK. Konfigurerat snabbval har lagts till i Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista).
Obs!
  • Om ditt telefonsystem kräver ett externt åtkomstprefix som t.ex. 0 eller 9 för att komma ut på linjen, ange prefixet i början av faxnumret. Om skrivaren har ställts in för # (hash) istället för att ange exakt prefixkod, ange # (hash).
  • Hastigheten som ställts in för varje post i snabbvalen används endast för att skicka ett fax direkt från en skrivare. FAX Utility ignorerar inställningen och använder den hastighet som ställts in på skrivaren.
Konfigurera gruppnummer:
Välj Group Dial (Gruppnummer), ange Name/Group Name (Namn/gruppnamn) och Index Word (Indexord), markera kryssrutan för det snabbval som du vill lägga till i gruppnumret och klicka sedan på OK. Det konfigurerade gruppnumret har lagts till i Speed Dial/Group Dial List (Snabbvals-/gruppnummerlista).
Obs!
Index Word (Indexord) och Fax Speed (Hastighet) är kanske inte tillgängligt beroende på skrivare.
När alla inställningar är slutförda, klicka på .
De snabbval och gruppnummer som visas i listan har sparats i skrivaren.
