Sändningsproblem

Felsökning enligt följande vid sändningsproblem.
Problem
Lösning
FAX XXXXXX (din skrivare) visas inte i skrivarlistan.
Se till att skrivar- och PC-FAX-drivrutinen är installerad och att PC-FAX-drivrutinen har lagts till i skrivarlistan.
Fax-knappen är gråmarkerad (inaktiverad).
Denna knapp är inte tillgänglig om inte en mottagare har lagts till i Recipient List (Mottagarlista). Ange minst en mottagare.
Kan inte skicka fax från skrivaren.
  • När "No dial tone detected (Kopplingston saknas)" visas, välj Fax Settings (Faxinställningar) - Dial Tone Detection Settings (Inställningar för avkänning av kopplingston) och avmarkera rutan Detect Dial Tone (Avkänn kopplingston).
  • Se till att du har ställt in Your Phone Number (Ditt telefonnummer) i Header Information Settings (Inställningar för rubrikinformation). Mottagarens faxmaskin kanske har konfigurerats för att blockera fax som inte inkluderar information om avsändarens telefon-/faxnummer.
Status för skickad data ändras inte i Waiting for printer response (Väntar på svar från skrivaren).
Waiting for printer response (Väntar på svar från skrivaren) visas när datorn inte kan kommunicera med skrivaren (FAX) och försvinner inte förrän kommunikationen är möjlig. Kontrollera följande.
  • Att skrivaren är påslagen.
  • Att skrivaren och datorn är korrekt anslutna.
  • Om skrivaren är en nätverksskrivare, att skrivarens nätverksadress är korrekt angiven.
  • Att skrivaren kan skriva ut som normalt.
Om ovanstånde är korrekt, stäng av och slå sedan på skrivaren igen.
Programmet fryser när "Cannot print other fax data because a transmission is being processed." visas.
Vissa program kan frysa vid faxsändning, samtidigt som en varning visas vid uppstart. (De kanske återställs efter ett litet tag.)
Överst 

Mottagningsproblem (endast för modeller med PC-FAX-mottagning)

Felsökning enligt följande om ett mottagningsproblem inträffar.
Problem
Lösning
Ikonen för Fax Receive Monitor (Övervakare för faxmottagning) visas inte i FAX Utility (huvudfönster).
Ikonen visas när du väljer en skrivare som stöder PC-FAX-mottagning.
Anslut en skrivare (fax) som stöder denna funktion och slå på skrivaren, välj sedan din skrivare.
Received Fax Output Settings (Utdatainställningar för mottagna fax) är inställd på att spara fax på datorn, men inga fax sparas.
Mottagna fax sparas endast i angiven mapp när Fax Receive Monitor (Övervakare för faxmottagning) körs. Vi rekommenderar att du alltid kör Fax Receive Monitor (Övervakare för faxmottagning).
Kan inte ta emot fax.
  • Välj kryssrutan Auto Answer (Auto-svar) i Fax Settings (Faxinställningar) - Auto Answer Settings (Inställningar för Auto-svar). Om ingen extern telefon är ansluten till skrivaren kan skrivaren inte ta emot fax utan inställningen Auto Answer (Auto-svar).
  • Kontrollera att skrivaren inte visar en felaktig status.
Kan inte ändra inställningarna för att skriva ut mottagna fax.
Du kan inte ändra inställningarna förrän alla mottagna fax har sparats på datorn. På en dator som är inställd för att spara mottagna fax, ändra inställningarna efter att du har kontrollerat efter nya fax, eller ändra inställningarna via skrivarens kontrollpanel.
"Cannot find the received fax folder (Kan inte hitta mappen för mottagna fax)." visas när en delad mapp i nätverket konfigureras som mapp för mottagna fax.
För att spara i en delad mapp i ett nätverk måste du ansluta till den i förväg. Välj menyraden på skrivbordet Go (Gå till) - Connect to Server (Anslut till server) och anslut till den delade mappen i nätverket.
Fönstret Fax Receive Monitor (Övervakare för faxmottagning) visas inte när Fax Receive Monitor (Övervakare för faxmottagning) startas automatiskt vid inloggning.
Fönstret Fax Receive Monitor (Övervakare för faxmottagning) visas inte automatiskt när datorn loggas in. Klicka på ikonen för Fax Receive Monitor (Övervakare för faxmottagning) i Dock för att öppna fönstret. Även om fönstret Fax Receive Monitor (Övervakare för faxmottagning) inte visas automatiskt så utför den åtgärder i bakgrunden som t.ex. kontrollerar efter nya fax osv.
Postskärmen för användarautentisering visas på datorn varje gång du tar emot ett fax.
Vid användning av en skrivare som är ansluten till nätverket, om PC-FAX-mottagningens användarautentisering (namn och lösenord för fildelning) är inställt på skrivaren, visas denna skärm när det mottagna faxet sparas på datorn. Det visas även när autentiseringsinformationen som ställts in i skrivaren skiljer sig från den information som sparats i nyckelringen.
Utför Received Fax Output Settings (Utdatainställningar för mottagna fax) igen och ange sedan korrekt information för användarautentisering när postskärmen visas. Kom ihåg att välja Remember this password in my keychain (Kom ihåg lösenordet i min nyckelring).

Felmeddelanden

När ett felmeddelande visas, lös problemet genom att följa felsökningsinstruktionerna.
Meddelande
Lösning
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (Kan inte hämta skrivarinformation (fax). Gå till "Utskrifts- och faxinställningar" och lägg till PC-FAX drivrutinen.)
Lägg till PC-FAX-drivrutinen i skrivarlistan.
Klicka på OK i meddelandets dialogruta. Dialogrutan lägg till skrivare visas.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (Faxsändningsposterna kan inte öppnas eftersom en annan FAX Utility-funktion redan används. Vänta tills den är klar och försök igen.)
En annan FAX Utility-funktion körs. Stäng den andra funktionen och försök sedan igen.
Cannot open the transmission record file. You have no access rights or the file does not exist. (Filen med sändningsposter kan inte öppnas. Du saknar behörighet eller filen saknas.)
Kontrollera att du har läsbehörighet för sändningsposter.
Failed to save the log file. Change the path and file name, and save it again. (Det går inte att spara loggfilen. Ändra sökvägen och filnamnet och spara den igen.)
Kontrollera destination där loggfilen sparas.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (Det går inte att radera loggfilen. Ett annat program använder den, du saknar behörighet eller så har den redan raderats.)
Om en loggfil finns kvar, kontrollera åtkomstinställningarna eller stänga andra program.
A printer communication error occurred. (Ett skrivarkommunikationsfel har inträffat.)
Stäng och slå sedan på skrivaren igen. Om det inte fungerar, vänta ett litet tag och försök sedan igen.
An error was detected while communicating with the printer. (Ett fel identifierades vid kommunikation med skrivaren.)
Vänta ett litet tag och försök sedan igen.
Failed to write the backup file. There is not enough free space or it may be write protected. (Det går inte att skriva till säkerhetskopieringsfilen. Det finns inte tillräckligt med ledigt utrymme eller filen är skrivskyddad.)
Kontrollera det lediga utrymmet på hårddisken. Eller ta bort skrivskyddet på säkerhetskopieringsfilen.
Failed to write the backup file. The file specified is being used by another application. (Det går inte att skriva till säkerhetskopieringsfilen. Den angivna filen används av ett annat program.)
Säkerhetskopieringsfilen används av ett annat program. Avsluta andra program.
Cannot recognize the file format. The file may be corrupt or created by another application. (Filformatet kan inte identifieras. Filen kan vara skadad eller så har den skapats i ett annat program.)
Endast filtillägget *.fpb stöds för säkerhetskopieringsfilen. Kontrollera säkerhetskopieringsfilen.
Failed to read the backup file. The specified file is being used by another application. (Det går inte att läsa säkerhetskopieringsfilen. Den angivna filen används av ett annat program.)
Säkerhetskopieringsfilen används av ett annat program. Avsluta andra program.
Processing canceled because an error occurred. (Bearbetningen avbröts eftersom ett fel inträffade.)
Om problemet kvarstår efter att skrivaren och datorn har startats om, kontakta Epson support. För kontaktinformation, se skrivarens bruksanvisning.
