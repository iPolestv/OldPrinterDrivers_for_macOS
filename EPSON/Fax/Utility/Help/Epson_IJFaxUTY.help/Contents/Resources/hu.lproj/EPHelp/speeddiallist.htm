Speed Dial/Group Dial List (Gyorshívó/Csoporthívó lista)

Menü
Eszközsor
A Speed Dial/Group Dial List (Gyorshívó/Csoporthívó lista) lehetővé teszi, hogy a fax gyorshívó és csoporthívó bejegyzéseit regisztrálja vagy szerkessze a nyomtató számára.
Megjegyzés:
  • Ez az ablak akkor jelenik meg, ha a Speed Dial/Group Dial List (Gyorshívó/Csoporthívó lista) elemre kattint a FAX Utility alkalmazásban (főablak).
  • A csatlakoztatott nyomtatóhoz (faxhoz) rendelkezésre álló bejegyzések maximális száma az ablak alján lévő sávon látható.

File (Fájl) menü

Elem
Magyarázat
Get Speed Dials (Gyorshívások fogadása)
Beolvassa a fax gyorshívó vagy csoporthívó bejegyzéseit a nyomtató számára.
Send Speed Dials (Gyorshívások küldése)
Menti a listán szereplő gyorshívókat és csoporthívókat a nyomtatóra.
Import (Importálás)
From Address Book (Címjegyzékből)
Importálja a Address Book (Címjegyzék) adatait a nyomtató gyorshívói közé.
Back up (Biztonsági mentés)
Save Backup (Biztonsági mentés készítése)
Biztonsági másolatot készít a konfigurált gyorshívókról és csoporthívókról.
Restore Backup (Mentés visszaállítása)
Beolvassa a mentett gyorshívót vagy csoporthívót.
Close (Bezár)
Bezárja az ablakot.
