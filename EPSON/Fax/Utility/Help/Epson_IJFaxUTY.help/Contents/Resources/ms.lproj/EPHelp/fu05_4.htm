Menyimpan Sandaran Dail Laju

Sandaran dail laju dan dail kumpulan yang di konfigurasi boleh disimpan. Senarai dail laju/kumpulan boleh hilang kerana pencetak tidak berfungsi. Kami mencadangkan supaya anda menyandarkan data setiap kali senarai dail dikemas kini. Epson tidak akan bertanggung jawab untuk sebarang data yang hilang, untuk menyandarkan atau memulihkan semula data dan/atau tetapan walaupun masih dalam tempoh jaminan.
Nota:
Data ditandakan sebagai "In use by another function (Sedang digunakan oleh fungsi lain)" tidak disimpan.
Mula FAX Utility.
Sedang dimulakan FAX Utility
Pilih FAX XXXXXX (pencetak anda) daripada Printer (Pencetak) dan kemudian klik Speed Dial/Group Dial List (Senarai Dail Laju/Dail Kumpulan).
Senarai dail laju/dail kumpulan disimpan dalam pencetak anda akan dipaparkan.
Nota:
Jika fungsi keselamatan terdapat pada pencetak anda yang membenarkan pentadbir mengehadkan pengguna dari mengubah tetapan faks pencetak, kata laluan perlu dimasukkan untuk meneruskan.
Klik File (Fail) - Back up (Sandaran) - Save Backup (Simpan Sandaran).
Tentukan nama fail dan direktori fail dan kemudian klik Save (Simpan).
