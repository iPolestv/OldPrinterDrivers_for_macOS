Speed Dial/Group Dial List (Senarai Dail Laju/Dail Kumpulan)

Menu
Bar alat
Speed Dial/Group Dial List (Senarai Dail Laju/Dail Kumpulan) membenarkan anda untuk mendaftar atau mengedit dail laju dan kumpulan faks untuk pencetak anda.
Nota:
  • Tetingkap ini dipaparkan semasa anda klik Speed Dial/Group Dial List (Senarai Dail Laju/Dail Kumpulan) di dalam FAX Utility (tetingkap utama).
  • Jumlah maksimum kemasukan tersedia untuk pencetak yang disambung (faks) dipaparkan pada bar di bawah tetingkap.

Menu

File (Fail) menu

Item
Penerangan
Get Speed Dials (Dapatkan Dail Laju)

Menyimpan paparan dail laju dan dail kumpulan dalam senarai pada pencetak.
Import
From Address Book (Daripada Buku Alamat)
Mengimport data Address Book (Buku Alamat) pada dail laju pencetak anda.
Back up (Sandaran)
Save Backup (Simpan Sandaran)
Simpan sandaran konfigurasi dail laju dan dail kumpulan.
Restore Backup (Pulihkan Sandaran)
Baca sandaran dail laju atau dail kumpulan.
Close (Tutup)
Tutup tetingkap.
