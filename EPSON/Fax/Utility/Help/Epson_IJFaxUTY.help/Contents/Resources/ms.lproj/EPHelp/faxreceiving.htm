Fax Receive Monitor (Monitor Terima Faks) (Hanya untuk model yang mempunyai penerima PC-FAKS)

Bar alat
Ikon dan Mesej pada Senarai Pencetak
Tetingkap ini membenarkan anda untuk menyemak status penerima faks dan membuat tetapan output untuk faks diterima.
Nota:
Tetingkap ini dipaparkan semasa anda klik Fax Receive Monitor (Monitor Terima Faks) di dalam FAX Utility (tetingkap utama).

Bar alat

Item
Penerangan
Open the received fax folder (Buka folder faks diterima)
Buka folder yang faks diterima disimpan.
Check new faxes (Semak faks baharu)
Semak untuk faks baharu.
Received Fax Output Settings (Tetapan Output Faks Diterima)
Setkan sama ada untuk menyimpan atau tidak faks yang diterima pada komputer.
Preferences (Keutamaan)
Mengkonfigurasi keutamaan Monitor Terima Faks.
Atas 

Ikon dan Mesej pada Senarai Pencetak

Ikon
Mesej
Penerangan
Waiting (Menunggu) / Checking for new faxes (Menyemak faks baharu)
Menyemak faks baharu atau tunggu sedia.
Received faxes saved successfully in the specified folder (Faks diterima berjaya disimpan di dalam folder tertentu).
Menerima faks baharu oleh pencetak (faks) telah disimpan ke folder ditentukan.
The printer is not responding (Pencetak tidak bertindak balas)
Semak sambungan di antara komputer dan pencetak anda atau semak pencetak dihidupkan.
The printer is processing data (Pencetak sedang memproses data)
Pencetak sedang memproses data yang lain. Sila tunggu.
Cannot find the received fax folder (Tidak menemui folder faks diterima).
Semak nama folder yang ditentukan adalah betul. Jika anda menggunakan folder yang dikongsi pada rangkaian, pastikan hak akses anda pada folder.
Failed to save faxes in the specified folder (Gagal menyimpan faks di dalam folder tertentu).
Semak folder yang ditentukan adalah tersedia atau anda mempunyai hak akses pada folder.
A printer communication error occurred (Ralat komunikasi pencetak berlaku)
Semak sambungan di antara komputer dan pencetak anda atau semak pencetak dihidupkan. Jika anda menggunakan sambungan rangkaian, semak sama ada boleh akses pencetak melalui rangkaian.
The "FAXRECV" disk is being used by another application. (Cakera "FAXRECV" sedang digunakan oleh aplikasi lain.)
Tutup program yang menggunakan cakera "FAXRECV".
Cannot save faxes. User authentication failed. (Tidak dapat simpan faks. Pengesahan pengguna gagal.)
Semak PC-FAX pengesahan pengguna penerima (nama dan kata laluan untuk perkongsian fail) yang diset dalam pencetak, buat Received Fax Output Settings (Tetapan Output Faks Diterima) sekali lagi dan kemudian masukkan maklumat pengesahan pengguna dengan betul. Jangan lupa untuk memilih Remember this password in my keychain (Ingat kata laluan ini dalam rantai kunci saya).
