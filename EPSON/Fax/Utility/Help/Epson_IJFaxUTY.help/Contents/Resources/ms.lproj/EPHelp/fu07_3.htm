Mesej Ralat Penerimaan (Hanya untuk model dengan penerimaan PC-FAKS)

Senarai mesej ralat yang berikut mungkin dipaparkan semasa menerima faks dan bagaimana untuk menangani masalah itu.
Mesej
Apa yang perlu dilakukan
A printer communication error occurred. (Ralat komunikasi pencetak berlaku.)
Matikan pencetak dan kemudian hidupkan. Tunggu sebentar dan kemudian cuba lagi.
The printer (fax driver) you are setting has been deleted. Add the printer (fax driver) in [System Preferences] - [Print & Fax]. (Pencetak (pemacu faks) yang sedang anda tetapkan telah dipadam. Tambah pencetak (pemacu faks) dalam [Keutamaan Sistem] - [Cetakan & Faks].)
Tambah pemacu PC-FAKS pada senarai pencetak. Klik OK dalam dialog mesej. Dialog untuk menambah pencetak dipaparkan.
Change not accepted. Some printer settings have been changed by another computer, print queue, or the printer. Please try again. (Pertukaran tidak diterima. Sesetengah tetapan pencetak telah ditukar oleh komputer lain, gilir cetakan, atau pencetak. Sila cuba semula.)
Received Fax Output Settings (Tetapan Output Faks Diterima) telah diubah daripada komputer lain pada rangkaian, dari giliran pencetak atau pencetak itu. Tunggu sebentar dan kemudian cuba lagi.
Faks diterima hanya boleh disimpan pada satu komputer.
Received faxes not saved due to an error. (Faks diterima tidak disimpan akibat ralat.)
Gagal untuk menyimpan faks yang diterima dalam folder ditentukan. Semak sambungan pada folder ditentukan untuk memastikannya tersedia dan semak hak akses anda pada folder.
Processing canceled because an error occurred. (Pemprosesan dibatalkan akibat berlakunya ralat.)
Jika pencetak anda dimulakan semula dan komputer tidak menyelesaikan masalah, hubungi sokongan Epson. Untuk maklumat kenalan, lihat manual pencetak anda.
Cannot save faxes. User authentication failed. (Tidak dapat simpan faks. Pengesahan pengguna gagal.)
Faks diterima tidak boleh disimpan kerana PC-FAX pengesahan pengguna penerima (nama dan kata laluan untuk perkongsian fail) tidak diset dalam rantai kunci komputer dengan betul.
Jadikan Fax Output Setting (Tetapan Output Faks Diterima.) sekali lagi dan kemudian masukkan maklumat pengesahan pengguna yang betul semasa skrin entri dipaparkan. Jangan lupa untuk memilih Remember this password in my keychain (Ingat kata laluan ini dalam rantai kunci saya).
