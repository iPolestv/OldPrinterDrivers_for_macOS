Sedang dimulakan FAX Utility

Untuk Mac OS X 10.6.x atau kemudian
Klik Apple - System Preferences (Keutamaan Sistem) - Print & Scan (Cetak & Imbas) atau Print & Fax (Cetak & Faks).
Pilih FAX XXXXXX (pencetak anda) daripada Printer (Pencetak) dan kemudian klik Options & Supplies (Pilihan & Bekalan) - Utility (Utiliti) - Open Printer Utility (Buka Utiliti Pencetak).
Untuk Mac OS X 10.5.x
Klik Apple - System Preferences (Keutamaan Sistem) - Print & Fax (Cetak & Faks)
Pilih FAX XXXXXX (pencetak anda) daripada Pencetak dan kemudian klik Open Print Queue (Buka Giliran Cetakan) - Utility (Utiliti).
Untuk Mac OS X 10.4.x
Dwiklik cakera Keras - Applications (Aplikasi) - Utilities (Utiliti) - Printer Setup Utility (Utiliti Persediaan Pencetak).
Pilih FAX XXXXXX (pencetak anda) daripada Pencetak dan kemudian klik Utility (Utiliti).
