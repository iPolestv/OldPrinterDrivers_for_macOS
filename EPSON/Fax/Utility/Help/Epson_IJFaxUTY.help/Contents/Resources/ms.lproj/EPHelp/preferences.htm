Preferences (Keutamaan) (Hanya untuk model yang mempunyai penerima PC-FAKS)

Tetingkap ini membenarkan anda mengkonfigurasi tetapan penerima faks, seperti bagaimana pemberitahuan dibuat semasa faks baharu diterima.
Nota:
  • Tetingkap ini dipaparkan semasa anda klik Fax Receive Monitor Preferences (Keutamaan Monitor Terima Faks) di Fax Receive Monitor (Monitor Terima Faks).
  • Tetapan ini digunakan kepada semua pencetak dan giliran cetak yang dipaparkan pada Fax Receive Monitor (Monitor Terima Faks).
Item
Penerangan
Specify an interval to check for new faxes (Tetapkan sela untuk menyemak faks baharu)
Tentukan selang di mana untuk menyemak faks baharu dari 1 minit ke 30 minit.
Notify me of new faxes via a dock icon (Maklumkan saya tentang faks baharu melalui ikon dok)
Pilih ini untuk membuat ikon Fax Receive Monitor (Monitor Terima Faks) pada Dok pintas semasa faks baharu sampai.
Notify me of errors via a dock icon (Maklumkan saya ralat melalui ikon dok)
Pilih ini untuk membuat ikon Fax Receive Monitor (Monitor Terima Faks) pada Dok pintas semasa ralat berlaku. Klik ikon untuk memaparkan tetingkap Monitor Terima Faks di mana anda boleh menyemak maklumat ralat.
Start at login (Mulakan pada log masuk)
Kami mengesyorkan tetapan ini kepada Automatic (Automatik). Semasa "Save" (save faxes on this computer) ("Simpan" (simpan faks pada komputer ini)) diset dalam Received Fax Output Settings (Tetapan Output Faks Diterima), Fax Receive Monitor (Monitor Terima Faks) bermula secara automatik semasa anda log masuk. Jika anda memilih On (Hidup), Fax Receive Monitor (Monitor Terima Faks) bermula dengan mengelog masuk, tanpa mengira tetapan output. Jika anda memilih Off (Mati), Fax Receive Monitor (Monitor Terima Faks) tidak dimulakan semasa anda mengelog masuk.
Wait time before checking new faxes (Masa tunggu sebelum menyemak faks baharu)
Item ini tersedia semasa Automatic (Automatik) atau On (Hidup) diset dalam Start at login (Mulakan pada log masuk). Tentukan masa menunggu (0 ke 10 minit) sebelum menyemak faks baharu selepas Fax Receive Monitor (Monitor Terima Faks) bermula semasa anda log masuk. Ini berguna ketika folder untuk menyimpan faks adalah pada rangkaian dan anda memerlukan masa untuk mendapatkan sambungan ke folder selepas mengelog masuk.
Nota:
Jika folder untuk menyimpan faks diterima ialah folder perkongsian pada rangkaian, sambungan ke folder boleh dibuat semasa anda log masuk dengan mendaftarkan folder ke senarai item log masuk. Klik Apple - System Preferences (Keutamaan Sistem) - Account (Akaun) - Login to register (Log masuk untuk mendaftar).
