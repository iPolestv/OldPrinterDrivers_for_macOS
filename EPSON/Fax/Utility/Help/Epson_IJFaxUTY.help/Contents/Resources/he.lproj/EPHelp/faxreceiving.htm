Fax Receive Monitor (מנטר קבלת הפקסים) (רק לדגמים שתומכים בקבלת פקסים למחשב)

סרגל כלים
הסמלים וההודעות ברשימת המדפסות
חלון זה מאפשר לבדוק את מצב קבלת הפקסים ולערוך את הגדרות הפלט של הפקסים הנכנסים.
הערה:
חלון זה מוצג כשלוחצים על Fax Receive Monitor (מנטר קבלת הפקסים) ביישום FAX Utility (בחלון הראשי).

הסמלים וההודעות ברשימת המדפסות

סמל
הודעה
הסבר
Waiting (ממתין) / Checking for new faxes (בודק אם יש פקסים חדשים)
המערכת בודקת אם יש פקסים חדשים או ממתינים.
Received faxes saved successfully in the specified folder (פקסים נכנסים נשמרים בהצלחה בתיקייה שצוינה).
פקסים חדשים שהתקבלו על ידי המדפסת (הפקס) נשמרו בתיקייה שצוינה.
The printer is not responding (המדפסת לא מגיבה)
בדוק את החיבור בין המחשב למדפסת ושהמדפסת דולקת.
The printer is processing data (המדפסת מעבדת נתונים)
המדפסת מעבדת נתונים אחרים. אנא המתן.
Cannot find the received fax folder (לא ניתן למצוא את תיקיית הפקסים הנכנסים).
בדוק שהשם של התיקייה שצוינה נכונה. אם אתה משתמש בתיקייה משותפת ברשת ודא שיש לך הרשאות גישה לתיקייה.
Failed to save faxes in the specified folder (כשל בשמירת פקסים בתיקייה שצוינה).
בדוק שהתיקייה שצוינה זמינה ושיש לך הרשאות גישה לתיקייה.
A printer communication error occurred (אירעה שגיאת תקשורת במדפסת)
בדוק את החיבור בין המחשב למדפסת ושהמדפסת דולקת. אם אתה משתמש בחיבור לרשת בדוק שניתן לגשת למדפסת דרך הרשת.
The "FAXRECV" disk is being used by another application. (הדיסק "FAXRECV"" נמצא בשימוש על-ידי יישום אחר.)
סגור את התוכנית שמשתמת בדיסק "FAXRECV".
Cannot save faxes. User authentication failed. (לא ניתן לשמור פקסים. אימות המשתמש נכשל.)
בדוק את פרטי אימות המשתמש ב-PC-FAX עבור קבלת פקסים (שם וסיסמה לשיתוף קבצים) שמוגדרים במדפסת, ערוך שוב את Received Fax Output Settings (הגדרות פלט פקסים נכנסים) והזן נכון את פרטי אימות המשתמש. זכור לבחור באפשרות Remember this password in my keychain (זכור את הסיסמה הזו במפתחות שלי).
