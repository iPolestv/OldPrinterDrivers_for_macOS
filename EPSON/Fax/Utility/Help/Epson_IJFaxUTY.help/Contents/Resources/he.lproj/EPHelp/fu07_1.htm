בעיות שידור

במקרה של בעיות בשידור/בשליחה, נסה את הפתרונות הבאים.
הבעיה
מה לעשות
FAX XXXXXX (המדפסת שלך) לא מוצגת ברשימת המדפסות.
ודא שמנהל ההתקן של המדפסת ומנהל ההתקן של PC-FAX מותקנים ושמנהל ההתקן של PC-FAX נוסף לרשימת המדפסות.
הלחצן Fax (פקס) מחוק באפור (לא פעיל).
לחצן זה לא פעיל עד שלא מוסיפים נמען אל Recipient List (רשימת הנמענים). יש לציין לפחות נמען אחד.
אי אפשר לשלוח פקסים מהמדפסת.
  • כאשר ההודעה "No dial tone detected (לא זוהה צליל חיוג)" מופיעה, בחר את Fax Settings (הגדרות פקס) - Dial Tone Detection Settings (הגדרות זיהוי צליל חיוג), ונקה את הסימון מהתיבה Detect Dial Tone (זהה צליל חיוג).
  • ודא שהגדרת את Your Phone Number (מספר הטלפון שלך) במסגרת Header Information Settings (הגדרות פרטי כותרת עליונה). ייתכן שמכשיר הפקס של הנמען הוגדר לחסום פקסים שאינם מכילים את מספר הטלפון/הפקס של השולח.
הסטטוס של הנתונים ששודרו לא משתנה מהסטטוס Waiting for printer response (ממתין לתגובת המדפסת).
Waiting for printer response (ממתין לתגובת המדפסת) מוצג כאשר המחשב לא מצליח לתקשר עם המדפסת (הפקס) והוא נותר עד שיש תקשורת. בדוק את הדברים הבאים.
  • האם המדפסת דולקת.
  • האם המדפסת והמחשב מחוברים היטב.
  • אם המדפסת היא מדפסת ברשת, האם כתובת המדפסת ברשת הוגדרה נכון.
  • האם המדפסת יכולה להדפיס.
אם בדקת את כל הדברים הנ"ל והם תקינים, כבה את המדפסת והדלק אותה מחדש.
היישום נתקע לאחר שמוצגת ההודעה "Cannot print other fax data because a transmission is being processed.".
חלק מהיישומים נתקעים במהלך שידור פקס ומוצגת אזהרת הפעלה. (הם עשויים להתאושש לבד אחרי זמן מה).

בעיות בקבלה (רק לדגמים שתומכים בקבלת פקסים למחשב)

במקרה של בעיות בקבלה, נסה את הפתרונות הבאים.
הבעיה
מה לעשות
הסמל של Fax Receive Monitor (מנטר קבלת הפקסים) לא מוצג ביישום FAX Utility (בחלון הראשי).
הסמל מוצג רק כשבוחרים מדפסת שתומכת בקבלת פקסים למחשב.
חבר מדפסת (פקס) שתומכת בפונקציה זו, הדלק את המדפסת ובחר אותה.
Received Fax Output Settings (הגדרות פלט פקסים נכנסים) לשמור פקסים במחשב אך הפקסים לא נשמרים.
הפקסים הנכנסים נשמרים בתיקייה שצוינה רק כאשר Fax Receive Monitor (מנטר קבלת הפקסים) פועל. מומלץ לשמור על Fax Receive Monitor (מנטר קבלת הפקסים) פועל.
אי אפשר לקבל פקסים.
  • סמן את התיבה Auto Answer (מענה אוטומטי) דרך Fax Settings (הגדרות פקס) - Auto Answer Settings (הגדרות מענה אוטומטי). אם לא מחובר למדפסת טלפון חיצוני המדפסת לא יכולה לקבל פקסים ללא ההגדרה Auto Answer (מענה אוטומטי).
  • בדוק שאין הודעות שגיאה במדפסת.
אי אפשר לשנות את ההגדרות כדי להדפיס את הפקסים הנכנסים.
אי אפשר לשנות את ההגדרות לפני שכל הפקסים הנכנסים נשמרו במחשב. בדוק אם יש פקסים חדשים במחשב שהוגדר לשמור את הפקסים הנכנסים ורק אז שנה את ההגדרות או שנה את ההגדרות באמצעות לוח הבקרה של המדפסת.
ההודעה "Cannot find the received fax folder (לא ניתן למצוא את תיקיית הפקסים הנכנסים)." מופיעה כשמוגדרת תיקייה משותפת ברשת בתור התיקייה לפקסים הנכנסים.
כדי לשמור בתיקייה משותפת ברשת עליך להתחבר אליה מראש. בחר את שורת התפריט בשולחן העבודה Go (עבור) - Connect to Server (התחבר לשרת) והתחבר לתיקייה המשותפת ברשת.
החלון של Fax Receive Monitor (מנטר קבלת הפקסים) לא מופיע כאשר Fax Receive Monitor (מנטר קבלת הפקסים) מוגדר לפעול אוטומטית בכניסה למערכת.
החלון של Fax Receive Monitor (מנטר קבלת הפקסים) לא מוצג אוטומטית כאשר אתה נכנס למערכת במחשב. לחץ על הסמל של Fax Receive Monitor (מנטר קבלת הפקסים) בשולחן העבודה כדי לפתוח את החלון. גם אם החלון של Fax Receive Monitor (מנטר קבלת הפקסים) לא מוצג אוטומטית, התוכנה עדיין פועלת ברקע ומבצעת פעולות כמו בדיקה אם יש פקסים חדשים וכיוצ"ב.
מסך הכניסה לאימות המשתמש מוצג במחשב בכל פעם שמקבלים פקס.
כשמשתמשים במדפסת שמחוברת לרשת, אם פרטי אימות המשתמש ב-PC-FAX עבור קבלת פקסים (שם וסיסמה לשיתוף קבצים) מוגדרים במדפסת, מסך זה מוצג כששומרים את הפקס הנכנס במחשב. הוא מוצג גם כאשר פרטי האימות שמוגדרים במדפסת שונים מהפרטים ששמורים במפתחות.
הגדר מחדש את Received Fax Output Settings (הגדרות פלט פקסים נכנסים) והזן את פרטי האימות הנכונים של המשתמש כאשר מסך הכניסה מוצג. זכור לבחור באפשרות Remember this password in my keychain (זכור את הסיסמה הזו במפתחות שלי).

הודעות שגיאה

כאשר מוצגת הודעה שגיאה, נסה לפתור את הבעיה באמצעות הפתרונות הבאים.
הודעה
מה לעשות
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (לא ניתן לקבל את פרטי המדפסת (פקס). פתח את "Print & Fax Preferences" והוסף את מנהל ההתקן של PC-FAX.)
הוסף את מנהל ההתקן של PC-FAX לרשימת המדפסות.
לחץ על OK (אישור) בתיבת הדו-שיח של ההודעה. תיבת הדו-שיח להוספת מדפסת תוצג.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (לא ניתן לפתוח את רשומת שידור הפקס משום שפונקציה אחרת של FAX Utility כבר פועלת. המתן לסיום הפעולה ונסה שוב.)
פונקציה אחרת של FAX Utility פועלת. סגור את הפונקציה האחרת ונסה שוב.
Cannot open the transmission record file. You have no access rights or the file does not exist. (לא ניתן לפתוח את קובץ רשומת השידור. אין לך זכויות גישה או שהקובץ לא קיים.)
בדוק שיש לך זכויות גישה (הרשאות קריאה) לרשומת השידור.
Failed to save the log file. Change the path and file name, and save it again. (כשל בשמירת קובץ היומן. שנה את הנתיב ואת שם הקובץ, ושמור אותו שוב.)
בדוק את היעד שבו נשמר קובץ היומן.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (כשל במחיקת קובץ היומן. ייתכן שיישום אחר משתמש בו, שאין לך זכויות גישה, או שהוא כבר נמחק.)
אם קובץ היומן לא נמחק בדוק את ההגדרות של זכויות הגישה או סגור את היישומים האחרים שפועלים.
A printer communication error occurred. (אירעה שגיאת תקשורת במדפסת.)
כבה את המדפסת והדלק אותה מחדש. אם הבעיה נמשכת, המתן זמן מה ונסה שוב.
An error was detected while communicating with the printer. (זוהתה שגיאה במהלך התקשורת עם המדפסת. קוד שגיאה (XXX))
המתן זמן מה ונסה שוב.
Failed to write the backup file. There is not enough free space or it may be write protected. (כשל בכתיבת קובץ הגיבוי. אין מספיק מקום פנוי או שהוא מוגן מפני כתיבה.)
בדוק שיש מקום פנוי בכונן הקשיח. לחלופין, הסר את ההגנה מפני כתיבה של קובץ הגיבוי.
Failed to write the backup file. The file specified is being used by another application. (כשל בכתיבת קובץ הגיבוי. הקובץ שצוין נמצא בשימוש על-ידי יישום אחר.)
קובץ הגיבוי שצוין נמצא בשימוש על-ידי תוכנית אחרת. צא מהתוכנית האחרת.
Cannot recognize the file format. The file may be corrupt or created by another application. (לא ניתן לזהות את פורמט הקובץ. ייתכן שהקובץ פגום או שהוא נוצר באמצעות יישום אחר.)
התוכנה תומכת רק בקבצי גיבוי עם סיומת fpb.*. בדוק את קובץ הגיבוי.
Failed to read the backup file. The specified file is being used by another application. (כשל בקריאת קובץ הגיבוי. הקובץ שצוין נמצא בשימוש על-ידי יישום אחר.)
קובץ הגיבוי שצוין נמצא בשימוש על-ידי תוכנית אחרת. צא מהתוכנית האחרת.
Processing canceled because an error occurred. (העיבוד בוטל משום שאירעה שגיאה.)
אם הפעלה מחדש של המדפסת והמחשב אינה פותרת את הבעיה פנה לתמיכה של Epson. לקבלת פרטי ההתקשרות עיין במדריך למשתמש של המדפסת.
