הודעות שגיאה בקבלה (רק לדגמים שתומכים בקבלת פקסים למחשב)

להלן רשימת הודעות השגיאה שעשויות להיות מוצגות כשמתקבל פקס והסברים לפתרון הבעיות.
הודעה
מה לעשות
A printer communication error occurred. (אירעה שגיאת תקשורת במדפסת.)
כבה את המדפסת והדלק אותה מחדש. המתן זמן מה ונסה שוב.
The printer (fax driver) you are setting has been deleted. Add the printer (fax driver) in [System Preferences] - [Print & Fax]. (המדפסת (מנהל התקן הפקס) שאתה מגדיר נמחקה. הוסף את המדפסת (מנהל התקן הפקס) תחת [העדפות מערכת] - [הדפסה ופקס].)
הוסף את מנהל ההתקן של PC-FAX לרשימת המדפסות. לחץ על OK (אישור) בתיבת הדו-שיח של ההודעה. תיבת הדו-שיח להוספת המדפסת תוצג.
Change not accepted. Some printer settings have been changed by another computer, print queue, or the printer. Please try again. (השינוי אינו קביל. הגדרות מדפסת מסוימות שונו על-ידי מחשב אחר, תור הדפסה או המדפסת. נסה שוב.)
ההגדרות של Received Fax Output Settings (הגדרות פלט פקסים נכנסים) שונו ממחשב אחר ברשת, מתור ההדפסה או מהמדפסת. המתן זמן מה ונסה שוב.
ניתן לשמור את הפקסים הנכנסים רק במחשב אחד.
Received faxes not saved due to an error. (פקסים נכנסים לא נשמרים עקב שגיאה.)
כשל בשמירת הפקסים הנכנסים בתיקייה שצוינה. בדוק את החיבור אל התיקייה שצוינה וודא שהיא זמינה. ודא גם שיש לך זכויות גישה לתיקייה.
Processing canceled because an error occurred. (העיבוד בוטל משום שאירעה שגיאה.)
אם הפעלה מחדש של המדפסת והמחשב אינה פותרת את הבעיה פנה לתמיכה של Epson. לקבלת פרטי ההתקשרות עיין במדריך למשתמש של המדפסת.
Cannot save faxes. User authentication failed. (לא ניתן לשמור פקסים. אימות המשתמש נכשל.)
אי אפשר לשמור את הפקסים הנכנסים כיוון שפרטי אימות המשתמש שנשמרו ב-PC-FAX עבור קבלת פקסים (שם וסיסמה לשיתוף קבצים) לא מוגדרים נכון במפתחות המחשב.
הגדר מחדש את Fax Output Setting (הגדרות פלט פקסים נכנסים.) והזן את פרטי האימות הנכונים של המשתמש כאשר מסך הכניסה מוצג. זכור לבחור באפשרות Remember this password in my keychain (זכור את הסיסמה הזו במפתחות שלי).
