שליחת מסמך שנוצר באמצעות יישום לעיבוד תמלילים או עבודה עם גיליונות נתונים

עבור Mac OS X 10.5.x ואילך
עבור Mac OS X 10.4.x
באמצעות בחירה במדפסת עם פקס מהתפריט Print (הדפסה) של חלק מהיישומים ניתן לשלוח נתונים ישירות מהיישום, למשל מסמכים, שרטוטים וטבלאות שיצרת.
הערה:
בהסבר הבא אנו משתמשים ב-TextEdit, יישום ברירת המחדל של Mac OS X כדוגמה. סדר הפעולות בפועל עשוי להשתנות בהתאם ליישום שבו משתמשים. למידע נוסף עיין בעזרה של היישום.

עבור Mac OS X 10.5.x ואילך

היעזר ביישום כדי ליצור מסמך לשליחה בפקס.
לחץ על Print (הדפס) בתפריט File (קובץ) של היישום.
תיבת הדו-שיח Print (הדפסה) של היישום תוצג.
בחר את FAX XXXXXX (המדפסת) בשדה Printer (מדפסת), ולחץ על כדי להציג את ההגדרות המתקדמות.
ערוך את ההגדרות לפי הצורך.
    • ציין את המספר 1 בשדה Copies (עותקים). גם אם תציין מספר אחר, כגון 2, או יותר, רק עותק אחד יישלח.
    • בהתאם למדפסת (פקס), ייתכן שלא תוכל לשלוח פקסים צבעוניים בדפים שגדולים מדף A4, ‏Letter או Legal.
    • ניתן לשלוח עד 100 עמודים בפקס אחד.
בחר את Fax Settings (הגדרות פקס) וערוך את ההגדרות לפי הצורך.
להסבר על כל הגדרה, ניתן לעיין בעזרה של מנהל ההתקן של PC-FAX. ניתן להציג את העזרה של מנהל ההתקן באמצעות לחיצה על ? בפינה השמאלית התחתונה של המסך אחרי בחירת האפשרות Fax Settings (הגדרות פקס).
בחר את Recipient Settings (הגדרות הנמענים) וציין נמען.
ציון נמען (שם, מספר פקס וכו') ישירות:
הזן את המידע הדרוש בחלק Add (הוסף) ולחץ על +. הנמען יתווסף אל Recipient List (רשימת הנמענים) שמוצגת בחלקה העליון של תיבת הדו-שיח.
הערה:
אם בחרת באפשרות Enter fax number twice (הזן את מספר הפקס פעמיים) בהגדרות מנהל ההתקן של PC-FAX עליך להזין את אותו מספר שוב.
אם בקו הטלפון צריך להזין קידומת גישה לקו חוץ, הזן את הקידומת בתיבה External Access Prefix (קידומת גישה מבחוץ).
הערה:
אם המדפסת מחוברת לקו במרכזיה פרטית (PBX) שהוגדר להשתמש ב-# (סולמית) במקום קידומת חיוג, הזן # (סולמית).
בחירת נמען (שם, מספר פקס וכו') דרך Address book (פנקס הכתובת) או Contacts (אנשי קשר):
אם הנמען שמור בפנקס הכתובות, לחץ על . בחר את הנמען מהרשימה, לחץ על Add (הוסף), ואז על OK (אישור).
אם בקו הטלפון צריך להזין קידומת גישה לקו חוץ, הזן את הקידומת בתיבה External Access Prefix (קידומת גישה מבחוץ).
הערה:
אם המדפסת מחוברת לקו במרכזיה פרטית (PBX) שהוגדר להשתמש ב-# (סולמית) במקום קידומת חיוג, הזן # (סולמית).
בדוק את הגדרות הנמען ולחץ על Fax (פקס).
לחץ על Fax (פקס) כדי להתחיל בשליחה. ודא שהשם ומספר הפקס של הנמען נכונים לפני השליחה.
הערה:
  • לחץ על הסמל של המדפסת ב- Dock (שולחן העבודה) כדי להציג את מצב שליחה. כדי להפסיק את השליחה בחר את הנתונים ששלחת ולחץ על Delete (מחק).
  • אם אירעה שגיאה בזמן השליחה, הודעת שגיאה תוצג. בדוק את רשומת שידור הפקס בחלון Fax Transmission Record (רשומת שידור פקס).
    בדיקת/מחיקת רשומות שליחה
    רשימת שגיאות שידור
  • אם תנסה לשלוח מסמכים המכילים דפים בגדלים שונים ייתכן שהם לא יישלחו טוב.
