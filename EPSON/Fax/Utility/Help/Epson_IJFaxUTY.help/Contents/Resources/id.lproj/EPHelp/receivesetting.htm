Received Fax Output Settings (Setelan Keluaran Faks Masuk) (Hanya untuk model dengan penerimaan FAKS-PC)

Jendela ini dapat digunakan untuk memilih apakah Anda akan menyimpan faks masuk di komputer.
Catatan:
Jendela ini akan ditampilkan bila Anda mengklik Received Fax Output Settings (Setelan Keluaran Faks Masuk) di Fax Receive Monitor (Monitor Penerimaan Faks).
Item
Penjelasan
Setting on the printer (Setelan di printer)
Menampilkan setelan Output faks masuk saat ini.
"Print" ("Cetak") *
Mencetak faks masuk di printer.
Not save faxes (Jgn simpan faks) *
Pilih item ini untuk berhenti menyimpan faks di komputer. Faks masuk akan dicetak atau disimpan di memori printer, tergantung pada setelan printer.
"Save" (save faxes on this computer) ("Simpan" (simpan faks di komputer ini))
Menyimpan faks masuk ke folder yang ditentukan. Faks akan disimpan sebagai file PDF.
Save to (Simpan ke)
Menentukan folder tempat menyimpan faks masuk.
Send the settings to the printer (Mengirimkan setelan ke printer)
Mentransfer setelan keluaran faks masuk ke printer.
* Item tersebut mungkin tidak ditampilkan, tergantung pada printer.
Bila otentikasi pengguna penerima PC-FAX (nama dan sandi untuk berbagi file) ditetapkan di printer, Anda harus mendaftarkan informasi tersebut ke rantai kode komputer.
Jika informasi belum terdaftar dalam rantai kode, maka layar entri pendaftaran akan ditampilkan. Tulis nama dan sandi, pilih Remember this password in my keychain (Ingat sandi ini dalam rantai kode saya), lalu klik Connect (Sambungkan).
Jika Anda tidak menyimpan informasi otentikasi ini dalam rantai kode, Anda akan diminta untuk menulisnya setiap kali menerima faks dan menyimpannya ke komputer.
