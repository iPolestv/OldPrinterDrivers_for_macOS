Setelan untuk menyimpan faks masuk di komputer

Anda dapat mengubah setelan keluaran faks untuk menyimpan faks masuk sebagai file PDF di komputer atau folder bersama di server (penerimaan FAKS-PC)
Catatan:
  • Jika printer Anda mendukung penerimaan FAKS-PC, Fax Receive Monitor (Monitor Penerimaan Faks) akan ditampilkan di jendela utama FAX Utility.
  • Bila menggunakan penerimaan FAKS-PC, tetapkan jawab otomatis.
  • Tergantung pada printer, data faks masuk yang disimpan ke komputer akan dihapus dari memori printer.
  • Karena faks masuk akan disimpan sebagai file PDF, Anda memerlukan Cover Flow (10.5 atau yang lebih baru), Quick Look (10.5 atau yang lebih baru), atau Preview.app yang sudah diinstal di Mac OS X.
Jalankan FAX Utility.
Persiapan FAX Utility
Klik Fax Receive Monitor (Monitor Penerimaan Faks).
Catatan:
  • Anda tidak dapat menjalankan beberapa fungsi Fax Receive Monitor (Monitor Penerimaan Faks) dalam lingkungan multi-pengguna.
  • Anda hanya dapat mengimpor dan menyimpan faks masuk bila Fax Receive Monitor (Monitor Penerimaan Faks) sedang berjalan.
  • Ikon akan muncul di Dock, bila Fax Receive Monitor (Monitor Penerimaan Faks) sedang berjalan. Menambahkan Fax Receive Monitor (Monitor Penerimaan Faks) ke Dock memungkinkan Anda untuk menjalankan Fax Receive Monitor (Monitor Penerimaan Faks) dengan mudah. Untuk menambahkan Fax Receive Monitor (Monitor Penerimaan Faks) ke Dock, klik ikon Fax Receive Monitor (Monitor Penerimaan Faks) sambil menekan terus tombol Control (Kontrol), lalu klik Add to Dock (Tambahkan ke Dock) di menu.
Tentukan printer, lalu pilih Receive Settings (Setelan Penerimaan).
Catatan:
  • Karena Anda hanya dapat menyimpan faks ke satu antrean cetak (faks) pada satu waktu, jika terdapat beberapa antrean cetak (faks) pada printer yang sama, misalnya "FAX XXXXXX (USB)" untuk sambungan USB dan "FAX XXXXXX(IP)" untuk koneksi jaringan, Anda harus memilih salah satu yang akan digunakan untuk menyimpan faks masuk.
  • Jika printer dilengkapi dengan fungsi keamanan di mana administrator dapat membatasi pengguna dari mengubah setelan faks printer, Anda mungkin harus menulis sandi administrator untuk melanjutkan.
Pilih "Save" (save faxes on this computer) ("Simpan" (simpan faks di komputer ini)), lalu klik Save to (Simpan ke) untuk memilih folder yang akan menyimpan faks masuk.
Catatan:
  • Folder berikut ditentukan sebagai default.
    \Pengguna\Nama login\FaksPC\Nama printer
  • Bila printer dibagi dalam jaringan, Anda dapat mengubah setelan dari komputer mana pun di jaringan. Namun, faks hanya dapat disimpan ke satu akun pada satu waktu. Jika "Save" (save faxes on this computer) ("Simpan" (simpan faks di komputer ini)) telah ditetapkan dan Received Fax Output Settings (Setelan Keluaran Faks Masuk) dipilih di komputer lain, maka jendela Received Fax Output Settings (Setelan Keluaran Faks Masuk) akan menampilkan pesan "Save faxes on another computer or print queue (Simpan faks di komputer lain atau antrean cetak)".
  • Di jendela, Anda dapat mengubah ke "Save" (save faxes on this computer) ("Simpan" (simpan faks di komputer ini)) bila tidak ada faks di printer lainnya. Jika faks masih ada di printer, pesan kesalahan akan ditampilkan yang memastikan bahwa Anda tidak dapat mengubah setelan.
    Bila berbagi printer (faks), sebaiknya Anda membuat folder di jaringan untuk menyimpan faks.
  • Anda dapat mengubah setelan Fax Receive Monitor (Monitor Penerimaan Faks) bila mengklik Preferences (Preferensi) di Fax Receive Monitor (Monitor Penerimaan Faks).
    Preferences (Preferensi) (Hanya untuk model dengan penerimaan FAKS-PC)
Klik Send (Kirim) untuk mengirim setelan ke printer.
Catatan:
Bila otentikasi pengguna penerima PC-FAX (nama dan sandi untuk berbagi file) ditetapkan di printer, Anda harus mendaftarkan informasi tersebut ke rantai kode komputer.
Jika informasi belum terdaftar dalam rantai kode, maka layar entri pendaftaran akan ditampilkan. Tulis nama dan sandi, pilih Remember this password in my keychain (Ingat sandi ini dalam rantai kode saya), lalu klik Connect (Sambungkan).
Jika Anda tidak menyimpan informasi otentikasi ini dalam rantai kode, Anda akan diminta untuk menulisnya setiap kali menerima faks dan menyimpannya ke komputer.
