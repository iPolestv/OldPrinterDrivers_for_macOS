> Awal > Mengatasi Masalah

Pesan Kesalahan Penerimaan (Hanya untuk model dengan penerimaan FAKS-PC)

Berikut adalah daftar pesan kesalahan yang dapat ditampilkan saat faks masuk, serta cara menangani masalanya.
Pesan
Cara mengatasi
A printer communication error occurred. (Terjadi galat komunikasi printer.)
Matikan, lalu hidupkan kembali printer. Tunggu sebentar, lalu coba lagi.
The printer (fax driver) you are setting has been deleted. Add the printer (fax driver) in [System Preferences] - [Print & Fax]. (Printer (driver faks) yang sdg Anda setel telah dihapus. Tambah printer (driver faks) di [Preferensi Sistem] - [Cetak & Faks])
Tambahkan driver FAKS-PC ke daftar printer. Klik OK dalam dialog pesan. Dialog untuk menambahkan printer akan ditampilkan.
Change not accepted. Some printer settings have been changed by another computer, print queue, or the printer. Please try again. (Perubahan tdk diterima. Beberapa setelan printer telah diubah oleh komputer lain, antrean cetak, atau printer itu. Harap coba lagi.)
Received Fax Output Settings (Setelan Keluaran Faks Masuk) telah diubah dari komputer lain di jaringan, dari antrean cetak, atau dari printer. Tunggu sebentar, lalu coba lagi.
Faks masuk hanya dapat disimpan di satu komputer.
Received faxes not saved due to an error. (Faks masuk tdk disimpan karena galat.)
Gagal menyimpan faks masuk di folder yang ditentukan. Periksa sambungan ke folder yang ditentukan untuk memastikan bahwa folder tersebut tersedia, lalu pastikan Anda memiliki hak akses ke folder tersebut.
Processing canceled because an error occurred. (Pemrosesan dibatalkan karena terjadi galat.)
Jika memulai ulang printer dan komputer tidak mengatasi masalah, hubungi dukungan Epson. Untuk mendapatkan informasi kontak, baca panduan printer Anda.
Cannot save faxes. User authentication failed. (Tdk dpt menyimpan faks. Otentikasi pengguna gagal.)
Tidak dapat menyimpan faks masuk karena otentikasi pengguna penerimaan PC-FAX (nama dan sandi untuk berbagi file) tidak ditetapkan dalam rantai kode komputer dengan benar.
Buat kembali Fax Output Setting (Setelan Keluaran Faks Masuk.), lalu masukkan informasi otentikasi pengguna yang benar saat layar entri ditampilkan. Ingat untuk memilih Remember this password in my keychain (Ingat sandi ini dalam rantai kode saya).
