Memeriksa Faks Baru

Meskipun Anda dapat memeriksa faks baru dengan membuka folder yang ditentukan dalam Received Fax Output Settings (Setelan Keluaran Faks Masuk), prosedur yang dijelaskan di sini adalah untuk memeriksa faks baru menggunakan Fax Receive Monitor (Monitor Penerimaan Faks). Fungsi ini hanya tersedia di komputer jika "Save" (save faxes on this computer) ("Simpan" (simpan faks di komputer ini)) dipilih.
Catatan:
Jika Notify me of new faxes via a dock icon (Beri tahu saya jika ada faks baru via ikon dok) dipilih dalam preferensi Fax Receive Monitor (Monitor Penerimaan Faks), ikon Fax Receive Monitor (Monitor Penerimaan Faks) akan "muncul" yang akan memberitahukan Anda bahwa faks baru masuk.
Ikuti langkah berikut agar dapat membuka folder untuk faks masuk.
1.
Klik ikon Fax Receive Monitor (Monitor Penerimaan Faks) di Dock untuk membuka Fax Receive Monitor (Monitor Penerimaan Faks).
2.
Pilih printer, lalu klik Open the received fax folder (Buka folder faks masuk), atau klik dua kali printer.
Periksa tanggal dan pengirim di nama file, lalu buka file PDF-nya.
Catatan:
  • Faks masuk akan secara otomatis diubah namanya menggunakan format nama berikut:
    TTTTBBHHJJMMDD_xxxxxxxxxx_nnnnn (Tahun/Bulan/Hari/Jam/Menit/Detik_nomor pengirim)
    "Nomor pengirim" menampilkan informasi dari pengirim. Nomor ini mungkin tidak ditampilkan, tergantung pada setelan pengirim.
  • Anda dapat segera memeriksa faks baru dengan memilih Check new faxes (Periksa faks baru) di Fax Receive Monitor (Monitor Penerimaan Faks).
