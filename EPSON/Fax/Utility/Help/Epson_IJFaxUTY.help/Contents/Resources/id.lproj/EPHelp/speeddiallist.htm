Speed Dial/Group Dial List (Daftar Panggilan Cepat/Grup)

Menu
Toolbar
Speed Dial/Group Dial List (Daftar Panggilan Cepat/Grup) digunakan untuk mendaftarkan atau mengedit panggilan cepat dan panggilan grup faks untuk printer.
Catatan:
  • Jendela ini akan ditampilkan bila Anda mengklik Speed Dial/Group Dial List (Daftar Panggilan Cepat/Grup) di (jendela utama) FAX Utility.
  • Jumlah maksimum entri yang tersedia untuk printer (faks) yang tersambung akan ditampilkan pada panel di bagian bawah jendela.
