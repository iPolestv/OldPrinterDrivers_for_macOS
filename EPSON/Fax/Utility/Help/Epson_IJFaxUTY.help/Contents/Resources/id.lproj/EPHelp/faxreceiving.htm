Fax Receive Monitor (Monitor Penerimaan Faks) (Hanya untuk model dengan penerimaan FAKS-PC)

Toolbar
Ikon dan Pesan dalam Daftar Printer
Dengan jendela ini, Anda dapat memeriksa status penerimaan faks dan membuat setelan keluar untuk faks masuk.
Catatan:
Jendela ini akan ditampilkan bila Anda mengklik Fax Receive Monitor (Monitor Penerimaan Faks) di (jendela utama) FAX Utility.

Toolbar

Item
Penjelasan
Open the received fax folder (Buka folder faks masuk)
Membuka folder tempat menyimpan faks masuk.
Check new faxes (Periksa faks baru)
Memeriksa faks baru.
Received Fax Output Settings (Setelan Keluaran Faks Masuk)
Memilih untuk menyimpan faks masuk di komputer atau tidak.
Preferences (Preferensi)
Mengatur preferensi Monitor Penerimaan Faks.

Ikon dan Pesan dalam Daftar Printer

Ikon
Pesan
Penjelasan
Waiting (Menunggu)/Checking for new faxes (Memeriksa faks baru)
Memeriksa faks baru atau siaga.
Received faxes saved successfully in the specified folder (Faks masuk berhasil disimpan di folder yg ditentukan).
Faks baru yang diterima oleh printer (faks) telah disimpan di folder yang ditentukan.
The printer is not responding (Printer tidak merespons)
Periksa sambungan antara komputer dan printer, atau pastikan printer telah dihidupkan.
The printer is processing data (Printer sdg memproses data)
Printer sedang memproses data lainnya. Silakan tunggu.
Cannot find the received fax folder (Tdk menemukan folder faks masuk).
Pastikan nama folder yang ditentukan sudah benar. Jika Anda menggunakan folder bersama di jaringan, pastikan Anda memiliki hak akses ke folder tersebut.
Failed to save faxes in the specified folder (Gagal menyimpan faks di folder yg ditentukan).
Pastikan folder yang ditentukan tersedia atau pastikan Anda memiliki hak akses ke folder tersebut.
A printer communication error occurred (Terjadi galat komunikasi printer)
Periksa sambungan antara komputer dan printer, atau pastikan printer telah dihidupkan. Jika Anda menggunakan sambungan jaringan, periksa apakah Anda dapat mengakses printer melalui jaringan.
The "FAXRECV" disk is being used by another application. (Disk "FAXRECV" sedang digunakan aplikasi lain.)
Tutup program yang menggunakan disk "FAXRECV".
Cannot save faxes. User authentication failed. (Tdk dpt menyimpan faks. Otentikasi pengguna gagal.)
Periksa otentikasi pengguna penerimaan PC-FAX (nama dan sandi untuk berbagi file) yang ditetapkan di printer, buat kembali Received Fax Output Settings (Setelan Keluaran Faks Masuk), lalu masukkan informasi otentikasi pengguna dengan benar. Ingat untuk memilih Remember this password in my keychain (Ingat sandi ini dalam rantai kode saya).
