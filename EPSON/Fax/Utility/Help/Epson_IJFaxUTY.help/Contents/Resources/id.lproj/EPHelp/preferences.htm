Preferences (Preferensi) (Hanya untuk model dengan penerimaan FAKS-PC)

Dengan jendela ini, Anda dapat mengatur setelan penerimaan faks, misalnya bagaimana notifikasi dibuat saat faks baru masuk.
Catatan:
  • Jendela ini akan ditampilkan bila Anda mengklik Fax Receive Monitor Preferences (Preferensi Monitor Penerimaan Faks) di Fax Receive Monitor (Monitor Penerimaan Faks).
  • Setelan ini berlaku ke semua printer dan antrean cetak akan ditampilkan di Fax Receive Monitor (Monitor Penerimaan Faks).
Item
Penjelasan
Specify an interval to check for new faxes (Tentukan interval utk periksa faks baru)
Tentukan interval untuk memeriksa faks baru mulai dari 1 hingga 30 menit.
Notify me of new faxes via a dock icon (Beri tahu saya jika ada faks baru via ikon dok)
Pilih item ini untuk membuat ikon Fax Receive Monitor (Monitor Penerimaan Faks) di Dock yang muncul bila faks baru masuk.
Notify me of errors via a dock icon (Beri tahu saya jika ada galat via ikon dok)
Pilih item ini untuk membuat ikon Fax Receive Monitor (Monitor Penerimaan Faks) di Dock yang muncul bila terjadi kesalahan. Klik ikon tersebut untuk menampilkan jendela Fax Receive Monitor (Monitor Penerimaan Faks) di mana Anda dapat memeriksa informasi kesalahan.
Start at login (Mulai ktk login)
Sebaiknya tetapkan ke Automatic (Otomatis). Bila "Save" (save faxes on this computer) ("Simpan" (simpan faks di komputer ini)) ditetapkan di Received Fax Output Settings (Setelan Keluaran Faks Masuk), Fax Receive Monitor (Monitor Penerimaan Faks) akan dijalankan secara otomatis bila Anda login. Jika Anda memilih On, Fax Receive Monitor (Monitor Penerimaan Faks) akan dijalankan saat login, apa pun setelan output-nya. Jika Anda memilih Off, Fax Receive Monitor (Monitor Penerimaan Faks) tidak akan dijalankan bila Anda log in.
Wait time before checking new faxes (Waktu tunggu sblm memeriksa faks baru)
Item ini tersedia bila Automatic (Otomatis) atau On ditetapkan di Start at login (Mulai ktk login). Tentukan lama waktu (0 hingga 10 menit) untuk menunggu sebelum memeriksa faks baru setelah Fax Receive Monitor (Monitor Penerimaan Faks) dijalankan saat Anda log in. Fungsi ini bermanfaat bila folder tempat menyimpan faks adalah di jaringan, dan Anda memerlukan waktu untuk membuat sambungan ke folder setelah log in.
Catatan:
Jika folder tempat menyimpan faks masuk merupakan folder bersama di jaringan, Anda dapat menyambung ke folder saat log in dengan mendaftarkan folder ke daftar item login. Klik Apple - System Preferences (Preferensi Sistem) - Account (Akun) - Login to register (Login untuk mendaftar).
