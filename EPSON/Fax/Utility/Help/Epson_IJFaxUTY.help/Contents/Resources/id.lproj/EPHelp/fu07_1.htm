Masalah pengiriman

Atasi dengan langkah berikut jika terjadi masalah pengiriman.
Masalah
Cara mengatasi
FAX XXXXXX (printer Anda) tidak ditampilkan dalam daftar Printer.
Pastikan driver printer dan driver FAKS-PC telah diinstal dan driver FAKS-PC telah ditambahkan ke daftar printer.
Tombol Fax (Faks) akan berubah abu-abu (nonaktif).
Tombol ini tidak tersedia kecuali penerima telah ditambahkan ke Recipient List (Dftr Penerima). Tentukan minimal satu penerima.
Tidak dapat mengirim faks dari printer.
  • Bila "No dial tone detected (Nada drg tdk terdeteksi)" ditampilkan, pilih Fax Settings (Setelan Faks) - Dial Tone Detection Settings (Setelan Deteksi Nada Panggil), lalu hapus centang kotak Detect Dial Tone (Deteksi Nada Panggil).
  • Pastikan Anda telah menetapkan Your Phone Number (Nomor Telepon Anda) dalam Header Information Settings (Setelan Informasi Header). Mesin faks penerima mungkin telah diatur untuk memblokir faks yang tidak mencantumkan informasi nomor faks/telepon pengirim.
Status data yang terkirim tidak akan berubah dari Waiting for printer response (Menunggu respons printer).
Waiting for printer response (Menunggu respons printer) akan ditampilkan bila komputer tidak dapat berkomunikasi dengan printer (FAKS) dan tidak berubah hingga komunikasi berfungsi. Periksa hal berikut.
  • Pastikan printer telah dihidupkan.
  • Pastikan printer dan komputer tersambung dengan benar.
  • Jika printer merupakan printer jaringan, pastikan alamat jaringan printer telah ditetapkan dengan benar.
  • Pastikan printer dapat mencetak dengan normal.
Jika semua hal di atas sudah benar, matikan, lalu hidupkan printer.
Aplikasi membeku setelah menampilkan "Cannot print other fax data because a transmission is being processed.".
Aplikasi tertentu dapat membeku saat pengiriman faks berlangsung, dengan beberapa peringatan pengaktifan yang ditampilkan. (Aplikasi tersebut akan pulih setelah beberapa waktu).

Masalah penerimaan (Hanya untuk Model dengan penerima FAKS-PC)

Atasi dengan langkah berikut jika terjadi masalah penerimaan
Masalah
Cara mengatasi
Ikon Fax Receive Monitor (Monitor Penerimaan Faks) tidak ditampilkan di (jendela utama) FAX Utility.
Ikon akan ditampilkan bila Anda memilih printer yang mendukung penerimaan FAKS-PC.
Sambungkan printer (faks) yang mendukung fungsi ini, hidupkan printer, lalu pilih printer.
Received Fax Output Settings (Setelan Keluaran Faks Masuk) ditetapkan untuk menyimpan faks di komputer, namun faks tidak disimpan.
Faks masuk hanya akan disimpan ke folder yang ditentukan saat Fax Receive Monitor (Monitor Penerimaan Faks) sedang berjalan. Sebaiknya jaga agar Fax Receive Monitor (Monitor Penerimaan Faks) selalu berjalan.
Tidak dapat menerima faks.
  • Centang kotak Auto Answer (Jwb Otmatis) dalam Fax Settings (Setelan Faks) - Auto Answer Settings (Setelan Jawab Otomatis). Jika tidak ada telepon eksternal yang tersambung ke printer, printer tidak dapat menerima faks tanpa setelan Auto Answer (Jwb Otmatis).
  • Pastikan printer tidak berada dalam status error/kesalahan.
Tidak dapat mengubah setelan agar faks masuk dicetak.
Anda tidak dapat mengubah setelan hingga semua faks masuk disimpan ke komputer. Di komputer yang ditetapkan untuk menyimpan faks masuk, ubah setelan setelah memeriksa faks baru, atau ubah setelan menggunakan panel kontrol printer.
"Cannot find the received fax folder (Tdk menemukan folder faks masuk)." ditampilkan bila folder bersama di jaringan diatur sebagai folder faks masuk.
Untuk menyimpan dalam folder bersama di jaringan, Anda harus tersambung terlebih dulu. Pilih panel menu di desktop Go (Buka) - Connect to Server (Sambungkan ke Server), lalu sambungkan ke folder bersama di jaringan.
Jendela Fax Receive Monitor (Monitor Penerimaan Faks) tidak ditampilkan bila Fax Receive Monitor (Monitor Penerimaan Faks) dijalankan secara otomatis saat log in.
Jendela Fax Receive Monitor (Monitor Penerimaan Faks) tidak ditampilkan saat komputer log in. Klik ikon Fax Receive Monitor (Monitor Penerimaan Faks) di Dock untuk membuka jendela. Meskipun jendela Fax Receive Monitor (Monitor Penerimaan Faks) tidak ditampilkan secara otomatis, fungsi ini menjalankan pengoperasian di latar belakang, misalnya memeriksa faks baru dan sebagainya.
Layar entri untuk otentikasi pengguna akan ditampilkan di komputer setiap kali Anda menerima faks.
Bila menggunakan printer yang tersambung ke jaringan, jika otentikasi pengguna penerimaan PC-FAX (nama dan sandi untuk berbagi file) ditetapkan di printer, maka layar ini akan ditampilkan saat Anda menyimpan faks masuk ke komputer. Hal ini juga terjadi bila informasi otentikasi yang ditetapkan di printer berbeda dengan informasi yang disimpan dalam rantai kode.
Buat kembali Received Fax Output Settings (Setelan Keluaran Faks Masuk), lalu masukkan informasi otentikasi pengguna yang benar saat layar entri ditampilkan. Ingat untuk memilih Remember this password in my keychain (Ingat sandi ini dalam rantai kode saya).

Pesan kesalahan

Bila pesan kesalahan ditampilkan, atasi masalah dengan mengikuti petunjuk mengatasi masalah berikut.
Pesan
Cara mengatasi
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (Tdk dpt mengambil informasi (faks) printer. Buka "Print & Fax Preferences" dan tambahkan driver FAKS-PC.)
Tambahkan driver FAKS-PC ke daftar printer.
Klik OK dalam dialog pesan. Dialog tambahkan printer akan ditampilkan.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (Tdk dpt membuka Riwayat Pengiriman Faks karena fungsi FAX Utility lain sedang berjalan. Harap tunggu hingga fungsi itu selesai dan coba lagi.)
Fungsi FAX Utility lainnya sedang berjalan. Tutup fungsi lainnya, lalu coba lagi.
Cannot open the transmission record file. You have no access rights or the file does not exist. (Tdk dpt membuka file riwayat pengiriman. Anda tidak punya hak akses atau file tidak ada.)
Pastikan Anda memiliki hak akses baca ke riwayat pengiriman.
Failed to save the log file. Change the path and file name, and save it again. (Gagal menyimpan file log. Ubah jalur dan nama file, lalu simpan lagi.)
Periksa tempat tujuan penyimpanan file log.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (Gagal menghapus file log. Aplikasi lain mgkn sdg menggunakannya, atau Anda tdk punya hak akses, atau mgkn sudah dihapus.)
Jika file log tidak terhapus, periksa setelan hak akses atau tutup aplikasi lainnya.
A printer communication error occurred. (Terjadi galat komunikasi printer.)
Matikan, lalu hidupkan kembali printer. Jika tidak berfungsi, tunggu sebentar, lalu coba lagi.
An error was detected while communicating with the printer. (Sebuah galat terdeteksi ketika berkomunikasi dgn printer.)
Tunggu sebentar, lalu coba lagi.
Failed to write the backup file. There is not enough free space or it may be write protected. (Gagal menulis file backup. Tidak cukup ruang bebas atau file mungkin terproteksi tulis.)
Periksa sisa memori hardisk Anda. Atau hapus status baca-saja file cadangan.
Failed to write the backup file. The file specified is being used by another application. (Gagal menulis file backup. File yang dimaksud sedang digunakan aplikasi lain.)
File backup sedang digunakan aplikasi lain. Tutup program lainnya.
Cannot recognize the file format. The file may be corrupt or created by another application. (Tidak dpt mengenali format file. File mungkin korup atau dibuat dengan aplikasi lain.)
Ekstensi yang didukung untuk file cadangan hanya *.fpb. Periksa file backup.
Failed to read the backup file. The specified file is being used by another application. (Gagal membaca file backup. File yang dimaksud sedang digunakan aplikasi lain.)
File backup sedang digunakan aplikasi lain. Tutup program lainnya.
Processing canceled because an error occurred. (Pemrosesan dibatalkan karena terjadi galat.)
Jika memulai ulang printer dan komputer tidak mengatasi masalah, hubungi dukungan Epson. Untuk mendapatkan informasi kontak, baca panduan printer Anda.
