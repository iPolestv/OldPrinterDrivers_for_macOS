Mengatur Fungsi Faks Printer

Anda dapat mengatur Header Information Settings (Setelan Informasi Header), Auto Answer Settings (Setelan Jawab Otomatis), dan Dial Tone Detection Settings (Setelan Deteksi Nada Panggil) fungsi FAKS untuk printer.
Jalankan FAX Utility.
Persiapan FAX Utility
Pilih FAX XXXXXX (printer Anda) dari Printer, lalu klik Fax Settings (Setelan Faks).
Ikuti petunjuk di layar untuk mengatur Header Information Settings (Setelan Informasi Header), Auto Answer Settings (Setelan Jawab Otomatis), dan Dial Tone Detection Settings (Setelan Deteksi Nada Panggil).
Catatan:
Jika printer dilengkapi dengan fungsi keamanan di mana administrator dapat membatasi pengguna dari mengubah setelan faks printer, Anda mungkin harus menulis sandi administrator untuk melanjutkan.
