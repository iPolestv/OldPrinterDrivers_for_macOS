FAX Utility uygulamasını başlatma

Mac OS X 10.6.x veya üstü için
Apple - System Preferences (Sistem Tercihleri) - Print & Scan (Yazdır ve Tara) veya Print & Fax (Yazdır ve Faks) öğesine tıklayın.
Printer (Yazıcı) kısmından FAX XXXXXX (yazıcınız) öğesini seçip, Options & Supplies (Seçenekler ve Gereçler) - Utility (Yardımcı Program) - Open Printer Utility (Yazıcı Yardımcı Programını Aç) öğesine tıklayın.
Mac OS X 10.5.x için
Apple - System Preferences (Sistem Tercihleri) - Print & Fax (Yazdır ve Faks) öğesine tıklayın.
Yazıcı kısmından FAX XXXXXX (yazıcınız) öğesini seçip, Open Print Queue (Yazdırma Kuyruğunu Aç) - Utility (Yardımcı Program) öğesine tıklayın.
Mac OS X 10.4.x için
Sabit disk - Applications (Uygulamalar) - Utilities (Yardımcı Programlar) - Printer Setup Utility (Yazıcı Kurulumu Yardımcı Programı) öğesine çift tıklayın.
Yazıcı kısmından FAX XXXXXX (yazıcınız) öğesini seçip, Utility (Yardımcı Program) öğesine tıklayın.
