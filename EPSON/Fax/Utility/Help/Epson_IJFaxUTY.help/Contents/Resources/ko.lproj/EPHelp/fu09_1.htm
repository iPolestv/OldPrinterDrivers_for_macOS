컴퓨터에 수신 팩스 저장 설정

팩스 출력 설정을 변경하여 수신된 팩스를 PDF 파일로 컴퓨터 혹은 서버(PC-FAX 수신)의 공유 폴더에 저장할 수 있습니다.
참고:
  • PC-FAX 수신을 지원하는 프린터인 경우, PC-FAX Utility 기본 창에 Fax 수신 모니터가 표시됩니다.
  • PC-FAX 수신을 사용할 때, 자동 응답을 설정하십시오.
  • 프린터에 따라 컴퓨터에 저장된 수신된 팩스 데이터는 프린터의 메모리에서 삭제됩니다.
  • 수신된 팩스가 PDF 파일로 저장되기 때문에, Cover Flow (10.5이상), Quick Look(10.5이상), 혹은 Mac OS X에 설치된 Preview.app(미리보기)가 필요합니다.
FAX Utility를 시작합니다.
FAX Utility 시작
Fax 수신 모니터를 클릭합니다.
참고:
  • 다중 사용자 환경에서 Fax 수신 모니터의 멀티 인스턴스는 시작할 수 없습니다.
  • Fax 수신 모니터가 실행 중일 때만 수신된 팩스를 불러오고 저장할 수 있습니다.
  • Fax 수신 모니터가 실행 중일 때 Dock에 아이콘이 표시됩니다. Dock에 Fax 수신 모니터를 추가하면, Fax 수신 모니터를 쉽게 시작할 수 있습니다. Dock에 Fax 수신 모니터를 추가하려면, Control 키를 누르고 Fax 수신 모니터 아이콘을 클릭하고, 메뉴에서 Dock에 추가를 클릭합니다.
프린터를 선택하고, 수신 설정을 클릭합니다.
참고:
  • 한번에 하나의 인쇄 대기열에 팩스를 저장할 수 있기 때문에, 만약 같은 프린터에 여러 개의 인쇄 대기열이 있는 경우, 예를 들어 USB 연결로 "FAX XXXXXX (USB)" 그리고 네트워크 연결로 "FAX XXXXXX(IP)", 수신 팩스 저장을 원하는 하나를 선택하십시오.
  • 사용 중인 프린터가 관리자가 프린터의 팩스 설정을 변경할 수 있는 사용자를 제한하는 보안 기능을 가지고 있다면, 계속하기 위하여 관리자 비밀번호 입력이 필요할 수 있습니다.
“저장”(이 컴퓨터에 팩스 저장)을 선택하고, 수신된 팩스가 저장될 폴더를 지정하려면 저장 대상을 클릭합니다.
참고:
  • 아래 폴더가 기본으로 지정됩니다.
    \User\Login name\PCFax\Printer name
  • 프린터가 네트워크로 공유되고 있으면, 네트워크의 다른 컴퓨터에서 설정을 변경할 수 있습니다. 그러나 팩스 저장은 한번에 하나의 계정만 허용됩니다. “저장”(이 컴퓨터에 팩스 저장)이 설정되어 있고 수신 팩스 출력 설정이 다른 컴퓨터에서 선택되었다면, 수신 팩스 설정 창은 "다른 컴퓨터 또는 인쇄 대기열에 팩스 저장" 메시지를 표시합니다.
  • 이 창에서, 프린터에 남아있는 팩스가 없으면 “저장”(이 컴퓨터에 팩스 저장)으로 변경할 수 있습니다. 프린터에 팩스가 남아 있으면, 설정을 변경할 수 없다는 오류 메시지가 표시됩니다.
    프린터(팩스)가 공유 중이면, 팩스 저장 폴더로 네트워크 폴더 지정을 권장합니다.
  • Fax 수신 모니터의 환경 설정을 클릭하여 Fax 수신 모니터 설정을 변경할 수 있습니다.
    환경 설정(PC-FAX 수신 모델에 한함)
설정을 프린터로 보내려면 송신을 클릭합니다.
참고:
PC-FAX 수신 사용자 인증(파일 공유에 대한 이름, 비밀번호)이 프린터에 설정된 경우, 컴퓨터의 키체인에 정보를 입력해야 합니다.
키체인에 정보가 입력되지 않으면 등록 입력 화면이 표시됩니다. 이름과 비밀번호를 입력하고 나의 키체인에 이 비밀번호 기억을 선택하고 연결을 클릭합니다.
키체인에 이 인증 정보를 입력하지 않으면 팩스를 수신하고 컴퓨터로 저장할 때마다 입력을 요청 받을 것입니다.
