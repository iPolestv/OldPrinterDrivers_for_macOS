> 홈 > 팩스 송신

워드프로세서 또는 스프레드시트 응용 프로그램을 사용하여 만든 문서 송신

Mac OS X 10.5.x 이상
Mac OS X 10.4.x
일부 응용 프로그램의 인쇄 메뉴에서 프린터 팩스를 선택하여 만들어 놓은 문서, 그림, 표와 같은 데이터를 직접 송신할 수 있습니다.
참고:
다음 설명은 표준 Mac OS X 응용 프로그램인 텍스트 편집기를 예로 든 것입니다. 실제 작업은 사용하는 응용 프로그램에 따라 다를 수 있습니다. 자세한 내용은 응용 프로그램의 도움말을 참조하십시오.

Mac OS X 10.5.x 이상

응용 프로그램을 사용하여 팩스로 송신할 문서를 만듭니다.
응용 프로그램의 파일 메뉴에서 프린트를 클릭합니다.
응용 프로그램의 인쇄 대화 상자가 표시됩니다.
프린터에서 FAX XXXXXX(사용 중인 프린터)를 선택한 다음 를 클릭하여 세부 설정을 표시합니다.
적절한 설정을 지정합니다.
    • 매수는 1로 지정합니다. 2 이상을 지정한 경우 1매만 송신됩니다.
    • 프린터(fax)에 따라 A4, Letter, 혹은 Legal 보다 큰 용지는 컬러 팩스가 지원되지 않을 수 있습니다.
    • 한 번의 팩스 송신에 최대 100페이지까지 송신할 수 있습니다.
팩스 설정을 선택하고 적절한 설정을 지정합니다.
각 설정에 대한 설명은 PC-FAX 드라이브 도움말을 참조하십시오. 팩스 설정을 선택한 후 화면의 왼쪽 아래에 있는 ?를 클릭하면 드라이버 도움말을 볼 수 있습니다.
수신처 설정을 선택하고 수신처를 지정합니다.
수신처를 직접 지정(이름, 팩스 번호 등):
필수 정보를 입력한 다음 +를 클릭합니다 대화 상자 위쪽 부분에 표시되는 수신처 목록에 수신처가 추가됩니다.
참고:
PC-FAX 드라이버 설정에서 팩스 번호 두 번 입력을 선택하면 같은 번호를 다시 입력해야 합니다.
연결된 팩스 선에 외부 액세스 국번이 필요한 경우 외부 액세스 번호 상자에 국번을 입력합니다.
참고:
사용 중인 프린터가 PBX 라인 형식으로 설정되어 있고, 외부 접속 키 대신 #(우물 정자)를 사용 중이면 #을 입력하십시오.
주소록 또는 연락처에서 수신처 선택(이름, 팩스 번호 등):
주소록에 수신처가 저장되어 있으면, 를 클릭합니다. 목록에서 수신처를 선택하고 추가를 클릭한 다음 승인을 클릭합니다.
연결된 팩스 선에 외부 액세스 국번이 필요한 경우 외부 액세스 번호 상자에 국번을 입력합니다.
참고:
사용 중인 프린터가 PBX 라인 형식으로 설정되어 있고, 외부 접속 키 대신 #(우물 정자)를 사용 중이면 #을 입력하십시오.
수신처 설정을 확인하고 팩스를 클릭합니다.
팩스를 클릭하여 송신을 시작합니다. 송신하기 전에 수신처의 이름과 팩스 번호가 올바른지 확인합니다.
참고:
  • 송신 상태를 표시하려면 Dock의 프린터 아이콘을 클릭하십시오. 송신을 중단하려면 보낸 데이터를 선택한 다음 삭제를 클릭하십시오.
  • 송신 도중 오류가 발생하면 오류 메시지가 표시됩니다. 팩스 송신 기록 창에서 송신 기록을 확인하십시오.
    송신 기록 확인/삭제
    송신 오류 목록
  • 여러 개의 다른 용지 크기가 포함된 문서는 바르게 송신되지 않을 수 있습니다.
