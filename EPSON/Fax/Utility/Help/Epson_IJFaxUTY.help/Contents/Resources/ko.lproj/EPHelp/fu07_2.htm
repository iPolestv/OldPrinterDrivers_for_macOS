송신 오류 목록

다음은 통신 오류 창에 간략한 개요와 함께 표시되는 오류 메시지의 목록입니다.
오류 메시지
개요
지원되지 않는 용지 크기
지원되지 않는 용지 크기를 선택하였기 때문에 송신할 수 없습니다. 선택한 용지 크기가 맞는지 확인하십시오.
지정한 수신처 없음
지정한 수신처가 없기 때문에 송신할 수 없습니다. 수신처 설정 메뉴에서 수신처를 지정합니다.
워드프로세서 또는 스프레드시트 응용 프로그램을 사용하여 만든 문서 송신
디스크 공간 부족
디스크 공간이 충분하지 않기 때문에 송신할 수 없습니다.
송신처 통화 중
수신처가 통화 중이거나 통신 선의 종류가 다르면 송신이 불가능합니다.
전화를 건 번호가 잘못됨
팩스 번호가 잘못되어 수신처 팩스가 응답하지 않기 때문에 송신할 수 없습니다.
컬러 송신 실패
수신처 팩스 기기에서 컬러 팩스를 지원하지 않으면 컬러 송신이 불가능합니다. PC-FAX 드라이버의 컬러 설정을 흑백으로 변경하여 다시 송신하십시오.
워드프로세서 또는 스프레드시트 응용 프로그램을 사용하여 만든 문서 송신
발신음 없음
발신음 확인에 실패했습니다. 프린터의 발신음 확인 설정은 프린터의 사용설명서를 참조하십시오.
응답 없음
수신처에서 주어진 시간 내에 응답 없음이 감지되었습니다. 프린터에서 팩스를 바로 송신(컴퓨터 없이)할 수 있는지 확인하십시오. 오류가 계속 발생하는 경우 전화선이 올바르게 연결되어 정상적으로 작동하는지 확인하십시오.
송신 데이터 용량 제한
데이터에 문제가 있거나 데이터가 100페이지를 초과하면 송신이 불가능합니다.
통신 오류(PC->프린터)
프린터 연결에 문제가 있거나 프린터가 꺼져 있어서 통신이 끊겼습니다.
통신 오류(프린터->수신처)
통신이 끊겼거나 전화 라인에 문제가 있어 송신할 수 없습니다. 전화 라인의 연결을 확인하고, 다시 시도하십시오.
PC-FAX 사용할 수 없음
컴퓨터를 사용한 팩스 송신을 프린터에서 사용할 수 없습니다.
통신 오류(450)
프린터가 컴퓨터에서 전송된 팩스 데이터를 처리할 수 없거나 컴퓨터가 프린터를 인식하지 못합니다.
통신 오류(454)
통신 오류(459)
전화선 연결 오류
프린터(Fax)에 연결된 전화 케이블이 바르게 연결되지 않았습니다. 벽의 전화 연결 잭에서 연결된 전화 선이 LINE 포트에 연결되었는지 확인하십시오.
