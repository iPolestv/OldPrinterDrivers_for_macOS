백업된 단축 다이얼 목록 복원

백업된 프린터 주소록 데이터(단축/그룹 다이얼 목록)를 복원할 수 있고 이것으로 프린터를 복원합니다.
참고:
  • Windows Fax Utility에서 백업한 데이터는 복원할 수 없습니다.
  • 복원을 원하는 백업된 단축/그룹 다이얼 목록에 프린터에서 지원되지 않는 항목이 포함된 경우 이 항목은 복원되지 않습니다.
  • 복원을 원하는 데이터의 백업된 단축/그룹 다이얼 목록에 프린터에서 지원하는 항목이 포함되지 않은 경우 아래와 같이 복원됩니다.
    색인 단어: 공백
    팩스 속도: 사전 설정값
  • 복원을 원하는 데이터에 이름/그룹 이름과 같은 항목에 프린터에서 지원되지 않는 문자가 포함된 경우 공백으로 복원됩니다.
  • FAX Utility에서 다른 기능에서 사용 중으로 표시되는 항목은 항목의 숫자가 맞으면 복원된 데이터로 덮어쓰기 됩니다.
  • 사용 중이거나 보호 중인 어떠한 항목도 복원된 데이터로 덮어쓰기 되지 않습니다.
FAX Utility를 시작합니다.
FAX Utility 시작
프린터에서 FAX XXXXXX(사용 중인 프린터)를 선택한 다음 단축 다이얼/그룹 다이얼 목록을 클릭합니다.
참고:
사용 중인 프린터가 관리자가 프린터의 팩스 설정을 변경할 수 있는 사용자를 제한하는 보안 기능을 가지고 있다면, 계속하기 위하여 관리자 비밀번호 입력이 필요할 수 있습니다.
파일 - 백업 - 백업 복원을 클릭합니다.
복원할 파일을 선택합니다.
단축 다이얼/그룹 다이얼 목록 창에 복원된 목록이 표시됩니다.
창의 버튼을 클릭합니다.
복원된 주소록 데이터는 프린터에 저장됩니다.
