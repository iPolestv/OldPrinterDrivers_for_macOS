송신 문제

송신 문제의 경우 다음과 같이 문제를 해결하십시오.
문제
해결 방법
FAX XXXXXX(사용 중인 프린터)가 프린터 목록에 표시되지 않습니다.
프린터 드라이버와 PC-FAX 드라이버가 설치되어 있고 PC-FAX 드라이버가 프린터 목록에 추가되었는지 확인하십시오.
팩스 버튼이 회색으로 표시됩니다(사용 불가).
수신처 목록에 수신처가 추가되지 않으면 이 버튼을 사용할 수 없습니다. 한 개 이상의 수신처를 지정하십시오.
프린터에서 팩스를 전송할 수 없습니다.
  • “발신음 없음”이 표시된 경우, 팩스 설정 - 발신음 확인 설정을 선택하고 발신음 확인의 확인란을 선택 해제하십시오.
  • 머리글 설정에서 전화번호를 설정하십시오. 수신처의 팩스 기기가 송신처의 전화/팩스 번호 정보가 없는 팩스를 차단하도록 구성되었을 수 있습니다.
프린터 응답 대기에서 송신 데이터의 상태가 변경되지 않습니다.
프린터 응답 대기는 컴퓨터가 프린터(팩스)와 통신을 할 수 없는 경우 표시되고 통신이 가능해질 때까지 지속됩니다. 다음을 확인하십시오.
  • 프린터가 켜졌는지 확인하십시오.
  • 프린터와 컴퓨터가 올바르게 연결되어 있는지 확인하십시오.
  • 프린터가 네트워크 프린터인 경우 프린터 네트워크 주소가 올바르게 지정되었는지 확인하십시오.
  • 프린터가 정상적으로 인쇄할 수 있는지 확인하십시오.
위의 모든 내용이 올바른 경우 프린터를 껐다가 다시 켜십시오.
"다른 팩스 데이터를 송신 중이기 때문에 인쇄할 수 없습니다."가 표시된 후 응용 프로그램이 정지됩니다.
시작 경고가 동시에 표시되면서 일부 응용 프로그램이 팩스 송신에 따라 정지할 수 있습니다. (잠시 후 복원됩니다.)

수신 문제(PC-FAX 수신 모델에 한함)

수신 문제의 경우 다음과 같이 문제를 해결하십시오.
문제
해결 방법
FAX Utility(기본 설정 창)에 팩스 수신 모니터 아이콘이 표시되지 않습니다.
PC-FAX 수신이 지원되는 프린터를 선택하면 아이콘이 표시됩니다.
이 기능을 지원하는 프린터(팩스)를 연결하고 프린터 전원 켜고, 프린터를 선택합니다.
수신 팩스 출력 설정에서 컴퓨터에 팩스 저장이 설정되었으나, 저장된 팩스가 없습니다.
Fax 수신 모니터가 실행 중일 때만 지정된 폴더에 수신된 팩스가 저장됩니다. Fax 수신 모니터 실행을 유지하는 것을 권장합니다.
팩스를 수신할 수 없습니다.
  • 팩스 설정 - 자동 응답 설정에서 자동 응답 확인란을 선택하십시오. 프린터에 외부 전화가 연결되지 않은 경우, 프린터는 자동 응답 설정 없이 팩스를 수신할 수 없습니다.
  • 프린터가 오류 상태에 있지 않은지 확인하십시오.
수신된 팩스가 인쇄되었기 때문에 설정을 변경할 수 없습니다.
모든 수신된 팩스가 컴퓨터에 저장 될 때까지 설정을 변경할 수 없습니다. 수신 팩스 저장으로 설정된 컴퓨터에서, 새로운 팩스를 확인 한 후 설정을 변경하거나, 프린터 제어판을 이용하여 설정을 변경하십시오.
네트워크로 공유된 폴더가 수신 팩스 폴더로 지정 되었을 때 "수신 팩스 폴더를 찾을 수 없습니다."가 표시됩니다.
네트워크의 공유 폴더에 저장하려면, 공유 폴더에 먼저 연결해야 합니다. 데스크 탑의 메뉴바에서 이동 - 서버에 연결을 선택하고 네트워크의 공유 폴더에 연결합니다.
로그인으로 Fax 수신 모니터가 자동으로 시작되어도 Fax 수신 모니터 창은 표시되지 않습니다.
컴퓨터가 로그인 되었을 때 Fax 수신 모니터 창은 자동으로 표시되지 않습니다. 창을 열려면 Dock의 Fax 수신 모니터를 클릭합니다. Fax 수신 모니터 창이 자동으로 표시되지 않아도, 새로운 팩스를 확인하는 등의 작업은 백그라운드로 실행 중입니다.
팩스를 받을 때 마다 컴퓨터에 사용자 인증 입력 화면이 표시됩니다.
프린터를 네트워크로 연결하여 사용 중인 경우, PC-FAX 수신 사용자 인증(파일 공유에 대한 이름, 비밀번호)이 프린터에 설정된 경우 컴퓨터에 수신된 팩스를 저장할 때 마다 이 화면이 표시됩니다. 프린터에 설정된 인증 정보가 키체인에 저장된 정보와 다를 경우에도 발생합니다.
수신 팩스 출력 설정을 다시 지정하고 입력 화면이 표시되면 바른 사용자 정보를 입력하십시오 나의 키체인에 이 비밀번호 기억을 선택하여 기억합니다.

오류 메시지

오류 메시지가 표시되면 다음 문제 해결 지침을 참조하여 문제를 해결하십시오.
메시지
해결 방법
프린터(팩스) 정보를 가져올 수 없습니다. "인쇄 & 팩스 환경 설정"을 열고 PC-FAX 드라이버를 추가하십시오.
PC-FAX 드라이버를 프린터 목록에 추가하십시오.
메시지 대화 상자의 승인을 클릭하십시오. 프린터 추가 대화 상자가 표시됩니다.
FAX Utility의 다른 기능이 이미 실행 중이기 때문에 팩스 전송 기록을 열 수 없습니다. 마칠 때까지 기다렸다가 다시 시도하십시오.
다른 FAX Utility 기능이 실행 중입니다. 다른 기능을 닫은 다음 다시 시도하십시오.
전송 기록 파일을 열 수 없습니다. 액세스 권한이 없거나 파일이 존재하지 않습니다.
송신 기록에 대한 읽기 액세스 권한이 있는지 확인하십시오.
로그 파일을 저장하지 못했습니다. 경로와 파일 이름을 변경하고 다시 저장하십시오.
로그 파일을 저장한 대상을 확인하십시오.
로그 파일을 삭제하지 못했습니다. 다른 응용 프로그램에서 사용 중이거나 액세스 권한이 없거나 이미 삭제되었을 수 있습니다.
로그 파일이 남아 있는 경우 액세스 권한 설정을 확인하거나 다른 응용 프로그램을 닫으십시오.
프린터 통신 오류가 발생하였습니다.
프린터를 껐다가 다시 켜십시오. 만약 동작하지 않으면, 잠시 후 다시 시도하십시오.
프린터와 통신 중 오류를 발견하였습니다.
잠시 후 다시 시도하십시오.
백업 파일 쓰기에 실패했습니다.
여유 공간이 부족하거나 쓰기 금지가 되어 있는지 확인하십시오.
하드 디스크의 여유 공간을 확인하십시오. 또는 백업 파일의 쓰기 금지를 해제하십시오.
백업 파일 쓰기에 실패했습니다.
지정한 파일은 다른 응용 프로그램에서 사용 중입니다.
다른 프로그램에서 백업 파일이 사용되고 있습니다. 다른 프로그램을 종료하십시오.
파일 형식을 인식할 수 없습니다.
파일이 손상되었거나 다른 응용 프로그램에서 생성되었을 수 있습니다.
백업 파일을 지원하는 확장명은 *.fpb 뿐입니다. 백업 파일을 확인하십시오.
백업 파일 읽기에 실패했습니다.
지정한 파일은 다른 응용 프로그램에서 사용 중입니다.
다른 프로그램에서 백업 파일이 사용되고 있습니다. 다른 프로그램을 종료하십시오.
오류가 발생해서 진행이 취소되었습니다.
프린터와 컴퓨터를 다시 시작하는 것으로 문제가 해결되지 않는 경우 Epson 고객 지원 센터에 문의하십시오. 연락처 정보는 프린터 사용설명서를 참조하십시오.
