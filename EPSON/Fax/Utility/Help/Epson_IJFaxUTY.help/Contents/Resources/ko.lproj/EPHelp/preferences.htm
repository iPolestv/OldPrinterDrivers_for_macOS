환경 설정(PC-FAX 수신 모델에 한함)

이 창은 새로운 팩스가 수신될 때 알림을 설정하는 방법 등 팩스 수신 설정의 환경을 설정할 수 있도록 합니다.
참고:
  • Fax 수신 모니터의 환경설정을 클릭하면 이 창이 표시됩니다.
  • 이 설정은 Fax 수신 모니터에 표시된 모든 프린터와 인쇄 큐에 적용됩니다.
항목
설명
새 팩스 확인 간격 지정
새로운 팩스를 확인하는 간격을 1 - 30 분에서 선택합니다.
Dock의 아이콘을 통하여 새 팩스 알림
새로운 팩스가 도착했을 때 Dock의 Fax 수신 모니터 아이콘이 강조되도록 하려면 선택합니다.
Dock의 아이콘을 통하여 오류 알림
오류가 발생했을 때 Dock의 Fax 수신 모니터 아이콘이 강조되도록 하려면 선택합니다. 오류 정보를 확인할 수 있도록 Fax 수신 모니터를 표시하려면 이 아dl콘을 클릭합니다.
로그인할 때 시작
자동 설정을 권장합니다. 수신 팩스 출력 설정에 "저장"(이 컴퓨터에 팩스 저장)이 설정되었으면, 로그인시 Fax 수신 모니터가 자동으로 시작됩니다. 사용을 선택하면, 출력 설정에 상관없이 로그인으로 Fax 수신 모니터는 시작됩니다. 사용 안 함을 선택하면, 로그인 했을 때 Fax 수신 모니터가 시작되지 않습니다.
새 팩스 확인 전 대기 시간
로그인할 때 시작이 자동 혹은 사용일 경우 이 항목은 사용 가능합니다. Fax 수신 모니터가 시작된 후 새 팩스를 확인하기 전 대기 시간을 지정합니다(0 - 10 분). 네트워크의 팩스 저장 폴더에 유용하고, 로그인 후 폴더에 연결하기 위해 시간이 필요합니다.
참고:
수신 팩스 저장 폴더가 네트워크로 공유되어 있을 경우, 로그인 항목 목록에 폴더를 등록하여 로그인 할 때 폴더에 접속할 수 있습니다. 애플 - 시스템 환경설정 - 계정 - 로그인 항목을 클릭합니다.
