Überprüfen der Faxfunktion Ihres Druckers

Sie können überprüfen, ob die Faxfunktion Ihres Druckers fehlerfrei funktioniert. Das Testergebnis wird als Bericht ausgegeben. Wird ein Fehler gefunden, beheben Sie diesen mithilfe der Informationen des Berichts.
Stellen Sie sicher, dass der Drucker angeschlossen ist.
  • Der Drucker muss über USB oder ein Netzwerk mit dem Computer verbunden werden.
  • Der Drucker muss an eine Telefonleitung angeschlossen sein und Faxe senden und empfangen können, ohne an einen Computer angeschlossen zu sein. Weitere Informationen finden Sie im Handbuch des Druckers.
Legen Sie Normalpapier der Größe A4 oder Letter (je nach Land/Region) in den Drucker ein.
Starten Sie FAX Utility.
Starten von FAX Utility
Wählen Sie FAX XXXXXX (Ihr Drucker) unter Printer (Drucker) und klicken Sie anschließend auf Check Fax Connection (Faxverbindung prüfen).
Prüfen Sie die Informationen im Dialogfeld und klicken Sie dann auf Start.
Die Überprüfung startet und die Ergebnisse werden in einem Fenster angezeigt. Der Diagnosebericht wird ebenfalls ausgedruckt. Wird ein Fehler gefunden, beheben Sie diesen mithilfe der Informationen des Berichts.
Hinweis:
Wenn Ihr Drucker über eine Sicherheitsfunktion verfügt, mit der Administratoren die Änderung der Faxeinstellungen des Druckers einschränken können, müssen Sie möglicherweise zum Fortsetzen ein Administratorkennwort eingeben.
