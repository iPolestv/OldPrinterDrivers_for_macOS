Einstellen, dass empfangene Faxe nicht auf einem Computer gespeichert werden

In diesem Abschnitt wird erläutert, wie Sie die Einstellungen von Ihrem Computer aus ändern können. Empfangene Faxe werden abhängig von den Einstellungen des Druckers gedruckt oder verbleiben im Speicher des Produkts.
Hinweis:
  • Sie können die Einstellung auch mithilfe des Druckers ändern. Weitere Informationen finden Sie im Handbuch des Druckers.
  • Wenn empfangene Faxe auf dem Drucker vorhanden sind, können Sie diese Einstellung nicht ändern.
Starten Sie FAX Utility.
Starten von FAX Utility
Klicken Sie auf Fax Receive Monitor (Monitor zum Faxempfang).
Wählen Sie unter Fax Receive Monitor (Monitor zum Faxempfang) Ihren Drucker aus und klicken Sie auf Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe).
Wählen Sie "Print" (print faxes on the printer) ("Drucken" (Faxe auf dem Drucker drucken)) oder "Not save faxes (Faxe nicht speichern)" unter Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe).
Die in Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) angezeigten Einstellungsoptionen variieren abhängig vom Drucker.
