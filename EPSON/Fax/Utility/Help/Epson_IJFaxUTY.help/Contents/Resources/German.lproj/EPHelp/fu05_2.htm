Kurzwahlnummern oder Gruppenwahlnummern anlegen und bearbeiten

Sie können die Kurzwahlnummern oder Gruppenwahlnummern des Faxgerätes für Ihren Drucker anlegen.
Nach dem Anlegen einer Kurzwahlnummer können Sie die Faxnummer beim Senden eines Faxes über den Drucker leicht eingeben. Nach dem Anlegen von Gruppenwahlnummern können Sie in einem Arbeitsschritt mehrere Empfänger eingeben.
Hinweis:
Sie können vor dem Anlegen von Kurzwahlnummern keine Gruppenwahlnummern anlegen.
Starten Sie FAX Utility.
Starten von FAX Utility
Wählen Sie FAX XXXXXX (Ihr Drucker) unter Printer (Drucker) und klicken Sie anschließend auf Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste).
Die im Drucker gespeicherten Kurzwahl-/Gruppenwahllisten werden angezeigt.
Hinweis:
Wenn Ihr Drucker über eine Sicherheitsfunktion verfügt, mit der Administratoren die Änderung der Faxeinstellungen des Druckers einschränken können, müssen Sie möglicherweise zum Fortsetzen ein Administratorkennwort eingeben.
Um einen neuen Eintrag anzulegen, markieren Sie eine leere Zeile. Zur Bearbeitung eines bestehenden Eintrages, wählen Sie den Eintrag, der bearbeitet werden soll.
Doppelklicken Sie dann auf die Zeile oder drücken Sie die Eingabetaste, um das Bearbeitungsfenster zu öffnen.
Hinweis:
  • Um einen bestehenden Eintrag zu löschen, markieren Sie den zu löschenden Eintrag und drücken anschließend die Löschtaste, oder klicken Sie auf das Menü Edit (Bearbeiten) und anschließend auf Delete (Löschen). Beim Löschen einer Gruppenwahlnummer werden jedoch nicht die Kurzwahlnummern gelöscht, die in der Gruppe enthalten waren.
  • Sie können einen geschützten Kurzwahleintrag oder Gruppenwahleintrag nicht bearbeiten oder löschen.
Konfigurieren einer Kurzwahlnummer oder Gruppenwahlnummer
Anlegen von Kurzwahlnummern:
Wählen Sie Speed Dial (Kurzwahl), geben Sie Name/Group Name (Name/Gruppenname), Index Word (Indexwort), Fax Number (Faxnummer) und Fax Speed (Faxgeschwindigkeit) ein und klicken Sie dann auf OK. Die angelegte Kurzwahlnummer wird der Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste) hinzugefügt.
Hinweis:
  • Wenn in Ihrem Telefonsystem eine Vorwahl wie 0 oder 9 für eine Amtsleitung gewählt werden muss, geben Sie die Vorwahl vor der Faxnummer ein. Wenn Ihr Drucker zur Verwendung von # (Raute) statt einer bestimmten Vorwahl eingerichtet ist, geben Sie # (Raute) ein.
  • Die für die einzelnen Kurzwahleinträge festgelegte Faxgeschwindigkeit wird nur verwendet, wenn ein Fax direkt von einem Drucker gesendet wird. FAX Utility ignoriert die Einstellung und verwendet die auf dem Drucker festgelegte Faxgeschwindigkeit.
Anlegen von Gruppenwahlnummern:
Wählen Sie Group Dial (Gruppenwahl), geben Sie Name/Group Name (Name/Gruppenname) und Index Word (Indexwort) ein, aktivieren Sie das Kontrollkästchen der Kurzwahlnummer, die Sie zu der Gruppenwahl hinzufügen möchten, und klicken Sie dann auf OK. Die angelegte Gruppenwahlnummer wird der Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste) hinzugefügt.
Hinweis:
Index Word (Indexwort) und Fax Speed (Faxgeschwindigkeit) sind abhängig vom Drucker möglicherweise nicht verfügbar.
Wenn Sie alle nötigen Einstellungen vorgenommen haben, klicken Sie auf .
Die in der Liste angezeigten Kurzwahlnummern und Gruppenwahlnummern werden im Drucker gespeichert.
