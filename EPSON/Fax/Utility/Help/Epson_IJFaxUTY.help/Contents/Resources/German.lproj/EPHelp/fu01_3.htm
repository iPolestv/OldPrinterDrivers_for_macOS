Starten von FAX Utility

Für Mac OS X 10.6.x oder später
Klicken Sie auf Apple - System Preferences (Systemeinstellungen) - Print & Scan (Drucken & Scannen) oder Print & Fax (Drucken & Faxen).
Wählen Sie unter Printer (Drucker) FAX XXXXXX (Ihr Drucker) und klicken Sie auf Options & Supplies (Optionen & Zubehör) - Utility (Dienstprogramm) - Open Printer Utility (Druckerdienstprogramm öffnen).
Mac OS X 10.5.x
Klicken Sie auf Apple - System Preferences (Systemeinstellungen) - Print & Fax (Drucken & Faxen).
Wählen Sie unter "Printer" (Drucker) die Option "FAX XXXXXX" (Ihren Drucker) und klicken Sie auf Open Print Queue (Drucker-Warteliste öffnen) - Utility (Dienstprogramm).
Mac OS X 10.4.x
Doppelklicken Sie auf "Hard disk (Festplatte)" - Applications (Programme) - Utilities (Dienstprogramme) - Printer Setup Utility (Dienstprogramm zur Druckerkonfiguration).
Wählen Sie unter "Printer" (Drucker) die Option "FAX XXXXXX" (Ihren Drucker) und klicken Sie anschließend auf Utility (Dienstprogramm).
