> Anfang > Faxübertragung

Übertragen eines mit einem Textverarbeitungs- oder Tabellenkalkulationsprogramm erstellten Dokumentes

Für Mac OS X 10.5.x oder später
Mac OS X 10.4.x
Durch Auswählen eines Druckerfaxgerätes im Menü Print (Drucken) in einigen Anwendungen können Sie Daten direkt übertragen, wie z. B. gerade erstellte Dokumente, Zeichnungen und Tabellen.
Hinweis:
In der folgenden Beschreibung dient als Beispiel das Mac OS X-Standardprogramm TextEdit. Die tatsächliche Verwendung kann je nach der von Ihnen verwendeten Anwendung davon abweichen. Weitere Informationen finden Sie in der Hilfe der Anwendung.

Für Mac OS X 10.5.x oder später

Erstellen Sie mithilfe einer Anwendung ein Dokument zur Faxübertragung.
Klicken Sie auf Print (Drucken) im Menü File (Ablage) der Anwendung.
Das Dialogfeld Print (Drucken) der Anwendung wird angezeigt.
Wählen Sie FAX XXXXXX (Ihr Drucker) unter Printer (Drucker) und klicken Sie anschließend auf , um die Detaileinstellungen anzuzeigen.
Ändern Sie die entsprechenden Einstellungen.
    • Geben Sie 1 in Copies (Kopien) an. Auch wenn Sie 2 oder mehr eingeben, wird nur ein Exemplar übertragen.
    • Abhängig vom Drucker (Fax) ist das Faxen in Farbe bei Papierformaten, die größer als A4, Letter oder Legal sind, eventuell nicht möglich.
    • In einer Faxübertragung können bis zu 100 Seiten gesendet werden.
Wählen Sie Fax Settings (Faxeinstellungen) und ändern Sie die entsprechenden Einstellungen.
Eine Erläuterung zu jeder Einstellung finden Sie in der Hilfe für den PC-FAX-Treiber. Die Treiberhilfe können Sie durch Klicken auf ? unten links im Bildschirm nach der Auswahl von Fax Settings (Faxeinstellungen) aufrufen.
Wählen Sie Recipient Settings (Empfängereinstellungen) und geben Sie einen Empfänger ein.
Direkteingabe eines Empfängers (Name, Faxnummer usw.):
Geben Sie die erforderlichen Informationen im Abschnitt Add (Hinzufügen) ein und klicken Sie anschließend auf +. Der Empfänger wird zu der im oberen Teil des Dialogfeldes angezeigten Recipient List (Empfängerliste) hinzugefügt.
Hinweis:
Wenn Sie Enter fax number twice (Geben Sie die Faxnummer zweimal ein) in den PC-FAX-Treibereinstellungen ausgewählt haben, müssen Sie die gleiche Nummer nochmals eingeben.
Wenn für die verbundene Faxleitung eine Amtsholung erforderlich ist, geben Sie die Vorwahl im Feld External Access Prefix (Vorwahl für externen Zugriff) ein.
Hinweis:
Wenn Ihr Drucker über eine Sammelanschlusseinstellung verfügt und zur Verwendung von # (Raute) statt einer bestimmten Vorwahl eingerichtet ist, geben Sie # (Raute) ein.
Auswahl eines Empfängers (Name, Faxnummer usw.) aus Address book (Adressbuch) oder Contacts (Kontakte):
Wenn der Empfänger im Adressbuch gespeichert ist, klicken Sie auf . Wählen Sie den Empfänger aus der Liste, klicken Sie auf Add (Hinzufügen) und anschließend auf OK.
Wenn für die verbundene Faxleitung eine Amtsholung erforderlich ist, geben Sie die Vorwahl im Feld External Access Prefix (Vorwahl für externen Zugriff) ein.
Hinweis:
Wenn Ihr Drucker über eine Sammelanschlusseinstellung verfügt und zur Verwendung von # (Raute) statt einer bestimmten Vorwahl eingerichtet ist, geben Sie # (Raute) ein.
Prüfen Sie die Empfängereinstellungen und klicken Sie auf Fax.
Klicken Sie auf Fax, um die Übertragung zu starten. Vergewissern Sie sich vor dem Absenden, dass Name und Faxnummer des Empfängers korrekt sind.
Hinweis:
  • Klicken Sie auf das Druckersymbol im Dock, um den Übertragungsstatus anzuzeigen. Um die Übertragung abzubrechen, markieren Sie die Daten, die gerade gesendet werden, und klicken Sie dann auf Delete (Löschen).
  • Wenn während der Übertragung ein Fehler auftritt, wird eine Fehlermeldung angezeigt. Prüfen Sie die Übertragungsaufzeichnung im Fenster Fax Transmission Record (Faxübertragungsaufzeichnung).
    Prüfen / Löschen der Übertragungsaufzeichnungen
    Übertragungsfehlerliste
  • Dokumente mit mehreren unterschiedlichen Papierformaten werden möglicherweise nicht korrekt übertragen.
