FAX Utility (Fenster Main (Haupteinstellungen))

In diesem Fenster wählen Sie die Funktion von FAX Utility aus, die Sie verwenden möchten.
Hinweis:
Das Fenster wird beim Start von FAX Utility angezeigt.
Starten von FAX Utility
Funktion
Erklärung
Fax Transmission Record (Faxübertragungsaufzeichnung)
Zeigt die Übertragungsaufzeichnungen an.
Fax Transmission Record (Faxübertragungsaufzeichnung)
Fax Receive Monitor (Monitor zum Faxempfang)
Dies wird nur angezeigt, wenn ein Drucker (Fax) ausgewählt ist, der PC-FAX-Empfang unterstützt. Ermöglicht Ihnen auszuwählen, ob empfangene Faxe auf einem Computer gespeichert werden.
Check Fax Connection (Faxverbindung prüfen)
Zur Überprüfung der einwandfreien Faxfunktion des angeschlossenen Druckers.
Fax Settings (Faxeinstellungen)
Zur Einstellung der Fax-Kopfzeile und der verschiedenen, im Drucker gespeicherten Faxkommunikationseinstellungen.
Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste)
Zur Speicherung oder Bearbeitung der im Drucker gespeicherten Kurz- oder Gruppenwahlnummern.
"Address Book (Adressbuch) synchronisieren" oder "Contacts (Kontakte) synchronisieren"
Synchronisiert Daten für Address Book (Adressbuch) oder Contacts (Kontakte) mit FAX Utility, sodass Sie Empfänger aus der Kontaktliste wählen können, wenn Sie ein Fax mit FAX Utility senden. Wenn Sie Address Book (Adressbuch) oder Contacts (Kontakte) nach dem Synchronisieren aktualisieren, müssen Sie diese Synchronisierungsfunktion erneut ausführen, um die aktuellen Daten in FAX Utility zu verwenden.
