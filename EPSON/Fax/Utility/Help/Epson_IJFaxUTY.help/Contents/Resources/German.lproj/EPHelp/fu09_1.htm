Einstellung zum Speichern eines empfangenen Faxes auf einem Computer

Sie können die Ausgabeeinstellungen für empfangene Faxe ändern, um empfangene Faxe als PDF-Dateien auf einem Computer oder in einem freigegebenen Ordner auf einem Server zu speichern (PC-FAX-Empfang).
Hinweis:
  • Wenn Ihr Drucker den PC-FAX-Empfang unterstützt, wird im FAX Utility-Hauptfenster Fax Receive Monitor (Monitor zum Faxempfang) angezeigt.
  • Wenn Sie den PC-FAX-Empfang verwenden, aktivieren Sie „Auto Answer“ (Auto-Antwort).
  • Abhängig vom Drucker werden empfangene Faxdaten aus dem Speicher des Druckers gelöscht, sobald sie auf dem Computer gespeichert sind.
  • Da empfangene Faxe als PDF-Dateien gespeichert werden, benötigen Sie Cover Flow (10.5 oder höher), Quick Look (10.5 oder höher) oder Preview.app, welche gemeinsam mit Mac OS X vorinstalliert werden.
Starten Sie FAX Utility.
Starten von FAX Utility
Klicken Sie auf Fax Receive Monitor (Monitor zum Faxempfang).
Hinweis:
  • Sie können in einer Umgebung mit mehreren Benutzern nicht mehrere Instanzen von Fax Receive Monitor (Monitor zum Faxempfang) öffnen.
  • Ein empfangenes Fax können Sie nur dann importieren und speichern, wenn Fax Receive Monitor (Monitor zum Faxempfang) ausgeführt wird.
  • Wenn Fax Receive Monitor (Monitor zum Faxempfang) ausgeführt wird, wird im Dock ein Symbol angezeigt. Wenn Sie dem Dock Fax Receive Monitor (Monitor zum Faxempfang) hinzufügen, können Sie Fax Receive Monitor (Monitor zum Faxempfang) jederzeit einfach starten. Klicken Sie zum Hinzufügen von Fax Receive Monitor (Monitor zum Faxempfang) zum Dock auf das Symbol für Fax Receive Monitor (Monitor zum Faxempfang), halten Sie zugleich die Control-Taste gedrückt und klicken Sie auf Add to Dock (Zum Dock hinzufügen) im Menü.
Wählen Sie Ihren Drucker und dann die Receive Settings (Empfangseinstellungen).
Hinweis:
  • Faxe können nur in einer Druck- bzw. Faxwarteliste gleichzeitig gespeichert werden. Wenn es auf demselben Drucker mehrere Druck- bzw. Faxwartelisten gibt, wie etwa "FAX XXXXXX (USB)" für USB-Verbindungen und "FAX XXXXXX(IP)" für Netzwerkverbindungen, müssen Sie deshalb wählen, welche der Wartelisten Sie zum Speichern empfangener Faxe verwenden möchten.
  • Wenn Ihr Drucker über eine Sicherheitsfunktion verfügt, mit der Administratoren die Änderung der Faxeinstellungen des Druckers einschränken können, müssen Sie möglicherweise zum Fortsetzen ein Administratorkennwort eingeben.
Wählen Sie "Save" (save faxes on this computer) ("Speichern" (Faxe auf diesem Computer speichern)) und klicken Sie dann auf Save to (Speichern in), um einen Ordner zum Speichern empfangener Faxe festzulegen.
Hinweis:
  • Standardmäßig ist der folgende Ordner angegeben.
    \Benutzer\Anmeldename\PCFax\Druckername
  • Wenn der Drucker innerhalb eines Netzwerks freigegeben ist, können Sie die Einstellungen von jedem Computer innerhalb dieses Netzwerks aus ändern. Faxe können jedoch immer nur unter einem Account gleichzeitig gespeichert werden. Wenn "Save" (save faxes on this computer) ("Speichern" (Faxe auf diesem Computer speichern)) eingestellt ist und Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) auf einem anderen Computer ausgewählt ist, wird im Fenster Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) die Meldung "Save faxes on another computer or print queue (Faxe auf einem anderen Computer oder in einer anderen Drucker-Warteliste speichern)" angezeigt.
  • In dem Fenster können Sie die Einstellung "Save" (save faxes on this computer) ("Speichern" (Faxe auf diesem Computer speichern)) wählen, vorausgesetzt es sind keine Faxe mehr auf dem Drucker vorhanden. Wenn Faxe auf dem Drucker vorhanden sind, wird eine Meldung angezeigt, dass Sie die Einstellung nicht ändern können.
    Bei gemeinsamer Nutzung des Druckers (Fax) empfehlen wir die Erstellung eines Ordners im Netzwerk zum Speichern von Faxen.
  • Sie können die Einstellungen von Fax Receive Monitor (Monitor zum Faxempfang) ändern, wenn Sie unter Fax Receive Monitor (Monitor zum Faxempfang) auf Preferences (Voreinstellungen) klicken.
    Preferences (Voreinstellungen) (Nur für Modelle mit PC-FAX-Empfang)
Klicken Sie auf Send (Senden), um die Einstellungen an den Drucker zu senden.
Note:
Wenn für den PC-FAX-Empfang eine Benutzerauthentifizierung (Name und Kennwort zur Dateifreigabe) im Drucker festgelegt ist, müssen Sie diese Informationen im Schlüsselbund des Computers registrieren.
Wenn die Informationen noch nicht im Schlüsselbund registriert sind, wird der Registrierungsbildschirm angezeigt. Geben Sie Name und Kennwort ein, aktivieren Sie die Option Remember this password in my keychain (Passwort im Schlüsselbund sichern) und klicken Sie auf Connect (Verbinden).
Wenn Sie die Authentifizierungsinformationen nicht im Schlüsselbund sichern, müssen Sie sie jedes Mal eingeben, wenn Sie ein erhaltenes Fax auf dem Computern speichern.
