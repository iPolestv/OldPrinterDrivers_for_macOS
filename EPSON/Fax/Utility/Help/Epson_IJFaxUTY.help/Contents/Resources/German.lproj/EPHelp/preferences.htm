Preferences (Voreinstellungen) (Nur für Modelle mit PC-FAX-Empfang)

In diesem Fenster können Sie die Einstellungen für den Faxempfang konfigurieren, wie etwa wie Benachrichtigungen bei Eingang eines Faxes erfolgen.
Hinweis:
  • Dieses Fenster wird angezeigt, wenn Sie auf Fax Receive Monitor Preferences (Einstellungen) im Fax Receive Monitor (Monitor zum Faxempfang) klicken.
  • Dies Einstellungen werden auf alle Drucker angewendet und die Wartelisten zum Drucken werden im Fax Receive Monitor (Monitor zum Faxempfang) angezeigt.
Funktion
Erklärung
Specify an interval to check for new faxes (Intervall für Überprüfung auf neue Faxe festlegen)
Legen Sie hier den Zeitabstand (zwischen 1 und 30 Minuten) fest, in dem geprüft werden soll, ob neue Faxe eingegangen sind.
Notify me of new faxes via a dock icon (Benachrichtigung per Dock-Symbol bei neuen Faxen)
Aktivieren Sie diese Option, damit das Symbol für Fax Receive Monitor (Monitor zum Faxempfang) im Dock "hüpft", wenn ein neues Fax eingeht.
Notify me of errors via a dock icon (Benachrichtigung per Dock-Symbol bei Fehlern)
Aktivieren Sie diese Option, damit das Symbol für Fax Receive Monitor (Monitor zum Faxempfang) im Dock "hüpft", wenn ein Fehler auftritt. Klicken Sie auf das Symbol, um das Fenster des Monitors für den Faxempfang anzuzeigen und darin den Fehler zu überprüfen.
Start at login (Start bei Anmeldung)
Für diese Einstellung wird der Wert Automatic (Automatisch) empfohlen. Wenn "Save" (save faxes on this computer) ("Speichern" (Faxe auf diesem Computer speichern)) in Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) eingestellt ist, wird Fax Receive Monitor (Monitor zum Faxempfang) automatisch bei der Anmeldung gestartet. Wenn Sie On (Ein) wählen, wird Fax Receive Monitor (Monitor zum Faxempfang) bei jeder Anmeldung gestartet, unabhängig von den Ausgabeeinstellungen. Wenn Sie Off (Aus) wählen, wird Fax Receive Monitor (Monitor zum Faxempfang) bei Anmeldung nicht gestartet.
Wait time before checking new faxes (Wartezeit vor Überprüfung auf neue Faxe)
Diese Funktion ist verfügbar, wenn für Start at login (Start bei Anmeldung) die Optionen Automatic (Automatisch) oder On (Ein) eingestellt sind. Geben Sie an, wie lange (0 bis 10 Minuten) nach dem durch Ihre Anmeldung ausgelösten Start von Fax Receive Monitor (Monitor zum Faxempfang) gewartet werden soll, bis geprüft wird, ob neue Faxe eingegangen sind. Dies ist sinnvoll, wenn der Ordner, in dem die Faxe gespeichert werden, sich in einem Netzwerk befindet und nach der Anmeldung für das Herstellen der Netzwerkverbindung eine gewisse Zeit benötigt wird.
Hinweis:
Wenn der Ordner, in dem empfangene Faxe gespeichert werden, ein freigegebener Ordner in einem Netzwerk ist, können Sie bei Ihrer Anmeldung eine Verbindung zu diesem Ordner herstellen, indem Sie ihn in der Liste der Anmeldeelemente registrieren. Klicken Sie auf Apple - System Preferences (Systemeinstellungen) - Account - Login to register (Zum Registrieren anmelden).
