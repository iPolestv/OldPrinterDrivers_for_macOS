Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste) - Edit Speed Dial/Group Dial (Kurzwahl/Gruppenwahl bearbeiten) -

In diesem Fenster können Sie die Kurzwahlnummern oder Gruppenwahlnummern des Faxgerätes für den Drucker anlegen oder bearbeiten.
Funktion
Erklärung
Entry Number (Eintragsnummer)
Zeigt die Eintragsnummer der Kurzwahlnummer an.
Speed Dial (Kurzwahl)
Zum Anlegen oder Bearbeiten eines Kurzwahleintrages auswählen.
Group Dial (Gruppenwahl)
Zum Anlegen oder Bearbeiten eines Gruppenwahleintrags auswählen. Eine Gruppenwahl kann nur angelegt werden, wenn Kurzwahleinträge angelegt sind.
Name/Group Name (Name/Gruppenname)
Geben Sie eine Kurzwahlnummer oder einen Gruppenwahlnamen ein, den Sie anlegen möchten.
Index Word (Indexwort) *3
Geben Sie ein Indexwort für den Empfänger ein, das Sie beim Suchen verwenden können.
Fax Number (Faxnummer) *1
Zur Eingabe der Faxnummer des Empfängers. Wenn Sie Nummern trennen möchten, wie z. B. Vorwahlnummern von persönlichen Nummern, verwenden Sie ein Leerzeichen anstelle eines Bindestrichs (-), da ein Bindestrich (-) beim Wählen als Pause interpretiert wird. Dieses Symbol wird angezeigt, wenn Sie Kurzwahlnummern bearbeiten.
Member (Mitglied) *2
Zeigt eine Liste der gespeicherten Kurzwahlnummern an. Wählen Sie die Kurzwahlnummer aus, die Sie zur Gruppenwahlnummer hinzufügen möchten und klicken Sie auf OK. Dieses Symbol wird angezeigt, wenn Sie Gruppenwahlnummern bearbeiten.
Fax Speed (Faxgeschwindigkeit) *3
Wählen Sie die Faxgeschwindigkeit zum Senden eines Faxes an den Empfänger. Die für die einzelnen Kurzwahleinträge festgelegte Faxgeschwindigkeit wird nur verwendet, wenn ein Fax direkt von einem Drucker gesendet wird. FAX Utility ignoriert die Einstellung und verwendet die auf dem Drucker festgelegte Faxgeschwindigkeit.
OK
Speichert die Einstellungen und schließt das Fenster.
Cancel (Abbrechen)
Bricht das Speichern oder die Bearbeitung ab und schließt das Fenster.
*1 Benötigter Parameter für das Speichern der Kurzwahlnummer.
*2 Achten Sie beim Speichern einer Gruppenwahlnummer darauf, dass stets eine oder mehrere Kurzwahlnummern ausgewählt sind.
*3 Diese Funktionen werden abhängig vom Drucker möglicherweise nicht angezeigt.
