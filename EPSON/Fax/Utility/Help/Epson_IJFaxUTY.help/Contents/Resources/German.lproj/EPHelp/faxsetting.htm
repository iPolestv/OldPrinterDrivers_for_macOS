Fax Transmission Record (Faxübertragungsaufzeichnung)

Menü
In diesem Fenster kann das Faxübertragungsprotokoll geprüft werden.
Hinweis:
  • Dieses Fenster wird angezeigt, wenn Sie auf Fax Transmission Record (Faxübertragungsaufzeichnung) im Haupteinstellungsfenster von FAX Utility klicken.
  • FAX Utility speichert nur Übertragungsprotokolle. Da Übertragungsdaten nicht gespeichert werden, können Sie die Daten, die Sie gesendet haben, auch nicht anzeigen.

Menü

Menü File (Datei)

Funktion

Save (Speichern)
Speichert die Übertragungsaufzeichnungen. Die gespeicherten Daten können dazu verwendet werden, bei der Faxübertragung aufgetretene Probleme zu analysieren.
