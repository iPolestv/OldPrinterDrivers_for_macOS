Fehlermeldungen zum Faxempfang (nur für Modelle mit PC-FAX-Empfang)

Die folgende Liste enthält Fehlermeldungen, die beim Empfang eines Faxes möglicherweise angezeigt werden, und Vorschläge zum Beheben dieser Fehler.
Meldung
Maßnahme
A printer communication error occurred. (Es ist ein Druckerkommunikationsfehler aufgetreten.)
Drucker aus- und wieder einschalten. Warten Sie ein paar Minuten und versuchen Sie es dann erneut.
The printer (fax driver) you are setting has been deleted. Add the printer (fax driver) in [System Preferences] - [Print & Scan] (Der von Ihnen festgelegte Drucker (Faxtreiber) wurde gelöscht. Fügen Sie den Drucker (Faxtreiber) in [Systemeinstellungen] - [Drucken & Scannen] hinzu) or [Print & Fax] ([Drucken & Faxen]) (Der von Ihnen festgelegte Drucker (Faxtreiber) wurde gelöscht. Fügen Sie den Drucker (Faxtreiber) in [Systemeinstellungen] - [Drucken & Scannen] oder [Drucken & Faxen] hinzu.)
Fügen Sie den PC-FAX-Treiber zur Druckerliste hinzu. Klicken Sie im Meldungsdialogfeld auf OK. Das Dialogfeld zum Hinzufügen des Druckers wird angezeigt.
Change not accepted. Some printer settings have been changed by another computer, print queue, or the printer. Please try again. (Änderung nicht akzeptiert. Einige Druckereinstellungen wurden von einem anderen Computer, einer anderen Drucker-Warteliste oder dem Drucker geändert. Bitte versuchen Sie es erneut.)
Die Einstellung unter Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) wurde von einem anderen Computer im Netzwerk, von der Drucker-Warteliste oder vom Drucker aus geändert. Warten Sie ein paar Minuten und versuchen Sie es dann erneut.
Empfangene Faxe können nur auf einem Computer gespeichert werden.
Received faxes not saved due to an error. (Empfangene Faxe aufgrund eines Fehlers nicht gedruckt.)
Das Speichern des empfangenen Fax im angegebenen Ordner ist fehlgeschlagen. Überprüfen Sie die Verbindung zum angegebenen Ordner und stellen Sie sicher, dass er verfügbar ist und dass Sie über die erforderlichen Zugriffsrechte für diesen Ordner verfügen.
Processing canceled because an error occurred. (Verarbeitung abgebrochen, weil ein Fehler aufgetreten ist.)
Wenn das Problem nicht durch den Neustart Ihres Druckers und Computers gelöst werden kann, wenden Sie sich an den Epson-Kundendienst. Die Kontaktinformationen finden Sie im Handbuch Ihres Druckers.
Cannot save faxes. User authentication failed. (Faxe können nicht gespeichert werden. Benutzerauthentifizierung fehlgeschlagen.)
Empfangene Faxe können nicht gespeichert werden, da die Benutzerauthentifizierung (Name und Kennwort zur Dateifreigabe) für den PC-FAX-Empfang nicht korrekt im Schlüsselbund des Computers festgelegt ist.
Nehmen Sie die Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) erneut vor und geben Sie die korrekten Authentifizierungsinformationen ein, wenn der Eingabebildschirm angezeigt wird. Aktivieren Sie die Option Remember this password in my keychain (Passwort im Schlüsselbund sichern).
