Wiederherstellen einer gesicherten Kurzwahlliste

Sie können gesicherte Drucker-Telefonbuchdaten (Kurzwahl-/Gruppenwahllisten) auf dem Drucker wiederherstellen.
Hinweis:
  • In Windows-Fax Utility gesicherte Daten können nicht wiederhergestellt werden.
  • Wenn die gesicherte Kurzwahl-/Gruppenwahlliste Elemente enthält, die nicht von dem Drucker, auf dem Sie die Liste wiederherstellen möchten, unterstützt werden, werden diese Elemente nicht wiederhergestellt.
  • Wenn die gesicherte Kurzwahl-/Gruppenwahlliste keine Elemente enthält, die von dem Drucker, auf dem Sie die Daten wiederherstellen möchten, unterstützt werden, führt eine Wiederherstellung zu folgendem Ergebnis.
    Index Word (Indexwort): leer
    Fax Speed (Faxgeschwindigkeit): voreingestellter Wert
  • Alle Werte, wie beispielsweise Name/Group Name (Name/Gruppenname), mit Zeichen, die von dem Drucker, auf dem Sie die Daten wiederherstellen möchten, nicht unterstützt werden, sind nach der Wiederherstellung leer.
  • Alle durch In use by another function (Wird von einer anderen Funktion verwendet) in FAX Utility gekennzeichneten Einträge werden mit den wiederhergestellten Daten überschrieben, wenn die Eintragsnummer übereinstimmt.
  • Alle Einträge, die gerade verwendet werden und geschützt sind, werden nicht mit den wiederhergestellten Daten überschrieben.
Starten Sie FAX Utility.
Starten von FAX Utility
Wählen Sie FAX XXXXXX (Ihren Drucker) unter Printer (Drucker) und klicken Sie anschließend auf Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste).
Hinweis:
Wenn Ihr Drucker über eine Sicherheitsfunktion verfügt, mit der Administratoren die Änderung der Faxeinstellungen des Druckers einschränken können, müssen Sie möglicherweise zum Fortsetzen ein Administratorkennwort eingeben.
Klicken Sie auf File (Datei) - Back up (Sicherung) - Restore Backup (Sicherung wiederherstellen).
Wählen Sie die wiederherzustellende Datei.
Die wiederhergestellte Liste wird im Fenster Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste) angezeigt.
Klicken Sie im Fenster auf .
Die wiederhergestellten Telefonbuch-Daten werden auf dem Drucker gespeichert.
