Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste)

Menü
Symbolleiste
Mithilfe der Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste) können Sie Kurzwahlnummern und Gruppenwahlnummern des Faxgerätes für den Drucker anlegen oder bearbeiten.
Hinweis:
  • Dieses Fenster wird angezeigt, wenn Sie auf Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste) im Haupteinstellungsfenster von FAX Utility klicken.
  • Die maximale Anzahl der für den angeschlossenen Drucker (Fax) verfügbaren Einträge wird in der Leiste unten im Fenster angezeigt.

Menü

Menü File (Datei)

Funktion
Erklärung
Get Speed Dials (Kurzwahlnummern lesen)
Liest die Kurzwahlnummern oder Gruppenwahlnummern des Faxgerätes für Ihren Drucker.
Send Speed Dials (Kurzwahlnummern schreiben)
Speichert die Kurzwahlnummern und Gruppenwahlnummern in der Liste im Drucker.
Import (Importieren)
From Address Book (Von Adressbuch)
Importiert Daten aus dem Address Book (Adressbuch) in die Kurzwahlnummern des Druckers.
Back up (Sicherung)
Save Backup (Sicherung speichern)
Speichert eine Sicherungskopie der gespeicherten Kurzwahlnummern und Gruppenwahlnummern.
Restore Backup (Sicherung wiederherstellen)
Liest die gesicherten Kurzwahlnummern oder Gruppenwahlnummern.
Close (Schließen)
Schließt das Fenster.

Menü Edit (Bearbeiten)

Funktion
Erklärung
Edit (Bearbeiten)
Zum Bearbeiten der in der Liste ausgewählten Kurzwahlnummern oder Gruppenwahlnummern.
Delete (Löschen)
Zum Löschen der in der Liste ausgewählten Kurzwahlnummern oder Gruppenwahlnummern.
Delete All (Alles löschen)
Zum Löschen aller Kurzwahl-/Gruppenwahlnummern in der Liste.
