Fax Receive Monitor (Monitor zum Faxempfang) (Nur für Modelle mit PC-FAX-Empfang)

Symbolleiste
Symbol und zugehörige Meldung in der Druckerliste
In diesem Fenster können Sie den Faxempfangsstatus prüfen und Ausgabeeinstellungen für empfangene Faxe vornehmen.
Hinweis:
Dieses Fenster wird angezeigt, wenn Sie auf Fax Receive Monitor (Monitor zum Faxempfang) im Haupteinstellungsfenster von FAX Utility klicken.

Symbolleiste

Funktion
Erklärung
Open the received fax folder (Ordner für empfangene Faxe öffnen)
Öffnet den Ordner, in dem die empfangenen Faxe gespeichert werden.
Check new faxes (Neue Faxe abrufen)
Ruft neue Faxe ab.
Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe)
Ermöglicht Ihnen auszuwählen, ob empfangene Faxe auf einem Computer gespeichert werden.
Preferences (Voreinstellungen)
Konfiguriert die Einstellungen für den Monitor zum Faxempfang.
Oben 

Symbol und zugehörige Meldung in der Druckerliste

Symbol
Meldung
Erklärung
Waiting (Warten) / Checking for new faxes (Es wird nach neuen Faxen gesucht)
Neue Faxe werden abgerufen oder der Drucker ist im Wartestatus.
Received faxes saved successfully in the specified folder (Empfangene Faxe wurden erfolgreich im angegebenen Ordner gespeichert).
Neue Faxe wurden von einem Drucker (Fax) empfangen und im angegebenen Ordner gespeichert.
The printer is not responding (Der Drucker antwortet nicht)
Überprüfen Sie die Verbindung zwischen Ihrem Computer und Drucker und vergewissern Sie sich, dass der Drucker eingeschaltet ist.
The printer is processing data (Der Drucker verarbeitet Daten)
Der Drucker verarbeitet andere Daten. Bitte warten Sie.
Cannot find the received fax folder (Der Ordner für empfangene Faxe wurde nicht gefunden).
Prüfen Sie, ob der Name des angegebenen Ordners korrekt ist. Wenn Sie einen freigegebenen Ordner im Netzwerk verwenden, stellen Sie sicher, dass Sie über die erforderlichen Zugriffsrechte für diesen Ordner verfügen.
Failed to save faxes in the specified folder (Speichern von Faxen im angegebenen Ordner fehlgeschlagen).
Stellen Sie sicher, dass der angegebene Ordner verfügbar ist und Sie über die erforderlichen Zugriffsrechte für diesen Ordner verfügen.
A printer communication error occurred (Es ist ein Druckerkommunikationsfehler aufgetreten)
Überprüfen Sie die Verbindung zwischen Ihrem Computer und Drucker und vergewissern Sie sich, dass der Drucker eingeschaltet ist. Wenn Sie eine Netzwerkverbindung verwenden, stellen Sie sicher, dass Sie über das Netzwerk auf den Drucker zugreifen können.
The "FAXRECV" disk is being used by another application. (Die Festplatte "FAXRECV" wird von einer anderen Anwendung verwendet.)
Schließen Sie das Programm, das die Festplatte "FAXRECV" verwendet.
Cannot save faxes. User authentication failed. (Faxe können nicht gespeichert werden. Benutzerauthentifizierung fehlgeschlagen.)
Überprüfen Sie die im Drucker festgelegte Benutzerauthentifizierung (Name und Kennwort zur Dateifreigabe) für den PC-FAX-Empfang, nehmen Sie die Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) erneut vor und geben Sie die Authentifizierungsinformationen korrekt ein. Aktivieren Sie die Option Remember this password in my keychain (Passwort im Schlüsselbund sichern).
