Speichern einer Sicherungskopie der Kurzwahlnummern

Sie können eine Sicherungskopie der gespeicherten Kurzwahlnummern und Gruppenwahlnummern speichern. Die Daten in der Kurzwahl-/Gruppenwahlliste können aufgrund einer Fehlfunktion des Druckers verlorengehen. Es wird empfohlen, bei jeder Aktualisierung der Kurzwahl-/Gruppenwahlliste eine Sicherheitskopie zu erstellen. Epson kann für den Verlust, für die Sicherung oder Wiederherstellung von Daten und/oder Einstellungen in keiner Weise verantwortlich gemacht werden. Dies gilt auch gegebenenfalls für den Garantiezeitraum.
Hinweis:
Daten, die durch "In use by another function (Wird von einer anderen Funktion verwendet)" gekennzeichnet sind, werden nicht gespeichert.
Starten Sie FAX Utility.
Starten von FAX Utility
Wählen Sie FAX XXXXXX (Ihren Drucker) unter Printer (Drucker) und klicken Sie anschließend auf Speed Dial/Group Dial List (Kurzwahl-/Gruppenwahlliste).
Die im Drucker gespeicherten Kurzwahl-/Gruppenwahllisten werden angezeigt.
Hinweis:
Wenn Ihr Drucker über eine Sicherheitsfunktion verfügt, mit der Administratoren die Änderung der Faxeinstellungen des Druckers einschränken können, müssen Sie möglicherweise zum Fortsetzen ein Administratorkennwort eingeben.
Klicken Sie auf File (Datei) - Back up (Sicherung) - Save Backup (Sicherung speichern).
Geben Sie den Dateinamen und das Verzeichnis ein und klicken Sie anschließend auf Save (Speichern).
