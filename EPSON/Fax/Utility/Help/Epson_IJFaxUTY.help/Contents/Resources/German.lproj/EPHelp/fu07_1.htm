Übertragungsprobleme

Gehen Sie zur Fehlerbehebung bei Übertragungsproblemen wie folgt vor.
Problem
Maßnahme
FAX XXXXXX (Ihr Drucker) wird in der Druckerliste nicht angezeigt.
Stellen Sie sicher, dass Druckertreiber und PC-FAX-Treiber installiert sind, und dass der PC-FAX-Treiber zur Druckerliste hinzugefügt worden ist.
Die Schaltfläche Fax ist abgeblendet (deaktiviert).
Diese Schaltfläche ist nur verfügbar, wenn ein Empfänger zur Recipient List (Empfängerliste) hinzugefügt worden ist. Geben Sie mindestens einen Empfänger ein.
Der Drucker sendet keine Faxe.
  • Wenn No dial tone detected (Kein Wählton) angezeigt wird, wählen Sie Fax Settings (Faxeinstellungen) - Dial Tone Detection Settings (Einstellungen für Wähltonerkennung) und deaktivieren Sie das Kontrollkästchen Detect Dial Tone (Wählton erkennen).
  • Vergewissern Sie sich, dass Sie Your Phone Number (Ihre Telefonnummer) in den Header Information Settings (Einstellungen der Kopfzeilendaten) eingegeben haben. Das Faxgerät des Empfängers könnte so eingestellt sein, dass es alle Faxe, die nicht die Telefon-/Faxnummer des Senders enthalten, blockiert.
Der Status der übertragenen Daten verbleibt bei Waiting for printer response (Warten auf Druckerantwort).
Waiting for printer response (Warten auf Druckerantwort) wird angezeigt, wenn der Computer keine Verbindung zum Drucker (FAX) herstellen kann, und wird so lange angezeigt, bis die Kommunikation wieder möglich ist. Überprüfen Sie, ob
  • der Drucker eingeschaltet ist.
  • der Drucker korrekt mit dem Computer verbunden ist.
  • der Drucker ein Netzwerkdrucker ist und die Netzwerkadresse des Druckers korrekt eingegeben ist.
  • der Drucker normal drucken kann.
Wenn alle diese Punkte korrekt ausgeführt worden sind, schalten Sie den Drucker aus und wieder an.
Das Programm stürzt nach der Anzeige von „Cannot print other fax data because a transmission is being processed. (Andere Faxdaten können nicht gedruckt werden, weil eine Übertragung gerade verarbeitet wird.)“ ab.
Einige Anwendungen hängen möglicherweise bei der Faxübertragung bei gleichzeitiger Anzeige einer Warnmeldung. (Das Problem ist möglicherweise nach einer Weile behoben.)

Empfangsprobleme (Nur für Modelle mit PC-FAX-Empfang)

Gehen Sie zur Fehlerbehebung bei Empfangsproblemen wie folgt vor:
Problem
Maßnahme
Das Symbol für Fax Receive Monitor (Monitor zum Faxempfang) wird im Haupteinstellungsfenster von FAX Utility nicht angezeigt.
Dies wird angezeigt, wenn Sie einen Drucker auswählen, der PC-FAX-Empfang unterstützt.
Schließen Sie einen Drucker (Fax) an, der diese Funktion unterstützt, schalten Sie den Drucker ein und wählen Sie ihn anschließend aus.
Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) ist so eingestellt, dass Faxe auf einem Computer empfangen werden sollen, es werden jedoch keine Faxe gespeichert.
Empfangene Faxe werden nur dann gespeichert, wenn Fax Receive Monitor (Monitor zum Faxempfang) ausgeführt wird. Es wird empfohlen, Fax Receive Monitor (Monitor zum Faxempfang) immer auszuführen.
Faxe können nicht empfangen werden.
  • Aktivieren Sie das Kontrollkästchen Auto Answer (Auto-Antwort) unter Fax Settings (Faxeinstellungen) - Auto Answer Settings (Auto-Antwort-Einstellungen). Wenn kein externes Telefon an den Drucker angeschlossen ist, kann der Drucker keine Faxe empfangen, wenn Auto Answer (Auto-Antwort) nicht eingestellt ist.
  • Stellen Sie sicher, dass der Drucker keinen Fehlerstatus aufweist.
Die Einstellungen zum Drucken empfangener Faxe können nicht geändert werden.
Sie können die Einstellungen erst dann ändern, wenn alle empfangene Faxe auf Ihrem Computer gespeichert worden sind. Auf einem Computer, der für den Faxempfang konfiguriert ist, können Sie die Einstellungen nach dem Überprüfen auf neue Faxe ändern. Sie können die Einstellungen auch über die Systemsteuerung des Druckers ändern.
"Cannot find the received fax folder. (Der Ordner für empfangene Faxe wurde nicht gefunden.)" wird angezeigt, wenn ein freigegebener Ordner im Netzwerk als Ordner für empfangene Faxe konfiguriert ist.
Zum Speichern von Faxen in einem freigegebenen Netzwerkordner müssen Sie zunächst eine Verbindung zum entsprechenden Netzwerk herstellen. Wählen Sie die Menüleiste auf dem Desktop Go (Gehe zu) - Connect to Server (Mit Server verbinden) und stellen Sie eine Verbindung zum freigegebenen Ordner im Netzwerk her.
Das Fenster Fax Receive Monitor (Monitor zum Faxempfang) wird nicht angezeigt, wenn Fax Receive Monitor (Monitor zum Faxempfang) bei Anmeldung automatisch gestartet wird.
Das Fenster Fax Receive Monitor (Monitor zum Faxempfang) wird nicht automatisch bei Anmeldung Ihres Computers angezeigt. Klicken Sie auf das Dock-Symbol für Fax Receive Monitor (Monitor zum Faxempfang), um das Fenster zu öffnen. Selbst wenn das Fenster Fax Receive Monitor (Monitor zum Faxempfang) nicht automatisch angezeigt wird, führt es im Hintergrund Vorgänge wie die Überprüfung auf neue Faxe aus.
Der Eingabebildschirm zur Benutzerauthentifizierung wird immer angezeigt, wenn Sie ein Fax erhalten.
Wenn bei Verwendung eines mit dem Netzwerk verbundenen Druckers für den PC-FAX-Empfang eine Benutzerauthentifizierung (Name und Kennwort zur Dateifreigabe) im Drucker festgelegt ist, wird dieser Bildschirm angezeigt, wenn Sie das empfangene Fax auf dem Computer speichern. Dies kommt auch dann vor, wenn die im Drucker festgelegten Authentifizierungsinformationen nicht mit den Informationen im Schlüsselbund übereinstimmen.
Nehmen Sie die Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) erneut vor und geben Sie die korrekten Authentifizierungsinformationen ein, wenn der Eingabebildschirm angezeigt wird. Aktivieren Sie die Option Remember this password in my keychain (Passwort im Schlüsselbund sichern).

Fehlermeldungen

Wenn eine Fehlermeldung angezeigt wird, lösen Sie das Problem mithilfe der folgenden Anweisungen zur Fehlerbehebung.
Meldung
Maßnahme
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (Drucker/Faxinformationen nicht abrufbar. Unter ;"Drucker- & Faxvoreinstellungen" den PC-FAX-Treiber hinzufügen.)
Fügen Sie den PC-FAX-Treiber zur Druckerliste hinzu.
Klicken Sie im Meldungsdialogfeld auf OK. Das Dialogfeld „Drucker hinzufügen“ wird angezeigt.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (Die Faxübertragungsaufzeichnung kann nicht geöffnet werden, da eine andere FAX Utility-Funktion bereits läuft. Bitte warten Sie, bis diese Funktion abgeschlossen ist, und versuchen Sie es dann erneut.)
Eine andere FAX Utility-Funktion wird ausgeführt. Beenden Sie die andere Funktion und versuchen Sie es erneut.
Cannot open the transmission record file. You have no access rights or the file does not exist. (Die Übertragungsaufzeichnungsdatei kann nicht geöffnet werden. Sie haben keine Zugriffsberechtigung oder die Datei existiert nicht.)
Prüfen Sie, ob Sie Lesezugriffsrechte auf die Übertragungsaufzeichnung haben.
Failed to save the log file. Change the path and file name, and save it again. (Protokolldatei konnte nicht gespeichert werden. Pfad- und Dateiname ändern und erneut speichern.)
Wählen Sie den Speicherort für die Protokolldatei aus.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (Protokolldatei konnte nicht gelöscht werden. Möglicherweise verwendet sie eine andere Anwendung bereits, haben Sie keine Zugriffsrechte oder ist sie bereits gelöscht worden.)
Wenn eine Protokolldatei verbleibt, prüfen Sie die Einstellung der Zugriffsrechte oder schließen Sie andere Anwendungen.
A printer communication error occurred. (Es ist ein Druckerkommunikationsfehler aufgetreten.)
Drucker aus- und wieder einschalten. Wenn dies nicht funktioniert, warten Sie ein paar Minuten und versuchen Sie es dann erneut.
An error was detected while communicating with the printer. (Während der Kommunikation mit dem Drucker wurde ein Fehler festgestellt.)
Warten Sie ein paar Minuten und versuchen Sie es dann erneut.
Failed to write the backup file. There is not enough free space or it may be write protected. (Fehler beim Schreiben der Sicherungsdatei. Es ist nicht genügend freier Speicherplatz vorhanden oder die Datei ist schreibgeschützt.)
Überprüfen Sie, wie viel Speicherplatz auf Ihrer Festplatte noch frei ist. Heben Sie alternativ den Schreibschutz der Sicherungskopie auf.
Failed to write the backup file. The file specified is being used by another application. (Sicherungsdatei konnte nicht geschrieben werden. Die angegebene Datei wird gerade von einer anderen Anwendung verwendet.)
Die Sicherungskopie wird von einem anderen Programm verwendet. Schließen Sie das andere Programm.
Cannot recognize the file format. The file may be corrupt or created by another application. (Dateiformat nicht erkannt. Die Datei ist möglicherweise defekt oder wurde von einer anderen Anwendung erstellt.)
Die für die Sicherungskopie verwendete Dateierweiterung ist ausschließlich *.fpb. Überprüfen Sie die Sicherungskopie.
Failed to read the backup file. The specified file is being used by another application. (Sicherungsdatei konnte nicht gelesen werden. Die angegebene Datei wird gerade von einer anderen Anwendung verwendet.)
Die Sicherungskopie wird von einem anderen Programm verwendet. Schließen Sie das andere Programm.
Processing canceled because an error occurred. (Verarbeitung abgebrochen, weil ein Fehler aufgetreten ist.)
Wenn das Problem nicht durch den Neustart Ihres Druckers und Computers gelöst werden kann, wenden Sie sich an den Epson-Kundendienst. Die Kontaktinformationen finden Sie im Handbuch Ihres Druckers.
