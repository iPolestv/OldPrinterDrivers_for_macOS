Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) (Nur für Modelle mit PC-FAX-Empfang)

In diesem Fenster können Sie einstellen, ob empfangene Faxe auf einem Computer gespeichert werden.
Hinweis:
Dieses Fenster wird angezeigt, wenn Sie auf Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) in Fax Receive Monitor (Monitor zum Faxempfang) klicken.
Funktion
Erklärung
Setting on the printer (Einstellung auf dem Drucker)
Zeigt die aktuelle Ausgabeeinstellung für empfangene Faxe an.
"Print" (Drucken) *
Druckt empfangene Faxe über den Drucker.
Not save faxes (Faxe nicht speichern) *
Wählen Sie diese Option, um Faxe nicht mehr auf einem Computer zu speichern. Empfangene Faxe werden abhängig von den Einstellungen des Druckers gedruckt oder im Drucker gespeichert.
"Save" (save faxes on this computer) ("Speichern" (Faxe auf diesem Computer speichern))
Speichert empfangene Faxe im angegebenen Ordner. Die Faxe werden als PDF-Dateien gespeichert.
Save to (Speichern in)
Legt den Ordner fest, in dem empfangene Faxe gespeichert werden.
Send the settings to the printer (Einstellungen an den Drucker senden)
Überträgt die Ausgabeeinstellungen für empfangene Faxe an den Drucker.
* Diese Funktionen werden abhängig vom Drucker möglicherweise nicht angezeigt.
Wenn für den PC-FAX-Empfang eine Benutzerauthentifizierung (Name und Kennwort zur Dateifreigabe) im Drucker festgelegt ist, müssen Sie diese Informationen im Schlüsselbund des Computers registrieren.
Wenn die Informationen noch nicht im Schlüsselbund registriert sind, wird der Registrierungsbildschirm angezeigt. Geben Sie Name und Kennwort ein, aktivieren Sie die Option Remember this password in my keychain (Passwort im Schlüsselbund sichern) und klicken Sie auf Connect (Verbinden).
Wenn Sie die Authentifizierungsinformationen nicht im Schlüsselbund sichern, müssen Sie sie jedes Mal eingeben, wenn Sie ein erhaltenes Fax auf dem Computern speichern.
