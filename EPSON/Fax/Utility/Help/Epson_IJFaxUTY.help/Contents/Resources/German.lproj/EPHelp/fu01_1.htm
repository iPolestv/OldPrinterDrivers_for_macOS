Produktübersicht

FAX Utility ist eine Anwendung, die verschiedene komfortable Funktionen zum Senden von Faxen bietet.
FAX Utility funktioniert nur mit Fax-fähigen Multifunktionsdruckern von Epson.
Wenn Sie einen Drucker verwenden, der den PC-FAX-Empfang unterstützt, können Sie empfangene Faxe als PDF-Datei auf Ihrem Computer speichern. So können Sie unnötiges Drucken vermeiden.
Hinweis:
Wenn Ihr Drucker den PC-FAX-Empfang unterstützt, wird im FAX Utility-Hauptfenster Fax Receive Monitor (Monitor zum Faxempfang) angezeigt.

Fax Receive Monitor (Monitor zum Faxempfang) (Nur für Modelle mit PC-FAX-Empfang)
Fax Receive Monitor (Monitor zum Faxempfang) wird angezeigt, wenn im FAX Utility-Hauptfenster ein Drucker mit PC-FAX-Empfang ausgewählt ist. Im Fenster Fax Receive Monitor (Monitor zum Faxempfang) können Sie Einstellungen zum Speichern von empfangenen Faxen auf dem Computer und zum Einstellen einer Benachrichtigung beim Eingang neuer Faxe vornehmen.

  • Sie können prüfen, ob der angeschlossene Drucker (Fax) sendebereit ist. Das Testergebnis wird als Bericht ausgegeben. Wird ein Fehler gefunden, beheben Sie diesen mithilfe der Informationen des Berichts.
    Überprüfen der Faxfunktion Ihres Druckers
  • Die im Drucker gespeicherten Faxeinstellungen, wie z. B. Auto Answer (Auto-Antwort) und Fax Header (Fax-Kopfzeile) können Sie konfigurieren oder ändern.
