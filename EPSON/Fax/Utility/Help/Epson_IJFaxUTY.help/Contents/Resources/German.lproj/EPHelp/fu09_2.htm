Abrufen neuer Faxe

Auch wenn das Abrufen neuer Faxe möglich ist, indem Sie den unter Received Fax Output Settings (Ausgabeeinstellungen für empfangene Faxe) angegebenen Ordner öffnen, wird hier das Abrufen neuer Faxe unter Verwendung von Fax Receive Monitor (Monitor zum Faxempfang) erläutert. Diese Funktion ist nur auf dem Computer verfügbar, auf dem die Option "Save" (save faxes on this computer) ("Speichern" (Faxe auf diesem Computer speichern)) aktiviert ist.
Hinweis:
Wenn die Option Notify me of new faxes via a dock icon (Benachrichtigung per Dock-Symbol bei neuen Faxen) in den Einstellungen für Fax Receive Monitor (Monitor zum Faxempfang) aktiviert ist, "hüpft" das Symbol für Fax Receive Monitor (Monitor zum Faxempfang) im Dock, wenn ein neues Fax vorhanden ist.
Gehen Sie zum Öffnen des Ordners für empfangene Faxe wie folgt vor.
1.
Klicken Sie im Dock auf das Symbol für Fax Receive Monitor (Monitor zum Faxempfang), um Fax Receive Monitor (Monitor zum Faxempfang) zu öffnen.
2.
Wählen Sie einen Drucker und klicken Sie auf Open the received fax folder (Ordner für empfangene Faxe öffnen) oder doppelklicken Sie auf den Drucker.
Prüfen Sie das Datum und den Absender im Dateinamen und öffnen Sie anschließend die PDF-Datei.
Hinweis:
  • Empfangene Faxe werden nach folgendem Namensschema automatisch umbenannt:
    JJJJMMTTSSMMSS_xxxxxxxxxx_nnnnn (Jahr/Monat/Tag/Stunde/Minute/Sekunde_Absendernummer)
    Als "Absendernummer" werden vom Absender übertragene Informationen angezeigt. Diese Nummer wird je nach den Einstellungen beim Absender möglicherweise nicht angezeigt.
  • Indem Sie unter Fax Receive Monitor (Monitor zum Faxempfang) die Option Check new faxes (Neue Faxe abrufen) auswählen, können Sie neue Faxe sofort abrufen.
