Obnova seznamu rychlých voleb ze zálohy

Můžete obnovit data telefonního seznamu tiskárny (seznamy rychlých/skupinových voleb) ze zálohy a potom je obnovit v tiskárně.
Poznámka:
  • Nelze obnovit data zálohovaná v aplikaci Fax Utility systému Windows.
  • Pokud zálohovaný seznam rychlých/skupinových voleb obsahuje položky, které nejsou podporovány tiskárnou, do které chcete seznam obnovit, tyto položky nebudou obnoveny.
  • Pokud zálohovaný seznam rychlých/skupinových voleb neobsahuje položky, které jsou podporovány tiskárnou, do které chcete data obnovit, budou obnoveny následujícím způsobem.
    Index Word (Indexové slovo): prázdné
    Fax Speed (Rychlost faxu): přednastavená hodnota
  • Všechny položky, například Name/Group Name (Jméno / název skupiny), které obsahují znaky nepodporované tiskárnou, do které chcete data obnovit, budou po obnovení prázdné.
  • Všechny položky označené jako In use by another function (Používáno jinou funkcí) v FAX Utility budou přepsány obnovenými daty, pokud se shoduje číslo položky.
  • Všechny položky, které jsou používány a chráněny, nebudou přepsány obnovenými daty.
Spusťte aplikaci FAX Utility.
Spuštění aplikace FAX Utility
Vyberte FAX XXXXXX (vaše tiskárna) v části Printer (Tiskárna) a potom klikněte na Speed Dial/Group Dial List (Seznam rychlých/skupinových voleb).
Poznámka:
Pokud je vaše tiskárna vybavena funkcí zabezpečení, která umožňuje správcům omezovat uživatele v provádění změny nastavení faxu tiskárny, pravděpodobně bude třeba před dalším postupem zadat heslo.
Klikněte na File (Soubor) - Back up (Zálohovat) - Restore Backup (Obnovit ze zálohy).
Vyberte soubor, který chcete obnovit.
Obnovený seznam se zobrazí v okně Speed Dial/Group Dial List (Seznam rychlých/skupinových voleb).
Klikněte na v okně.
Obnovený telefonní seznam je uložen do tiskárny.
