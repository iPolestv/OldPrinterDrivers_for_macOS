Speed Dial/Group Dial List (Seznam rychlých/skupinových voleb)

Menu (Nabídka)
Panel nástrojů
Speed Dial/Group Dial List (Seznam rychlých/skupinových voleb) umožňuje zaregistrovat nebo upravit rychlou volbu nebo skupinovou volbu faxu pro vaši tiskárnu.
Poznámka:
  • Toto okno se zobrazí, když kliknete na Speed Dial/Group Dial List (Seznam rychlých/skupinových voleb) v aplikaci FAX Utility (hlavní okno).
  • Maximální počet položek dostupný pro připojenou tiskárnu (fax) je zobrazen na panelu v dolní části okna.

Menu (Nabídka)

File (Soubor)

Položka
Popis
Get Speed Dials (Získat rychlé volby)
Načte rychlou volbu nebo skupinovou volbu faxu pro vaši tiskárnu.
Send Speed Dials (Odeslat rychlé volby)
Uloží rychlé volby a skupinové volby zobrazené v seznamu do tiskárny.
Import
From Address Book (Z adresáře)
Importuje data z Address Book (Adresář) do rychlých voleb v tiskárně.
Back up (Zálohovat)
Save Backup (Uložit zálohu)
Uloží zálohu nakonfigurované rychlé volby a skupinové volby.
Restore Backup (Obnovit ze zálohy)
Načte zálohovanou rychlou nebo skupinovou volbu.
Close (Zavřít)
Zavře okno.
