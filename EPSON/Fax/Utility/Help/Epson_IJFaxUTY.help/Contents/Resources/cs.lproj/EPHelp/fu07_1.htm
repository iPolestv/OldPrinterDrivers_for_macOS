Problémy s přenosem

V případě problému s přenosem postupujte podle následujících pokynů.
Problém
Řešení
FAX XXXXXX (vaše tiskárna) není zobrazeno v seznamu tiskáren.
Zkontrolujte, zda je nainstalován ovladač tiskárny a ovladač PC-FAX a zda byl ovladač PC-FAX přidán do seznamu tiskáren.
Tlačítko Fax (Faxovat) je zobrazeno šedě (deaktivováno).
Toto tlačítko není k dispozici, dokud nepřidáte příjemce do Recipient List (Seznam příjemců). Zadejte alespoň jednoho příjemce.
Z tiskárny nelze odesílat faxy.
  • Když je zobrazeno „No dial tone detected (Nebyl rozpoznán oznamovací tón)“, vyberte Fax Settings (Nastavení faxu) - Dial Tone Detection Settings (Nastavení detekce oznamovacího tónu) a zrušte zaškrtnutí políčka Detect Dial Tone (Detekovat oznamovací tón).
  • Zkontrolujte, zda jste nastavili Your Phone Number (Vaše telefonní číslo) v Header Information Settings (Nastavení záhlaví zpráv). Faxovací zařízení příjemce je pravděpodobně nakonfigurováno tak, aby blokovalo veškeré faxy, které neobsahují informace o telefonním/faxovém číslu odesilatele.
Stav přenášených dat Waiting for printer response (Čekání na odezvu tiskárny) se nemění.
Waiting for printer response (Čekání na odezvu tiskárny) je zobrazeno, když počítač nemůže komunikovat s tiskárnou (FAX) a tato situace bude zachována, dokud komunikace nebude možná. Zkontrolujte následující.
  • Zda je tiskárna zapnutá.
  • Zda jsou tiskárna a počítač správně propojeny.
  • Pokud je tiskárna síťová, zda je správně určena síťová adresa.
  • Zda tiskárna normálně tiskne.
Pokud jsou všechny výše uvedené podmínky splněny, vypněte a znovu zapněte tiskárnu.
Aplikace zobrazí zprávu „Cannot print other fax data because a transmission is being processed.“ a zablokuje se.
Některé aplikace se mohou po faxovém přenosu zablokovat a zároveň zobrazí varování při spuštění. (Po chvíli se mohou zotavit.)

Problémy s přijímáním (Pouze pro modely s příjmem PC-FAX)

V případě problému s příjmem postupujte podle následujících pokynů.
Problém
Řešení
Ikona Fax Receive Monitor (Sledování přijímání faxů) není zobrazena v aplikaci FAX Utility (hlavní okno).
Tato ikona se zobrazí, když vyberete tiskárnu, která podporuje příjem PC-FAX.
Připojte tiskárnu (fax), která podporuje tuto funkci, zapněte tiskárnu a potom vyberte vaši tiskárnu.
Received Fax Output Settings (Nastavení výstupu přijatého faxu) je nastaveno na ukládání faxů do počítače, ale faxy se neukládají.
Přijaté faxy jsou ukládány do určené složky pouze, když je spuštěno Fax Receive Monitor (Sledování přijímání faxů). Doporučujeme vám ponechat Fax Receive Monitor (Sledování přijímání faxů) spuštěné.
Nelze přijmout fax.
  • Zaškrtněte políčko Auto Answer (Automatická odpověď) v Fax Settings (Nastavení faxu) - Auto Answer Settings (Nastavení automatické odpovědi). Pokud není k tiskárně připojen žádný externí telefon, tiskárna nemůže přijímat faxy bez nastavení Auto Answer (Automatická odpověď).
  • Zkontrolujte, zda se tiskárna nenachází v chybovém stavu.
Nelze změnit nastavení tak, aby se přijaté faxy tiskly.
Nastavení nelze změnit, dokud nejsou všechny přijaté faxy uloženy do počítače. V počítači, který je nastaven pro příjímání faxů, změňte nastavení po kontrole nových faxů nebo změňte nastavení pomocí ovládacího panelu tiskárny.
Když je jako složka s přijatými faxy nakonfigurována sdílená složka, zobrazí se zpráva "Cannot find the received fax folder (Nelze najít složku s přijatými faxy)".
Chcete-li ukládat do sdílené složky v síti, je třeba se k ní předem připojit. Vyberte panel nabídek na pracovní ploše Go (Přejít) - Connect to Server (Připojit k serveru) a připojte se ke sdílené složce v síti.
Okno Fax Receive Monitor (Sledování přijímání faxů) se nezobrazí, když je Fax Receive Monitor (Sledování přijímání faxů) automaticky spuštěno při přihlášení.
Okno Fax Receive Monitor (Sledování přijímání faxů) se nezobrazí automaticky, když se počítač přihlásí. Kliknutím na ikonu Fax Receive Monitor (Sledování přijímání faxů) v části Dock (Dok) otevřete okno. I když se okno Fax Receive Monitor (Sledování přijímání faxů) nezobrazí automaticky, probíhají operace na pozadí, například vyhledávání nových faxů atd.
Při každém přijetí faxu se v počítači zobrazí obrazovka pro zadání ověření uživatele.
Používáte-li tiskárnu, která je připojena k síti, a v tiskárně je nastaveno ověřování uživatele příjmu PC-FAX (jméno a heslo pro sdílení souborů), při uložení přijatého faxu do počítače se zobrazí tato obrazovka. Rovněž k tomu dojde, když se ověřovací informace nastavené v tiskárně liší od informací uložených ve svazku klíčů .
Znovu proveďte Received Fax Output Settings (Nastavení výstupu přijatého faxu) a když se zobrazí obrazovka pro zadání, zadejte správné ověřovací informace uživatele. Nezapomeňte vybrat Remember this password in my keychain (Uložit heslo do svazku klíčů).

Chybové zprávy

Když se zobrazí chybová zpráva, odstraňte problém podle následujících pokynů.
Zpráva
Řešení
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (Nelze získat informace o tiskárně (faxu). Otevřete "Print & Fax Preferences" a přidejte ovladač PC-FAX.)
Přidejte ovladač PC-FAX do seznamu tiskáren.
V dialogovém okně se zprávou klikněte na OK. Zobrazí se dialogové okno pro přidání tiskárny.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (Nelze otevřít spustit Záznam přenosu faxu, protože je již spuštěna jiná funkce programu FAX Utility. Prosím počkejte, než bude ukončena, a zkuste to znovu.)
Je spuštěna jiná funkce aplikace FAX Utility. Ukončete jinou funkci a zkuste to znovu.
Cannot open the transmission record file. You have no access rights or the file does not exist. (Nelze otevřít soubor se záznamem přenosu. Nemáte oprávnění k přístupu nebo soubor neexistuje.)
Zkontrolujte, zda máte oprávnění k přístupu k záznamu přenosu.
Failed to save the log file. Change the path and file name, and save it again. (Uložení souboru protokolu se nezdařilo. Změňte umístění a název souboru a znovu jej uložte.)
Zkontrolujte cílové umístění souboru protokolu.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (Odstranění souboru protokolu se nezdařilo. Pravděpodobně jej používá jiná aplikace, nemáte oprávnění k přístupu nebo již byl odstraněn.)
Pokud soubor protokolu zůstane, zkontrolujte oprávnění k přístupu nebo ukončete ostatní aplikace.
A printer communication error occurred. (Došlo k chybě komunikace tiskárny.)
Vypněte tiskárnu a znovu ji zapněte. Pokud nefunguje, chvilku počkejte a potom to zkuste znovu.
An error was detected while communicating with the printer. (Při komunikaci s tiskárnou byla zjištěna chyba.)
Chvilku počkejte a zkuste to znovu.
Failed to write the backup file. There is not enough free space or it may be write protected. (Zápis záložního souboru se nezdařil. Není dostatek volného místa nebo je chráněno proti zápisu.)
Zkontrolujte volné místo na vašem pevném disku. Nebo zrušte ochranu proti zápisu záložního souboru.
Failed to write the backup file. The file specified is being used by another application. (Zápis do záložního souboru se nezdařil. Určený soubor je používán jinou aplikací.)
Záložní soubor je používán jinou aplikací. Ukončete program.
Cannot recognize the file format. The file may be corrupt or created by another application. (Nelze rozpoznat formát souboru. Soubor je pravděpodobně poškozen nebo byl vytvořen jinou aplikací.)
Podporovaná přípona záložního souboru je pouze *.fpb. Zkontrolujte záložní soubor.
Failed to read the backup file. The specified file is being used by another application. (Záložní soubor se nepodařilo načíst. Určený soubor je používán jinou aplikací.)
Záložní soubor je používán jinou aplikací. Ukončete program.
Processing canceled because an error occurred. (Zpracovávání bylo zrušeno z důvodu chyby.)
Pokud problém nelze odstranit restartováním tiskárny a počítače, kontaktujte odbornou pomoc Epson. Kontaktní informace viz příručka k tiskárně.
