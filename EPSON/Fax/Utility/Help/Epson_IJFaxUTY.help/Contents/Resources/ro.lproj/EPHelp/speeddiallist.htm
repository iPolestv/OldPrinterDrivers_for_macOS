Speed Dial/Group Dial List (Listă Apelare rapidă/Apelare grup)

Meniu
Bara de instrumente
Speed Dial/Group Dial List (Listă Apelare rapidă/Apelare grup) vă permite să înregistraţi sau să editaţi numere de apelare rapidă şi numere de apelare grup al faxului pentru imprimantă.
Notă:
  • Această fereastră este afişată atunci când faceţi clic pe Speed Dial/Group Dial List (Listă Apelare rapidă/Apelare grup) din FAX Utility (fereastra principală).
  • Numărul maxim de intrări disponibile pentru imprimanta (faxul) conectată este afişat în bara din partea inferioară a ferestrei.

Meniu

Meniul File (Fişier)

Element
Explicaţie
Get Speed Dials (Primire numere apelare rapidă)
Citeşte numărul de apelare rapidă sau numărul de apelare grup al faxului pentru imprimantă.
Send Speed Dials (Trimitere numere apelare rapidă)
Salvează în imprimantă numerele de apelare rapidă şi numerele de apelare a grupurilor afişate în listă.
Import
From Address Book (Din agendă)
Importă datele din Address Book (Agendă) în numerele de apelare rapidă ale imprimantei.
Back up (Copie de rezervă)
Save Backup (Salvare copie de rezervă)
Salvează o copie de rezervă a apelării rapide sau apelării grup.
Restore Backup (Restaurare copie de rezervă)
Citeşte apelarea rapidă sau apelarea grup pentru care s-a efectuat o copie de rezervă.
Close (Închidere)
Închide fereastra.

Meniul Edit (Editare)

Element
Explicaţie
Edit (Editare)
Editează numărul de apelare rapidă sau numărul de apelare grup selectat din listă.
Delete (Ştergere)
Şterge numărul de apelare rapidă sau numărul de apelare grup selectat din listă.
Delete All (Ştergere tot)
Şterge toate numerele de apelare rapidă şi apelare grup selectate din listă.
