Probleme de transmisie

În cazul unei probleme de transmisie, remediaţi-o după cum urmează.
Problemă
Ce trebuie să faceţi
FAX XXXXXX (imprimanta dvs.) nu este afişată în lista Printer (Imprimantă).
Asiguraţi-vă că sunt instalate driverul imprimantei şi driverul PC-FAX, şi că driverul PC-FAX este adăugat în lista de imprimante.
Butonul Fax este gri (dezactivat).
Acest buton este disponibil numai dacă în Recipient List (Listă destinatari) este adăugat un destinatar. Specificaţi cel puţin un destinatar.
Nu puteţi trimite faxuri de la imprimantă.
  • Când se afişează „No dial tone detected (Nu a fost detectat tonul de apel)”, selectaţi Fax Settings (Setări fax) - Dial Tone Detection Settings (Setări detectare ton de apel) şi debifaţi caseta de validare pentru Detect Dial Tone (Detectare ton de apel).
  • Asiguraţi-vă că aţi setat Your Phone Number (Număr de telefon) în Header Information Settings (Setări informaţii antet). Este posibil ca aparatul de fax al destinatarului să fi fost configurat pentru a bloca faxurile care nu includ informaţiile despre numărul de telefon sau numărul de fax al expeditorului.
Starea datelor transmise nu se modifică în timpul afişării mesajului Waiting for printer response (Se aşteaptă răspunsul de la imprimantă).
Waiting for printer response (Se aşteaptă răspunsul de la imprimantă) este afişat atunci când calculatorul nu poate comunica cu imprimanta (FAX) şi se menţine până când imprimarea devine posibilă. Verificaţi următoarele.
  • Dacă imprimanta este pornită.
  • Dacă imprimanta şi calculatorul sunt conectate corect.
  • În cazul în care imprimanta este o imprimantă de reţea, dacă adresa de reţea a imprimantei este specificată corect.
  • Dacă imprimanta poate imprima normal.
Dacă toate cele de mai sus sunt corecte, opriţi şi apoi porniţi imprimanta.
Aplicaţia se blochează după afişarea mesajului „Cannot print other fax data because a transmission is being processed.”.
Este posibil ca anumite aplicaţii să se blocheze în momentul transmiterii de faxuri, afişând simultan un avertisment de pornire. (Este posibil ca acestea să îşi revină după un timp.)

Probleme de primire (numai pentru modele cu recepţie PC-FAX)

În cazul survenirii unei probleme de recepţie, remediaţi-o după cum urmează.
Problemă
Ce trebuie să faceţi
Pictograma Fax Receive Monitor (Monitor primire faxuri) nu este afişată în FAX Utility (fereastra principală).
Pictograma este afişată în momentul selectării unei imprimante care suportă recepţia PC-FAX.
Conectaţi o imprimantă (un fax) care suportă această funcţie şi porniţi imprimanta, iar apoi selectaţi imprimanta.
Opţiunea Received Fax Output Settings (Setări ieşire fax primite) este setată pentru salvarea faxurilor pe un calculator, însă nu este salvat niciun fax.
Faxurile primite sunt salvate în directorul specificat numai când Fax Receive Monitor (Monitor primire faxuri) este activ. Vă recomandăm să păstraţi Fax Receive Monitor (Monitor primire faxuri) activ.
Nu se pot primi faxuri.
  • Bifaţi caseta de validare Auto Answer (Răspuns automat) din Fax Settings (Setări fax) - Auto Answer Settings (Setări răspuns automat). Dacă niciun telefon extern nu este conectat la imprimantă, imprimanta nu poate recepţiona faxuri fără setarea Auto Answer (Răspuns automat).
  • Verificaţi dacă imprimanta nu se află într-o stare de eroare.
Nu se pot modifica setările pentru ca faxurile primite să fie imprimate.
Nu puteţi modifica setările înainte ca toate faxurile primite să fi fost salvate pe calculator. Pe un calculator setat să salveze faxurile primite, modificaţi setările după ce verificaţi dacă există faxuri noi sau modificaţi setările folosind panoul de control al imprimantei.
"Cannot find the received fax folder (Directorul pentru faxurile primite nu a fost găsit)." se afişează atunci când un director partajat este configurat drept director de fax.
Pentru a salva într-un director partajat dintr-o reţea, trebuie să vă conectaţi la acesta în prealabil. Selectaţi bara de meniu de pe spaţiul de lucru Go (Accesare) - Connect to Server (Conectare la server) şi conectaţi-vă la directorul partajat din reţea.
Fereastra Fax Receive Monitor (Monitor primire faxuri) nu este afişată atunci când funcţia Fax Receive Monitor (Monitor primire faxuri) este pornită automat în momentul conectării.
Fereastra Fax Receive Monitor (Monitor primire faxuri) nu este afişată automat în momentul conectării calculatorului. Faceţi clic pe pictograma Fax Receive Monitor (Monitor primire faxuri) din Dock pentru a deschide fereastra. Chiar dacă fereastra Fax Receive Monitor (Monitor primire faxuri) nu este afişată automat, aceasta efectuează operaţii în fundal, cum ar fi verificarea existenţei unor faxuri noi etc.
Ecranul de introducere pentru autentificarea utilizatorului este afişat pe calculator de fiecare dată când primiţi un fax.
Când utilizaţi o imprimantă conectată la reţea, dacă autentificarea utilizatorului recepţiei PC-FAX (numele şi parola pentru partajarea fişierelor) este setat în imprimantă, acest ecran este afişat în momentul salvării faxului primit pe calculator. Acest lucru se întâmplă şi când informaţiile de autentificare setate în imprimantă diferă de informaţiile salvate în date.
Efectuaţi din nou Received Fax Output Settings (Setări ieşire fax primite) şi apoi introduceţi informaţiile corecte de identificare a utilizatorului când se afişează ecranul de introducere. Reţineţi să selectaţi Remember this password in my keychain (Memorare această parolă în datele mele).

Mesaje de eroare

Când se afişează un mesaj de eroare, rezolvaţi problema urmând instrucţiunile de remediere.
Mesaj
Ce trebuie să faceţi
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (Obţinerea informaţiilor imprimantei (faxului) a fost imposibilă. Deschideţi "Print & Fax Preferences" şi adăugaţi driverul PC-FAX.)
Adăugaţi driverul PC-FAX în lista de imprimante.
Faceţi clic pe OK în dialogul mesajului. Va fi afişat dialogul pentru adăugarea unei imprimante.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (Înregistrare transmisie fax nu poate fi deschisă deoarece o altă funcţie a FAX Utility rulează deja. Vă rugăm să aşteptaţi terminarea acesteia şi să reîncercaţi.)
O altă funcţie FAX Utility este în curs de execuţie. Închideţi cealaltă funcţie şi apoi încercaţi din nou.
Cannot open the transmission record file. You have no access rights or the file does not exist. (Fişierul cu înregistrarea transmisiei nu poate fi deschis. Nu aveţi drepturi de acces sau fişierul nu există.)
Verificaţi dacă aveţi drepturi de acces pentru citire asupra înregistrării transmisiei.
Failed to save the log file. Change the path and file name, and save it again. (Salvarea fişierului jurnal a eşuat. Modificaţi numele şi calea fişierului şi salvaţi din nou.)
Verificaţi destinaţia în care este salvat fişierul jurnal.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (Ştergerea fişierului jurnal a eşuat. Probabil este utilizat de o altă aplicaţie sau nu aveţi drepturi de acces sau poate a fost deja şters.)
Dacă rămâne un fişier jurnal, verificaţi setările pentru drepturi de acces sau închideţi alte aplicaţii.
A printer communication error occurred. (S-a produs o eroare de comunicare cu imprimanta.)
Opriţi imprimanta şi apoi porniţi-o din nou. Dacă acest lucru nu dă rezultate, aşteptaţi un timp şi apoi încercaţi din nou.
An error was detected while communicating with the printer. (În timpul comunicării cu imprimanta a fost detectată o eroare. )
Aşteptaţi un timp şi apoi reîncercaţi.
Failed to write the backup file. There is not enough free space or it may be write protected. (Scrierea fişierului cu copii de rezervă a eşuat. Nu este suficient spaţiu disponibil sau fişierul este protejat la scriere.)
Verificaţi spaţiul liber disponibil pe unitatea de disc. Sau eliminaţi protecţia la scriere a fişierului de rezervă.
Failed to write the backup file. The file specified is being used by another application. (Scrierea fişierului cu copii de rezervă a eşuat. Fişierul specificat este utilizat de o altă aplicaţie.)
Fişierul cu copii de rezervă este utilizat de un alt program. Închideţi celălalt program.
Cannot recognize the file format. The file may be corrupt or created by another application. (Formatul de fişier nu a putut fi recunoscut. Fişierul poate fi corupt sau creat cu o altă aplicaţie.)
Singura extensie suportată pentru fişierul cu copii de rezervă este *.fpb. Verificaţi fişierul cu copii de rezervă.
Failed to read the backup file. The specified file is being used by another application. (Citirea fişierului cu copii de rezervă a eşuat. Fişierul specificat este utilizat de o altă aplicaţie.)
Fişierul cu copii de rezervă este utilizat de un alt program. Închideţi celălalt program.
Processing canceled because an error occurred. (Procesarea a fost anulată deoarece s-a produs o eroare.)
Dacă repornirea imprimantei şi a calculatorului nu rezolvă problema, contactaţi asistenţa Epson. Pentru informaţii de contact, consultaţi manualul imprimantei.
