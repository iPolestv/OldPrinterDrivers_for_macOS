FAX Utility (Ecranul Main (principal))

Aceasta este fereastra în care selectaţi funcţia FAX Utility de utilizat.
Notă:
Această fereastră apare când lansaţi FAX Utility.
Cum se lansează FAX Utility
Element
Explicaţie
Fax Transmission Record (Înregistrare transmisie fax)
Afişează înregistrările transmisiilor.
Fax Transmission Record (Înregistrare transmisie fax)
Fax Receive Monitor (Monitor primire faxuri)
Se afişează numai atunci când este selectată o imprimantă (un fax) care suportă recepţia PC-FAX. Vă permite să selectaţi dacă salvaţi sau nu faxurile primite pe un calculator.
Check Fax Connection (Verificare fax Conexiune)
Verifică dacă funcţia de fax a imprimantei conectate funcţionează normal.
Fax Settings (Setări fax)
Setează antetul faxului şi diferite setări de comunicare prin fax salvate în imprimantă.
Speed Dial/Group Dial List (Listă Apelare rapidă/Apelare grup)
Salvează sau editează numere de apelare rapidă sau numere de apelare grup pentru fax, salvate în imprimantă.
Sincronizare Address Book (Agendă) sau Sincronizare Contacts (Contacte)
Sincronizează datele pentru Address Book (Agendă) sau Contacts (Contacte) cu FAX Utility pentru a vă permite să selectaţi destinatari din lista de contacte la trimiterea unui fax folosind FAX Utility. Dacă actualizaţi lista Address Book (Agendă) sau Contacts (Contacte) după sincronizare, trebuie să executaţi această funcţie de sincronizare din nou pentru a utiliza datele actualizate cu FAX Utility.
