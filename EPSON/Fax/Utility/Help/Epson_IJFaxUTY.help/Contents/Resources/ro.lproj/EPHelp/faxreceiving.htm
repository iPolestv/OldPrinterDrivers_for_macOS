Fax Receive Monitor (Monitor primire faxuri) (Numai pentru modelele cu recepţie PC-FAX)

Bara de instrumente
Pictograme şi mesaje în lista imprimantei
Această fereastră vă permite să verificaţi starea recepţiei faxurilor şi să efectuaţi setări de ieşire pentru faxurile primite.
Notă:
Această fereastră este afişată atunci când faceţi clic pe Fax Receive Monitor (Monitor primire faxuri) din FAX Utility (fereastra principală).

Bara de instrumente

Element
Explicaţie
Open the received fax folder (Deschideţi directorul cu faxuri primite)
Deschide directorul în care sunt salvate faxurile primite.
Check new faxes (Verificare faxuri noi)
Verifică dacă există faxuri noi.
Received Fax Output Settings (Setări ieşire fax primite)
Setează dacă se salvează sau nu faxurile primite pe un calculator.
Preferences (Preferinţe)
Configurează preferinţele pentru Fax Receive Monitor (Monitor primire faxuri).

Pictograme şi mesaje în lista imprimantei

Pictogramă
Mesaj
Explicaţie
Waiting (Aşteptare) / Checking for new faxes (Verificare faxuri noi)
Se verifică existenţa unor faxuri noi sau se aşteaptă.
Received faxes saved successfully in the specified folder (Faxurile primite au fost salvate cu succes în directorul specificat).
Faxurile noi primite de o imprimantă (un fax) au fost salvate în directorul specificat.
The printer is not responding (Imprimanta nu răspunde)
Verificaţi conexiunea dintre calculator şi imprimantă sau verificaţi dacă imprimanta este pornită.
The printer is processing data (Imprimanta procesează datele)
Imprimanta procesează alte date. Aşteptaţi.
Cannot find the received fax folder (Directorul pentru faxurile primite nu a fost găsit).
Verificaţi dacă numele directorului specificat este corect. Dacă utilizaţi un director partajat din reţea, asiguraţi-vă că deţineţi drepturi de acces la director.
Failed to save faxes in the specified folder (Salvarea faxurilor în directorul specificat a eşuat).
Verificaţi dacă directorul specificat este disponibil şi dacă deţineţi drepturi de acces la director.
A printer communication error occurred (S-a produs o eroare de comunicare cu imprimanta)
Verificaţi conexiunea dintre calculator şi imprimantă sau verificaţi dacă imprimanta este pornită. Dacă utilizaţi o conexiune în reţea, verificaţi dacă aveţi acces la imprimantă prin reţea.
The "FAXRECV" disk is being used by another application. (Discul "FAXRECV" este utilizat de o altă aplicaţie.)
Închideţi programul care utilizează discul „FAXRECVˮ.
Cannot save faxes. User authentication failed. (Faxurile nu au putut fi salvate. Autentificarea utilizatorului a eşuat.)
Verificaţi autentificarea (numele şi parola pentru partajarea de fişiere) utilizatorului care recepţionează PC-FAX setată în imprimantă, efectuaţi din nou Received Fax Output Settings (Setări ieşire fax primite) şi apoi introduceţi corect informaţiile de autentificare a utilizatorului. Reţineţi să selectaţi Remember this password in my keychain (Memorare această parolă în datele mele).
