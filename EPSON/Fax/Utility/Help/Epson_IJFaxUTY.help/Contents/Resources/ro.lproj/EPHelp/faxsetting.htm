Fax Transmission Record (Înregistrare transmisie fax)

Meniu
În această fereastră, puteţi consulta jurnalul de transmisii prin fax.
Notă:
  • Această fereastră este afişată atunci când faceţi clic pe Fax Transmission Record (Înregistrare transmisie fax) din FAX Utility (fereastra principală).
  • FAX Utility salvează numai jurnale ale transmisiilor. Datele transmise nu sunt salvate, astfel încât nu puteţi vizualiza datele pe care le-aţi trimis.

Meniu

Meniul File (Fişier)

Element
Explicaţie
Close (Închidere)
Închide fereastra.
Save (Salvare)
Salvează înregistrările transmise. Datele salvate pot fi utilizate pentru a analiza probleme survenite în timpul transmiterii faxului.

Meniul Edit (Editare)

Element
Explicaţie
Delete log (Ştergere jurnal)
Şterge toate înregistrările transmise.
