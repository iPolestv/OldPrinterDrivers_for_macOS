Cum se lansează FAX Utility

Pentru Mac OS X 10.6.x sau versiuni ulterioare
Faceţi clic pe Apple - System Preferences (Preferinţe sistem) - Print & Scan (Imprimare şi scanare) sau Print & Fax (Imprimare şi fax).
Selectaţi FAX XXXXXX (imprimanta dvs.) din Printer (Imprimată) şi apoi faceţi clic pe Options & Supplies (Opţiuni şi consumabile) - Utility (Utilitar) - Open Printer Utility (Deschidere utilitar imprimantă).
Pentru Mac OS X 10.5.x
Faceţi clic pe Apple - System Preferences (Preferinţe sistem) - Print & Fax (Imprimare şi fax).
Selectaţi FAX XXXXXX (imprimanta dvs.) din Imprimată şi apoi faceţi clic pe Open Print Queue (Deschidere coadă de imprimare) - Utility (Utilitar).
Pentru Mac OS X 10.4.x
Faceţi dublu clic pe unitatea de disc - Applications (Aplicaţii) - Utilities (Utilitare) - Printer Setup Utility (Utilitar de configurare imprimantă).
Selectaţi FAX XXXXXX (imprimanta dvs.) din Imprimată şi apoi faceţi clic pe Utility (Utilitar).
