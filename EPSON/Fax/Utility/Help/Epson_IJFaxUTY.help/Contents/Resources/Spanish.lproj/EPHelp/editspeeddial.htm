Speed Dial/Group Dial List (List Marc rápid/Marc grupo) - Edit Speed Dial/Group Dial (Editar Marc rápid/Marc grupo)-

En esta ventana puede registrar o editar una entrada de marcado rápido o de grupo del fax en su impresora.
Elemento
Explicación
Entry Number (Nº de entrada)
Muestra el número de entrada del marcado rápido.
Speed Dial (Marcado rápido)
Seleccione esta opción si desea crear o modificar una entrada de marcado rápido.
Group Dial (Marcado grupo)
Seleccione esta opción si desea crear o modificar una entrada de marcado de grupo. No puede crear un marcado de grupo si no ha creado entradas de marcado rápido.
Name/Group Name (Nombre/Nombre de grupo)
Escriba el nombre de un marcado rápido o de grupo que desee registrar.
Index Word (Palabra índice) *3
Escriba una palabra de índice para el destinatario para poder utilizarla cuando realice búsquedas.
Fax Number (Número de fax) *1
Introduzca el número de fax del destinatario. Si desea separar números (códigos postales de números personales, por ejemplo), use un espacio en vez de un guión (-), puesto que un guión (-) se interpreta como una pausa al marcar. Este elemento se muestra cuando edita el marcado rápido.
Member (Miembro) *2
Muestra una lista de marcados rápidos registrados. Seleccione el marcado rápido que quiera añadir al marcado de grupo y después, haga clic en OK (Aceptar). Este elemento se muestra cuando edita el marcado de grupo.
Fax Speed (Velocidad del fax) *3
Seleccione la velocidad con la que se enviarán los faxes al destinatario. La velocidad de fax establecida para cada entrada de marcado rápido solo se utiliza cuando se envía un fax directamente desde una impresora. FAX Utility ignora la configuración y utiliza la velocidad de fax establecida en la impresora.
OK (Aceptar)
Guarda los ajustes y cierra la ventana.
Cancel (Cancelar)
Cancela el registro o edita y cierra la ventana.
*1 Dato imprescindible para registrar el marcado rápido.
*2 Si está registrando un marcado de grupo, confirme que haya uno o varios marcados rápidos seleccionados.
*3 Es posible que no se muestren estos elementos en determinadas impresoras.
Nota:
  • Para información sobre los caracteres y el número de caracteres que pueden introducirse, consulte el siguiente apartado.
    Caracteres permitidos
  • El número máximo de entradas de marcado rápido por grupo aparece en la barra en la parte inferior de la ventana de edición de marcado de grupo.
