Problemas de transmisión

Solución de averías en caso de producirse un problema de transmisión.
Problema
Qué hacer
FAX XXXXXX (su impresora) no figura en la lista Impresora.
Confirme que estén instalados el controlador de impresora y el controlador de PC-FAX, y que el controlador de PC-FAX se haya añadido a la lista de impresoras.
El botón Fax está atenuado (inhabilitado).
Este botón solamente está disponible si se ha añadido algún recipiente a la Recipient List (Lista destinatarios). Especifique un destinatario como mínimo.
No se pueden enviar faxes desde la impresora.
  • Si aparece No dial tone detected (No hay tono marcado) seleccione Fax Settings (Ajustes de fax) - Dial Tone Detection Settings (Ajustes de Detección tono marcado) y desmarque la casilla de verificación de Detect Dial Tone (Detectar tono marc).
  • Asegúrese de haber definido Your Phone Number (Su número de teléfono) en Header Information Settings (Ajustes de Datos de membrete). La máquina de fax del destinatario puede estar configurada para bloquear cualquier fax que no incluya el número de teléfono/fax del remitente.
El estado de los datos de transmisión no cambia de Waiting for printer response (Esperando respuest impresr).
Waiting for printer response (Esperando respuest impresr) se muestra cuando el ordenador no se puede comunicar con la impresora (FAX) y permanece hasta que se establece la comunicación. Comprobar lo siguiente.
  • Si la impresora está conectada.
  • Si la impresora y el ordenador están correctamente conectados.
  • Si la impresora es una impresora de red, y si la dirección de dicha impresora está correctamente especificada.
  • Si la impresora puede imprimir correctamente.
Si todos los datos anteriores son correctos, desconectar y volver a conectar la impresora.
La aplicación se bloquea después de mostrar el mensaje Cannot print other fax data because a transmission is being processed. (No se pueden imprimir otros datos de fax porque se está procesando una transmisión.)
Puede que algunas aplicaciones queden suspendidas, con un aviso simultáneo de advertencia visualizado a su inicio. (Puede que se recuperen transcurrido un rato.)

Problemas de recepción (sólo para modelos con recepción de PC-FAX)

Solución de problemas en caso de que se produzca un problema de recepción.
Problema
Qué hacer
El icono de Fax Receive Monitor (Monitor de recepción de faxes) no se muestra en la pantalla principal de FAX Utility.
El icono se muestra al seleccionar una impresora que admite la recepción de PC-FAX.
Conecte una impresora (fax) que admita esta función y encienda la impresora. A continuación, seleccione la impresora.
La opción Received Fax Output Settings (Ajustes de salida de faxes recibidos) está definida para guardar los faxes en un ordenador, pero no se guardan.
Los faxes recibidos solamente se guardan en la carpeta especificada si se está ejecutando el Fax Receive Monitor (Monitor de recepción de faxes). Se recomienda mantener el Fax Receive Monitor (Monitor de recepción de faxes) en funcionamiento.
No se pueden recibir faxes.
  • Seleccione la casilla de verificación Auto Answer (Resp autom) en Fax Settings (Ajustes de fax) - Auto Answer Settings (Ajustes de Resp autom). Si no hay conectado ningún teléfono externo a la impresora, la impresora no podrá recibir faxes sin configurar el ajuste Auto Answer (Resp autom).
  • Compruebe que no se haya producido ningún error en la impresora.
No puede cambiar los ajustes de manera que se imprimen los faxes recibidos.
No puede cambiar los ajustes hasta que no se hayan guardado todos los faxes recibidos en el ordenador. En un ordenador que esté configurado para guardar los faxes recibidos, cambie los ajustes después de comprobar si hay faxes nuevos, o bien cambie los ajustes utilizando el panel de control de la impresora.
"El mensaje Cannot find the received fax folder (No se encuentra la carpeta de faxes recibidos) aparece cuando una carpeta compartida en una red está configurada como carpeta de faxes recibidos.
Para guardar los faxes en una carpeta compartida en una red, primero debe conectarse a dicha red. Seleccione la barra de menús en el escritorio: Go (Ir) - Connect to Server (Conectar al servidor) y conéctese a la carpeta compartida en la red.
La ventana Fax Receive Monitor (Monitor de recepción de faxes) no se muestra cuando el Fax Receive Monitor (Monitor de recepción de faxes) se inicia automáticamente al iniciar sesión.
La ventana Fax Receive Monitor (Monitor de recepción de faxes) no se muestra automáticamente cuando el ordenador inicia sesión. Haga clic en el icono del Fax Receive Monitor (Monitor de recepción de faxes) en el Dock para abrir la ventana. Aunque la ventana Fax Receive Monitor (Monitor de recepción de faxes) no se muestre automáticamente, sí que realiza operaciones en segundo plano, como la comprobación de faxes nuevos, etc.
Cada vez que recibe un fax se muestra en el ordenador la pantalla de entradas de autenticación de usuario.
En una impresora conectada a la red, si la autenticación del usuario de recepción de PC-Fax (nombre y contraseña para compartir archivos) está establecida en la impresora, se muestra esta pantalla al guardar en el ordenador el fax recibido. Lo mismo sucede cuando la información de autenticación definida en la impresora difiere de la información guardada en el administrador de contraseñas.
Especifique de nuevo los Received Fax Output Settings (Ajuste de salida de faxes recibidos) y, a continuación, introduzca la información correcta de autenticación de usuario cuando se muestre la pantalla de entradas. No olvide seleccionar Remember this password in my keychain (Guardar esta contraseña en mi llavero).

Mensajes de error

Cuando se muestra un mensaje de error, se puede solucionar el problema siguiendo las instrucciones detalladas a continuación.
Mensaje
Qué hacer
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (No se pueden obtener datos de la impresora (fax). Vaya a "Preferencias de impresión y fax" y añada el controlador PC-FAX.)
Añada el controlador de PC-FAX a la lista de impresoras.
En el cuadro de mensaje, haga clic en OK (Aceptar). Se abrirá el cuadro de diálogo para añadir impresoras.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (No se puede abrir el Registro de transmisión de fax porque está ejecutándose otra función de FAX Utility. Espere a que termine e inténtelo de nuevo.)
Se está ejecutando otra función de FAX Utility. Cierre la otra función y vuelva a intentarlo.
Cannot open the transmission record file. You have no access rights or the file does not exist. (No se puede abrir el archivo de registro de transmisión. O usted no tiene derechos para acceder a él o el archivo no existe.)
Confirme que tiene derecho de acceso de lectura al registro de transmisión.
Failed to save the log file. Change the path and file name, and save it again. (No se ha podido guardar el archivo de registro. Cambie el nombre y la ruta del archivo y vuelva a guardarlo.)
Compruebe la ubicación donde desea guardar el archivo de registro.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (No se ha podido borrar el archivo de registro. O lo está usando otra aplicación, o usted no tiene derechos para acceder a él o ya se había borrado.)
Si queda un archivo de registro, revise los ajustes del derecho de acceso o cierre otras aplicaciones.
A printer communication error occurred. (Error de comunicación de la impresora.)
Apague la impresora y luego vuelva a encenderla. Si no funciona, espere un rato y vuelva a intentarlo.
An error was detected while communicating with the printer. (Error durante la comunicación con la impresora.)
Wait for a while and then try again. (Error durante la comunicación con la impresora. Espere un rato y vuelva a intentarlo.)
Failed to write the backup file. There is not enough free space or it may be write protected. (No se ha escrito archivo d copia de seguridad. El disco no tiene bastante espacio libre o está protegido contra escritura.)
Compruebe si hay suficiente espacio en el disco duro o elimine la protección contra escritura del archivo de copia de seguridad.
Failed to write the backup file. The file specified is being used by another application. (No se ha podido escribir el archivo de copia de seguridad. Está usándolo otra aplicación.)
Otro programa está usando el archivo de copia de seguridad. Salga del otro programa.
Cannot recognize the file format. The file may be corrupt or created by another application. (No se detecta el formato del archivo. El archivo puede estar dañado o ha sido creado con otra aplicación.)
Para los archivos de copia de seguridad, solamente se admite la extensión *.fpb. Compruebe el archivo de seguridad.
Failed to read the backup file. The specified file is being used by another application. (No se ha podido leer el archivo de copia de seguridad. Está usándolo otra aplicación.)
Otro programa está usando el archivo de copia de seguridad. Salga del otro programa.
Processing canceled because an error occurred. (Error. Procesamiento cancelado.)
Si al reiniciar la impresora y el ordenador no se resuelve el problema, póngase en contacto con el Servicio de asistencia de Epson. Para obtener la información de contacto, consulte el manual de la impresora.
