Visión general del producto

FAX Utility es una aplicación que ofrece varias funciones muy cómodas para enviar faxes.
FAX Utility solamente funciona con las impresoras multifunción Epson con fax.
Si se usa una impresora compatible con la recepción de PC-FAX, se pueden guardar los faxes recibidos como archivos PDF en el ordenador. Esto le permitirá reducir el número de impresiones innecesarias.
Nota:
Si su impresora admite la recepción de PC-FAX, el Fax Receive Monitor (Monitor de recepción de faxes) aparece en la ventana principal de FAX Utility.

Fax Receive Monitor (Monitor de recepción de faxes) (solo para modelos con recepción de PC-FAX)
Fax Receive Monitor (Monitor de recepción de faxes) se muestra cuando se selecciona una impresora con recepción de PC-FAX en la ventana principal de FAX Utility. En la ventana Fax Receive Monitor (Monitor de recepción de faxes), puede definir ajustes para guardar los faxes recibidos en su ordenador y establecer un mensaje de notificación que se mostrará cuando se reciban faxes nuevos.
