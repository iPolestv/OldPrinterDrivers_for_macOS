Preferences (Preferencias) (sólo para modelos con recepción de PC-FAX)

Esta ventana permite configurar los ajustes de recepción de faxes, como por ejemplo la forma en que se realizan las notificaciones cuando se recibe un fax nuevo.
Nota:
  • Esta ventana se visualiza al hacer clic en Fax Receive Monitor Preferences (Preferencias del Monitor de recepción de faxes) en Fax Receive Monitor (Monitor de recepción de faxes).
  • Estos ajustes se aplican a todas las impresoras y colas de impresión mostradas en el Fax Receive Monitor (Monitor de recepción de faxes).
Elemento
Explicación
Specify an interval to check for new faxes (Especifique el intervalo en el que se comprobará si hay faxes nuevos)
Especifique cada cuánto tiempo desea comprobar si hay faxes nuevos (entre 1 y 30 minutos).
Notify me of new faxes via a dock icon (Notificarme los faxes nuevos con un icono en el dock)
Seleccione esta opción para que aparezca el icono del Fax Receive Monitor (Monitor de recepción de faxes) en el Dock cuando llegue un fax nuevo.
Notify me of errors via a dock icon (Notificarme los errores con un icono en el dock)
Seleccione esta opción para que aparezca el icono del Fax Receive Monitor (Monitor de recepción de faxes) en el Dock cuando se produzca un error. Haga clic en el icono para mostrar la ventana Monitor de recepción de faxes donde se indica la información del error.
Start at login (Arrancar al iniciar sesión)
Se recomienda configurar este ajuste en Automatic (Automático). Si la opción "Save" (save faxes on this computer) ("Guardar" (guardar los faxes que hay en este ordenador)) está definida en Received Fax Output Settings (Ajustes de salida de faxes recibidos), Fax Receive Monitor (Monitor de recepción de faxes) se iniciará automáticamente cuando inicie sesión. Si selecciona On (Act), Fax Receive Monitor (Monitor de recepción de faxes) se iniciará al iniciar sesión, independientemente de los ajustes de salida. En cambio, si selecciona Off (Des), el Fax Receive Monitor (Monitor de recepción de faxes) no se iniciará al iniciar sesión.
Wait time before checking new faxes (Esperar este tiempo para comprobar si hay faxes nuevos)
Este elemento está disponible cuando la opción Automatic (Automático) o On (Act) está definida en Start at login (Arrancar al iniciar sesión). Especifique cuánto tiempo (0 a 10 minutos) se debe esperar antes de comprobar si hay faxes nuevos después de que se inicie el Fax Receive Monitor (Monitor de recepción de faxes) al iniciar sesión. Esto resulta útil cuando la carpeta donde se deben guardar los faxes se encuentra en una red, y usted necesita tiempo para establecer una conexión con la carpeta después de iniciar sesión.
Nota:
Si la carpeta donde se deben guardar los faxes recibidos es una carpeta compartida en una red, puede conectarse a la carpeta cuando inicie sesión; para ello, registre la carpeta en la lista de elementos de inicio de sesión. Haga clic en Apple - System Preferences (Preferencias del Sistema) - Account (Cuenta) - Login to register (Iniciar sesión para registrar).
