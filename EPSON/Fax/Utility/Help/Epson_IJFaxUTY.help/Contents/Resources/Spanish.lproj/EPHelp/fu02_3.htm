Sincronizar Address Book (Agenda) o Contacts (Contactos) con FAX Utility (para Mac OS X 10.7.x o posterior)

Si desea seleccionar un destinatario desde Address Book (Agenda) o Contacts (Contactos) para enviar un fax con FAX Utility, primero debe sincronizarlo. Si actualiza Address Book (Agenda) o Contacts (Contactos) después de sincronizar, debe ejecutar esta función de sincronización de nuevo para utilizar los datos actualizados con FAX Utility.
Inicie FAX Utility.
Iniciar FAX Utility
Seleccione Synchronize (Sincronizar) Address Book (Agenda) o Synchronize (Sincronizar) Contacts (Contactos).
Siga las instrucciones en pantalla para proceder a la sincronización.
