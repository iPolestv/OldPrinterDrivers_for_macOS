> หน้าแรก > การรับแฟกซ์ (เฉพาะรุ่นที่มีฟังก์ชันการรับ PC-FAX)

การตั้งค่าเพื่อบันทึกแฟกซ์ที่ได้รับบนคอมพิวเตอร์

คุณสามารถเปลี่ยนแปลงการตั้งค่าเอาต์พุตแฟกซ์ที่ได้รับ เพื่อบันทึกแฟกซ์ที่ได้รับในรูปแบบไฟล์ PDF บนคอมพิวเตอร์ หรือในแชร์โฟลเดอร์บนเซิร์ฟเวอร์ (การรับ PC-FAX)
หมายเหตุ:
  • ถ้าเครื่องพิมพ์ของคุณสนับสนุนการรับ PC-FAX Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) จะแสดงบนหน้าต่างหลัก FAX Utility
  • คุณต้องตั้งค่าการรับสายอัตโนมัติ ในขณะที่ใช้ฟังก์ชันการรับ PC-FAX
  • ระบบจะทำการลบข้อมูลแฟกซ์ที่บันทึกไปยังคอมพิวเตอร์แล้วออกจากหน่วยความจำของเครื่องพิมพ์ ขึ้นอยู่กับเครื่องพิมพ์แต่ละรุ่น
  • เนื่องจากแฟกซ์ที่ได้รับบันทึกในรูปแบบไฟล์ PDF คุณจำเป็นต้องใช้ Cover Flow (10.5 หรือใหม่กว่า), Quick Look (10.5 หรือใหม่กว่า) หรือ Preview.app ที่มีการติดตั้งล่วงหน้ามาพร้อมกับMac OS X
เริ่ม FAX Utility
การเริ่มต้นใช้งาน FAX Utility
คลิกที่ Fax Receive Monitor (การตรวจสอบการรับแฟกซ์)
หมายเหตุ:
  • คุณไม่สามารถเริ่ม Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) พร้อมกันหลายรายการภายใต้สภาพแวดล้อมผู้ใช้งานหลายรายได้
  • คุณสามารถนำเข้าและบันทึกแฟกซ์ที่ได้รับเฉพาะเมื่อ Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) กำลังทำงานอยู่เท่านั้น
  • ในขณะที่ Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) กำลังทำงานอยู่ ไอคอนจะปรากฏขึ้นบนด็อค การเพิ่ม Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) ไปยังด็อค ช่วยให้คุณเริ่มต้นใช้งาน Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) ได้อย่างง่ายดาย ในการเพิ่ม Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) ไปยังด็อค ให้คลิกที่ไอคอน Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) ในขณะที่กดปุ่ม Control ค้างไว้ จากนั้นคลิกที่ Add to Dock (เพิ่มไปยังด็อค) บนเมนู
เลือกเครื่องพิมพ์ของคุณ จากนั้นเลือก Receive Settings (รับการตั้งค่า)
หมายเหตุ:
  • เนื่องจากคุณสามารถบันทึกแฟกซ์ไปยังลำดับการพิมพ์ (แฟกซ์) ได้ครั้งละหนึ่งรายการเท่านั้น ถ้ามีลำดับการพิมพ์ (แฟกซ์) หลายรายการบนเครื่องพิมพ์เดียวกัน เช่น "FAX XXXXXX (USB)" สำหรับการเชื่อมต่อ USB และ "FAX XXXXXX(IP)" สำหรับการเชื่อมต่อเครือข่าย ให้เลือกรายการที่คุณต้องการใช้เพื่อบันทึกแฟกซ์ที่ได้รับ
  • ถ้าเครื่องพิมพ์ของคุณมีฟังก์ชันด้านความปลอดภัยที่อนุญาตให้ผู้ดูแลระบบจำกัดผู้ใช้ไม่ให้เปลี่ยนแปลงการตั้งค่าแฟกซ์ของเครื่องพิมพ์ คุณอาจจำเป็นต้องป้อนรหัสผ่านผู้ดูแลระบบเพื่อดำเนินการต่อ
เลือก "Save" (save faxes on this computer) ("บันทึก" (บันทึกแฟกซ์ที่คอมพิวเตอร์เครื่องนี้)) จากนั้นคลิกที่ Save to (บันทึกลงใน) เพื่อระบุโฟลเดอร์ที่จะบันทึกแฟกซ์ที่ได้รับ
หมายเหตุ:
  • ระบบระบุโฟลเดอร์ดังต่อไปนี้ตามค่าเริ่มต้น
    \ผู้ใช้งาน\ชื่อล็อกอิน\PCFax\ชื่อเครื่องพิมพ์
  • เมื่อเครื่องพิมพ์ได้รับการแชร์บนเครือข่าย คุณสามารถเปลี่ยนแปลงการตั้งค่าต่างๆ จากคอมพิวเตอร์เครื่องใดก็ได้ที่อยู่บนเครือข่าย อย่างไรก็ตาม แฟกซ์สามารถบันทึกได้หนึ่งบัญชีต่อหนึ่งครั้งเท่านั้น ถ้ามีการตั้งค่า "Save" (save faxes on this computer) ("บันทึก" (บันทึกแฟกซ์ที่คอมพิวเตอร์เครื่องนี้)) และมีการเลือก Received Fax Output Settings (การตั้งค่าเอาต์พุตแฟกซ์ที่ได้รับ) บนคอมพิวเตอร์อีกเครื่องหนึ่ง หน้าต่าง Received Fax Output Settings (การตั้งค่าเอาต์พุตแฟกซ์ที่ได้รับ) จะแสดงข้อความ "Save faxes on another computer or print queue (บันทึกแฟกซ์ที่คอมพิวเตอร์หรือลำดับการพิมพ์อื่น)"
  • บนหน้าต่าง คุณสามารถเปลี่ยนไปยัง "Save" (save faxes on this computer) ("บันทึก" (บันทึกแฟกซ์ที่คอมพิวเตอร์เครื่องนี้)) ในขณะที่ไม่มีแฟกซ์เหลืออยู่บนเครื่องพิมพ์ ถ้ามีแฟกซ์เหลืออยู่บนเครื่องพิมพ์ ข้อความผิดพลาดจะแสดงขึ้นเพื่อแจ้งให้คุณทราบว่าคุณไม่สามารถเปลี่ยนแปลงการตั้งค่าได้
    ในขณะที่แชร์เครื่องพิมพ์ (แฟกซ์) เราขอแนะนำให้คุณสร้างโฟลเดอร์บนเครือข่ายสำหรับการบันทึกแฟกซ์
  • คุณสามารถเปลี่ยนแปลงการตั้งค่า Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) ได้โดยการคลิกที่ Preferences (รายการตัวเลือก) บน Fax Receive Monitor (การตรวจสอบการรับแฟกซ์)
    Preferences (รายการตัวเลือก) (เฉพาะรุ่นที่มีคุณสมบัติการรับ PC-FAX)
คลิก Send (ส่ง) เพื่อส่งการตั้งค่าต่างๆ ไปยังเครื่องพิมพ์
หมายเหตุ:
เมื่อการยืนยันตัวตนผู้ใช้ของการรับ PC-FAX (ชื่อและรหัสผ่านสำหรับการแชร์ไฟล์) ได้รับการตั้งค่าในเครื่องพิมพ์ คุณจำเป็นต้องลงทะเบียนข้อมูลนั้นไปยังคีย์เชนของคอมพิวเตอร์
ถ้าข้อมูลไม่ได้ลงทะเบียนในคีย์เชน หน้าจอการลงทะเบียนจะปรากฏขึ้น ป้อนชื่อและรหัสผ่านของคุณ เลือก Remember this password in my keychain (จำรหัสผ่านนี้ในคีย์เชนของฉัน) จากนั้นคลิกที่ Connect (เชื่อมต่อ)
ถ้าคุณไม่ได้บันทึกข้อมูลการยืนยันตัวตนนี้ในคีย์เชน ระบบจะขอให้คุณป้อนข้อมูลนี้ทุกครั้งที่คุณได้รับแฟกซ์ และบันทึกไปยังคอมพิวเตอร์
