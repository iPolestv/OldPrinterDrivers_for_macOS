ปัญหาการรับส่งข้อมูล

ในกรณีที่เกิดปัญหาในการรับส่งข้อมูล คุณควรแก้ไขปัญหาดังนี้
ปัญหา
สิ่งที่ต้องทำ
FAX XXXXXX (เครื่องพิมพ์ของคุณ) ไม่แสดงในรายการเครื่องพิมพ์
ตรวจสอบให้แน่ใจว่าไดรเวอร์เครื่องพิมพ์และไดรเวอร์ PC-FAX ติดตั้งอยู่ และไดรเวอร์ PC-FAX ถูกเพิ่มไปยังรายการเครื่องพิมพ์
ปุ่ม Fax (แฟกซ์) เป็นสีเทาจาง (ปิดทำงาน)
ปุ่มนี้จะไม่สามารถใช้งานได้ถ้าไม่มีการเพิ่มผู้รับไปยัง Recipient List (รายการผู้รับ) ระบุผู้รับอย่างน้อยหนึ่งราย
ไม่สามารถส่งแฟกซ์จากเครื่องพิมพ์ได้
  • เมื่อ “No dial tone detected (ไม่พบสัญญาณการโทร)” แสดงขึ้น คุณต้องเลือก Fax Settings (การตั้งค่าแฟกซ์) - Dial Tone Detection Settings (การตั้งค่าการตรวจจับสัญญาณการโทร) และลบเครื่องหมายออกจากกล่องเครื่องหมาย Detect Dial Tone (ตรวจจับสัญญาณการโทร)
  • ตรวจสอบให้แน่ใจว่าคุณได้ตั้งค่า Your Phone Number (หมายเลขโทรศัพท์ของท่าน) ใน Header Information Settings (การตั้งค่าข้อมูลหัวกระดาษ) เครื่องแฟกซ์ของผู้รับอาจได้รับการกำหนดค่าให้บล็อกแฟกซ์ใดๆ ที่ไม่มีข้อมูลหมายเลขโทรศัพท์/แฟกซ์ของผู้ส่ง
สถานะของข้อมูลที่รับส่งไม่เปลี่ยนจาก Waiting for printer response (รอการตอบสนองของเครื่องพิมพ์)
Waiting for printer response (รอการตอบสนองของเครื่องพิมพ์) แสดงขึ้นเมื่อคอมพิวเตอร์ไม่สามารถสื่อสารกับเครื่องพิมพ์ (FAX) และจะยังคงสถานะนั้นต่อไปจนกว่าการสื่อสารจะสามารถทำได้ ตรวจสอบสิ่งต่อไปนี้
  • เครื่องพิมพ์เปิดอยู่หรือไม่
  • เครื่องพิมพ์และคอมพิวเตอร์เชื่อมต่อกันอย่างถูกต้องหรือไม่
  • ถ้าเครื่องพิมพ์เป็นเครื่องพิมพ์เครือข่าย มีการระบุที่อยู่เครือข่ายของเครื่องพิมพ์อย่างถูกต้องหรือไม่
  • เครื่องพิมพ์สามารถพิมพ์ได้ตามปกติหรือไม่
ถ้าทุกสิ่งที่กล่าวมาด้านบนถูกต้อง คุณต้องปิดเครื่องพิมพ์จากนั้นให้เปิดขึ้นมาใหม่อีกครั้ง
แอพพลิเคชันค้างหลังจากที่แสดงข้อความ “Cannot print other fax data because a transmission is being processed.”
แอพพลิเคชันบางตัวอาจหยุดทำงานขณะที่รับส่งแฟกซ์ โดยมีการแจ้งเตือนเกิดขึ้นมาพร้อมกัน (อาจคืนค่าหลังจากเวลาผ่านไปครู่หนึ่ง)
ด้านบน 

ปัญหาในการรับ (เฉพาะรุ่นที่มีคุณสมบัติการรับ PC-FAX)

หากเกิดปัญหาในการรับ คุณควรแก้ไขปัญหาดังนี้
ปัญหา
สิ่งที่ต้องทำ
ไอคอน Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) ไม่แสดงขึ้นบน FAX Utility (หน้าต่างหลัก)
ไอคอนแสดงเฉพาะเมื่อคุณเลือกเครื่องพิมพ์ที่สนับสนุนการรับ PC-FAX เท่านั้น
เชื่อมต่อเครื่องพิมพ์ (แฟกซ์) ที่สนับสนุนฟังก์ชั่นนี้ และเปิดเครื่องพิมพ์จากนั้นเลือกเครื่องพิมพ์ของคุณ
Received Fax Output Settings (การตั้งค่าเอาต์พุตแฟกซ์ที่ได้รับ) ได้รับการตั้งค่าให้บันทึกแฟกซ์บนคอมพิวเตอร์ แต่ไม่มีการบันทึกแฟกซใดๆ์
ระบบจะบันทึกแฟกซ์ที่ได้รับไปยังโฟลเดอร์ที่ระบุเฉพาะเมื่อ Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) กำลังทำงานอยู่เท่านั้น เราแนะนำให้คุณเปิดใช้งาน Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) ไว้ตลอดเวลา
ไม่สามารถรับแฟกซ์ได้
  • เลือกกล่องเครื่องหมาย Auto Answer (การตอบรับอัตโนมัติ) ใน Fax Settings (การตั้งค่าแฟกซ์) - Auto Answer Settings (การตั้งค่าการตอบรับอัตโนมัติ) ถ้าไม่มีโทรศัพท์ภายนอกเชื่อมต่อไปยังเครื่องพิมพ์ เครื่องพิมพ์จะไม่สามารถรับแฟกซ์ได้โดยที่ไม่ตั้งค่า Auto Answer (การตอบรับอัตโนมัติ)
  • ตรวจสอบว่าเครื่องพิมพ์ไม่ได้อยู่ในสถานะผิดพลาด
ไม่สามารถเปลี่ยนแปลงการตั้งค่าเพื่อทำการพิมพ์แฟกซ์ที่ได้รับได้
คุณไม่สามารถเปลี่ยนแปลงการตั้งค่าได้จนกว่าแฟกซ์ทั้งหมดจะได้รับการบันทึกไปยังคอมพิวเตอร์แล้ว บนคอมพิวเตอร์ที่ได้รับการตั้งค่าไว้ให้บันทึกแฟกซ์ที่ได้รับ คุณต้องเปลี่ยนแปลงการตั้งค่าหลังจากการตรวจสอบแฟกซ์ใหม่ หรือเปลี่ยนแปลงการตั้งค่าโดยใช้แผงควบคุมของเครื่องพิมพ์
"Cannot find the received fax folder (ไม่สามารถค้นหาโฟลเดอร์แฟกซ์ที่ได้รับได้)" จะแสดงขึ้นมาเมื่อแชร์โฟลเดอร์บนเครือข่ายได้รับการกำหนดค่าเป็นโฟลเดอร์ของแฟกซ์ที่ได้รับ
ในการบันทึกในแชร์โฟลเดอร์บนเครือข่าย คุณจำเป็นต้องเชื่อมต่อไปยังโฟลเดอร์นั้นล่วงหน้า เลือกแถบเมนูบนเดสก์ทอป Go (ไป) - Connect to Server (เชื่อมต่อไปยังเซิร์ฟเวอร์) และเชื่อมต่อไปยังแชร์โฟลเดอร์บนเครือข่าย
หน้าต่าง Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) จะไม่แสดงขึ้นเมื่อ Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) เริ่มทำงานโดยอัตโนมัติเมื่อเข้าสู่ระบบ
หน้าต่าง Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) จะไม่แสดงขึ้นมาโดยอัตโนมัติเมื่อคอมพิวเตอร์ของคุณเข้าสู่ระบบ คลิกไอคอน Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) ในด็อคเพื่อเปิดหน้าต่าง แม้ว่าหน้าต่าง Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) จะไม่แสดงขึ้นมาโดยอัตโนมัติ แต่ก็ยังทำงานอยู่บนพื้นหลัง เช่น การตรวจสอบแฟกซ์ใหม่ เป็นต้น
หน้าจอเข้าระบบสำหรับการยืนยันตัวตนผู้ใช้แสดงขึ้นบนคอมพิวเตอร์ทุกครั้งที่คุณได้รับแฟกซ์
เมื่อใช้เครื่องพิมพ์ที่เชื่อมต่อไปยังเครือข่าย ถ้าการยืนยันตัวตนผู้ใช้ (ชื่อและรหัสผ่านสำหรับการแชร์ไฟล์) ของการรับ PC-FAX ได้รับการตั้งค่าในเครื่องพิมพ์ หน้าจอนี้จะแสดงขึ้นเมื่อคุณบันทึกแฟกซ์ที่ได้รับไปยังคอมพิวเตอร์ เหตุการณ์นี้จะเกิดขึ้นเมื่อข้อมูลการยืนยันตัวตนที่ตั้งค่าในเครื่องพิมพ์แตกต่างจากข้อมูลที่บันทึกในคีย์เชนอีกด้วย
ดำเนินการ Received Fax Output Settings (การตั้งค่าเอาต์พุตแฟกซ์ที่ได้รับ) อีกครั้ง จากนั้นป้อนข้อมูลการยืนยันตัวตนผู้ใช้ที่ถูกต้อง เมื่อหน้าจอสำหรับป้อนข้อมูลแสดงขึ้นมา โปรดจำไว้ว่าคุณต้องเลือก Remember this password in my keychain (จำรหัสผ่านนี้ในคีย์เชนของฉัน)

ข้อความผิดพลาด

เมื่อข้อความผิดพลาดแสดงขึ้น ให้แก้ไขปัญหาโดยทำตามขั้นตอนการแก้ไขปัญหาต่อไปนี้
ข้อความ
สิ่งที่ต้องทำ
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (ไม่สามารถรับข้อมูลเครื่องพิมพ์ (แฟกซ์) ได้ เปิด "รายการตัวเลือกการพิมพ์ & แฟกซ์" และเพิ่มไดรเวอร์ PC-FAX)
เพิ่มไดรเวอร์ PC-FAX ไปยังรายการเครื่องพิมพ์
คลิกที่ OK (ตกลง) ในกล่องโต้ตอบข้อความ กล่องโต้ตอบในการเพิ่มเครื่องพิมพ์จะแสดงขึ้น
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (ไม่สามารถเปิดบันทึกการรับส่งข้อมูลแฟกซ์ได้ เนื่องจากมีการใช้งานฟังก์ชัน FAX Utility อื่นอยู่ โปรดรอจนกว่าจะดำเนินการเสร็จสิ้น แล้วลองอีกครั้ง)
ฟังก์ชัน FAX Utility อื่นกำลังทำงานอยู่ ปิดฟังก์ชันอื่น จากนั้นลองอีกครั้ง
Cannot open the transmission record file. You have no access rights or the file does not exist. (เปิดไฟล์บันทึกการรับส่งข้อมูลไม่ได้ ท่านไม่มีสิทธิ์เข้าใช้งานหรือไม่มีไฟล์อยู่)
ตรวจสอบว่าคุณมีสิทธิ์เข้าใช้งานด้วยการอ่านการบันทึกการรับส่งข้อมูลหรือไม่
Failed to save the log file. Change the path and file name, and save it again. (การบันทึกล็อกไฟล์ (Log) ล้มเหลว เปลี่ยนที่อยู่ไฟล์และชื่อไฟล์ แล้วบันทึกอีกครั้ง)
ตรวจสอบปลายทางที่ใช้บันทึกล็อกไฟล์
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (การลบล็อกไฟล์ (Log) ล้มเหลว แอพพลิเคชันอื่นอาจใช้งานอยู่ หรือท่านไม่มีสิทธิ์เข้าใช้งาน หรือไฟล์อาจถูกลบไปแล้ว)
ถ้าล็อกไฟล์ยังอยู่ ให้ตรวจสอบการตั้งค่าสิทธิ์เข้าใช้งาน หรือปิดแอพพลิเคชันอื่น
A printer communication error occurred. (เกิดข้อผิดพลาดเกี่ยวกับการสื่อสารของเครื่องพิมพ์)
ปิดเครื่องพิมพ์ จากนั้นเปิดเครื่องใหม่ ถ้าไม่ได้ผล ให้รอสักครู่ จากนั้นลองอีกครั้ง
An error was detected while communicating with the printer. (ตรวจพบข้อผิดพลาดขณะสื่อสารกับเครื่องพิมพ์)
รอสักครู่ จากนั้นลองอีกครั้ง
Failed to write the backup file. There is not enough free space or it may be write protected. (การเขียนไฟล์สำรองข้อมูลล้มเหลว พื้นที่ว่างไม่เพียงพอหรืออาจมีการป้องกันการเขียนไว้)
ตรวจสอบพื้นที่ว่างของฮาร์ดดิสก์ของคุณ หรือนำการป้องกันการเขียนของไฟล์สำรองออก
Failed to write the backup file. The file specified is being used by another application. (การเขียนไฟล์สำรองข้อมูลล้มเหลว แอพพลิเคชันอื่นกำลังใช้งานไฟล์ที่ระบุอยู่)
โปรแกรมอื่นกำลังใช้งานไฟล์สำรองอยู่ ออกจากโปรแกรมอื่น
Cannot recognize the file format. The file may be corrupt or created by another application. (ไม่สามารถรับรู้รูปแบบไฟล์ได้ ไฟล์อาจเสียหายหรือสร้างโดยแอพพลิเคชันอื่น)
นามสกุลที่สนับสนุนสำหรับไฟล์สำรองคือ *.fpb เท่านั้น ตรวจสอบไฟล์สำรอง
Failed to read the backup file. The specified file is being used by another application. (การอ่านไฟล์สำรองข้อมูลล้มเหลว แอพพลิเคชันอื่นกำลังใช้งานไฟล์ที่ระบุอยู่)
โปรแกรมอื่นกำลังใช้งานไฟล์สำรองอยู่ ออกจากโปรแกรมอื่น
Processing canceled because an error occurred. (ยกเลิกการดำเนินการ เนื่องจากเกิดข้อผิดพลาด)
ถ้าการรีสตาร์ทเครื่องพิมพ์และคอมพิวเตอร์ของคุณใหม่ไม่ได้ช่วยแก้ปัญหา ให้ติดต่อฝ่ายสนับสนุนของ Epson สำหรับข้อมูลในการติดต่อ โปรดดูที่คู่มือเครื่องพิมพ์ของคุณ
