ข้อความผิดพลาดในการรับ (เฉพาะรุ่นที่มีคุณสมบัติการรับ PC-FAX)

ต่อไปนี้เป็นรายการของข้อความผิดพลาดที่อาจแสดงเมื่อได้รับแฟกซ์ และวิธีการจัดการกับปัญหา
ข้อความ
สิ่งที่ต้องทำ
A printer communication error occurred. (เกิดข้อผิดพลาดเกี่ยวกับการสื่อสารของเครื่องพิมพ์)
ปิดเครื่องพิมพ์ จากนั้นเปิดเครื่องใหม่ รอสักครู่ จากนั้นลองอีกครั้ง
The printer (fax driver) you are setting has been deleted. Add the printer (fax driver) in [System Preferences] - [Print & Fax]. (เครื่องพิมพ์ (ไดรเวอร์แฟกซ์) ที่ท่านกำลังตั้งค่าถูกลบออกไปแล้ว เพิ่มเครื่องพิมพ์ (ไดรเวอร์แฟกซ์) ใน [รายการตัวเลือกระบบ] - [พิมพ์ & แฟกซ์])
เพิ่มไดรเวอร์ PC-FAX ไปยังรายการเครื่องพิมพ์ คลิกที่ OK (ตกลง) ในกล่องโต้ตอบข้อความ กล่องโต้ตอบเพื่อเพิ่มเครื่องพิมพ์จะแสดงขึ้น
Change not accepted. Some printer settings have been changed by another computer, print queue, or the printer. Please try again. (ไม่ยอมรับการเปลี่ยนแปลง การตั้งค่าเครื่องพิมพ์บางรายการถูกเปลี่ยนแปลงโดยคอมพิวเตอร์ ลำดับการพิมพ์ หรือเครื่องพิมพ์อื่น โปรดลองอีกครั้ง)
Received Fax Output Settings (การตั้งค่าเอาต์พุตแฟกซ์ที่ได้รับ) ได้รับการเปลี่ยนแปลงจากคอมพิวเตอร์เครือข่าย จากลำดับการพิมพ์ หรือจากเครื่องพิมพ์อื่น รอสักครู่ จากนั้นลองอีกครั้ง
แฟกซ์ที่ได้รับสามารถบันทึกบนคอมพิวเตอร์ได้หนึ่งเครื่องเท่านั้น
Received faxes not saved due to an error. (บันทึกแฟกซ์ที่ได้รับไม่ได้ เนื่องจากมีข้อผิดพลาด)
ไม่สามารถบันทึกแฟกซ์ที่ได้รับในโฟลเดอร์ที่ระบุได้ ตรวจสอบการเชื่อมต่อไปยังโฟลเดอร์ที่ระบุเพื่อให้มั่นใจว่าสามารถใช้งานได้ และตรวจสอบว่าคุณมีสิทธิ์ในการเข้าถึงโฟลเดอร์นั้น
Processing canceled because an error occurred. (ยกเลิกการดำเนินการ เนื่องจากเกิดข้อผิดพลาด)
ถ้าการรีสตาร์ทเครื่องพิมพ์และคอมพิวเตอร์ของคุณใหม่ไม่ได้ช่วยแก้ปัญหา ให้ติดต่อฝ่ายสนับสนุนของ Epson สำหรับข้อมูลในการติดต่อ โปรดดทีู่คู่มือเครื่องพิมพ์ของคุณ
Cannot save faxes. User authentication failed. (ไม่สามารถบันทึกแฟกซ์ได้ การยืนยันตัวตนผู้ใช้ไม่ผ่าน)
ไม่สามารถบันทึกแฟกซ์ที่ได้รับ เนื่องจากการยืนยันตัวตนผู้ใช้ (ชื่อและรหัสผ่านสำหรับการแชร์ไฟล์) ของการรับ PC-FAX ไม่ได้รับการตั้งค่าในคีย์เชนของคอมพิวเตอร์อย่างถูกต้อง
ดำเนินการ Fax Output Setting (การตั้งค่าเอาต์พุตแฟกซ์ที่ได้รับ) อีกครั้ง จากนั้นป้อนข้อมูลการยืนยันตัวตนผู้ใช้ที่ถูกต้อง เมื่อหน้าจอการป้อนแสดงขึ้น โปรดจำไว้ว่าคุณต้องเลือก Remember this password in my keychain (จำรหัสผ่านนี้ในคีย์เชนของฉัน)
