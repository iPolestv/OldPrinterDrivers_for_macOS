Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม) - แก้ไขรายการโทรด่วน/การโทรกลุ่ม -

หน้าต่างนี้อนุญาตให้คุณลงทะเบียนหรือแก้ไขรายการโทรด่วนหรือการโทรกลุ่มของแฟกซ์สำหรับเครื่องพิมพ์ของคุณ
รายการ
คำอธิบาย
Entry Number (หมายเลขรายการ)
แสดงหมายเลขรายการของการโทรด่วน
Speed Dial (การโทรด่วน)
เลือกเพื่อสร้างหรือแก้ไขรายการการโทรด่วน
Group Dial (การโทรกลุ่ม)
เลือกเพื่อสร้างหรือแก้ไขรายการการโทรกลุ่ม คุณไม่สามารถสร้างการโทรกลุ่มได้ถ้าคุณไม่ได้สร้างรายการการโทรด่วน
Name/Group Name (ชื่อ/ชื่อกลุ่ม)
ป้อนชื่อการโทรด่วนหรือชื่อกลุ่มที่คุณต้องการลงทะเบียน
Index Word (ดัชนีคำ) *3
ป้อนดัชนีคำสำหรับผู้รับเพื่อที่คุณสามารถใช้เมื่อทำการค้นหา
Fax Number (หมายเลขแฟกซ์) *1
ป้อนหมายเลขแฟกซ์ของผู้รับ ถ้าคุณต้องการแยกหมายเลข เช่น การแยกรหัสเมืองออกจากหมายเลขส่วนตัว ให้ใช้ช่องว่างแทนที่จะใช้เครื่องหมายขีดกลาง (-) เนื่องจากเครื่องหมายขีดกลาง (-) ทำหน้าที่เป็นตัวหยุดชั่วคราวในขณะที่โทร รายการนี้จะแสดงขึ้นเมื่อคุณแก้ไขการโทรด่วน
Member (สมาชิก) *2
แสดงรายการของการโทรด่วนที่ลงทะเบียนไว้ เลือกการโทรด่วนที่คุณต้องการเพิ่มไปยังการโทรกลุ่ม จากนั้นคลิกที่ OK (ตกลง) รายการนี้จะแสดงขึ้นเมื่อคุณแก้ไขการโทรกลุ่ม
Fax Speed (ความเร็วแฟกซ์) *3
เลือกความเร็วแฟกซ์ที่ใช้สำหรับการส่งแฟกซ์ไปยังผู้รับ ระบบจะใช้ความเร็วแฟกซ์ที่ตั้งค่าสำหรับรายการโทรด่วนแต่ละรายการเฉพาะเมื่อมีการส่งแฟกซ์โดยตรงจากเครื่องพิมพ์เท่านั้น FAX Utility ไม่สนใจการตั้งค่าและจะใช้ความเร็วแฟกซ์ที่ตั้งค่าไว้บนเครื่องพิมพ์
OK (ตกลง)
บันทึกการตั้งค่า และปิดหน้าต่าง
Cancel (ยกเลิก)
ยกเลิกการลงทะเบียนหรือแก้ไข และปิดหน้าต่าง
*1 รายการที่จำเป็นต้องใส่สำหรับการลงทะเบียนการโทรด่วน
*2 เมื่อลงทะเบียนการโทรกลุ่ม ตรวจสอบให้แน่ใจว่ามีการเลือกการโทรด่วนหนึ่งหรือหลายรายการอยู่เสมอ
*3 รายการเหล่านี้อาจไม่แสดงขึ้นมา ขึ้นอยู่กับเครื่องพิมพ์
หมายเหตุ:
  • สำหรับข้อมูลเกี่ยวกับตัวอักษร และจำนวนตัวอักษรที่สามารถป้อนเข้าไปได้ โปรดดูทีู่ส่วนต่อไปนี้
    ตัวอักษรที่สามารถใช้ได้
  • จำนวนสูงสุดของรายการการโทรด่วนต่อหนึ่งกลุ่มจะแสดงขึ้นบนแถบที่อยู่ตรงส่วนล่างของหน้าต่างแก้ไขการโทรกลุ่ม
