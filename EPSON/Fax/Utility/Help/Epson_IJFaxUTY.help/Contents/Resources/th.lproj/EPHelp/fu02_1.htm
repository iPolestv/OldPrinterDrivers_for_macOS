การส่งเอกสารที่สร้างโดยใช้แอพพลิเคชันการประมวลผลคำหรือสเปรดชีต

สำหรับ Mac OS X 10.5.x หรือใหม่กว่า
สำหรับ Mac OS X 10.4.x
คุณสามารถส่งข้อมูลโดยตรง เช่น เอกสาร ภาพวาด และตารางต่างๆ ที่คุณสร้างขึ้นมาได้โดยการเลือกแฟกซ์เครื่องพิมพ์จากเมนู Print (พิมพ์) ของแอพพลิเคชันบางตัว
หมายเหตุ:
การอธิบายต่อไปนี้ใช้ TextEdit ซึ่งเป็นแอพพลิเคชันมาตรฐานของ Mac OS X เป็นตัวอย่าง การทำงานที่แท้จริงอาจแตกต่างจากนี้ ขึ้นอยู่กับแอพพลิเคชันที่คุณใช้ สำหรับรายละเอียดเพิ่มเติม โปรดดูที่คำแนะนำของแอพพลิเคชัน

สำหรับ Mac OS X 10.5.x หรือใหม่กว่า

การใช้แอพพลิเคชันในการสร้างเอกสารเพื่อส่งทางแฟกซ์
คลิกที่ Print (พิมพ์) จากเมนู File (ไฟล์) ในแอพพลิเคชันของคุณ
กล่องโต้ตอบ Print (พิมพ์) ในแอพพลิเคชันจะแสดงขึ้นมา
เลือก FAX XXXXXX (เครื่องพิมพ์ของคุณ) จาก Printer (เครื่องพิมพ์) จากนั้นคลิกที่ เพื่อแสดงการตั้งค่าอย่างละเอียด
ทำการตั้งค่าอย่างเหมาะสม
    • ระบุ 1 ใน Copies (สำเนา) แม้ว่าคุณระบุ 2 หรือมากกว่า แต่จะมีการส่งเพียงหนึ่งชุดเท่านั้น
    • การแฟกซ์สีอาจใช้ไม่ได้สำหรับกระดาษที่มีขนาดใหญ่กว่า A4, Letter หรือ Legal ขึ้นอยู่กับเครื่องพิมพ์ (แฟกซ์) แต่ละรุ่น
    • คุณสามารถส่งเอกสารได้มากที่สุด 100 หน้าในการส่งแฟกซ์หนึ่งครั้ง
เลือก Fax Settings (การตั้งค่าแฟกซ์) และทำการตั้งค่าที่เหมาะสม
สำหรับคำอธิบายในการตั้งค่าแต่ละรายการ โปรดดูที่คำแนะนำของไดรเวอร์ PC-FAX คุณสามารถแสดงคำแนะนำของไดรเวอร์ได้โดยการคลิก ? ที่มุมซ้ายล่างของหน้าจอ หลังจากที่เลือก Fax Settings (การตั้งค่าแฟกซ์)
เลือก Recipient Settings (การตั้งค่าผู้รับ) และระบุผู้รับ
การระบุผู้รับโดยตรง (ชื่อ หมายเลขแฟกซ์ เป็นต้น):
ป้อนข้อมูลที่จำเป็นในส่วน Add (เพิ่ม) จากนั้นคลิก + ระบบจะทำการเพิ่มผู้รับไปยัง Recipient List (รายการผู้รับ) ที่จะแสดงในส่วนบนของกล่องโต้ตอบ
หมายเหตุ:
ถ้าคุณเลือก Enter fax number twice (ใส่หมายเลขแฟกซ์สองครั้ง) ในการตั้งค่าไดรเวอร์ PC-FAX คุณจำเป็นต้องป้อนหมายเลขเดิมซ้ำอีกครั้ง
ถ้าสายแฟกซ์ที่เชื่อมต่ออยู่จำเป็นต้องมีการป้อนรหัสนำหน้าใช้งานสายนอก คุณต้องป้อนรหัสนำหน้าในกล่อง External Access Prefix (รหัสนำหน้าใช้งานสายนอก)
หมายเหตุ:
ถ้าเครื่องพิมพ์ของคุณมีการตั้งค่าชนิดสาย PBX และตั้งค่าให้ใช้ # (เครื่องหมายสี่เหลี่ยม) คุณต้องป้อน # (เครื่องหมายสี่เหลี่ยม) แทนการป้อนรหัสนำหน้าที่แท้จริง
การเลือกผู้รับ (ชื่อ หมายเลขแฟกซ์ เป็นต้น) จาก Address book (สมุดที่อยู่) หรือ Contacts (รายชื่อ):
ถ้ารายชื่อผู้รับบันทึกอยู่ในสมุดที่อยู่ คลิกที่ เลือกผู้รับจากรายการ คลิกที่ Add (เพิ่ม) จากนั้นคลิกที่ OK (ตกลง)
ถ้าสายแฟกซ์ที่เชื่อมต่ออยู่จำเป็นต้องมีการป้อนรหัสนำหน้าใช้งานสายนอก คุณต้องป้อนรหัสนำหน้าในกล่อง External Access Prefix (รหัสนำหน้าใช้งานสายนอก)
หมายเหตุ:
ถ้าเครื่องพิมพ์ของคุณมีการตั้งค่าชนิดสาย PBX และตั้งค่าให้ใช้ # (เครื่องหมายสี่เหลี่ยม) คุณต้องป้อน # (เครื่องหมายสี่เหลี่ยม) แทนการป้อนรหัสนำหน้าที่แท้จริง
ตรวจสอบการตั้งค่าผู้รับ และคลิกที่ Fax (แฟกซ์)
คลิกที่ Fax (แฟกซ์) เพื่อเริ่มการส่งข้อมูล ตรวจสอบให้แน่ใจว่าชื่อและหมายเลขแฟกซ์ของผู้รับนั้นถูกต้องก่อนที่จะส่ง
หมายเหตุ:
  • คลิกไอคอนเครื่องพิมพ์ใน Dock (ด็อค) เพื่อแสดงสถานะการรับส่งข้อมูล ในการหยุดการรับส่งข้อมูล เลือกข้อมูลที่คุณส่ง จากนั้นคลิกที่ Delete (ลบ)
  • ถ้าเกิดข้อผิดพลาดระหว่างการรับส่งข้อมูล ข้อความผิดพลาดจะแสดงขึ้น ตรวจสอบบันทึกการรับส่งข้อมูลจากหน้าต่าง Fax Transmission Record (บันทึกการรับส่งข้อมูลแฟกซ์)
    การตรวจสอบ / การลบบันทึกการรับส่งข้อมูล
    รายการผิดพลาดการรับส่งข้อมูล
  • เอกสารที่ประกอบด้วยขนาดกระดาษที่แตกต่างกันหลายขนาดอาจไม่สามารถทำการส่งได้อย่างถูกต้อง
