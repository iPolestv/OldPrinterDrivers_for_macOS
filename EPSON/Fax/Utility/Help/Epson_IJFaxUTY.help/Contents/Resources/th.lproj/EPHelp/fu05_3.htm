> หน้าแรก > Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)

การลงทะเบียนการโทรด่วนโดยการนำเข้าจาก Address Book (สมุดที่อยู่) หรือ Contacts (รายชื่อ)

คุณสามารถลงทะเบียนข้อมูลใน Address Book (สมุดที่อยู่) หรือ Contacts (รายชื่อ) ไปยังการโทรด่วน
เริ่ม FAX Utility
การเริ่มต้นใช้งาน FAX Utility
เลือก FAX XXXXXX (เครื่องพิมพ์ของคุณ) จาก Printer (เครื่องพิมพ์) จากนั้นคลิก Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)
รายการโทรด่วน/การโทรกลุ่มที่บันทึกในเครื่องพิมพ์ของคุณจะแสดงขึ้นมา
หมายเหตุ:
ถ้าเครื่องพิมพ์ของคุณมีฟังก์ชันด้านความปลอดภัยที่อนุญาตให้ผู้ดูแลระบบจำกัดผู้ใช้ไม่ให้เปลี่ยนแปลงการตั้งค่าแฟกซ์ของเครื่องพิมพ์ คุณอาจจำเป็นต้องป้อนรหัสผ่านผู้ดูแลระบบเพื่อดำเนินการต่อ
เพื่อเพิ่มรายการโทรด่วนใหม่โดยการนำเข้าข้อมูลจาก Address Book (สมุดที่อยู่) หรือ Contacts (รายชื่อ) คุณต้องเลือกบรรทัดที่ว่าง จากนั้นคลิกที่ เพื่อเขียนทับรายการที่มีอยู่ด้วยข้อมูล Address Book (สมุดที่อยู่) หรือข้อมูล Contacts (รายชื่อ) คุณต้องเลือกรายการเป้าหมาย และคลิกที่
เลือกข้อมูลที่คุณต้องการลงทะเบียนไปยังการโทรด่วน จากนั้นคลิกที่ Import (นำเข้า)
ระบบจะเพิ่มข้อมูลที่เลือกไปยัง Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)
หลังจากที่ทำการตั้งค่าทั้งหมดเสร็จแล้ว คลิกที่ ในหน้าต่าง Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)
ระบบจะบันทึกการโทรด่วนและการโทรกลุ่มที่แสดงในรายการไปยังเครื่องพิมพ์
หมายเหตุ
  • ถ้าข้อมูล Address Book (สมุดที่อยู่) หรือข้อมูล Contacts (รายชื่อ) ที่นำเข้ามาไม่มีข้อมูลชื่อ ระบบจะนำข้อมูลเข้ามาโดยไม่มีชื่อ
  • ถ้าข้อมูล Address Book (สมุดที่อยู่) หรือข้อมูล Contacts (รายชื่อ) ที่นำเข้ามาประกอบด้วยตัวอักษรที่ไม่ได้รับการสนับสนุนโดย Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม) ระบบจะนำข้อมูลเข้ามาโดยไม่มีชื่อ
    สำหรับข้อมูลเพิ่มเติมเกี่ยวกับตัวอักษรที่สนับสนุน โปรดดูที่ส่วนต่อไปนี้
    ตัวอักษรที่สามารถใช้ได้
  • ถ้าชื่อของข้อมูล Address Book (สมุดที่อยู่) หรือข้อมูล Contacts (รายชื่อ) ที่นำเข้ามามีตัวอักษรมากกว่า 31 ตัว ระบบจะนำเข้าเฉพาะตัวอักษร 30 ตัวแรกและจะลบส่วนที่เหลือไป
