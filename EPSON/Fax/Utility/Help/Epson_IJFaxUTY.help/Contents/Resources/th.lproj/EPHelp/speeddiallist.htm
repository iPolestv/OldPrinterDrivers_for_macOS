Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)

เมนู
แถบเครื่องมือ
Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม) อนุญาตให้คุณลงทะเบียนหรือแก้ไขการโทรด่วนหรือการโทรกลุ่มของแฟกซ์สำหรับเครื่องพิมพ์ของคุณ
หมายเหตุ:
  • หน้าต่างนี้จะแสดงขึ้นเมื่อคุณคลิกที่ Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม) ใน FAX Utility (หน้าต่างหลัก)
  • จำนวนสูงสุดของรายการที่ใช้ได้สำหรับเครื่องพิมพ์ (แฟกซ์) ที่เชื่อมต่อจะแสดงบนแถบที่ส่วนล่างของหน้าต่าง

เมนู

เมนู File (ไฟล์)

รายการ
คำอธิบาย
Get Speed Dials (รับการโทรด่วน)
อ่านข้อมูลการโทรด่วนหรือการโทรกลุ่มของแฟกซ์สำหรับเครื่องพิมพ์ของคุณ
Send Speed Dials (ส่งการโทรด่วน)
บันทึกการโทรด่วนและการโทรกลุ่มที่แสดงในรายการไปยังเครื่องพิมพ์
Import (นำเข้า)
From Address Book (จากสมุดที่อยู่)
นำเข้าข้อมูล Address Book (สมุดที่อยู่) ไปยังการโทรด่วนของเครื่องพิมพ์ของคุณ
Back up (สำรองข้อมูล)
Save Backup (กำลังบันทึกการสำรองข้อมูล)
บันทึกข้อมูลสำรองของการโทรด่วนและการโทรกลุ่มที่กำหนดค่าไว้
Restore Backup (กำลังคืนค่าการสำรองข้อมูล)
อ่านข้อมูลการโทรด่วนหรือการโทรกลุ่มที่สำรองไว้
Close (ปิด)
ปิดหน้าต่าง

เมนู Edit (แก้ไข)

รายการ
คำอธิบาย
Edit (แก้ไข)
แก้ไขการโทรด่วนหรือการโทรกลุ่มที่เลือกในรายการ
Delete (ลบ)
ลบการโทรด่วนหรือการโทรกลุ่มที่เลือกในรายการ
Delete All (ลบทั้งหมด)
ลบการโทรด่วน/การโทรกลุ่มทั้งหมดในรายการ
