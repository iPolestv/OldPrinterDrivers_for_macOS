Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) (เฉพาะรุ่นที่มีคุณสมบัติการรับ PC-FAX)

แถบเครื่องมือ
ไอคอนและข้อความบนรายการเครื่องพิมพ์
หน้าต่างนี้อนุญาตให้คุณตรวจสอบสถานะการรับแฟกซ์ และทำการตั้งค่าเอาต์พุตสำหรับแฟกซ์ที่ได้รับ
หมายเหตุ:
หน้าต่างนี้จะแสดงขึ้นเมื่อคุณคลิกที่ Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) ใน FAX Utility (หน้าต่างหลัก)

แถบเครื่องมือ

รายการ
คำอธิบาย
Open the received fax folder (เปิดโฟลเดอร์แฟกซ์ที่ได้รับ)
เปิดโฟลเดอร์ที่ใช้เพื่อบันทึกแฟกซ์ที่ได้รับ
Check new faxes (ตรวจสอบแฟกซ์ใหม่)
ตรวจสอบแฟกซ์ใหม่
Received Fax Output Settings (การตั้งค่าเอาต์พุตแฟกซ์ที่ได้รับ)
ตั้งค่าว่าจะบันทึกแฟกซ์ที่ได้รับบนคอมพิวเตอร์หรือไม่
Preferences (รายการตัวเลือก)
กำหนดค่ารายการตัวเลือกการตรวจสอบการรับแฟกซ์
ด้านบน 

ไอคอนและข้อความบนรายการเครื่องพิมพ์

ไอคอน
ข้อความ
คำอธิบาย
Waiting (กำลังรอ) / Checking for new faxes (กำลังตรวจสอบหาแฟกซ์ใหม่)
ตรวจสอบแฟกซ์ใหม่ หรือสแตนด์บาย
Received faxes saved successfully in the specified folder (บันทึกแฟกซ์ที่ได้รับในโฟลเดอร์ที่ระบุเรียบร้อยแล้ว)
ระบบจะบันทึกแฟกซ์ใหม่ที่เครื่องพิมพ์ (แฟกซ์) ได้รับไปยังโฟลเดอร์ที่กำหนดไว้แล้ว
The printer is not responding (เครื่องพิมพ์ไม่ตอบสนองการทำงาน)
ตรวจสอบการเชื่อมต่อระหว่างคอมพิวเตอร์และเครื่องพิมพ์ของคุณ หรือตรวจสอบว่าเครื่องพิมพ์เปิดอยู่
The printer is processing data (เครื่องพิมพ์กำลังประมวลผลข้อมูล)
เครื่องพิมพ์กำลังประมวลผลข้อมูลอื่นอยู่ โปรดรอ
Cannot find the received fax folder (ไม่สามารถค้นหาโฟลเดอร์แฟกซ์ที่ได้รับได้)
ตรวจสอบว่าชื่อของโฟลเดอร์ที่ระบุนั้นถูกต้อง ถ้าคุณกำลังใช้แชร์โฟลเดอร์บนเครือข่าย ตรวจดูให้แน่ใจว่าคุณมีสิทธิ์ในการเข้าถึงโฟลเดอร์นั้น
Failed to save faxes in the specified folder (การบันทึกแฟกซ์ในโฟลเดอร์ที่ระบุล้มเหลว)
ตรวจสอบว่าโฟลเดอร์ที่ระบุสามารถใช้งานได้ หรือคุณมีสิทธิ์ในการเข้าถึงโฟลเดอร์นั้น
A printer communication error occurred (เกิดข้อผิดพลาดเกี่ยวกับการสื่อสารของเครื่องพิมพ์)
ตรวจสอบการเชื่อมต่อระหว่างคอมพิวเตอร์และเครื่องพิมพ์ของคุณ หรือตรวจสอบว่าเครื่องพิมพ์เปิดอยู่ ถ้าคุณกำลังใช้การเชื่อมต่อเครือข่าย ตรวจสอบว่าคุณสามารถเข้าถึงเครื่องพิมพ์ผ่านเครือข่ายได้
The "FAXRECV" disk is being used by another application. (แอพพลิเคชันอื่นกำลังใช้ดิสก์ "FAXRECV" อยู่)
ปิดโปรแกรมที่กำลังใช้ดิสก์ "FAXRECV"
Cannot save faxes. User authentication failed. (ไม่สามารถบันทึกแฟกซ์ได้ การยืนยันตัวตนผู้ใช้ไม่ผ่าน)
ตรวจสอบการยืนยันตัวตนผู้ใช้ของผู้รับ PC-FAX (ชื่อและรหัสผ่านสำหรับการแชร์ไฟล์) ที่มีการตั้งค่าไว้ในเครื่องพิมพ์ ทำการ Received Fax Output Settings (การตั้งค่าเอาต์พุตแฟกซ์ที่ได้รับ) อีกครั้ง จากนั้นป้อนข้อมูลการยืนยันตัวตนผู้ใช้อย่างถูกต้อง โปรดจำไว้ว่าคุณต้องเลือก Remember this password in my keychain (จำรหัสผ่านนี้ในคีย์เชนของฉัน)
