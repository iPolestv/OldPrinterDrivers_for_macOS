> หน้าแรก > Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)

การบันทึกข้อมูลสำรองของการโทรด่วน

คุณสามารถบันทึกข้อมูลสำรองของการโทรด่วนและการโทรกลุ่มที่กำหนดค่าไว้ได้ ข้อมูลรายการโทรด่วน/การโทรกลุ่มอาจหายไปเนื่องจากเครื่องพิมพ์ทำงานผิดพลาด เราแนะนำให้คุณทำการสำรองข้อมูลไว้ทุกครั้งที่คุณอัพเดตรายการโทร Epson ไม่ขอรับผิดชอบต่อการสูญหายของข้อมูล สำหรับการสำรองหรือการคืนค่าข้อมูล และ/หรือการตั้งค่าต่างๆ แม้ว่าจะยังอยู่ในระหว่างระยะเวลารับประกันก็ตาม
หมายเหตุ:
ข้อมูลที่ระบุเป็น "In use by another function (กำลังถูกใช้โดยฟังก์ชันอื่น)" จะไม่ได้รับการบันทึก
เริ่ม FAX Utility
การเริ่มต้นใช้งาน FAX Utility
เลือก FAX XXXXXX (เครื่องพิมพ์ของคุณ) จาก Printer (เครื่องพิมพ์) จากนั้นคลิก Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)
รายการโทรด่วน/การโทรกลุ่มที่บันทึกในเครื่องพิมพ์ของคุณจะแสดงขึ้นมา
หมายเหตุ:
ถ้าเครื่องพิมพ์ของคุณมีฟังก์ชันด้านความปลอดภัยที่อนุญาตให้ผู้ดูแลระบบจำกัดผู้ใช้ไม่ให้เปลี่ยนแปลงการตั้งค่าแฟกซ์ของเครื่องพิมพ์ คุณอาจจำเป็นต้องป้อนรหัสผ่านผู้ดูแลระบบเพื่อดำเนินการต่อ
คลิก File (ไฟล์) - Back up (สำรองข้อมูล) - Save Backup (กำลังบันทึกการสำรองข้อมูล)
ระบุชื่อไฟล์และไดเรกทอรีของไฟล์ จากนั้นคลิก Save (บันทึก)
