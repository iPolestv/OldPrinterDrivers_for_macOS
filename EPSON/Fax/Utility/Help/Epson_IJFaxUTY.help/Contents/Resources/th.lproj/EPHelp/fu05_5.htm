> หน้าแรก > Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)

การคืนค่ารายการโทรด่วนที่สำรองไว้

คุณสามารถเรียกคืนค่าข้อมูลสมุดโทรศัพท์ของเครื่องพิมพ์ที่สำรองไว้ (รายการโทรด่วน/การโทรกลุ่ม) โดยการคืนค่าข้อมูลลงบนเครื่องพิมพ์
หมายเหตุ:
  • คุณไม่สามารถคืนค่าข้อมูลที่สำรองใน Windows Fax Utility
  • ถ้ารายการโทรด่วน/การโทรกลุ่มที่สำรองไว้ ประกอบด้วยรายการที่ไม่ได้รับการสนับสนุนโดยเครื่องพิมพ์ซึ่งคุณต้องการคืนค่ารายการ รายการเหล่านั้นจะไม่ได้รับการคืนค่า
  • ถ้ารายการโทรด่วน/การโทรกลุ่มที่สำรองไว้ ไม่ได้ประกอบด้วยรายการที่ได้รับการสนับสนุนโดยเครื่องพิมพ์ซึ่งคุณต้องการคืนค่าข้อมูล ข้อมูลจะได้รับการคืนค่าดังที่แสดงไว้ด้านล่าง
    Index Word (ดัชนีคำ): ว่าง
    Fax Speed (ความเร็วแฟกซ์): ค่าที่ตั้งล่วงหน้า
  • รายการใดๆ เช่น Name/Group Name (ชื่อ/ชื่อกลุ่ม) ที่มีตัวอักษรที่ไม่ได้รับการสนับสนุนโดยเครื่องพิมพ์ซึ่งคุณต้องการคืนค่าข้อมูล จะว่างเปล่าเมื่อข้อมูลได้รับการคืนค่ากลับมา
  • รายการใดๆ ที่ระบุว่า In use by another function (กำลังถูกใช้โดยฟังก์ชันอื่น) บน FAX Utility จะได้รับการเขียนทับด้วยข้อมูลที่คืนค่ามาตราบเท่าที่หมายเลขรายการตรงกัน
  • รายการใดๆ ที่กำลังใช้งานอยู่ และได้รับการป้องกันจะไม่เขียนทับด้วยข้อมูลที่คืนค่ามา
เริ่ม FAX Utility
การเริ่มต้นใช้งาน FAX Utility
เลือก FAX XXXXXX (เครื่องพิมพ์ของคุณ) จาก Printer (เครื่องพิมพ์) จากนั้นคลิก Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)
หมายเหตุ:
ถ้าเครื่องพิมพ์ของคุณมีฟังก์ชันด้านความปลอดภัยที่อนุญาตให้ผู้ดูแลระบบจำกัดผู้ใช้ไม่ให้เปลี่ยนแปลงการตั้งค่าแฟกซ์ของเครื่องพิมพ์ คุณอาจจำเป็นต้องป้อนรหัสผ่านผู้ดูแลระบบเพื่อดำเนินการต่อ
คลิก File (ไฟล์) - Back up (สำรองข้อมูล) - Restore Backup (กำลังคืนค่าการสำรองข้อมูล)
เลือกไฟล์ที่คุณต้องการคืนค่า
รายการที่คืนค่าจะแสดงในหน้าต่าง Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)
คลิกที่ ในหน้าต่าง
ระบบจะบันทึกข้อมูลสมุดโทรศัพท์ที่คืนค่ากลับมาไปยังเครื่องพิมพ์
