Preferences (รายการตัวเลือก) (เฉพาะรุ่นที่มีคุณสมบัติการรับ PC-FAX)

หน้าต่างนี้อนุญาตให้คุณกำหนดค่าการตั้งค่าการรับแฟกซ์ เช่น วิธีการแจ้งเตือนเมื่อได้รับแฟกซ์ใหม่
หมายเหตุ:
  • หน้าต่างนี้จะแสดงขึ้นเมื่อคุณคลิกที่ Fax Receive Monitor Preferences (รายการตัวเลือกการตรวจสอบการรับแฟกซ์) บน Fax Receive Monitor (การตรวจสอบการรับแฟกซ์)
  • การตั้งค่าเหล่านี้จะนำไปใช้กับเครื่องพิมพ์และลำดับการพิมพ์ทั้งหมดที่แสดงบน Fax Receive Monitor (การตรวจสอบการรับแฟกซ์)
รายการ
คำอธิบาย
Specify an interval to check for new faxes (ระบุช่วงเวลาในการตรวจสอบหาแฟกซ์ใหม่)
ระบุช่วงเวลาซึ่งจะตรวจสอบหาแฟกซ์ใหม่ระหว่าง 1 ถึง 30 นาที
Notify me of new faxes via a dock icon (แจ้งเตือนฉันว่ามีแฟกซ์ใหม่ผ่านไอคอนด็อค)
เลือกคุณสมบัตินี้เพื่อทำให้ไอคอน Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) บนด็อคแสดงขึ้นมาเมื่อได้รับแฟกซ์ใหม่
Notify me of errors via a dock icon (แจ้งเตือนฉันว่ามีข้อผิดพลาดผ่านไอคอนด็อค)
เลือกคุณสมบัตินี้เพื่อทำให้ไอคอน Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) บนด็อคแสดงขึ้นมาเมื่อมีข้อผิดพลาดเกิดขึ้น คลิกไอคอนเพื่อแสดงหน้าต่างการตรวจสอบการรับแฟกซ์ซึ่งคุณสามารถตรวจสอบข้อมูลความผิดพลาดได้
Start at login (เริ่มต้นที่การเข้าสู่ระบบ )
เราแนะนำให้ตั้งค่านี้เป็น Automatic (อัตโนมัติ) เมื่อ "Save" (save faxes on this computer) ("บันทึก" (บันทึกแฟกซ์ที่คอมพิวเตอร์เครื่องนี้)) ได้รับการตั้งค่าใน Received Fax Output Settings (การตั้งค่าเอาต์พุตแฟกซ์ที่ได้รับ), Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) จะเริ่มทำงานโดยอัตโนมัติเมื่อคุณเข้าสู่ระบบ ถ้าคุณเลือก On (ใช้), Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) จะเริ่มทำงานเมื่อมีการเข้าสู่ระบบ โดยไม่คำนึงถึงการตั้งค่าเอาต์พุต ถ้าคุณเลือก Off (ไม่ใช้), Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) จะไม่เริ่มทำงานเมื่อคุณเข้าสู่ระบบ
Wait time before checking new faxes (เวลารอก่อนตรวจสอบหาแฟกซ์ใหม่)
รายการนี้ใช้ได้เมื่อตั้งค่าเป็น Automatic (อัตโนมัติ) หรือ On (ใช้) ใน Start at login (เริ่มต้นที่การเข้าสู่ระบบ ) ระบุว่าจะให้รอนานเท่าไร (0 ถึง 10 นาที) ก่อนที่จะตรวจสอบแฟกซ์ใหม่หลังจาก Fax Receive Monitor (การตรวจสอบการรับแฟกซ์) เริ่มทำงานเมื่อคุณเข้าสู่ระบบ คุณสมบัตินี้มีประโยชน์เมื่อโฟลเดอร์ซึ่งใช้บันทึกแฟกซ์อยู่บนเครือข่าย และคุณจำเป็นต้องใช้เวลาในการสร้างการเชื่อมต่อไปยังโฟลเดอร์หลังจากที่เข้าสู่ระบบ
หมายเหตุ:
ถ้าโฟลเดอร์ซึ่งใช้บันทึกแฟกซ์ที่ได้รับเป็นแชร์โฟลเดอร์บนเครือข่าย คุณสามารถเชื่อมต่อไปยังโฟลเดอร์เมื่อคุณเข้าสู่ระบบโดยการลงทะเบียนโฟลเดอร์ไปยังรายการเมื่อเข้าสู่ระบบ คลิก Apple - System Preferences (รายการตัวเลือกระบบ) - Account (บัญชี) - Login to register (เข้าสู่ระบบเพื่อลงทะเบียน)
