> หน้าแรก > Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)

การลงทะเบียนหรือการแก้ไขการโทรด่วนและการโทรกลุ่ม

คุณสามารถลงทะเบียนหรือแก้ไขการโทรด่วนหรือการโทรกลุ่มของแฟกซ์สำหรับเครื่องพิมพ์ของคุณได้
หลังจากที่ลงทะเบียนการโทรด่วนแล้ว คุณสามารถป้อนหมายเลขแฟกซ์อย่างง่ายดายเมื่อคุณส่งแฟกซ์จากเครื่องพิมพ์ หลังจากที่ลงทะเบียนการโทรกลุ่มแล้ว คุณสามารถระบุผู้รับหลายรายได้ในการทำงานครั้งเดียว
หมายเหตุ:
คุณไม่สามารถกำหนดค่าการโทรกลุ่มก่อนที่จะลงทะเบียนการโทรด่วนได้
เริ่ม FAX Utility
การเริ่มต้นใช้งาน FAX Utility
เลือก FAX XXXXXX (เครื่องพิมพ์ของคุณ) จาก Printer (เครื่องพิมพ์) จากนั้นคลิก Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)
รายการโทรด่วน/การโทรกลุ่มที่บันทึกในเครื่องพิมพ์ของคุณจะแสดงขึ้นมา
หมายเหตุ:
ถ้าเครื่องพิมพ์ของคุณมีฟังก์ชันด้านความปลอดภัยที่อนุญาตให้ผู้ดูแลระบบจำกัดผู้ใช้ไม่ให้เปลี่ยนแปลงการตั้งค่าแฟกซ์ของเครื่องพิมพ์ คุณอาจจำเป็นต้องป้อนรหัสผ่านผู้ดูแลระบบเพื่อดำเนินการต่อ
คุณต้องเลือกบรรทัดที่ว่างเพื่อสร้างรายการใหม่ เลือกรายการที่คุณต้องการแก้ไขเพื่อแก้ไขรายการที่มีอยู่
จากนั้นดับเบิลคลิกที่บรรทัด หรือกดปุ่ม Enter เพื่อเปิดกล่องโต้ตอบแก้ไข
หมายเหตุ:
  • เพื่อลบรายการที่มีอยู่ เลือกรายการที่คุณต้องการลบ จากนั้นกดปุ่มลบ หรือคลิกที่เมนู Edit (แก้ไข) จากนั้นคลิกที่ Delete (ลบ) อย่างไรก็ตาม การลบการโทรกลุ่มจะไม่ลบรายการโทรด่วนที่ใช้สำหรับสร้างกลุ่มขึ้นมา
  • คุณสามารถแก้ไขหรือลบรายการโทรด่วนที่ได้รับการป้องกันหรือรายการโทรกลุ่ม
กำหนดค่าการโทรด่วนหรือการโทรกลุ่ม
การกำหนดค่าการโทรด่วน:
เลือก Speed Dial (การโทรด่วน) ป้อน Name/Group Name (ชื่อ/ชื่อกลุ่ม), Index Word (ดัชนีคำ), Fax Number (หมายเลขแฟกซ์) และ Fax Speed (ความเร็วแฟกซ์) จากนั้นคลิกที่ OK (ตกลง) การโทรด่วนที่กำหนดค่าแล้วจะถูกเพิ่มไปยัง Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)
หมายเหตุ:
  • ถ้าระบบโทรศัพท์ของคุณจำเป็นต้องมีการใส่รหัสนำหน้าใช้งานสายนอก เช่น 0 หรือ 9 เพื่อโทรออกไปยังสายภายนอก ให้ป้อนรหัสนำหน้าที่ส่วนเริ่มต้นของหมายเลขแฟกซ์ ถ้าเครื่องพิมพ์ของคุณได้รับการตั้งค่าให้ใช้ # (เครื่องหมายสี่เหลี่ยม) คุณต้องป้อน # (เครื่องหมายสี่เหลี่ยม)แทนการป้อนรหัสนำหน้าที่แท้จริง
  • ระบบจะใช้ความเร็วแฟกซ์ที่ตั้งค่าสำหรับรายการโทรด่วนแต่ละรายการเฉพาะเมื่อมีการส่งแฟกซ์โดยตรงจากเครื่องพิมพ์เท่านั้น FAX Utility ไม่สนใจการตั้งค่าและจะใช้ความเร็วแฟกซ์ที่ตั้งค่าไว้บนเครื่องพิมพ์
การกำหนดค่าการโทรกลุ่ม:
เลือก Group Dial (การโทรกลุ่ม) ป้อน Name/Group Name (ชื่อ/ชื่อกลุ่ม) และ Index Word (ดัชนีคำ) เลือกกล่องเครื่องหมายของการโทรด่วนที่คุณต้องการเพิ่มไปยังการโทรกลุ่ม จากนั้นคลิก OK (ตกลง) ระบบจะเพิ่มการโทรกลุ่มที่กำหนดค่าแล้วไปยัง Speed Dial/Group Dial List (รายการโทรด่วน/การโทรกลุ่ม)
หมายเหตุ:
Index Word (ดัชนีคำ) และ Fax Speed (ความเร็วแฟกซ์) อาจใช้ไม่ได้ ขึ้นอยู่กับเครื่องพิมพ์แต่ละรุ่น
หลังจากที่ทำการตั้งค่าทั้งหมดเสร็จเรียบร้อยแล้ว คลิกที่
ระบบจะบันทึกการโทรด่วนและการโทรกลุ่มที่แสดงในรายการไปยังเครื่องพิมพ์
