Fax Receive Monitor (Faksin vastaanoton monitori) (Vain malleissa, joissa on PC-FAX-vastaanotto)

Työkalurivi
Tulostinluettelon kuvake ja viesti
Tämä ikkuna antaa tarkistaa faksin vastaanoton tilan ja asettaa vastaanotetuille fakseille tulostusasetuksia.
Huomautus:
Tämä ikkuna näkyy, kun napsautat Fax Receive Monitor (Faksin vastaanoton monitori) FAX Utility-ikkunassa (pääikkuna).

Työkalurivi

Kohde
Selitys
Open the received fax folder (Avaa vastaanotetun faksin kansio)
Avaa kansion, johon vastaanotetut faksit on tallennettu.
Check new faxes (Tarkista uudet faksit)
Tarkistaa uudet faksit.
Received Fax Output Settings (Vastaanotetun faksin tulostusasetukset)
Valita tallennetaanko vastaanotetut faksit tietokoneeseen.
Preferences (Asetukset)
Määrittää Faksin vastaanoton monitorin asetukset.

Tulostinluettelon kuvake ja viesti

Kuvake
Viesti
Selitys
Waiting (Odotetaan) / Checking for new faxes (Tarkistetaan uudet faksit)
Uusien faksien tarkistus tai valmiustila.
Received faxes saved successfully in the specified folder (Vastaanotetut faksit tallennettu onnistuneesti määritettyyn kansioon).
Uudet tulostimen (faksin) vastaanottamat faksit on tallennettu määritettyyn kansioon.
The printer is not responding (Tulostin ei vastaa Varmista)
Tarkista tietokoneen ja tulostimen välinen yhteys tai onko tulostin kytketty päälle.
The printer is processing data (Tulostin käsittelee tietoja)
Tulostin käsittelee muita tietoja. Odota.
Cannot find the received fax folder (Ei löydä vastaanotetun faksin kansiota).
Tarkista, onko määritetyn kansion nimi oikein. Jos käytät verkossa jaettua kansiota varmista, että sinulla on kansion käyttöoikeudet.
Failed to save faxes in the specified folder (Faksien tallennus määritettyyn kansioon epäonnistui).
Tarkista onko määritetty kansio käytettävissä tai onko sinulla käyttöoikeudet kansioon.
A printer communication error occurred (Tapahtui tulostimen yhteysvirhe)
Tarkista tietokoneen ja tulostimen välinen yhteys tai onko tulostin kytketty päälle. Jos käytät verkkoyhteyttä, tarkista voitko käyttää tulostinta verkon kautta.
The "FAXRECV" disk is being used by another application. (Toinen sovellus käyttää "FAXRECV"-levyä.)
Sulje ohjelma, joka käyttää "FAXRECV"-levyä.
Cannot save faxes. User authentication failed. (Fakseja ei voi tallentaa. Käyttäjän todentaminen epäonnistui.)
Tarkista PC-FAX-vastaanoton käyttäjän todentaminen (nimi ja salasana tiedoston jakamiseen), joka on asetettu tulostimeen, aseta Received Fax Output Settings (Vastaanotetun faksin tulostusasetukset) uudelleen ja syötä sitten käyttäjän todentamistiedot oikein. Muista valita Remember this password in my keychain (Muista tämä salasana avainnipussani).
