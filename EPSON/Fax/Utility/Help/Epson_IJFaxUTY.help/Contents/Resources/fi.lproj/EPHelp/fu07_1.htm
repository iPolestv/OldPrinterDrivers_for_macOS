Lähetysongelmat

Mikäli ilmenee lähetysongelma, suorita vianetsintä seuraavasti.
Ongelma
Mitä tehdä
FAX XXXXXX (tulostimesi) ei näy tulostinluettelossa.
Varmista, että tulostinohjain ja PC-FAX-ohjain on asennettu, ja että PC-FAX-ohjain on lisätty tulostinluetteloon.
Fax (Faks)-ipainike on himmennetty (pois käytöstä).
Tämä painike ei ole käytettävissä, jollei vastaanottajaa ole lisätty Recipient List (Vastaanottajaluettelo). Määritä vähintään yksi vastaanottaja.
Tulostimesta ei voi lähettää fakseja.
  • Kun "No dial tone detected (Valintaääntä ei havaittu)" näkyy näytössä, valitse Fax Settings (Faksiasetukset) - Dial Tone Detection Settings (Valintaäänen tunnistuksen asetukset) ja tyhjennä Detect Dial Tone (Tunnista valintaääni) -valintaruutu.
  • Varmista, että olet asettanut Your Phone Number (Puhelinnumerosi) Header Information Settings (Otsikkotietojen asetus) -kohdassa. Vastaanottajan faksilaite on voitu määrittää estämään kaikki faksit, jotka eivät sisällä lähettäjän puhelin-/faksinumerotietoja.
Lähetettyjen tietojen tila muutu tilasta Waiting for printer response (Odotetaan tulostimen vastausta).
Waiting for printer response (Odotetaan tulostimen vastausta) näkyy näytössä, kun tietokone ei pysty kommunikoimaan tulostimen (faksin) kanssa ja se pysyy näytössä, kunnes kommunikointi on taas mahdollista. Tarkista seuraava.
  • Onko tulostin kytketty päälle.
  • Onko tulostin ja tietokone liitetty oikein toisiinsa.
  • Jos tulostin on verkkotulostin, onko tulostimen verkko-osoite määritetty oikein.
  • Pystyykö tulostin tulostamaan normaalisti.
Jos kaikki yllä mainitut ovat oikein, sammuta tulostin ja käynnistä uudelleen.
Sovellus pysähtyy, kun näyttöön tulee näkyviin "Cannot print other fax data because a transmission is being processed.".
Jotkut sovellukset voivat pysähtyä faksia lähetettäessä, kun samanaikaisesti näytetään käynnistysvaroitus. (Ne voivat palautua hetken kuluttua.)

Vastaanotto-ongelmia (vain malleissa, joissa on PC-FAX-vastaanotto)

Jos ilmenee vastaanotto-ongelmia, suorita vianetsintä seuraavasti.
Ongelma
Mitä tehdä
Fax Receive Monitor (Faksin vastaanoton monitori) -kuvaketta ei näy FAX Utility-ikkunassa (pääikkuna).
Kuvake tulee näkyviin, kun valitset tulostimen, joka tukee PC-FAX-vastaanottoa.
Liitä tulostin (faksi), joka tukee tätä toimintoa ja kytke tulostin päälle ja valitse sitten toinen tulostin.
Received Fax Output Settings (Vastaanotetun faksin tulostusasetukset) on asetettu tallentamaan faksit tietokoneeseen, mutta fakseja ei ole tallennettu.
Vastaanotetut faksit tallennetaan ainoastaan määritettyyn kansioon, kun Fax Receive Monitor (Faksin vastaanoton monitori) on käynnissä. On suositeltavaa pitää Fax Receive Monitor (Faksin vastaanoton monitori) käynnissä.
Ei voi vastaanottaa fakseja.
  • Valitse Auto Answer (Automaattinen vastaus) -valintaruutu kohdassa Fax Settings (Faksiasetukset) - Auto Answer Settings (Automaattiset vastausasetukset). Jos ulkoista puhelinta ei ole yhdistetty tulostimeen, tulostin ei voi vastaanottaa fakseja ilman Auto Answer (Automaattinen vastaus) -asetusta.
  • Varmista, ettei tulostin ole virhetilassa.
Asetuksia ei voi asettaa niin, että vastaanotetut faksit tulostetaan.
Et voi muuttaa asetuksia ennen kuin kaikki vastaanotetut faksit on tallennettu tietokoneeseen. Muuta tietokoneessa, johon saapuneet faksit on asetettu tallennettavaksi, asetuksia tarkistettuasi uudet faksit tai muuta asetuksia käyttämällä tulostimen ohjauspaneelia.
"Cannot find the received fax folder (Ei löydä vastaanotetun faksin kansiota)." näkyy näytössä, kun verkossa jaettu kansio on määritetty vastaanotetun faksin kansioksi.
Tallentaaksesi verkossa olevaan jaettuun kansioon, yhteys on muodostettava etukäteen. Valitse työpöydän valikkopalkissa Go (Siirry) - Connect to Server (Muodosta yhteys palvelimeen) ja muodosta yhteys verkossa olevaan jaettuun kansioon.
Fax Receive Monitor (Faksin vastaanoton monitori) -ikkuna ei näy, kun Fax Receive Monitor (Faksin vastaanoton monitori) on käynnistynyt automaattisesti kirjauduttaessa.
Fax Receive Monitor (Faksin vastaanoton monitori) -ikkuna ei näyt automaattisesti, kun tietokoneeseen kirjaudutaan sisään. Napsauta Fax Receive Monitor (Faksin vastaanoton monitori) -kuvaketta Dock-kuvakkeella avataksesi ikkunan. Vaikka Fax Receive Monitor (Faksin vastaanoton monitori) -ikkuna ei näkyisi automaattisesti, se suorittaa taustatehtäviä, kuten uusien faksien tarkistamisen jne.
Käyttäjän todentamisen tietuenäyttö näkyy tietokoneessa joka kerta, kun vastaanotat faksin.
Kun käytetään tulostinta, joka on yhdistetty verkkoon, jos PC-FAX-vastaanoton käyttäjän todentaminen (nimi ja salasana tiedoston jakamiseen) on asetettu tulostimessa, tämä näyttö näytetään, kun tallennat vastaanotetun faksin tietokoneeseen. Näin käy myös, kun tulostimessa asetetut varmennustiedot eroavat avainnippuun tallennetuista tiedoista.
Tee Received Fax Output Settings (Vastaanotetun faksin tulostusasetukset) uudelleen ja syötä sitten oikeat käyttäjän varmennustiedot, kun tietuenäyttö tulee näkyviin. Muista valita Remember this password in my keychain (Muista tämä salasana avainnipussani).

Virheviestit

Kun virheviesti tulee näkyviin, ratkaise ongelma seuraamalla vianetsintäohjeita.
Viesti
Mitä tehdä
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (Tulostimen (faksin) tietoja ei voi saada. Mene kohtaan "Tulostus- & faksivalinnat" ja lisää PC-FAX-ohjain.)
Lisää PC-FAX-ohjain tulostinluetteloon.
Napsauta OK viestin valintaikkunassa. Lisää tulostin -valintaikkuna tulee näkyviin.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (Faksilähetyksen tietuetta ei voi avata, koska toinen FAX Utility -toiminto on jo käynnissä. Odota, kunnes se on päättynyt ja yritä uudelleen.)
Toinen FAX Utility-toiminto on käynnissä. Sulje toinen toiminto ja yritä sitten uudelleen.
Cannot open the transmission record file. You have no access rights or the file does not exist. (Lähetystietuetiedostoa ei voi avata. Sinulla ei ole pääsyoikeuksia tai tiedostoa ei ole.)
Tarkista onko sinulla lähetystietueen lukukäyttöoikeudet.
Failed to save the log file. Change the path and file name, and save it again. (Lokitiedoston tallennus epäonnistui. Muuta polku ja tiedoston nimi ja tallenna se uudelleen.)
Tarkista kohde, johon lokitiedosto on tallennettu.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (Lokitiedoston poisto epäonnistui. Toinen sovellus saattaa käyttää sitä tai sinulla ei ole pääsyoikeuksia tai se on ehkä jo poistettu.)
Jos lokitiedosto säilyy, tarkista käyttöoikeudet tai sulje muita sovelluksia.
A printer communication error occurred. (Tapahtui tulostimen yhteysvirhe.)
Kytke tulostin pois päältä ja takaisin päälle. Jos se ei toimi, odota hetki ja yritä sitten uudelleen.
An error was detected while communicating with the printer. (Havaittiin virhe kommunikoinnissa tulostimen kanssa.)
Odota hetki ja yritä uudelleen.
Failed to write the backup file. There is not enough free space or it may be write protected. (Varmuuskopiotiedoston kirjoitus epäonnistui. Vapaata tilaa ei ole tarpeeksi tai se saattaa olla kirjoitussuojattu.)
Tarkista kiintolevyn vapaa tila. Tai poista varmuuskopiotiedoston kirjoitussuojaus.
Failed to write the backup file. The file specified is being used by another application. (Varmuuskopiotiedoston kirjoitus epäonnistui. Toinen sovellus käyttää määritettyä tiedostoa.)
Toinen ohjelma käyttää varmuuskopiotiedostoa. Sulje toinen ohjelma.
Cannot recognize the file format. The file may be corrupt or created by another application. (Tiedostomuotoa ei voi tunnistaa. Tiedosto saattaa olla viallinen tai se on luotu toisella sovelluksella.)
Varmuuskopiotiedoston tuettu tunniste on vain *.fpb. Tarkista varmuuskopiotiedosto.
Failed to read the backup file. The specified file is being used by another application. (Varmuuskopiotiedoston kirjoitus epäonnistui. Toinen sovellus käyttää määritettyä tiedostoa.)
Toinen ohjelma käyttää varmuuskopiotiedostoa. Sulje toinen ohjelma.
Processing canceled because an error occurred. (Käsittely peruutettu, koska tapahtui virhe.)
Jos tulostimen ja tietokoneen uudelleenkäynnistys ei ratkaise ongelmaa, ota yhteys Epson-tukeen. Katso yhteystiedot tulostimen käyttöoppaasta.
