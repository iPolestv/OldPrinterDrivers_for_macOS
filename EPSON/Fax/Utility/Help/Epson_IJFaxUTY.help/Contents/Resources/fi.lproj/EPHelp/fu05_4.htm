Pikavalinnan varmuuskopion tallentaminen

Voit tallentaa määritetyn pikavalinnan ja ryhmävalinnan varmuuskopion. Pikavalinnan/ryhmävalinnan luettelon tiedot voivat kadota tulostimen toimintahäiriön vuoksi. On suositeltavaa varmuuskopioida tiedot aina valintaluetteloa päivitettäessä. Epson ei ole vastuussa mistään tietojen katoamisesta, tietojen varmuuskopioinnista tai palauttamisesta ja/tai asetuksista edes takuuaikana.
Huomautus:
Tietoja, joiden on ilmoitettu olevan "In use by another function (Käytössä toisessa toiminnossa)" ei tallenneta.
Käynnistä FAX Utility.
FAX Utility:n käynnistäminen
Valitse FAX XXXXXX (tulostimesi) kohdasta Printer (Tulostin) ja napsauta sitten Speed Dial/Group Dial List (Pikavalinnan/ryhmävalinnan luettelo).
Tulostimeen tallennettu pikavalinnan/ryhmävalinnan luettelo näkyy näytössä.
Huomautus:
Jos tulostimessa on suojaustoiminto, joka antaa järjestelmänvalvojan estää käyttäjiä muuttamasta tulostimen faksiasetuksia, sinun on annettava järjestelmänvalvojan salasana jatkaaksesi.
Napsauta File (Tiedosto) - Back up (Varmuuskopio) - Save Backup (Tallenna varmuuskopio).
Määritä tiedostonimi ja tiedostohakemisto ja napsauta sitten Save (Tallenna).
