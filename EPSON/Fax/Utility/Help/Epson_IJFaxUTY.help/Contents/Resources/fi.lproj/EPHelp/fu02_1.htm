Tekstinkäsittely- tai taulukkolaskentasovelluksella luotujen asiakirjojen lähettäminen

Mac OS X 10.5.x tai uudempi
Mac OS X 10.4.x
Valitsemalla tulostinfaksin jonkin sovelluksen Print (Tulosta)-valikosta, voit lähettää suoraan tietoja, kuten luomiasi asiakirjoja, piirroksia ja taulukoita.
Huomautus:
Seuraavassa selityksessä käytetään esimerkkinä TextEditiä, Mac OS X -käyttöjärjestelmän vakiosovellusta. Todellinen toiminta voi olla erilaista käyttämäsi sovelluksen mukaan. Katso lisätietoja sovelluksen ohjeesta.

Mac OS X 10.5.x tai uudempi

Luo sovelluksella faksilla lähetettävä asiakirja.
Napsauta Print (Tulosta) sovelluksesi File (Tiedosto)-valikossa.
Sovelluksen Print (Tulosta)-valintaikkuna tulee näkyviin.
Valitse FAX XXXXXX (tulostimesi) kohdasta Printer (Tulostin) ja napsauta sitten näyttääksesi yksityiskohtaiset asetukset.
Tee asianmukaiset asetukset.
    • Määritä 1 Copies (Kopioita)-kohdassa. Vaikka määrittäisit 2 tai useamman, vain yksi kopio lähetetään.
    • Tulostimen (faksin) mukaan värifaksaus ei ehkä ole käytettävissä paperikoolla, joka on suurempi kuin A4, Letter tai Legal.
    • Voit lähettää enintään 100 sivua yhdessä faksilähetyksessä.
Valitse Fax Settings (Faksiasetukset) ja tee asianmukaiset asetukset.
Katso kunkin asetuksen selitys PC-FAX-ohjaimen ohjeesta. Voit tuoda ohjaimen ohjeen näkyviin napsauttamalla ? näytössä vasemmalla alhaalla, kun olet valinnut Fax Settings (Faksiasetukset).
Valitse Recipient Settings (Vastaanottaja-asetukset) ja määritä vastaanottaja.
Vastaanottajan määrittäminen suoraan (nimi, faksinumero jne.):
Syötä tarvittavat tiedot Add (Lisää)-osassa ja napsauta +. Vastaanottaja on lisätty Recipient List (Vastaanottajaluettelo) valintaikkunan yläosassa.
Huomautus:
Jos olet valinnut Enter fax number twice (Anna faksinumero kaksi kertaa) PC-FAX-ohjainasetuksissa, sinun on syötettävä tämä numero uudelleen.
Jos yhdistetty faksilinja vaatii ulkoisen käytön etuliitteen, syötä etuliite External Access Prefix (Ulkoisen käytön etuliite) -ruutuun.
Huomautus:
Jos tulostimellasi on PBX-linjatyyppiasetus ja se on asetettu käyttämään #-merkkiä (hashtagia) tarkan etuliitteen asemesta, syötä # (hashtag).
Vastaanottajan valitseminen (nimi, faksinumero jne.) Address book (Osoitekirja)- tai Contacts (Yhteystiedot)-kohdasta:
Jos vastaanottaja on tallennettu osoitekirjaan, napsauta . Valitse vastaanottaja luettelosta, napsauta Add (Lisää) ja sitten OK.
Jos yhdistetty faksilinja vaatii ulkoisen käytön etuliitteen, syötä etuliite External Access Prefix (Ulkoisen käytön etuliite) -ruutuun.
Huomautus:
Jos tulostimellasi on PBX-linjatyyppiasetus ja se on asetettu käyttämään #-merkkiä (hashtagia) tarkan etuliitteen asemesta, syötä # (hashtag).
Tarkista vastaanottaja-asetukset ja napsauta Fax (Faksaa).
Napsauta Fax (Faksaa) käynnistääksesi lähetyksen. Varmista, että vastaanottajan nimi ja faksinumero ovat oikein ennen lähetystä.
Huomautus:
  • Napsauta tulostinkuvaketta Dock-kuvakkeella näyttääksesi lähetyksen tilan. Voit pysäyttää lähetyksen valitsemalla lähettämäsi datan ja napsauttamalla Delete (Poista).
  • Jos lähetyksen aikana ilmenee virhe, näyttöön tulee virheviesti. Tarkista lähetystallenne Fax Transmission Record (Faksilähetyksen tietue) -ikkunasta.

  • Asiakirjoja, jotka sisältävät useita eri paperikokoja, ei ehkä lähetetä oikein.
