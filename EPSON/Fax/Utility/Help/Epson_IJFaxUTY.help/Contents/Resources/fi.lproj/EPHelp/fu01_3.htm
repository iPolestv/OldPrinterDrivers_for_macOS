FAX Utility:n käynnistäminen

Mac OS X 10.6.x tai uudempi
Napsauta Apple - System Preferences (Järjestelmäasetukset) - Print & Scan (Tulosta ja skannaa) tai Print & Fax (Tulosta ja faksaa).
Valitse FAX XXXXXX (tulostimesi) kohdasta Printer (Tulostin) ja napsauta sitten Options & Supplies (Valinnat ja varusteet) - Utility (Apuohjelma) - Open Printer Utility (Avaa tulostinapuohjelma).
Mac OS X 10.5.x
Napsauta Apple - System Preferences (Järjestelmäasetukset) - Print & Fax (Tulosta ja faksaa).
Valitse FAX XXXXXX (tulostimesi) kohdasta Tulostin ja napsauta sitten Open Print Queue (Avaa tulostusjono) - Utility (Apuohjelma) .
Mac OS X 10.4.x
Kaksoisnapsauta Hard disk - Applications (Sovellukset) - Utilities - Printer Setup Utility (Tulostinasetusapuohjelma).
Valitse FAX XXXXXX (tulostimesi) kohdasta Tulostin ja napsauta sitten Utility (Apuohjelma).
