Consultation des nouveaux fax

Bien qu'il soit possible de consulter les nouveaux fax en ouvrant le dossier spécifié dans Received Fax Output Settings (Paramètres de sortie des fax reçus), la procédure suivante utilise Fax Receive Monitor (Moniteur de réception de fax). Cette méthode n'est disponible que si l'option "Save" (save faxes on this computer) ("Enregistrer" (enregistre les fax sur cet ordinateur)) est sélectionnée sur l'ordinateur.
Remarque :
si l'option Notify me of new faxes via a dock icon (M'informer des nouveaux fax via une icône du Dock) est sélectionnée dans les préférences de Fax Receive Monitor (Moniteur de réception de fax), l'icône Fax Receive Monitor (Moniteur de réception de fax) clignote pour vous avertir de la réception d'un nouveau fax.
Suivez la procédure suivante pour ouvrir le dossier contenant les fax reçus.
1.
Cliquez sur l'icône Fax Receive Monitor (Moniteur de réception de fax) figurant sur le Dock pour ouvrir Fax Receive Monitor (Moniteur de réception de fax).
2.
Sélectionnez une imprimante, puis cliquez sur Open the received fax folder (Ouvrir le dossier des fax reçus) ou cliquez deux fois sur l'imprimante.
Vérifiez la date et l'expéditeur dans le nom du fichier, puis ouvrez le fichier PDF.
Remarque :
  • les fax reçus sont automatiquement renommés dans le format suivant :
    YYYYMMDDHHMMSS_xxxxxxxxxx_nnnnn (Année/Mois/Jour/Heure/Minutes/Secondes_numéro de l'expéditeur)
    « Sender's number (numéro de l'expéditeur) » affiche les informations envoyées par l'expéditeur. Ce numéro ne s'affiche pas toujours car il dépend des paramètres de l'expéditeur.
  • Vous pouvez consulter les nouveaux fax instantanément en sélectionnant Check new faxes (Vérifier les nouveaux fax) dans Fax Receive Monitor (Moniteur de réception de fax).
