Transmission d'un document créé à l'aide d'un traitement de texte ou d'un tableur

Pour Mac OS X 10.5.x ou version supérieure
Pour Mac OS X 10.4.x
En sélectionnant une imprimante fax dans le menu Print (Imprimer) de certaines applications, vous pouvez transmettre directement des données, telles que des documents, des dessins et des tableaux que vous avez créés.
Remarque :
L'explication suivante utilise TextEdit, une application Mac OS X standard, en guise d'exemple. Le fonctionnement réel peut être être différent en fonction de l'application utilisée. Pour plus de détails, reportez-vous à l'aide de l'application.

Pour Mac OS X 10,5.x ou version supérieure

Créez un document à transmettre par fax à l'aide d'une application.
Cliquez sur Print (Imprimer) dans le menu File (Fichier) de l'application.
La boîte de dialogue Print (Imprimer) de l'application s'affiche.
Sélectionnez FAX XXXXXX (votre imprimante) sous Printer (Imprimante) et cliquez sur pour afficher les paramètres détaillés.
Définissez les paramètres adaptés.
    • Spécifiez 1 dans Copies. Seule une copie est transmise, même si vous sélectionnez 2 ou plus.
    • En fonction de l'imprimante (fax), l'envoi de fax en couleur n'est pas disponible pour les tailles de papier supérieures à A4, Letter ou Legal.
    • Vous pouvez envoyer un maximum de 100 pages au cours d'une transmission par fax.
Sélectionnez Fax Settings (Paramètres fax) et définissez les paramètres appropriés.
Pour obtenir des explications concernant les différents paramètres, reportez-vous à l'aide du pilote PC-FAX. Vous pouvez afficher l'aide du pilote en cliquant sur ?, dans la partie inférieure gauche de l'écran, une fois l'option Fax Settings (Paramètres fax) sélectionnée.
Sélectionnez Recipient Settings (Paramètres du destinataire) et spécifiez un destinataire.
Définition directe d'un destinataire (nom, numéro de fax, etc.) :
Saisissez les informations nécessaires dans la section Add (Ajouter) et cliquez sur +. Le destinataire est ajouté à la Recipient List (Liste des destinataires), affichée dans la partie supérieure de la boîte de dialogue.
Remarque :
Si vous avez sélectionné Enter fax number twice (Saisissez le numéro de fax deux fois) dans les paramètres de pilote PC-FAX, vous devez saisir de nouveau le même numéro.
Si la ligne fax connectée nécessite la saisie d'un préfixe d'accès externe, saisissez le préfixe dans la zone External Access Prefix (Préfixe d'accès externe).
Remarque :
Si votre imprimante dispose d'un paramètre de type de ligne PBX et a été définie pour utiliser # (dièse) au lieu d'entrer le code de préfixe exact, entrez # (dièse).
Sélection d'un destinataire (nom, numéro de fax, etc.) dans Address book (Carnet d'adresses) ou Contacts :
Si le destinataire est enregistré dans le carnet d'adresses, cliquez sur . Sélectionnez le destinataire dans la liste et cliquez sur Add (Ajouter), puis sur OK.
Si la ligne fax connectée nécessite la saisie d'un préfixe d'accès externe, saisissez le préfixe dans la zone External Access Prefix (Préfixe d'accès externe).
Remarque :
Si votre imprimante dispose d'un paramètre de type de ligne PBX et a été définie pour utiliser # (dièse) au lieu d'entrer le code de préfixe exact, entrez # (dièse).
Vérifiez les paramètres relatifs au destinataire et cliquez sur Fax (Faxer).
Cliquez sur Fax (Faxer) pour démarrer la transmission. Vérifiez que le nom et le numéro de fax du destinataire sont corrects avant de procéder à la transmission.
Remarque :
  • Cliquez sur l'icône d'imprimante dans le Dock pour afficher l'état de la transmission. Pour arrêter la transmission, sélectionnez les données envoyées et cliquez sur Delete (Supprimer).
  • Si une erreur survient lors de la transmission, un message d'erreur s'affiche. Consultez le registre de transmission dans la fenêtre Fax Transmission Record (Registre de transmission fax).

    Liste des erreurs de transmission
  • Les documents contenant plusieurs formats de papier différents risquent de ne pas être transmis correctement.

Pour Mac OS X 10.4.x

Créez un document à transmettre par fax à l'aide de l'application.
Cliquez sur l'option Page Setup (Mise en page) du menu File (Fichier) de l'application.
La boîte de dialogue Page Setup (Mise en page) de l'application s'affiche.
Sélectionnez FAX XXXXXX (votre imprimante) sous Printer (Imprimante), définissez les paramètres adaptés et cliquez sur OK.
Remarque :
En fonction de l'imprimante, l'envoi de fax en couleur n'est pas disponible pour les tailles de papier supérieures à A4, Letter ou Legal.
