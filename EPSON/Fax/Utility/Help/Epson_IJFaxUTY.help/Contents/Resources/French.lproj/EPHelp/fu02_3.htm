Synchronisation de Address Book (Carnet d'adresses) ou Contacts avec FAX Utility (pour Mac OS X 10.7.x ou versions ultérieures)

Si voulez sélectionner un destinataire à partir de Address Book (Carnet d'adresses) ou Contacts pour envoyer un fax via FAX Utility, vous devez le synchroniser à l'avance. Si vous mettez à jour Address Book (Carnet d'adresses) ou Contacts après une synchronisation, vous devez exécuter cette fonction de synchronisation à nouveau pour utiliser les données mises à jour avec FAX Utility.
Lancez FAX Utility
Démarrage de FAX Utility
Sélectionnez Synchroniser Address Book (Carnet d'adresses) ou Synchroniser Contacts.
Suivez les instructions à l'écran pour synchroniser.
