Paramètre d'enregistrement d'un fax reçu sur un ordinateur

Vous pouvez changer les paramètres de sortie de fax pour enregistrer les fax reçus au format PDF sur un ordinateur ou dans un dossier partagé sur un serveur (réception PC-FAX).
Remarque :
  • si votre imprimante prend en charge la réception PC-FAX, Fax Receive Monitor (Moniteur de réception de fax) s'affiche dans la fenêtre principale de FAX Utility.
  • Lorsque vous utilisez la réception PC-FAX, activez la réponse auto.
  • En fonction de l'imprimante, les données de fax reçu enregistrées sur l'ordinateur sont supprimées de la mémoire de l'imprimante.
  • Les fax reçus étant enregistrés au format PDF, vous devez utiliser les applications Cover Flow (10.5 ou version plus récente), Quick Look (10.5 ou version plus récente) ou Preview, toutes préinstallées avec Mac OS X.
Lancez FAX Utility.
Démarrage de FAX Utility
Cliquez sur Fax Receive Monitor (Moniteur de réception de fax).
Remarque :
  • il est impossible de démarrer plusieurs instances de Fax Receive Monitor (Moniteur de réception de fax) dans un environnement multi-utilisateur.
  • Vous ne pouvez importer et enregistrer un fax reçu que lorsque Fax Receive Monitor (Moniteur de réception de fax) est exécuté.
  • Fax Receive Monitor (Moniteur de réception de fax) est exécuté, son icône s'affiche sur le Dock. L'ajout de Fax Receive Monitor (Moniteur de réception de fax) au Dock permet de démarrer Fax Receive Monitor (Moniteur de réception de fax) facilement. Pour ajouter Fax Receive Monitor (Moniteur de réception de fax) au Dock, cliquez sur l'icône Fax Receive Monitor (Moniteur de réception de fax) tout en maintenant la touche Control enfoncée, puis cliquez sur l'option de menu Add to Dock (Ajouter au Dock).
Sélectionnez votre imprimante, puis cliquez sur Receive Settings (Paramètres de réception).
Remarque :
  • vous ne pouvez enregistrer les fax que dans une seule file d'attente d'impression (de fax) à la fois ; pour cette raison, si plusieurs files d'attente d'impression (de fax) correspondent à une imprimante donnée, par exemple « FAX XXXXXX (USB) » pour les connexions USB et « FAX XXXXXX(IP) » pour les connexions réseau, vous devez sélectionner celle qui doit être utilisée pour l'enregistrement des fax reçus.
  • Si votre imprimante dispose d'une fonction de sécurité qui permet aux administrateurs d'empêcher les utilisateurs de changer les paramètres de fax de l'imprimante, vous aurez peut-être besoin d'un mot de passe d'administrateur pour continuer.
Sélectionnez "Save" (save faxes on this computer) ("Enregistrer" (enregistre les fax sur cet ordinateur)), puis cliquez sur Save to (Enreg. sous) pour spécifier le dossier dans lequel enregistrer les fax reçus.
Remarque :
  • le dossier suivant est spécifié par défaut.
    \User\Login name\PCFax\Printer name (\utilisateur\nom de connexion\PCFax\nom d'imprimante)
  • Lorsque l'imprimante est partagée en réseau, vous pouvez modifier les paramètres à partir de tout ordinateur du réseau. Toutefois, les fax ne peuvent être enregistrés que dans un compte à la fois. Si l'option "Save" (save faxes on this computer) ("Enregistrer" (enregistre les fax sur cet ordinateur)) est définie et l'option Received Fax Output Settings (Paramètres de sortie des fax reçus) est sélectionnée sur un autre ordinateur, la fenêtre Received Fax Output Settings (Paramètres de sortie des fax reçus) affiche le message Save faxes on another computer or print queue (Enregistrer les fax sur un autre ordinateur ou file d'attente d'impression).
  • Cette fenêtre vous permet de remplacer la sélection par "Save" (save faxes on this computer) ("Enregistrer" (enregistre les fax sur cet ordinateur)) si aucun fax ne figure sur l'imprimante. Si des fax figurent encore sur l'imprimante, un message d'erreur s'affiche pour vous signaler que vous ne pouvez pas modifier le paramètre.
    En cas de partage de l'imprimante (fax), il est conseillé de créer un dossier réservé à l'enregistrement des fax sur le réseau.
  • Vous pouvez modifier le paramètre de Fax Receive Monitor (Moniteur de réception de fax) en cliquant sur Preferences (Préférences) dans Fax Receive Monitor (Moniteur de réception de fax).
    Preferences (Préférences) (modèles dotés de la réception PC-FAX uniquement)
Cliquez sur Send (Envoyer) pour envoyer les paramètres à l'imprimante.
Note:
Lorsque l'authentification utilisateur pour la réception PC-FAX (nom et mot de passe pour le partage de fichiers) est définie sur l'imprimante, vous devez enregistrer ces informations sur le trousseau de l'ordinateur.
Si ces informations ne sont pas enregistrées sur le trousseau, l'écran de saisie de l'enregistrement s'affiche. Saisissez votre nom et votre mot de passe, sélectionnez Remember this password in my keychain (Conserver ce mot de passe dans le trousseau), puis cliquez sur Connect (Connexion).
Si vous n'enregistrez pas ces informations d'authentification dans le trousseau, vous serez invité à les saisir à chaque fois que vous recevrez un fax et que vous l'enregistrerez sur l'ordinateur.
