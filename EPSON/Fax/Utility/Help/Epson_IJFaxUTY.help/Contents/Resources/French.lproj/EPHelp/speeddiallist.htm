Speed Dial/Group Dial List (Liste N˚ abrégés/N° grpés)

Menu
Toolbar (Barre d'outils)
Speed Dial/Group Dial List (Liste N˚ abrégés/N° grpés) vous permet d'enregistrer ou de modifier la numérotation abrégée et la numérotation groupée du fax de votre imprimante.
Remarque :
  • Cette fenêtre s'ouvre lorsque vous cliquez sur Speed Dial/Group Dial List (Liste N˚ abrégés/N° grpés) dans FAX Utility (fenêtre principale).
  • Le nombre maximal d'entrées disponibles pour l'imprimante connectée (fax) s'affiche dans la barre située en bas de de la fenêtre.

Menu

Menu File (Fichier)

Élément
Explication
Get Speed Dials (Lecture des numérotations rapides)
Permet de lire la numérotation abrégée ou groupée du fax de votre imprimante.
Send Speed Dials (Écriture des numérotations rapides)
Permet d'enregistrer dans l'imprimante les numérotations abrégées et groupées affichées dans la liste.
Import (Importer)
From Address Book (À partir du Carnet d’adresses)
Permet d'importer les données du Address Book (Carnet d'adresses) dans les numérotations abrégées de l'imprimante.
Back up (Sauvegarde)
Save Backup (Enregistrer la sauvegarde)
Permet d'enregistrer la sauvegarde des numérotations abrégées et groupées configurées.
Restore Backup (Restaurer la sauvegarde)
Permet de lire la numérotation abrégée ou groupée enregistrée.
Close (Fermer)
Permet de fermer la fenêtre.

Menu Edit (Modifier)

Élément
Explication
Edit (Modifier)
Permet de modifier la numérotation abrégée ou groupée sélectionnée dans la liste.
Delete (Supprimer)
Permet de supprimer la numérotation abrégée ou groupée sélectionnée dans la liste.
Delete All (Supprimer tout)
Permet de supprimer toutes les numérotations abrégées/groupées dans la liste.
