Enregistrement de numérotations abrégées par importation à parti de Address Book (Carnet d'adresses) ou Contacts

Vous pouvez enregistrer les informations de Address Book (Carnet d'adresses) ou Contacts dans la numérotation abrégée.
Lancez FAX Utility
Démarrage de FAX Utility
Sélectionnez FAX XXXXXX (votre imprimante) sous Printer (Imprimante) et cliquez sur Speed Dial/Group Dial List (Liste N˚ abrégés/N° grpés).
Les listes de numéros abrégés/groupés enregistrées au niveau de l'imprimante s'affichent.
Remarque :
Si votre imprimante dispose d'une fonction de sécurité qui permet aux administrateurs d'empêcher les utilisateurs de changer les paramètres de fax de l'imprimante, vous aurez peut-être besoin d'un mot de passe d'administrateur pour continuer.
Pour ajouter une entrée de numérotation abrégée en important les données de Address Book (Carnet d’adresses) ou Contacts, sélectionnez une ligne vide et cliquez sur . Pour écraser une entrée existante avec les données de Address Book (Carnet d’adresses) ou Contacts, sélectionnez l'entrée cible et cliquez sur .
Sélectionnez les données que vous souhaitez enregistrer dans la numérotation abrégée, puis cliquez sur Import (Importer).
Les données sélectionnées sont ajoutées à Speed Dial/Group Dial List (Liste N˚ abrégés/N° grpés).
Une fois le paramétrage terminé, cliquez sur dans la fenêtre Speed Dial/Group Dial List (Liste N˚ abrégés/N° grpés).
Les numérotations abrégées et groupées affichées dans la liste sont enregistrées dans l'imprimante.
Remarque
  • Si les données de Address Book (Carnet d’adresses) ou Contacts importées n'incluent aucune information au sujet du nom, elles sont importées sans nom.
  • Si les données importées à partir de Address Book (Carnet d’adresses) ou Contacts incluent des caractères non pris en charge par la fenêtre Speed Dial/Group Dial List (Liste N˚ abrégés/N° grpés), les données sont importées sans nom.
    Pour obtenir des d'informations sur les caractères pris en charge, reportez-vous à la section suivante.
    Caractères acceptables
  • Si le nom des données importées à partir de Address Book (Carnet d’adresses) ou Contacts contient 31 caractères ou plus, seuls les 30 premiers caractères sont importés.
