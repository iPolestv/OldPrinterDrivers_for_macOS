Received Fax Output Settings (Paramètres de sortie des fax reçus) (modèles dotés de la réception PC-FAX uniquement)

Cette fenêtre permet de choisir d'enregistrer ou non les fax reçus sur un ordinateur.
Remarque :
cette fenêtre s'affiche lorsque vous cliquez sur Received Fax Output Settings (Paramètres de sortie des fax reçus) dans Fax Receive Monitor (Moniteur de réception de fax).
Élément
Explication
Setting on the printer (Paramétrage de l'imprimante)
Affiche le paramètre actuel de sortie des fax reçus.
"Print" (Imprimer) *
Imprime les fax reçus sur l'imprimante.
Not save faxes (Ne pas enregistrer les fax) *
Sélectionnez cette option pour ne plus enregistrer de fax sur un ordinateur. Les fax reçus sont imprimés ou enregistrés dans la mémoire du produit en fonction des paramètres de l'imprimante.
"Save" (save faxes on this computer) ("Enregistrer" (enregistre les fax sur cet ordinateur))
Enregistre les fax reçus dans le dossier spécifié. Les fax sont enregistrés au format PDF.
Save to (Enreg. sous)
Spécifie le dossier dans lequel enregistrer les fax reçus.
Send the settings to the printer (Envoyer les paramètres à l'imprimante)
Transfère les paramètres de sortie des fax reçus à l'imprimante.
* Il est possible que ces éléments ne s'affichent pas en fonction de l'imprimante.
Lorsque l'authentification utilisateur pour la réception PC-FAX (nom et mot de passe pour le partage de fichiers) est définie sur l'imprimante, vous devez enregistrer ces informations sur le trousseau de l'ordinateur.
Si ces informations ne sont pas enregistrées sur le trousseau, l'écran de saisie de l'enregistrement s'affiche. Saisissez votre nom et votre mot de passe, sélectionnez Remember this password in my keychain (Conserver ce mot de passe dans le trousseau), puis cliquez sur Connect (Connexion).
Si vous n'enregistrez pas ces informations d'authentification dans le trousseau, vous serez invité à les saisir à chaque fois que vous recevrez un fax et que vous l'enregistrerez sur l'ordinateur.
