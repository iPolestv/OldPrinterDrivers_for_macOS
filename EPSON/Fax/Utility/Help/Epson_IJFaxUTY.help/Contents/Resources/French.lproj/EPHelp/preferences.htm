Preferences (Préférences) (modèles dotés de la réception PC-FAX uniquement)

Cette fenêtre vous permet de configurer les paramètres de réception de fax, notamment la manière dont vous êtes informé de la réception d'un nouveau fax.
Remarque :
  • Cette fenêtre s'affiche lorsque vous cliquez sur Fax Receive Monitor Preferences (Préférences du Moniteur de réception de fax) de Fax Receive Monitor (Moniteur de réception de fax).
  • Ces paramètres sont appliqués à l'ensemble des imprimantes et files d'attente d'impression affichées dans Fax Receive Monitor (Moniteur de réception de fax).
Élément
Explication
Specify an interval to check for new faxes (Indiquez un intervalle de recherche des nouveaux fax)
Indiquez un intervalle compris entre 1 et 30 minute(s) auquel rechercher la réception de nouveaux fax.
Notify me of new faxes via a dock icon (M'informer des nouveaux fax via une icône du Dock)
Sélectionnez cette option pour que l'icône de Fax Receive Monitor (Moniteur de réception de fax) figurant sur le Dock clignote à la réception d'un nouveau fax.
Notify me of errors via a dock icon (M'informer des erreurs via une icône du Dock)
Sélectionnez cette option pour que l'icône de Fax Receive Monitor (Moniteur de réception de fax) figurant sur le Dock clignote lorsqu'une erreur se produit. Cliquez sur l'icône pour afficher la fenêtre Fax Receive Monitor (Moniteur de réception de fax) dans laquelle vous pouvez consulter la description de l'erreur.
Start at login (Démarrer à la connexion)
Il est conseillé de donner à ce paramètre la valeur Automatic (Automatique). Si l'option "Save" (save faxes on this computer) ("Enregistrer" (enregistre les fax sur cet ordinateur)) est sélectionnée dans Received Fax Output Settings (Paramètres de sortie des fax reçus), Fax Receive Monitor (Moniteur de réception de fax) démarre automatiquement lorsque vous vous connectez. Si vous sélectionnez On (Oui), Fax Receive Monitor (Moniteur de réception de fax) démarre lorsque vous vous connectez, indépendamment des paramètres de sortie. Si vous sélectionnez Off (Non), Fax Receive Monitor (Moniteur de réception de fax) ne démarre pas lorsque vous vous connectez.
Wait time before checking new faxes (Délai d'attente avant la vérification des nouveaux fax)
Cette option est disponible lorsque l'option Automatic (Automatique) ou On (Oui) est sélectionnée dans Start at login (Démarrer à la connexion). Indiquez la durée d'attente comprise en 0 et 10 minute(s) après laquelle rechercher les nouveaux fax suivant le démarrage de Fax Receive Monitor (Moniteur de réception de fax) lorsque vous êtes connecté. Cette option est utile lorsque le dossier dans lequel les fax sont enregistrés se trouve sur un réseau et qu'un certain temps est nécessaire pour établir une connexion avec le dossier une fois que vous êtes connecté.
Remarque :
si le dossier dans lequel les fax reçus sont enregistrés est un dossier partagé sur un réseau, vous pouvez vous connecter au dossier lorsque vous vous connectez en l'incluant à la liste des éléments de connexion. Cliquez sur Apple - System Preferences (Préférences système) - Account (Compte) - Login to register (Se connecter pour s'inscrire).
