Fax Transmission Record (Registre de transmission fax)

Menu
Cette fenêtre vous permet de consulter le journal des transmissions par fax.
Remarque :
  • Cette fenêtre s'ouvre lorsque vous cliquez sur Fax Transmission Record (Registre de transmission fax) dans FAX Utility (fenêtre principale).
  • FAX Utility enregistre uniquement des journaux de transmission. Les données transmises ne sont pas enregistrées, vous ne pouvez donc pas consulter les données envoyées.

Menu

Menu File (Fichier)

Élément
Explication
Close (Fermer)
Permet de fermer la fenêtre.
Save (Enregistrer)
Permet d'enregistrer les registres transmis. Les données enregistrées permettent d'analyser les problèmes qui surviennent lors de la transmission de fax.

Menu Edit (Modifier)

Élément
Explication
Delete log (Supprimer le journal)
Permet de supprimer tous les registres transmis.
