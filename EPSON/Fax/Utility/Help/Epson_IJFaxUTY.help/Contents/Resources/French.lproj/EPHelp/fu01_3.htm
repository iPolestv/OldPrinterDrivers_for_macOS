Démarrage de FAX Utility

Pour Mac OS X 10,6.x ou version supérieure
Cliquez sur Apple - System Preferences (Préférences système) - Print & Scan (Imprimantes et scanners) ou Print & Fax (Imprimantes et fax).
Sélectionnez FAX XXXXXX (votre imprimante) sous Printer (Imprimante), puis cliquez sur Options & Supplies (Options et fournitures) - Utility (Utilitaire) - Open Printer Utility (Ouvrir l'utilitaire d'impression).
Pour Mac OS X 10.5.x
Cliquez sur Apple - System Preferences (Préférences système) - Print & Fax (Imprimer & Faxer).
Sélectionnez FAX XXXXXX (votre imprimante) sous Printer (Imprimante), puis cliquez sur Open Print Queue (Ouvrir la file d'attente d'impression) - Utility (Utilitaire).
Pour Mac OS X 10.4.x
Cliquez deux fois sur le disque dur - Applications - Utilities (Utilitaires) - Printer Setup Utility (Utilitaire de configuration de l'imprimante).
Sélectionnez FAX XXXXXX (votre imprimante) sous Printer (Imprimante), puis cliquez sur Utility (Utilitaire).
