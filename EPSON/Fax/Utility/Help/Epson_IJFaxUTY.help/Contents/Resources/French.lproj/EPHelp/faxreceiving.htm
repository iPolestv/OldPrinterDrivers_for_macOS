Fax Receive Monitor (Moniteur de réception de fax) (modèles dotés de la réception PC-FAX uniquement)

Toolbar (Barre d'outils)
Icône et message de la liste des imprimantes
Cette fenêtre vous permet de vérifier l'état de la réception des fax et de définir les paramètres de sortie des fax reçus.
Remarque :
Cette fenêtre s'affiche lorsque vous cliquez sur Fax Receive Monitor (Moniteur de réception de fax) dans FAX Utility (fenêtre principale).

Toolbar (Barre d'outils)

Élément
Explication
Open the received fax folder (Ouvrir le dossier des fax reçus)
Ouvre le dossier dans lequel les fax reçus sont enregistrés.
Check new faxes (Vérifier les nouveaux fax)
Recherche les nouveaux fax.
Received Fax Output Settings (Paramètres de sortie des fax reçus)
Permet de choisir d'enregistrer ou non les fax reçus sur un ordinateur.
Preferences (Préférences)
Configure les préférences de Fax Receive Monitor (Moniteur de réception de fax).

Icône et message de la liste des imprimantes

Icône
Message
Explication
Waiting (Attente) / Checking for new faxes (Vérification des nouveaux fax)
Recherche les nouveaux fax ou les fax en attente.
Received faxes saved successfully in the specified folder (Fax reçus enregistrés avec succès dans le dossier indiqué).
Les nouveaux fax reçus par une imprimante (fax) ont été enregistrés dans le dossier spécifié.
The printer is not responding (L'imprimante ne répond pas)
Vérifiez la connexion entre votre ordinateur et votre imprimante ou vérifiez que l'imprimante est allumée.
The printer is processing data (L'imprimante traite les données)
L'imprimante traite d'autres données. Veuillez patienter.
Cannot find the received fax folder (Le dossier des fax reçus est introuvable)
Vérifiez que le nom du dossier spécifié est correct. Si vous utilisez un dossier partagé en réseau, vérifiez que vous disposez des droits d'accès appropriés.
Failed to save faxes in the specified folder (Echec de l'enregistrement des fax dans le dossier indiqué)
Vérifiez que le dossier spécifié est disponible ou que vous disposez des droits d'accès appropriés.
A printer communication error occurred (Erreur de communication au niveau de l'imprimante)
Vérifiez la connexion entre votre ordinateur et votre imprimante ou vérifiez que l'imprimante est allumée. Si vous utilisez une connexion réseau, vérifiez que vous pouvez accéder à l'imprimante via le réseau.
The "FAXRECV" disk is being used by another application. (Le disque "FAXRECV" est utilisé par une autre application.)
Fermez le programme qui utilise le disque « FAXRECV ».
Cannot save faxes. User authentication failed. (Enregistrement des télécopies impossible. Échec de l'authentification de l'utilisateur.)
Consultez les informations d'authentification utilisateur pour la réception PC-FAX (nom et mot de passe pour le partage de fichiers) définies sur l'imprimante, définissez les Received Fax Output Settings (Paramètres de sortie des fax reçus) une nouvelle fois, puis saisissez les informations d'authentification utilisateur correctement. Pensez à sélectionner Remember this password in my keychain (Conserver ce mot de passe dans le trousseau).
