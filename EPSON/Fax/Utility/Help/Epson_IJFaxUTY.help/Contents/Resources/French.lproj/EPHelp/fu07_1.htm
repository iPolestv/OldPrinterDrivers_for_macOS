Problèmes de transmission

En cas de problème de transmission, consultez le tableau suivant pour le résoudre.
Problème
Que faire
FAX XXXXXX (votre imprimante) ne s'affiche pas dans la liste des imprimantes.
Assurez-vous que le pilote d'imprimante et le pilote PC-FAX sont installés et que le pilote PC-FAX est ajouté à la liste des imprimantes.
Le bouton Fax est grisé (désactivé).
Ce bouton n'est disponible que si un destinataire est ajouté à Recipient List (Liste des destinataires). Sélectionnez au moins un destinataire.
Impossible d'envoyer des fax de l'imprimante.
  • Lorsque le message No dial tone detected (Pas de tonalité) s'affiche, sélectionnez Fax Settings (Paramètres fax) - Dial Tone Detection Settings (Paramètres Détection de la tonalité), puis désactivez la case à cocher Detect Dial Tone (Détecter la tonalité).
  • Vérifiez que Your Phone Number (Votre numéro de téléphone) est défini dans Header Information Settings (Paramètres Informations de l'en-tête). Il est possible que le télécopieur du destinataire soit configuré pour bloquer les fax qui n'indiquent pas le numéro de téléphone/fax de l'expéditeur.
L'état des données transmises reste Waiting for printer response (Attente rép. impr.)
Le message Waiting for printer response (Attente rép. impr.) s'affiche lorsque l'ordinateur ne peut pas communiquer avec l'imprimante (fax) et reste affiché jusqu'à ce que la communication soit possible. Vérifiez ce qui suit.
  • Si l'imprimante est sous tension.
  • Si l'imprimante et l'ordinateur sont correctement connectés.
  • Si l'imprimante est une imprimante réseau, si l'adresse réseau de l'imprimante est correctement indiquée.
  • Si l'imprimante peut imprimer normalement.
Si tout ce qui précède est correct, mettez l'imprimante hors tension puis sous tension.
L'application se fige après l'affichage du message suivant : Cannot print other fax data because a transmission is being processed. (Impossible d'imprimer d'autres données fax : une transmission est en cours de traitement.)
Certaines applications risquent de se figer au cours de la transmission d'un fax et un avertissement simultané de démarrage s'affiche. (Elles peuvent redémarrer après quelques instants.)

Réception d'erreurs (modèles dotés de la réception PC-FAX uniquement)

Résolvez le problème de la manière suivante en cas de problème de réception.
Problème
Que faire
L'icône de Fax Receive Monitor (Moniteur de réception de fax) ne s'affiche pas dans FAX Utility (fenêtre principale).
L'icône s'affiche lorsque vous sélectionnez une imprimante prenant en charge la réception PC-FAX.
Connectez une imprimante (fax) qui prend en charge cette fonction, puis allumez-la et sélectionnez-la.
Received Fax Output Settings (Paramètres de sortie des fax reçus) est défini de sorte à enregistrer les fax sur un ordinateur, pourtant aucun fax n'est enregistré.
Les fax reçus ne sont enregistrés dans le dossier spécifié que lorsque Fax Receive Monitor (Moniteur de réception de fax) est exécuté. Nous vous recommandons de ne pas interrompre l'exécution de Fax Receive Monitor (Moniteur de réception de fax).
Impossible de recevoir des fax.
  • Cochez la case Auto Answer (Réponse auto) dans Fax Settings (Paramètres fax) - Auto Answer Settings (Paramètres de Réponse auto). Si aucun téléphone externe n'est connecté à l'imprimante, celle-ci ne peut pas recevoir de fax sans la sélection du paramètre Auto Answer (Réponse auto).
  • Vérifiez que l'imprimante ne se trouve pas dans un état d'erreur.
Impossible de modifier les paramètres afin d'imprimer les fax reçus.
Les paramètres ne peuvent pas être modifiés tant que l'ensemble des fax reçus n'a pas été enregistré sur l'ordinateur. Sur un ordinateur configuré pour enregistrer les fax reçus, modifiez les paramètres après avoir vérifié les nouveaux fax ou modifiez-les à partir du panneau de contrôle de l'imprimante.
"Cannot find the received fax folder. (Le dossier des fax reçus est introuvable.) Ce message s'affiche lorsqu'un dossier partagé en réseau est configuré comme dossier des fax reçus.
Pour effectuer un enregistrement dans un dossier partagé en réseau, vous devez au préalable vous y connecter. Cliquez dans la barre de menus sur Go (Aller) - Connect to Server (Se connecter au serveur), puis connectez-vous au dossier partagé sur le réseau.
La fenêtre Fax Receive Monitor (Moniteur de réception de fax) ne s'affiche pas lorsque Fax Receive Monitor (Moniteur de réception de fax) est démarré automatiquement à la connexion.
La fenêtre Fax Receive Monitor (Moniteur de réception de fax) ne s'affiche pas automatiquement lorsque votre ordinateur se connecte. Cliquez sur l'icône de Fax Receive Monitor (Moniteur de réception de fax) dans le Dock pour ouvrir la fenêtre. Même si la fenêtre Fax Receive Monitor (Moniteur de réception de fax) ne s'affiche pas automatiquement, le moniteur effectue des opérations en arrière-plan telles que la vérification des nouveaux fax, etc.
L'écran de saisie pour l'authentification utilisateur s'affiche sur l'ordinateur à chaque fois que vous recevez un fax.
Lorsque vous utilisez une imprimante connectée au réseau, si les informations d'authentification de l'utilisateur pour la réception PC-FAX (nom et mot de passe pour le partage de fichiers) sont définies sur l'imprimante, cet écran s'affiche lorsque vous enregistrez le fax reçu sur l'ordinateur. Cela se produit également lorsque les informations d'authentification définies sur l'imprimante sont différentes de celles qui sont enregistrées dans le trousseau.
Définissez à nouveau les Received Fax Output Settings (Paramètres de sortie des fax reçus…), puis saisissez les informations d'authentification utilisateur correctes lorsque l'écran de saisie s'affiche. Pensez à sélectionner Remember this password in my keychain (Conserver ce mot de passe dans le trousseau).

Messages d'erreur

Si un message d'erreur s'affiche, résolvez le problème en respectant les consignes de dépannage.
Message
Que faire
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (Impossible d'obtenir les informations de l'imprimante (fax). Accédez à "Préférences de fax et d'impression" et ajoutez le pilote PC-FAX.)
Ajoutez le pilote PC-FAX à la liste des imprimantes.
Cliquez sur OK dans la zone d'affichage du message. La boîte de dialogue d'ajout d'imprimantes s'affiche.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (Impossible d'ouvrir le registre de transmission du fax : une autre fonction FAX Utility est déjà en cours d'exécution. Veuillez attendre la fin de la fonction et réessayer.)
Une autre fonction FAX Utility est en cours d'exécution. Fermez l'autre fonction et réessayez.
Cannot open the transmission record file. You have no access rights or the file does not exist. (Impossible d'ouvrir le fichier du registre de la transmission. Vous ne disposez pas de droits d'accès ou le fichier n'existe pas.)
Vérifiez que vous disposez de droits d'accès en lecture au registre de transmission.
Failed to save the log file. Change the path and file name, and save it again. (Échec de l'enregistrement du fichier journal. Modifiez le chemin et le nom du fichier et réessayez.)
Vérifiez la destination d'enregistrement du fichier journal.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (Échec de la suppression du fichier journal. Il est possible qu'il soit utilisé par une autre application, que vous ne disposiez pas de droits d'accès ou qu'il ait déjà été supprimé.)
Si un fichier journal n'est pas supprimé, vérifiez les droits d'accès ou fermez les autres applications.
A printer communication error occurred. (Erreur de communication au niveau de l'imprimante.)
Mettez l'imprimante hors tension, puis de nouveau sous tension. Si cela ne fonctionne pas, patientez un peu et réessayez.
An error was detected while communicating with the printer. (Une erreur a été détectée lors de la communication avec l'imprimante.)
Patientez un peu et réessayez.
Failed to write the backup file. There is not enough free space or it may be write protected. (Impossible de rédiger le fichier de sauvegarde. Il n'y a pas suffisamment d'espace disponible ou le fichier est protégé en écriture.)
Vérifiez la quantité d'espace disponible sur votre disque dur ou désactivez la protection en écriture du fichier de sauvegarde.
Failed to write the backup file. The file specified is being used by another application. (Échec de l'écriture du fichier de sauvegarde. Le fichier indiqué est déjà utilisé par une autre application.)
Le fichier de sauvegarde est utilisé par un autre programme. Quittez l'autre programme.
Cannot recognize the file format. The file may be corrupt or created by another application. (Impossible de reconnaître le format du fichier. Il est possible que le fichier soit corrompu ou ait été créé par une autre application.)
La seule extension prise en charge pour le fichier de sauvegarde est *.fpb. Vérifiez le fichier de sauvegarde.
Failed to read the backup file. The specified file is being used by another application. (Échec de la lecture du fichier de sauvegarde. Le fichier indiqué est déjà utilisé par une autre application.)
Le fichier de sauvegarde est utilisé par un autre programme. Quittez l'autre programme.
Processing canceled because an error occurred. (Traitement annulé en raison d'une erreur.)
Si un redémarrage de votre imprimante et de votre ordinateur ne résout pas le problème, contactez le service après-vente Epson. Pour obtenir des informations sur les personnes à contacter, reportez-vous au manuel de votre imprimante.
