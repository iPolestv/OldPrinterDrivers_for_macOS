Restauration d'une liste de numérotation abrégée sauvegardée

Vous pouvez restaurer des données de répertoire d'imprimante (listes de numérotations abrégées/groupées) sauvegardées sur l'imprimante.
Remarque :
  • Vous ne pouvez pas restaurer les données sauvegardées dans la version Windows de l'application Fax Utility.
  • Si la liste de numérotations abrégées/groupées sauvegardée contient des éléments non pris en charge par l'imprimante vers laquelle vous voulez restaurer la liste, ces éléments ne sont pas restaurés.
  • Si la liste de numérotations abrégées/groupées sauvegardée ne contient pas d'éléments pris en charge par l'imprimante vers laquelle vous voulez restaurer les données, ces dernières sont restaurées comme suit.
    Index Word (Mot d'index) : vide
    Fax Speed (Vitesse de fax) : valeur préconfigurée
  • Les éléments, tels que Name/Group Name (Nom/nom du groupe), comprenant des caractères non pris en charge par l'imprimante vers laquelle vous voulez restaurer les données sont vides quand ils sont restaurés.
  • Les entrées indiquées comme In use by another function (Utilisé par une autre fonction) sur FAX Utility sont écrasées par les données restaurées dans la mesure où le numéro d'entrée correspond.
  • Les entrées en cours d'utilisation ou protégées ne sont pas écrasées par les données restaurées.
Lancez FAX Utility
Démarrage de FAX Utility
Sélectionnez FAX XXXXXX (votre imprimante) sous Printer (Imprimante) et cliquez sur Speed Dial/Group Dial List (Liste N˚ abrégés/N° grpés).
Remarque :
Si votre imprimante dispose d'une fonction de sécurité qui permet aux administrateurs d'empêcher les utilisateurs de changer les paramètres de fax de l'imprimante, vous aurez peut-être besoin d'un mot de passe d'administrateur pour continuer.
Cliquez sur File (Fichier) - Back up (Sauvegarde) - Restore Backup (Restaurer la sauvegarde).
Sélectionnez le fichier que vous voulez restaurer.
La liste restaurée est affichée dans la boîte de dialogue Speed Dial/Group Dial List (Liste N˚ abrégés/N° grpés).
Cliquez sur dans la fenêtre.
Les données de répertoire restaurées sont enregistrées dans l'imprimante.
