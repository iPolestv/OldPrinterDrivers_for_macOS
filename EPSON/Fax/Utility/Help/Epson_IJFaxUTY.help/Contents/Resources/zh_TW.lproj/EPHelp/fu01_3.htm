啟動 FAX Utility

Mac OS X 10.6.x 以上版本
點選〝Apple 圖示〞-〝系統偏好設定〞-〝列印和掃描〞或〝列印與傳真〞。
從〝印表機〞選擇〝FAX XXXXXX〞(您的印表機)，然後點選〝選項與耗材〞-〝工具程式〞-〝打開印表機工具程式〞。
Mac OS X 10.5.x
點選〝Apple 圖示〞-〝系統偏好設定〞-〝列印與傳真〞。
從〝印表機〞選擇〝FAX XXXXXX〞(您的印表機)，然後點選〝打開列印佇列〞-〝工具〞。
Mac OS X 10.4.x
雙擊〝硬碟圖示〞-〝應用程式〞-〝工具程式〞-〝印表機設定工具程式〞。
從〝印表機〞選擇〝FAX XXXXXX〞(您的印表機)，然後點選〝工具〞。
