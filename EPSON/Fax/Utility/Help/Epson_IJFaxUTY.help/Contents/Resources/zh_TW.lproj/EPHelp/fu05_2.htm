登錄或編輯快速撥號和群組撥號

可以登錄或編輯印表機傳真的快速撥號或群組撥號。
登錄快速撥號後，從印表機端傳送傳真時，便可輕易地輸入傳真號碼。登錄群組撥號後，可以在同一個操作中指定多位收件人。
附註：
登錄快速撥號之前，無法設定群組撥號。
啟動 FAX Utility。
啟動 FAX Utility
從〝印表機〞選擇〝FAX XXXXXX〞(您的印表機)，然後點選〝快速撥號 / 群組撥號清單〞。
印表機中儲存的快速撥號 / 群組撥號清單會隨即顯示。
附註：
若您的印表機具有安全性功能，允許管理員限制使用者變更印表機的傳真設定，則您可能需要輸入管理員密碼才能繼續操作。
若要建立新的項目，請選擇空白行。若要編輯現有項目，請選擇您要編輯的項目。
然後雙擊該行或按下『Enter』鍵開啟編輯對話框。
附註：
  • 若要刪除現有項目，請選擇您要刪除的項目並按下刪除鍵，或點選〝編輯〞功能表並按下『刪除』。不過，刪除群組撥號不會刪除組成該群組的快速撥號。
  • 您無法編輯或刪除受保護的快速撥號項目或群組撥號項目。
設定快速撥號或群組撥號。
設定快速撥號：
選擇〝快速撥號〞，輸入〝名稱 / 群組名稱〞、〝索引詞〞、〝傳真號碼〞和〝傳真速度〞，然後按下『確定』鍵。設定的快速撥號會被新增至〝快速撥號 / 群組撥號清單〞清單中。
附註：
  • 若您的電話系統需要使用外接存取首碼 (如 0 或 9) 才可撥出外線電話，請在傳真號碼的開頭輸入首碼。若您的印表機設為使用 # (井字鍵)，而不是使用首碼，則請輸入 # (井字鍵)。
  • 當從印表機直接傳送傳真時，只會使用針對各快速撥號項目所設定的傳真速度。FAX Utility 會忽略設定值並使用在印表機上設定的傳真速度。
設定群組撥號：
選擇〝群組撥號〞，輸入〝名稱 / 群組名稱〞和〝索引詞〞，勾選欲加入至群組撥號的快速撥號檢查盒，然後按下『確定』鍵。設定的群組撥號會被新增至〝快速撥號 / 群組撥號清單〞清單中。
附註：
根據印表機而定，可能無法使用〝索引詞〞和〝傳真速度〞。
完成所有設定後，請按下『』圖示鍵。
顯示在清單中的快速撥號和群組撥號會被儲存至印表機中。
