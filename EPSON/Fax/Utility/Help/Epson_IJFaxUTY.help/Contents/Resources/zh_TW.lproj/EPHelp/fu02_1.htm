> 首頁 > 傳真傳送

傳送由文書處理或試算表應用程式所建立的文件

Mac OS X 10.5.x 以上版本
Mac OS X 10.4.x
從某些應用程式的〝列印〞功能表選擇印表機傳真後，您可直接傳送已建立的資料，例如文件、圖畫和表格。
附註：
下列說明以標準 Mac OS X 應用程式 TextEdit 為範例。實際的操作可能會因所使用的應用程式而不同。 詳細說明，請參考應用程式的輔助說明。

Mac OS X 10.5.x 以上版本

使用應用程式建立欲透過傳真傳送的文件。
從應用程式的〝檔案〞功能表中選擇〝列印〞。
應用程式的〝列印〞對話框會隨即顯示。
從〝印表機〞選擇〝FAX XXXXXX〞(您的印表機)，然後點選 顯示詳細設定。
進行適當的設定。
    • 在〝份數〞中指定〝1〞。 即使指定〝2〞或以上，也只能傳送一份。
    • 根據印表機 (傳真) 而定，紙張大小超過 A4、Letter 或 Legal 時，可能無法使用彩色傳真。
    • 單次傳真傳送最多可傳送 100 頁。
選擇〝傳真設定〞並進行適當的設定。
關於各設定的說明，請參考電腦傳真驅動程式輔助說明。 選擇〝傳真設定〞後按下畫面左下角的『?』，可顯示驅動程式輔助說明。
選擇〝收件人設定〞並指定收件人。
直接指定收件人 (姓名、傳真號碼等)：
在〝新增〞欄位中輸入必要的資訊，然後按下『+』。 收件人會被新增至對話框上方所顯示的〝收件人清單〞中。
附註：
若您在電腦傳真驅動程式設定中選擇〝輸入兩次傳真號碼〞，您必須重新輸入相同的號碼。
如果連接的傳真線路需要外接存取首碼，請在〝外接存取首碼〞方塊中輸入首碼。
附註：
若您的印表機具有 PBX 線路類型設定並設為使用 # (井字鍵)，而不是使用首碼，則請輸入 # (井字鍵)。
從〝通訊錄〞或〝聯絡人〞選擇收件人 (名稱、傳真號碼等)：
若收件人已儲存在通訊錄中，請按下『』。 從清單中選擇收件人、按下『新增』鍵，然後按下『確定』鍵。
如果連接的傳真線路需要外接存取首碼，請在〝外接存取首碼〞方塊中輸入首碼。
附註：
若您的印表機具有 PBX 線路類型設定並設為使用 # (井字鍵)，而不是使用首碼，則請輸入 # (井字鍵)。
檢查收件人設定並按下『傳真』。
按下『傳真』開始傳送。 傳送前，請確定收件人的姓名和傳真號碼正確。
附註：
  • 按下 Dock 工具列上的印表機圖示，即可顯示傳送狀態。 若要停止傳送，請選擇您傳送的資料並按下『刪除』。
  • 如果傳送時發生錯誤，將會顯示錯誤訊息。 從〝傳真傳送記錄〞視窗查看傳送記錄。

    傳送錯誤清單
  • 含有多種不同紙張大小的文件可能無法正確傳送。

Mac OS X 10.4.x

使用應用程式建立欲透過傳真傳送的文件。
從應用程式的〝檔案〞功能表中選擇〝設定頁面〞。
應用程式的〝設定頁面〞對話框會隨即顯示。
從〝印表機〞選擇〝FAX XXXXXX〞(您的印表機)、進行適當的設定，然後按下『確定』鍵。
附註：
根據印表機而定，紙張大小超過 A4、Letter 或 Legal 時，可能無法使用彩色傳真。
從應用程式的〝檔案〞功能表中選擇〝列印〞。 應用程式的〝列印〞對話框會隨即顯示。
從〝印表機〞選擇〝FAX XXXXXX〞(您的印表機)，然後進行適當的設定。
    • 在〝份數〞中指定〝1〞。 即使指定〝2〞或以上，也只能傳送一份。
    • 單次傳真傳送最多可傳送 100 頁。
選擇〝傳真設定〞並進行適當的設定。
關於各設定的說明，請參考電腦傳真驅動程式輔助說明。 選擇〝傳真設定〞後按下畫面左下角的『?』，可顯示驅動程式輔助說明。
選擇〝收件人設定〞並指定收件人。
直接指定收件人 (姓名、傳真號碼等)：
在〝新增〞欄位中輸入必要的資訊，然後按下『+』。 收件人會被新增至對話框上方所顯示的〝收件人清單〞中。
如果連接的傳真線路需要外接存取首碼，請在〝外接存取首碼〞方塊中輸入首碼。
附註：
若您的印表機具有 PBX 線路類型設定並設為使用 # (井字鍵)，而不是使用首碼，則請輸入 # (井字鍵)。
從通訊錄選擇收件人 (名稱、傳真號碼等)：
若收件人已儲存在通訊錄中，請按下『』圖示鍵。從清單中選擇收件人並按下『新增』鍵，然後再按下『好』鍵關閉清單視窗。
如果連接的傳真線路需要外接存取首碼，請在〝外接存取首碼〞方塊中輸入首碼。
附註：
  • 若您的印表機具有 PBX 線路類型設定並設為使用 # (井字鍵)，而不是使用首碼，則請輸入 # (井字鍵)。
  • 有關如何將收件人儲存至通訊錄的詳細說明，請參考通訊錄的輔助說明。
檢查收件人設定並按下『列印』。
按下『列印』開始傳送。 傳送前，請確定收件人的姓名和傳真號碼正確。
