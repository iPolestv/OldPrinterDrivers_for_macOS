启动FAX Utility

对于Mac OS X 10.6.x或更高版本
单击 Apple - 系统偏好 - 打印和扫描或打印和传真。
从打印机中选择FAX XXXXXX(您的打印机), 然后单击 选件与耗材 - 应用工具 - 打开应用工具。
对于 Mac OS X 10.5.x
单击Apple - 系统偏好 - 打印和传真。
从打印机中选择FAX XXXXXX(您的打印机），然后单击打开打印队列 - 应用工具。
对于Mac OS X 10.4.x
双击硬盘 - 实用程序 - 应用工具 - 打印机设置应用工具。
从打印机中选择FAX XXXXXX(您的打印机），然后单击应用工具。
