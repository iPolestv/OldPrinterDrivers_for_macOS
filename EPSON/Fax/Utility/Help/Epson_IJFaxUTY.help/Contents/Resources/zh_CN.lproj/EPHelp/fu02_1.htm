传输使用字处理程序或电子制表软件创建的文档

对于Mac OS X 10.5.x 或更高版本
对于Mac OS X 10.4.x
通过从某些应用程序的打印菜单中选择一台多功能传真一体机，您可直接地传输您创建的文档、图形和表格数据。
注释：
下面描述使用TextEdit,以标准Mac OS X 应用程序作为样例。 根据您使用的应用程序不同实际的操作可能不同。 有关详细信息，参见应用程序的帮助。

对于Mac OS X 10.5.x或更高版本

使用一个应用程序，创建一个要传真传输的文档。
从应用程序的文件菜单中单击打印。
显示应用程序的打印对话框。
从打印机中选择FAX XXXXXX(您的打印机），然后单击可显示详细设置。
进行合话的设置。
    • 指定份数1 。 即使您指定2或更多，也仅传输一份备份。
    • 根据打印机（传真）不同，可能对于大A4, 信纸或Legal尺寸的打印纸不能使用彩色传真。
    • 在一次传真传输可多达100页。
选择传真设置并进行合适的设置。
有关每一设置的描述，参见PC-FAX驱动程序帮助。 在选择传真设置后，通过单击屏幕左下部?的，您可看到驱动程序帮助。
选择收件人设置 并指定一个收件人。
直接地指定一个收件人（姓名，传真号等等）:
在添加部分输入必需的信息，然后单击+。 收件人被添加到窗口上部显示的收件人列表中。
注释：
如果您在PC-FAX驱动程序设置中选择输入传真号二次，您需要再次输入相同的号码。
如果连接的传真线路需要外线号码，在外部访问前缀中输入您拨外线前需要先拨的那个号码。
注释：
如果您的打印机有PBX线路类型设置并已设置前缀代码使用#（井号）来代替，输入#（井号）。
从地址簿或联系人中选择一个收件人 (姓名，传真号等) :
如果在地址簿中保存有收件人，单击。 从列表中选择收件人，单击添加，然后确定。
如果连接的传真线路需要外线号码，在外部访问前缀中输入您拨外线前需要先拨的那个号码。
注释：
如果您的打印机有PBX线路类型设置并已设置前缀代码使用#（井号）来代替，输入#（井号）。
检查收件人设置并单击传真。
单击传真可开始传输。 在传输前，确保收件人的姓名和传真号是正确的。
注释：
  • 在Dock中单击打印机，可显示传输状态。 要停止传输，选择您发送的数据并单击删除。
  • 如果在传输过程序发生错误，将显示错误信息。 从传真传输记录窗口中检查传输记录。
