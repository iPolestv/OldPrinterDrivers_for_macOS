Ripristino del backup dell'elenco di numeri rapidi

È possibile ripristinare il backup dei dati della rubrica della stampante (velocità/numeri di gruppo) e salvarli nuovamente nella stampante.
Nota:
  • Non è possibile ripristinare i dati di backup con Windows Fax Utility.
  • Se il backup dell'elenco numeri rapidi/numeri di gruppo contiene elementi che non sono supportati dalla stampante di cui si desidera ripristinare l'elenco, tali elementi non vengono ripristinati.
  • Se il backup dell'elenco numeri rapidi/numeri di gruppo non contiene elementi supportati dalla stampante di cui si desidera ripristinare i dati, tali elementi vengono ripristinati come indicato di seguito.
    Index Word (Parole di riferimento): vuoto
    Fax Speed (Velocità fax): valore preimpostato
  • Tutte le voci, come ad esempio Name/Group Name (Nome/Nome gruppo), che includono caratteri non supportati dalla stampante di cui si desidera ripristinare i dati, risultano vuote quando vengono ripristinate.
  • Tutte le voci indicate come In use by another function (In uso da altra funzione) in FAX Utility vengono sovrascritte dai dati ripristinati a condizione che il numero delle voci corrisponda.
  • Tutte le voci che vengono utilizzate e protette non vengono sovrascritte dai dati ripristinati.
Avviare FAX Utility.
Avvio di FAX Utility
Selezionare FAX XXXXXX (stampante in uso) da Printer (Stampante), quindi fare clic su Speed Dial/Group Dial List (Elenco num. rap./CompGruppo).
Nota:
Se la stampante dispone di una funzione di sicurezza che consente agli amministratori di impedire agli utenti di modificare le impostazioni fax della stampante, potrebbe essere necessario immettere una password amministratore per continuare.
Fare clic su File - Back up (Backup) - Restore Backup (Ripristina backup).
Selezionare il file da ripristinare.
L'elenco ripristinato viene visualizzato nella finestra di dialogo Speed Dial/Group Dial List (Elenco num. rap./CompGruppo).
Fare clic su nella finestra.
I dati ripristinati della rubrica vengono slavati nella stampante.
