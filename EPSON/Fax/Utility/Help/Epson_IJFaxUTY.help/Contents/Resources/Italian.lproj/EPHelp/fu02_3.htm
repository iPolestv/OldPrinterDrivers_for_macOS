Sincronizzazione di Address Book (Rubrica Indirizzi) o Contacts (Contatti) con FAX Utility (per Mac OS X 10.7.x o versione successiva)

Se si desidera selezionare un destinatario da Address Book (Rubrica Indirizzi) o Contacts (Contatti) per inviare un fax utilizzando FAX Utility, è necessario eseguire prima la sincronizzazione. Se si esegue l'aggiornamento di Address Book (Rubrica Indirizzi) o Contacts (Contatti) dopo la sincronizzazione, è necessario eseguire nuovamente la funzione di sincronizzazione per utilizzare i dati aggiornati con FAX Utility.
Avviare FAX Utility.
