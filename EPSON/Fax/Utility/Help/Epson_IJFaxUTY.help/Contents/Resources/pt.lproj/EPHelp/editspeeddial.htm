Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo) - Editar Marcação Rápida/Marcação de Grupo -

Nesta janela, pode registar ou editar uma entrada de marcação rápida ou de marcação de grupo do fax para a impressora.
Item
Explicação
Entry Number (Núm. Entrada)
Mostra o número de entrada da marcação rápida.
Speed Dial (Marc ráp)
Seleccione para criar ou editar uma entrada de marcação rápida.
Group Dial (Marc Grupo)
Seleccione para criar ou editar uma entrada de marcação de grupo. Só pode criar uma marcação de grupo se tiver criado entradas de marcação rápida.
Name/Group Name (Nome/Nome de grupo)
Introduza um nome para a marcação rápida ou de grupo que pretende registar.
Index Word (Palavra de índice) *3
Introduza uma palavra de índice para o destinatário para a utilizar quando efectuar pesquisas.
Fax Number (Número de fax) *1
Introduza o número de fax do destinatário. Se quiser separar números, por exemplo, separar códigos postais de números pessoais, utilize um espaço em vez de um hífen (-), porque um hífen (-) funciona como uma pausa aquando da marcação. Este item aparece quando edita a marcação rápida.
Member (Membro) *2
Para ver uma lista de marcações rápidas registadas. Seleccione a marcação rápida que pretende adicionar à marcação de grupo e depois clique em OK. Este item aparece quando edita a marcação de grupo.
Fax Speed (Velocidade do Fax) *3
Seleccione a velocidade do fax utilizada para enviar um fax para o destinatário. A velocidade do fax definida para cada entrada de marcação rápida apenas é utilizada quando enviar um fax directamente a partir da impressora. O FAX Utility ignora a definição e utiliza a velocidade do fax definida na impressora.
OK
Para guardar as definições e fechar a janela.
Cancel (Cancelar)
Para cancelar o registo ou a edição e fechar a janela.
*1 Item necessário para registar a marcação rápida.
*2 Ao registar uma marcação de grupo, certifique-se de que estão sempre uma ou mais marcações rápidas seleccionadas.
*3 Estes itens podem não ser apresentados dependendo da impressora.
Nota:
  • Para obter informações sobre os caracteres e o número de caracteres que pode introduzir, consulte a secção que se segue.
    Caracteres aceitáveis
  • O número máximo de entradas de marcação rápida por grupo é apresentado na barra na parte inferior da janela de edição da marcação de grupo.
