Restaurar uma lista de marcação rápida copiada

Pode restaurar os dados da agenda de uma impressora com uma cópia de segurança (listas de marcação rápida/grupo) na impressora.
Nota:
  • Não pode repor dados copiados com o Fax Utility em Windows.
  • Se a lista de marcação rápida/grupo incluir itens não suportados pela impressora onde pretende restaurar a lista, estes itens não serão restaurados.
  • Se a lista de marcação rápida/grupo copiada não incluir itens suportados pela impressora onde pretende restaurar os dados, estes serão restaurados como apresentado abaixo.
    Index Word (Palavra de índice): vazio
    Fax Speed (Velocidade do Fax): valor predefinido
  • Todos os itens, como Name/Group Name (Nome/Nome de grupo) que incluam caracteres não suportados pela impressora onde pretende restaurar os dados, estarão vazios quando restaurados.
  • As entradas indicadas como In use by another function (A ser utilizada por outra função) no FAX Utility são substituídas com os dados restaurados, desde que os números da entrada sejam correspondentes.
  • As entradas que estão a ser utilizadas e estão protegidas não são substituídas com os dados restaurados.
Iniciar o FAX Utility.
Iniciar o FAX Utility
Seleccione FAX XXXXXX (a sua impressora) Printer (Impres.) e depois clique em Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo).
Nota:
Se a sua impressora inclui uma função de segurança que permite aos administradores restringir os utilizadores de alterar as definições de fax da impressora, poderá ser necessário introduzir uma palavra-passe de administrador para continuar.
Clique em File (Fich) - Back up (Cópia) - Restore Backup (Repor cópia).
Seleccione o ficheiro que pretende repor.
A lista restaurada é apresentada na janela Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo).
Clique em na janela.
Os dados da agenda restaurados são guardados na impressora.
