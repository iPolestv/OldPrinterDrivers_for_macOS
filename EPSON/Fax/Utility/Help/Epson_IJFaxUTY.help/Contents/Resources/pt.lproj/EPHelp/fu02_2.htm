Verificar/apagar registos de transmissões

Pode verificar e apagar a lista de registos de transmissões por fax.
Iniciar o FAX Utility.
Iniciar o FAX Utility
Seleccione FAX XXXXXX (a sua impressora) a partir de Printer (Impres.) e depois clique em Fax Transmission Record (Registo transmissão fax).
Verifique os registos das transmissões.
A lista contém até 1000 registos de transmissões.
Para apagar os registos de transmissões por fax, seleccione Delete (Apagar) a partir do menu Edit (Editar). Todos os dados são apagados. (Não é possível eliminar cada registo individualmente.)
