Preferences (Preferências) (Apenas para modelos com recepção PC-FAX)

Esta janela permite configurar as definições de recepção de fax, por exemplo, como são realizadas as notificações quando um novo fax é recebido.
Nota:
  • Esta janela aparece quando clica em Fax Receive Monitor Preferences (Preferências do Ecrã de recepção de faxes) no Fax Receive Monitor (Ecrã de recepção de faxes).
  • Estas definições são aplicadas a todas as impressoras e filas de impressão apresentadas no Fax Receive Monitor (Ecrã de recepção de faxes).
Item
Explicação
Specify an interval to check for new faxes (Especifique um intervalo para procurar novos faxes)
Especifique um intervalo para procurar novos faxes, entre 1 e 30 minutos.
Notify me of new faxes via a dock icon (Notificar sobre novos faxes através de um ícone no Dock)
Seleccione para que o ícone Fax Receive Monitor (Ecrã de recepção de faxes) no Dock salte quando é recebido um novo fax.
Notify me of errors via a dock icon (Notificar sobre erros através de um ícone no Dock)
Seleccione para que o ícone Fax Receive Monitor (Ecrã de recepção de faxes) no Dock salte quando ocorrer um erro. Clique no ícone para apresentar a janela Ecrã de recepção de faxes onde pode consultar as informações de erro.
Start at login (Iniciar no início sessão)
Recomendamos que defina para Automatic (Automático). Quando "Save" (save faxes on this computer) ("Guardar" (guardar faxes neste computador)) está definido para Received Fax Output Settings (Definições de saída dos faxes recebidos), Fax Receive Monitor (Ecrã de recepção de faxes) é iniciado automaticamente ao iniciar a sessão. Se seleccionar On (Sim), o Fax Receive Monitor (Ecrã de recepção de faxes) inicia com o início da sessão, independentemente das definições de saída. Se seleccionar Off (Não), o Fax Receive Monitor (Ecrã de recepção de faxes) não inicia ao iniciar a sessão.
Wait time before checking new faxes (Tempo de espera antes de ver novos faxes)
Este item está disponível quando Automatic (Automático) ou On (Sim) estiver definido em Start at login (Iniciar no início sessão). Especifique o intervalo de espera (0 a 10 minutos) antes de verificar novos faxes, após Fax Receive Monitor (Ecrã de recepção de faxes) iniciar durante o início de sessão. Isto é útil quando a pasta em que guarda faxes está numa rede, e necessita de tempo para estabelecer uma ligação para a pasta após iniciar sessão.
Nota:
Se a pasta em que guarda os faxes recebidos for uma pasta partilhada numa rede, pode efectuar a ligação à pasta ao iniciar sessão, registando-a na lista de itens de início de sessão. Clique em Apple - System Preferences (Preferências do Sistema) - Account (Conta) - Login to register (Iniciar sessão para registar).
