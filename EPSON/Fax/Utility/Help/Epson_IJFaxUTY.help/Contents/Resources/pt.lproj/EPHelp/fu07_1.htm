Problemas de transmissão

Se ocorrer um problema de transmissão, procure resolvê-lo conforme explicado de seguida.
Problema
O que fazer
FAX XXXXXX (a sua impressora) não aparece na lista Impressora.
Certifique-se de que o controlador de impressão e o controlador PC-FAX estão instalados, e ainda que o controlador PC-FAX foi adicionado à lista de impressoras.
O botão Fax aparece a cinzento (desactivado).
Este botão apenas está disponível se for adicionado um destinatário à Recipient List (Lista destinatários). Especifique pelo menos um destinatário.
Não consegue enviar faxes a partir da impressora.
  • Se aparecer “No dial tone detected (Sem tom de marcação)”, seleccione Fax Settings (Definições de fax) - Dial Tone Detection Settings (Definições detecção tom marc.) e retire o visto da caixa de Detect Dial Tone (Detectar tom marc).
  • Certifique-se de que definiu Your Phone Number (Núm. de telefone) em Header Information Settings (Definições de dados cabeç.). O fax do destinatário pode estar configurado para bloquear todos os faxes que não incluam o número de telefone/fax do remetente.
O estado dos dados transmitidos mantém-se em Waiting for printer response (A aguardar resposta impres).
A mensagem Waiting for printer response (A aguardar resposta impres) aparece quando o computador não consegue comunicar com a impressora (FAX) e permanece inalterado até que a comunicação seja possível. Verifique o seguinte.
  • Se a impressora está ligada.
  • Se a impressora está ligada correctamente ao computador.
  • Se a impressora for uma impressora de rede, verifique se o endereço de rede foi especificado correctamente.
  • Se a impressora consegue imprimir normalmente.
Se todos os elementos acima estiverem correctos, desligue e volte a ligar a impressora.
A aplicação fica bloqueada depois de aparecer a mensagem “Cannot print other fax data because a transmission is being processed. (Não pode imprimir outros dados fax porque está a processar transmissão.)”.
Algumas aplicações podem ficar bloqueadas na altura de transmissão de um fax, aparecendo simultaneamente um aviso de arranque. (Podem recuperar algum tempo depois.)

Problemas de recepção (Apenas para modelos com recepção PC-FAX)

Se ocorrer um problema de recepção, procure resolvê-lo conforme explicado de seguida.
Problema
O que fazer
O ícone Fax Receive Monitor (Ecrã de recepção de faxes) não é apresentado no FAX Utility (janela principal).
O ícone é apresentado quando seleccionar a impressora que suporta recepção PC-FAX.
Ligue uma impressora (fax) que suporte esta função e, em seguida, seleccione-a.
As Received Fax Output Settings (Definições de saída dos faxes recebidos) estão configuradas para guardar faxes num computador, mas os faxes não são guardados.
Os faxes recebidos apenas são guardados na pasta especificada quando Fax Receive Monitor (Ecrã de recepção de faxes) estiver a ser executado. Recomendamos que mantenha o Fax Receive Monitor (Ecrã de recepção de faxes) em execução.
Não é possível receber faxes.
  • Seleccione a caixa de verificação Auto Answer (Resposta auto) em Fax Settings (Definições de fax) - Auto Answer Settings (Definições Resposta auto). Se não houver nenhum telefone externo ligado à impressora, a impressora não pode receber faxes sem configurar Auto Answer (Resposta auto).
  • Verifique se a impressora não se encontra num estado de erro.
Não é possível alterar as definições para imprimir os faxes recebidos.
Não pode alterar as definições até todos os faxes recebidos terem sido guardados no computador. Num computador configurado para receber faxes, altere as definições após verificar a existência de novos faxes ou altere as definições, utilizando o painel de controlo da impressora.
"Cannot find the received fax folder. (Não encontra a pasta do fax recebido.)" é apresentado quando uma pasta partilhada na rede está configurada como pasta do fax recebido.
Para guardar numa pasta partilhada numa rede, necessita de efectuar previamente a respectiva ligação. Seleccione a barra de menus no ambiente de trabalho Go (Ir) - Connect to Server (Ligar ao Servidor) e ligue à pasta partilhada na rede.
A janela Fax Receive Monitor (Ecrã de recepção de faxes) não é apresentada quando Fax Receive Monitor (Ecrã de recepção de faxes) tiver iniciado automaticamente durante o início de sessão.
A janela Fax Receive Monitor (Ecrã de recepção de faxes) não é apresentada automaticamente quando os computadores iniciam sessão. Clique no ícone Fax Receive Monitor (Ecrã de recepção de faxes) na Dock para abrir a janela. Mesmo no caso da janela Fax Receive Monitor (Ecrã de recepção de faxes) não ser apresentada automaticamente, está a realizar operações em segundo plano, tais como verificação de novos faxes, etc.
O ecrã de entrada da autenticação de utilizador é apresentado no computador de cada vez que recebe um fax.
Quando utiliza uma impressora ligada à rede, se a autenticação de recepção PC-FAX (nome e palavra-passe da partilha de ficheiros) estiver configurada na impressora, este ecrã é apresentado quando guarda o fax recebido no computador. Isto também acontece quando as informações de autenticação configuradas na impressora são diferentes das informações guardadas no porta-chaves.
Defina novamente as Received Fax Output Settings (Definições de saída dos faxes recebidos) e, em seguida, introduza as informações de autenticação de utilizador correctas quando o ecrã de entrada for apresentado. Lembre-se de seleccionar a opção Remember this password in my keychain (Guardar palavra-passe no porta-chaves).

Mensagens de erro

Se aparecer uma mensagem de erro, procure solucionar o problema seguindo as instruções de resolução de problemas.
Mensagem
O que fazer
Unable to obtain printer (fax) information. Open "Print & Fax Preferences" and add the PC-FAX driver. (Impossível obter dados da impressora (fax). Vá para "Impr. e Preferências de fax" e adicione o controlador PC-FAX.)
Adicione o controlador PC-FAX à lista de impressoras.
Clique em OK na caixa de mensagens. Aparece a caixa de diálogo para adicionar a impressora.
Cannot open Fax Transmission Record because another FAX Utility function is already running. Please wait until it has finished and try again. (Não pode abrir o Registo transmissão fax porque outra função do FAX Utility está em execução. Aguarde até que termine e tente de novo.)
Outra função do FAX Utility está em execução. Feche a outra função e tente de novo.
Cannot open the transmission record file. You have no access rights or the file does not exist. (O ficheiro de registo de transmissão de fax não abre. Não tem direitos de acesso ou o ficheiro não existe.)
Verifique se tem direitos de acesso de leitura do registo de transmissão.
Failed to save the log file. Change the path and file name, and save it again. (Falha a guardar o ficheiro de registo. Altere o caminho e o nome do ficheiro e volte a guardá-lo.)
Verifique o destino onde o ficheiro de registo é guardado.
Failed to delete the log file. Another application may be using it, or you have no access rights, or it may already have been deleted. (Falha a apagar o ficheiro de registo. É possível que outra aplicação esteja a usá-lo, que não tenha direitos de acesso ou que já tenha sido apagado.)
Se um ficheiro de registo continuar a existir, verifique as definições de direito de acesso ou feche outras aplicações.
A printer communication error occurred. (Ocorreu erro comunicação da impressora.)
Desligue a impressora e volte a ligá-la. Se não resultar, tente novamente passado algum tempo.
An error was detected while communicating with the printer. (Detectado erro ao comunicar c/ a impressora.)
Aguarde um pouco e tente de novo.
Failed to write the backup file. There is not enough free space or it may be write protected. (Falha a escrever no fich. cópia. Não há espaço ou pode estar protegido contra escrita.)
Verifique se existe espaço livre no disco rígido. Ou retire a protecção contra gravação do ficheiro de cópia.
Failed to write the backup file. The file specified is being used by another application. (Falha a gravar o ficheiro de cópia. O ficheiro especificado está a ser utilizado por outra aplicação.)
O ficheiro de cópia está a ser utilizado por outro programa. Feche o outro programa.
Cannot recognize the file format. The file may be corrupt or created by another application. (Não reconhece o formato de ficheiro. O ficheiro pode estar danificado ou ter sido criado por outra aplicação.)
A única extensão suportada para o ficheiro de cópia é *.fpb. Verifique o ficheiro de cópia.
Failed to read the backup file. The specified file is being used by another application. (Falha a ler o ficheiro de cópia. O ficheiro especificado está a ser utilizado por outra aplicação.)
O ficheiro de cópia está a ser utilizado por outro programa. Feche o outro programa.
Processing canceled because an error occurred. (Processamento cancelado porque ocorreu erro.)
Se não conseguir resolver o problema reiniciando a impressora e o computador, contacte a assistência técnica da Epson. Para ver as informações de contacto, consulte o manual da impressora.
