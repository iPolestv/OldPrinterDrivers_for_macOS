Guardar uma cópia da marcação rápida

Pode guardar uma cópia da marcação rápida e da marcação de grupo configuradas. Os dados da lista de marcação rápida/marcação de grupos podem-se perder devido a uma avaria da impressora. Recomendamos que faça uma cópia de segurança dos dados sempre que actualizar a lista de marcação. A Epson não se responsabiliza pela perda de quaisquer dados, para criar cópias de segurança ou recuperar dados e/ou definições durante um período de garantia.
Nota:
Os dados indicados como "In use by another function (A ser utilizada por outra função)" não são guardados.
Iniciar o FAX Utility.
Iniciar o FAX Utility
Seleccione FAX XXXXXX (a sua impressora) a partir de Printer (Impres.) e depois clique em Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo).
Aparecem as lista de marcação rápida/marcação de grupo guardadas na impressora.
Nota:
Se a sua impressora inclui uma função de segurança que permite aos administradores restringir os utilizadores de alterar as definições de fax da impressora, poderá ser necessário introduzir uma palavra-passe de administrador para continuar.
Clique em File (Fich) - Back up (Cópia) - Save Backup (Guardar cóp).
Especifique o nome do ficheiro e o directório do ficheiro e clique em Save (Guardar).
