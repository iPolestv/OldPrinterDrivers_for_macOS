Registar marcações rápidas através da importação do Address Book (Livro de endereços) ou Contacts (Contactos)

Pode registar a informação contida no Address Book (Livro de endereços) ou Contacts (Contactos) para marcação rápida.
Iniciar o FAX Utility.
Iniciar o FAX Utility
Seleccione FAX XXXXXX (a sua impressora) a partir de Printer (Impres.) e depois clique em Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo).
Aparecem as lista de marcação rápida/marcação de grupo guardadas na impressora.
Nota:
Se a sua impressora inclui uma função de segurança que permite aos administradores restringir os utilizadores de alterar as definições de fax da impressora, poderá ser necessário introduzir uma palavra-passe de administrador para continuar.
Para adicionar uma nova entrada de marcação rápida importando dados através do Address Book (Livro de endereços) ou Contacts (Contactos), seleccione uma linha vazia e clique em . Para substituir uma entrada já existente com dados do Address Book (Livro de endereços) ou Contacts (Contactos), seleccione a entrada alvo e clique em .
Seleccione os dados que pretende registar para marcação rápida e depois clique em Import (Importar).
Os dados seleccionados são adicionados à Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo).
Depois de efectuar todas as definições, clique em na janela Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo).
As marcações rápidas e marcações de grupo apresentadas na lista são guardadas na impressora.
Nota
  • Se os dados importados do Address Book (Livro de endereços) ou Contacts (Contactos) não contiverem informação de nomes, os dados são importados sem nome.
  • Se os dados importados do Address Book (Livro de endereços) ou Contacts (Contactos) incluírem caracteres que não são suportados pela Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo), os dados são importados sem nome.
    Para mais informações sobre os caracteres suportados, consulte a secção seguinte.
    Caracteres aceitáveis
  • Se o nome dos dados importados do Address Book (Livro de endereços) ou Contacts (Contactos) contiver 31 ou mais caracteres, os primeiros 30 caracteres são importados e os restantes são excluídos.
