FAX Utility (Janela Main (Princ))

Esta é a janela onde pode seleccionar a função do FAX Utility que pretende utilizar.
Nota:
Esta janela aparece quando inicia o FAX Utility.
Iniciar o FAX Utility
Item
Explicação
Fax Transmission Record (Registo transmissão fax)
Mostra os registos das transmissões.
Fax Transmission Record (Registo transmissão fax)
Fax Receive Monitor (Ecrã de recepção de faxes)
Apenas é apresentado quando uma impressora (fax) que suporta recepção PC-FAX está seleccionada. Permite seleccionar se pretende guardar os faxes recebidos no seu computador.
Check Fax Connection (Ver Ligação Fax)
Verifica se a função de fax da impressora ligada está a funcionar normalmente.
Fax Settings (Definições de fax)
Define o cabeçalho do fax e várias definições de comunicação de fax guardadas na impressora.
Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo)
Guarda ou edita a marcação rápida de fax ou a marcação de grupo de fax guardadas na impressora.
Sincronizar Address Book (Livro de endereços) ou Sincronizar Contacts (Contactos)
Sincroniza os dados do Address Book (Livro de endereços) ou Contacts (Contactos) com o FAX Utility para que possa seleccionar os destinatários da lista de contactos quando enviar um fax utilizando o FAX Utility. Se actualizar a Address Book (Livro de endereços) ou Contacts (Contactos) após sincronizar, tem de executar esta função de sincronização novamente para utilizar os dados actualizados com o FAX Utility.
