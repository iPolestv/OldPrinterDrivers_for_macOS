Sincronizar a Address Book (Livro de endereços) ou Contacts (Contactos) com o FAX Utility (para Mac OS X 10.7.x ou posterior)

Se pretende seleccionar um destinatário na Address Book (Livro de endereços) ou em Contacts (Contactos) para enviar um fax utilizando o FAX Utility, tem de sincronizar primeiro. Se actualizar o Address Book (Livro de endereços) ou Contacts (Contactos) após sincronizar, tem de executar esta função de sincronização novamente para utilizar os dados actualizados com o FAX Utility.
Iniciar o FAX Utility.
Iniciar o FAX Utility
Seleccione Sincronizar Address Book (Livro de endereços) ou Sincronizar Contacts (Contactos).
Siga as instruções no ecrã para sincronizar.
