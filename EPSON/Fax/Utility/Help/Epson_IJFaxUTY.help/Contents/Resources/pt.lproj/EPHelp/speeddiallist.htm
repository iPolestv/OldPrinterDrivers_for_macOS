Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo)

Menu
Barra de ferramentas
Em Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo) pode registar ou editar a marcação rápida e a marcação de grupo do fax para a impressora.
Nota:
  • Esta janela aparece quando clica em Speed Dial/Group Dial List (Lista Marc Ráp./Marc Grupo) no FAX Utility (janela principal).
  • O número máximo de entradas disponíveis da impressora ligada (fax) é apresentado na barra na parte inferior da janela.

Menu

Menu File (Fich)

Item
Explicação
Get Speed Dials (A ler marc. rápidas)
Para ler a marcação rápida ou a marcação de grupo do fax para a impressora.
Send Speed Dials (A escrever marc. rápidas)
Para guardar na impressora as marcações rápidas e marcações de grupo apresentadas na lista.
Import (Importar)
From Address Book (Da Agenda)
Importa os dados da Address Book (Livro de endereços) para as marcações rápidas da impressora.
Back up (Cópia)
Save Backup (Guardar cóp)
Para guardar uma cópia da marcação rápida e da marcação de grupo configuradas.
Restore Backup (Repor cópia)
Para ler a cópia da marcação rápida ou da marcação de grupo.
Close (Fechar)
Para fechar a janela.

Menu Edit (Editar)

Item
Explicação
Edit (Editar)
Para editar a marcação rápida ou a marcação de grupo seleccionada na lista.
Delete (Apagar)
Para apagar a marcação rápida ou a marcação de grupo seleccionada na lista.
Delete All (Apagar tudo)
Apaga todas as marcações rápidas/marcações de grupo apresentadas na lista.
