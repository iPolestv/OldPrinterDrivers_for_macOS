Fax Receive Monitor (Монітор прийому факсів) (лише для моделей з підтримкою отримання PC-FAX)

Панель інструментів
Піктограми та повідомлення у списку принтера
у цьому вікні можна перевірити стан прийому факсів та налаштувати результат для прийнятих факсів.
Примітка.
Це вікно з'являється, якщо натиснути Fax Receive Monitor (Монітор прийому факсів) в FAX Utility (вікно «Основний»).

Панель інструментів

Пункт
Пояснення
Open the received fax folder (Відкриття папки для отриманих факсів)
Відкриття папки, де зберігаються отримані факси.
Check new faxes (Перевірити наявність нових факсів)
Перевірка наявності нових факсів.
Received Fax Output Settings (Параметри виведення отриманих факсів)
Налаштування зберігання отриманих факсів на комп'ютері.
Preferences (Параметри)
Налаштування параметрів Fax Receive Monitor (Монітор прийому факсів).
Угору 

Піктограми та повідомлення у списку принтера

Піктограма
Повідомлення
Пояснення
Waiting (Очікування) / Checking for new faxes (Перевірка наявності нових факсів)
Перевірка наявності нових факсів або режим очікування.
Received faxes saved successfully in the specified folder (Отримані факси успішно збережено у вказаній папці).
Нові факси, отримані за допомогою принтера (факсу), було збережено у вказаній папці.
The printer is not responding (Принтер не відповідає)
Перевірте з'єднання між комп'ютером і принтером, або переконайтеся, що принтер увімкнено.
The printer is processing data (Принтер обробляє дані)
Принтер обробляє інші дані. Зачекайте.
Cannot find the received fax folder (Не вдається знайти папку для отриманих факсів).
Переконайтеся, що ім'я зазначеної папки правильне. Якщо ви використовуєте папку, до якої надано спільний доступ у мережі, переконайтеся, що у вас є права доступу до папки.
Failed to save faxes in the specified folder (Не вдалося зберегти факси у вказаній папці).
Переконайтеся, що вказана папка доступна, або що у вас є права доступу до папки.
A printer communication error occurred (Сталася помилка зв'язку із принтером)
Перевірте з'єднання між комп'ютером і принтером, або переконайтеся, що принтер увімкнено. Якщо ви використовуєте підключення до мережі, переконайтеся, що ви можете отримати доступ до принтера по мережі.
The "FAXRECV" disk is being used by another application. (Диск "FAXRECV" використовує інша програма.)
Закрийте програму, яка використовує диск «FAXRECV».
Cannot save faxes. User authentication failed. (Не вдається зберегти факси. Невдала автентифікація користувача.)
Перевірте автентифікацію користувача в PC-FAX для отримання (ім'я та пароль для надання спільного доступу до файлів), що встановлена на принтері, встановіть параметри Received Fax Output Settings (Параметри виведення отриманих факсів) ще раз, а потім введіть правильну інформацію для автентифікації користувача. Не забудьте встановити прапорець поруч із пунктом Remember this password in my keychain (Зберегти пароль у в’язці).
