Speed Dial/Group Dial List (Список номерів швидкого або групового набору)

Меню
Панель інструментів
Speed Dial/Group Dial List (Список номерів швидкого або групового набору) дозволяє зареєструвати або змінити швидкий/груповий набір факсу на принтері.
Примітка.
  • Це вікно з'являється, якщо натиснути Speed Dial/Group Dial List (Список номерів швидкого або групового набору) в FAX Utility (вікно «Основний»).
  • Максимальна кількість записів, доступних для підключеного принтера (факсу) відображається на панелі в нижній частині вікна.

Меню

Меню File (Файл)

Пункт
Пояснення
Get Speed Dials (Отримати номери швидкого набору)
Зчитування швидкого чи групового набору факсу на принтері.
Send Speed Dials (Надіслати номери швидкого набору)
Номери швидкого та групового набору, що відображаються у списку, буде збережено на принтері.
Import (Імпорт)
From Address Book (З адресної книги)
Імпортування даних із Address Book (Адресна книга) в номери швидкого набору принтера.
Back up (Резервна копія)
Save Backup (Зберегти резервну копію)
Зберігання резервної копії налаштованого швидкого та групового набору.
Restore Backup (Відновити резервну копію)
Зчитування резервної копії швидкого чи групового набору.
Close (Закрити)
Закриття вікна.

Меню Edit (Змінити)

Пункт
Пояснення
Edit (Змінити)
Редагування швидкого чи групового набору, вибраного у списку.
Delete (Видалити)
Видалення швидкого чи групового набору, вибраного у списку.
Delete All (Видалити все)
Видалення всіх номерів швидкого чи групового набору зі списку.
