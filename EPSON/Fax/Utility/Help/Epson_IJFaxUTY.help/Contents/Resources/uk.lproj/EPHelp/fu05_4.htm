Збереження резервної копії швидкого набору

Ви можете зберегти резервну копію налаштованого швидкого та групового набору. Дані списку швидкого/групового набору може бути втрачено через несправність принтера. Рекомендуємо створювати резервну копію даних під час кожного оновлення списку набору. Компанія Epson не несе відповідальності за втрату будь-яких даних, резервне копіювання або відновлення даних та/або параметрів, навіть протягом гарантійного терміну.
Примітка.
Дані з позначкою In use by another function (Використовує інша функція) не зберігаються.
Запустіть FAX Utility.
Запуск FAX Utility
Виберіть FAX XXXXXX (ваш принтер) у пункті Printer (Принтер), а потім натисніть Speed Dial/Group Dial List (Список номерів швидкого або групового набору).
Відображаються списки швидкого/групового набору, збережені на принтері.
Примітка.
Якщо принтер оснащено функцією безпеки, що дозволяє адміністраторам обмежувати права користувачів на зміну параметрів факсу принтера, можливо, вам потрібно буде ввести пароль адміністратора, щоб продовжити.
Натисніть File (Файл) – Back up (Резервна копія) – Save Backup (Зберегти резервну копію).
Вкажіть ім'я файлу та каталог файлу, а потім натисніть Save (Зберегти).
