Синхронізація програм Address Book (Адресна книга) або Contacts (Контакти) з FAX Utility (для Mac OS X 10.7.x або пізнішої версії)

Щоб вибрати одержувача із програм Address Book (Адресна книга) або Contacts (Контакти) для надсилання факсу за допомогою FAX Utility, необхідно синхронізувати їх заздалегідь. У разі оновлення програм Address Book (Адресна книга) або Contacts (Контакти) після синхронізації необхідно ще раз запустити функцію синхронізації, щоб використовувати оновлені дані в FAX Utility.
Запустіть FAX Utility.
Запуск FAX Utility
Установіть прапорець у полі синхронізації програм Address Book (Адресна книга) або Contacts (Контакти).
Дотримуйтеся вказівок на екрані, щоб виконати синхронізацію.
