Передавання документа, створеного в текстовому редакторі чи електронній таблиці

Для Mac OS X 10.5.x або пізнішої версії
Для Mac OS X 10.4.x
Після вибору факсу принтера в меню Print (Друк) деяких програм можна передавати дані напряму, наприклад документи, малюнки або таблиці, які ви створили.
Примітка.
У наведеному нижче поясненні для прикладу використовується TextEdit, стандартна програма Mac OS X. Фактична операція може відрізнятися залежно від того, яку програму ви використовуєте. Докладніше див. у довідці програми.

Для Mac OS X 10.5.x або пізнішої версії

Створюйте документи, щоб передавати їх за допомогою факсу, використовуючи цю програму.
Натисніть кнопку Print (Друк) у меню File (Файл) програми.
Відкриється діалогове вікно Print (Друк) програми.
Виберіть FAX XXXXXX (ваш принтер) у пункті Printer (Принтер), а потім натисніть , щоб відкрити подробиці налаштувань.
Виконайте потрібні налаштування.
    • Вкажіть цифру 1 у меню Copies (Копії). Навіть якщо ви вкажете цифру 2 або більше, все одно буде передано лише одну копію.
    • Залежно від моделі принтера (факсу), передавання кольорового зображення факсом може бути недоступне, якщо розмір паперу більший за такі: A4, Letter або Legal.
    • Під час одного сеансу передавання факсу можна передати до 100 сторінок.
Установіть прапорець у полі Fax Settings (Параметри факсу) та виконайте потрібні налаштування.
Див. пояснення щодо кожного параметра в довідці драйвера PC-FAX. Щоб відкрити довідку драйвера, натисніть ? в нижньому лівому куті екрана після встановлення прапорця в полі Fax Settings (Параметри факсу).
Установіть прапорець поруч із пунктом Recipient Settings (Параметри одержувачів) та вкажіть одержувача.
Безпосереднє вказання одержувача (ім'я, номер факсу тощо):
введіть потрібну інформацію в розділі Add (Додати), а потім натисніть+. Одержувача буде додано в розділ Recipient List (Список одержувачів), що відображається у верхній частині діалогового вікна.
Примітка.
Якщо в налаштуваннях драйвера PC-FAX встановлено прапорець Enter fax number twice (Ввести номер факсу двічі), потрібно ввести номер ще раз.
Якщо підключена лінія факсу вимагає ввести префікс зовнішнього доступу, введіть його в полі External Access Prefix (Префікс зовнішнього доступу).
Примітка.
Якщо принтер підтримує налаштування лінії типу PBX та його налаштовано на використання знаку "#" (решітка) замість коду для зовнішнього доступу, введіть знак "#" (решітка).
Вибір одержувача (ім'я, номер факсу тощо) у програмах Address book (Адресна книга) або Contacts (Контакти)
Якщо одержувач наявний в адресній книзі, натисніть . Виберіть одержувача зі списку, натисніть кнопку Add (Додати), а потім OK.
Якщо підключена лінія факсу вимагає ввести префікс зовнішнього доступу, введіть його в полі External Access Prefix (Префікс зовнішнього доступу).
Примітка.
Якщо принтер підтримує налаштування лінії типу PBX та його налаштовано на використання знаку "#" (решітка) замість коду для зовнішнього доступу, введіть знак "#" (решітка).
Перевірте параметри одержувачів і натисніть Fax (Факс).
Натисніть Fax (Факс), щоб почати передавання. Перед передаванням переконайтеся, що ім'я та номер факсу одержувача правильні.
Примітка.
  • Клацніть піктограму принтера в меню Dock (Підключення), щоб відкрити стан передавання. Щоб припинити передавання, виберіть дані, які ви надіслали, та натисніть Delete (Видалити).
  • Якщо під час передавання станеться помилка, відобразиться повідомлення про помилку. Перевірте запис про передавання у вікні Fax Transmission Record (Запис передавання факсу).
    Перевірка/видалення записів про передавання
    Перелік помилок під час передавання
  • Документи, які містять папір різного розміру, може бути передано неправильно.
