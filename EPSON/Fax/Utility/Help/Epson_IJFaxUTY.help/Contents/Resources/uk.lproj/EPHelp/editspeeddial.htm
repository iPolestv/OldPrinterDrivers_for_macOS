Speed Dial/Group Dial List (Список номерів швидкого або групового набору) – Edit Speed Dial/Group Dial (Змінити список номерів швидкого або групового набору) –

У цьому вікні можна зареєструвати або змінити запис швидкого/групового набору факсу на принтері.
Пункт
Пояснення
Entry Number (Номер запису)
Відображення номера запису швидкого набору.
Speed Dial (Швидкий набір)
Виберіть, щоб створити або змінити запис швидкого набору.
Group Dial (Груповий набір)
Виберіть, щоб створити або змінити запис групового набору. Не можна створити груповий набір, якщо не створено записів швидкого набору.
Name/Group Name (Ім'я/ім'я групи)
Введіть ім'я швидкого або групового набору, який потрібно зареєструвати.
Index Word (Індексне слово) *3
Введіть індексне слова для одержувача, яке ви можете використовувати для виконання пошуку.
Fax Number (Номер факсу)*1
Введіть номер факсу одержувача. Якщо потрібно відокремити цифри, наприклад, відділити коди міст від особистих номерів, використовуйте пробіл замість дефіса (-), тому що дефіс (-) виступає як пауза під час набору номера. Цей пункт відображається під час зміни швидкого набору.
Member (Учасник) *2
Відображення списку зареєстрованих номерів швидкого набору. Виберіть швидкий набір, який потрібно додати до групового набору, а потім натисніть кнопку OK. Цей пункт відображається під час зміни групового набору.
Fax Speed (Швидк. передав. факсу) *3
Вибір швидкості передавання факсу, що використовується для надсилання факсу одержувачу. Швидкість факсу, встановлена для кожного запису швидкого набору, використовується тільки під час надсилання факсу безпосередньо з принтера. FAX Utility ігнорує налаштування та використовує швидкість факсу, встановлену на принтері.
OK
Збереження параметрів і закриття вікна.
Cancel (Скасувати)
Скасування реєстрації або зміни та закриття вікна.
*1 Обов'язковий пункт для реєстрації швидкого набору.
*2 Під час реєстрації групового набору переконайтеся, що прапорець завжди встановлено поруч із одним або кількома номерами швидкого набору.
*3 Ці елементи можуть не відображатися, залежно від моделі принтера.
Примітка.
  • Див. інформацію про припустимі символи та їх кількість у наступному розділі.
    Прийнятні символи
  • Максимальна кількість записів швидкого набору у групі відображається на панелі в нижній частині вікна зміни групового набору.
