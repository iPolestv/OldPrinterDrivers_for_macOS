Menetapkan penerima faks dalam menu Recipient Settings (Setelan Penerima).
Catatan:
Baca bantuan FAX Utility untuk informasi selengkapnya tentang cara mengirim faks dari komputer. Ikuti langkah-langkah di bawah ini untuk menjalankan FAX Utility.
Untuk Mac OS X 10.6.x atau yang lebih baru
1. Klik Apple - System Preferences (Preferensi Sistem) - Print & Scan (Cetak & Pindai) atau Print & Fax (Cetak & Faks).
2. Pilih FAX XXXXXX (printer Anda) dari Printer, lalu klik Options & Supplies (Opsi & Persediaan) - Utility (Utilitas) - Open Printer Utility (Buka Utilitas Printer).
Untuk Mac OS X 10.5.x
1. Klik Apple - System Preferences (Preferensi Sistem) - Print & Fax (Cetak & Faks).
2. Pilih FAX XXXXXX (printer Anda) dari Printer, lalu klik Open Print Queue (Buka Antrean Cetak) - Utility (Utilitas).
Untuk Mac OS X 10.4.x
1. Klik dua kali Hard disk - Applications (Aplikasi) - Utilities (Utilitas) - Printer Setup Utility (Utilitas Konfigurasi Printer).
2. Pilih FAX XXXXXX (printer Anda) dari Printer, lalu klik Utility (Utilitas).

Buka Address Book (Buku Alamat) atau Contacts (Kontak)

Menampilkan daftar kontak untuk Address Book (Buku Alamat) atau Contacts (Kontak) dan supaya Anda dapat menetapkan penerima dari daftar. Untuk Mac OS 10.7.x atau yang lebih baru, Anda harus mensinkronisasi Address Book (Buku Alamat) atau Contacts (Kontak) terlebih dulu menggunakan FAX Utility untuk menampilkan konten. Baca FAX Utility untuk informasi selengkapnya.
Address Book (Buku Alamat) dan Contacts (Kontak) adalah aplikasi buku alamat untuk Mac OS X. Aplikasi Address Book (Buku Alamat) atau Contacts (Kontak) memiliki fungsi impor/ekspor masing-masing. Anda dapat menemukan aplikasi ini di dalam folder Applications (Aplikasi). Baca bantuan yang diberikan bersama aplikasi untuk informasi selengkapnya.
Atas 

+ / -

+ tersedia saat memasukkan informasi penerima ke dalam bagian Add (Tambah). Klik + untuk menambahkan data ke Recipient List (Dftr Penerima).
- tersedia jika Anda memilih informasi penerima dalam Recipient List (Dftr Penerima). Klik - untuk menghapus data dari daftar.
Atas 

Add (Tambah)

Menambahkan informasi penerima, misalnya nama dan nomor faks.
Masukkan informasi yang diperlukan, lalu klik +.
Item
Penjelasan
Name (Nama)*1
Tulis nama penerima.
Anda dapat menulis hingga 60 karakter dalam UTF-8 (kode karakter).
Company/Corp. (Prsahaan/Korp.)
Tulis nama perusahaan penerima.
Anda dapat menulis hingga 60 karakter dalam UTF-8 (kode karakter).
Fax Number (Nomor Faks)*2
Masukkan nomor faks penerima.
Jika Anda ingin menjeda printer (faks) sebentar sebelum menekan nomor, sisipkan tanda penghubung (-) di awal nomor faks. Tanda penghubung (-) apa pun yang disisipkan di antara nomor akan dihapus secara otomatis.
cth. -81-123-45-6789 -81123456789
*1 Pesan konfirmasi akan ditampilkan jika tidak ada Name (Nama) yang dimasukkan.
*2 Item yang diperlukan.

External Access Prefix (Prefiks Akses Eksternal)

Masukkan nomor akses eksternal jika saluran faks yang tersambung memerlukan prefiks akses eksternal. Misalnya, jika saluran tersambung ke sistem private branch exchange (PBX), yang digunakan oleh sebagian besar perusahaan, dan Anda harus menekan prefiks akses eksternal, misalnya 0 atau 9 untuk mencapai saluran luar, masukkan nomor prefiks.
Jika printer Anda memiliki setelan jenis saluran PBX dan telah diatur untuk menggunakan # (hash), bukan memasukkan kode prefiks yang sama, tulis # (hash).
Saat mengirim faks ke lebih dari satu penerima, printer (faks) akan memasukkan nomor prefiks sebelum memasukkan setiap nomor faks.
