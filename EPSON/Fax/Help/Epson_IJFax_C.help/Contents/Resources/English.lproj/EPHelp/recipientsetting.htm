For Mac OS X 10.6.x or later
1. Click Apple - System Preferences - Print & Scan or Print & Fax.
2. Select FAX XXXXXX (your printer) from Printer, and then click Options & Supplies - Utility - Open Printer Utility.
For Mac OS X 10.5.x
1. Click Apple - System Preferences - Print & Fax.
2. Select FAX XXXXXX (your printer) from Printer, and then click Open Print Queue - Utility.
For Mac OS X 10.4.x
1. Double-click Hard disk - Applications - Utilities - Printer Setup Utility.
2. Select FAX XXXXXX (your printer) from Printer, and then click Utility.

Open Address Book or Contacts

Displays the contact list for the Address Book or Contacts and allows you to specify a recipient from the list. For Mac OS 10.7.x or later, you need to synchronize Address Book or Contacts in advance using FAX Utility to display the contents. See the FAX Utility help for more information.
Address Book and Contacts are address book applications for Mac OS X. The Address Book or Contacts applications have their own import/export functions. You can find these applications in the Applications folder. See the help provided with the application for more information.

+ / -

+ is available when entering recipient’s information in Add section. Click + to add the data to the Recipient List.
- is available if you select the recipient’s information in Recipient List. Click - to delete the data from the list.

Add

Adds the recipient’s information, such as name and fax number.
Enter the necessary information, and then click +.
Item
Explanation
Name*1
Enter the recipient’s name.
You can enter up to 60 characters in UTF-8 (character code).
Company/Corp.
Enter the recipient’s company name.
You can enter up to 60 characters in UTF-8 (character code).
Fax Number*2
Enter the recipient’s fax number.
If you want to pause the printer (fax) for a moment before dialing the number, insert a hyphen (-) at the beginning of the fax number. Any hyphens (-) inserted between numbers are automatically deleted.
ex. -81-123-45-6789 -81123456789
*1 A confirmation message is displayed if no Name is entered.
*2 Required item.

External Access Prefix

Enter an external access number if the connected fax line requires an external access prefix. For example, if the line is connected to a PBX (private branch exchange) system, which is used by most companies, and you need to dial an external access prefix such as 0 or 9 to get an outside line, enter the prefix number.
If your printer has a PBX line type setting and has been set to use # (hash) instead of entering the exact prefix code, enter # (hash).
When sending a fax to more than one recipient, the printer (fax) dials the prefix number before dialing each fax number.
