Obs!
Se FAX Utility-hjälpen för mer information om hur man skickar fax från en dator. Följ stegen nedan för att starta FAX Utility.
För Mac OS X 10.6.x eller senare
1. Klicka på Apple - System Preferences (Systeminställningar) - Print & Scan (Skriv ut och skanna) eller Print & Fax (Skriv ut och faxa).
2. Välj FAX XXXXXX (din skrivare) från Printer (Skrivare) och klicka sedan på Options & Supplies (Alternativ och förbrukningsmaterial) - Utility (Verktyg) - Open Printer Utility (Öppna skrivarverktyg).
För Mac OS X 10.5.x
1. Klicka på Apple - System Preferences (Systeminställningar) - Print & Fax (Skriv ut och faxa).
2. Välj FAX XXXXXX (din skrivare) från Skrivare och klicka sedan på Open Print Queue (Öppna utskriftskö) - Utility (Verktyg).
För Mac OS X 10.4.x
1. Dubbelklicka på hårddisken - Applications (Program) - Utilities (Verktyg) - Printer Setup Utility (Inställningsverktyg för skrivaren).
2. Välj FAX XXXXXX (din skrivare) från Skrivare och klicka sedan på Utility (Verktyg).

Recipient List (Mottagarlista)

Listar information att skicka till mottagaren.
Överst 

Öppna Address Book (Adressbok) eller Contacts (Kontakter)

Visar kontaktlistan för Address Book (Adressbok) eller Contacts (Kontakter) och låter dig ange en mottagare i listan. För Mac OS 10.7.x eller senare måste du synkronisera Address Book (Adressbok) eller Contacts (Kontakter) i förväg via FAX Utility för att visa innehållet. Se FAX Utility-hjälpen för mer information.
Address Book (Adressbok) och Contacts (Kontakter) är adressboksprogram för Mac OS X. Programmen Address Book (Adressbok) och Contacts (Kontakter) har sina egna importerings-/exporteringsfunktioner. Du hittar dessa program i mappen Applications (Program). Se hjälpen som medföljer programmet för mer information.
Överst 

+ / -

+ är tillgängligt när du anger mottagarens information i sektionen Add (Lägg till). Klicka på + för att lägga till data i Recipient List (Mottagarlista).
- är tillgängligt om du väljer mottagarens information i Recipient List (Mottagarlista). Klicka på - för att ta bort data från listan.
Överst 

Add (Lägg till)

Lägger till mottagarens information såsom namn och faxnummer.
Ange nödvändig information och klicka sedan på +.
Objekt
Förklaring
Name (Namn)*1
Ange mottagarens namn.
Du kan ange upp till 60 tecken i UTF-8 (teckenkod).
Company/Corp. (Företag.)
Ange mottagarens företagsnamn.
Du kan ange upp till 60 tecken i UTF-8 (teckenkod).
Fax Number (Faxnummer)*2
Ange mottagarens faxnummer.
Om du vill pausa skrivaren (fax) ett ögonblick innan numret rings, infoga ett bindestreck (-) i början av faxnumret. Alla bindestreck (-) som infogas mellan nummer tas automatiskt bort.
T.ex. -81-123-45-6789 -81123456789
*1 Ett bekräftelsemeddelande visas om inget Name (Namn) anges.
*2 Objekt som krävs.
Överst 

External Access Prefix (Externt åtkomstprefix)

Ange ett externt åtkomstnummer om den anslutna faxlinjen kräver ett externt åtkomstprefix. T.ex. om linjen är ansluten till ett system med telefonväxel (företagsväxel), som används av de flesta företag, och du måste ringa ett externt åtkomstprefix, t.ex. 0 eller 9 för att komma ut på linjen, ange prefixnumret.
Om skrivaren har en telefonväxel-inställning och har ställts in för att använda # (hash) istället för att ange den exakta prefixkoden, ange # (hash).
När du skickar ett fax till fler än en mottagare ringer skrivaren (fax) prefixnumret innan den ringer varje faxnummer.
