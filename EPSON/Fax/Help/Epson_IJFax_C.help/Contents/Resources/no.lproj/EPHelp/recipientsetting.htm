Merk:
Se hjelpen for FAX Utility for mer informasjon om hvordan du sender fakser fra en datamaskin. Følg fremgangsmåten under for å starte FAX Utility.
For Mac OS X 10.6.x eller nyere
1. Klikk Apple - System Preferences (Systemvalg) – Print & Scan (Skrivere og skannere) eller Print & Fax (Utskrift og faks).
2. Velg FAX XXXXXX (skriveren) fra Printer (Skriver), og klikk Options & Supplies (Valg og forbruksmateriell) - Utility (Verktøy) - Open Printer Utility (Åpne Skriververktøy).
For Mac OS X 10.5.x
1. Klikk Apple - System Preferences (Systemvalg) - Print & Fax (Utskrift og faks).
2. Velg FAX XXXXXX (skriveren) fra Printer (Skriver), og klikk deretter på Open Print Queue (Åpne utskriftskø) - Utility (Verktøy).
For Mac OS X 10.4.x
1. Dobbeltklikk på harddisken - Applications (Programmer) - Utilities (Verktøy) - Printer Setup Utility (Skriveroppsettverktøy).
2. Velg FAX XXXXXX (skriveren) fra Printer (Skriver), og klikk på Utility (Verktøy).

Åpne Address Book (Adressebok) eller Contacts (Kontakter)

Viser kontaktlisten for Address Book (Adressebok) eller Contacts (Kontakter) og lar deg angi en mottaker fra listen. I Mac OS 10.7.x eller nyere må du synkronisere Address Book (Adressebok) eller Contacts (Kontakter) på forhånd ved hjelp av FAX Utility for å vise innholdet. Se i hjelpen til FAX Utility for mer informasjon.
Address Book (Adressebok) og Contacts (Kontakter) adressebokprogrammene i Mac OS X. Programmene Address Book (Adressebok) og Contacts (Kontakter) har egne funksjoner for importering/eksportering. Du finner disse programmene i Applications (Programmer)-mappen. Se i hjelpen som fulgte med programmet for mer informasjon.

+ / -

+ Er tilgjengelig når du taster inn mottakerens informasjon i delen Add (Legg til). Klikk på + for å legge til data i Recipient List (Mottakerliste).
- kan klikkes når du har lagt til mottakerinformasjon i Recipient List (Mottakerliste). Klikk - for å slette data fra listen.

Add (Legg til)

Legger til mottakerens informasjon, for eksempel navn og faksnummer.
Skriv inn nødvendig informasjon, og klikk deretter +.
Navn
Beskrivelse
Name (Navn)*1
Skriv inn mottakerens navn.
Du kan skrive inn opptil 60 tegn i UTF-8 (tegnkode).
Company/Corp. (Firma/bedrift)
Angi mottakerens firmanavn.
Du kan skrive inn opptil 60 tegn i UTF-8 (tegnkode).
Fax Number (Faksnummer)*2
Angi mottakerens faksnummer.
Hvis du vil at skriveren (faksmaskinen) skal ta en liten før den slår nummeret, setter du inn en bindestrek (-) i begynnelsen av faksnummeret. Eventuelle bindestreker (-) satt inn mellom tallene blir automatisk slettet.
f.eks. -81-123-45-6789 -81123456789
*1 En bekreftelsesmelding vises hvis Name (Navn) ikke er fylt inn.
*2 Obligatorisk.

External Access Prefix (Retningsnummer for ekstern tilgang)

Skriv inn et retningsnummer for ekstern tilgang hvis den tilkoblede fakslinjen krever dette. Hvis linjen for eksempel er koblet til et PBX-system (Private Branch Exchange), som brukes av de fleste selskapene, og du må taste inn et retningsnummer for ekstern tilgang, som for eksempel 0 eller 9, for å få en ekstern linje, angir du dette.
Hvis skriveren har en PBX-linjetypeinnstilling og har blitt satt til å bruke # (firkant) uten å oppgi en eksakt prefikskode, taster du # (firkant).
Når du sender en faks til mer enn én mottaker, slår skriveren (faksmaskinen) slår retningsnummeret før den ringer hvert faksnummer.
