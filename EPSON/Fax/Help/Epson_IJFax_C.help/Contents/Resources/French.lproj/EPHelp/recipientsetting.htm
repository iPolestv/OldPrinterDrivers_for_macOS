Permet de définir le destinataire du fax dans le menu Recipient Settings (Paramètres du destinataire).
Remarque :
Reportez-vous à l'aide de FAX Utility pour en savoir plus sur l'envoi de fax à partir d'un ordinateur. Reportez-vous aux étapes ci-dessous pour démarrer FAX Utility.
Pour Mac OS X 10.6.x ou version supérieure
1. Cliquez sur Apple - System Preferences (Préférences système) - Print & Scan (Imprimantes et scanners) ou Print & Scan (Imprimantes et fax).
2. Sélectionnez FAX XXXXXX (votre imprimante) sous Printer (Imprimante), puis cliquez sur Options & Supplies (Options et fournitures) - Utility (Utilitaire) - Open Printer Utility (Ouvrir l'utilitaire d'impression).
Pour Mac OS X 10.5.x
1. Cliquez sur Apple - System Preferences (Préférences système) - Print & Fax (Imprimante et fax).
2. Sélectionnez FAX XXXXXX (votre imprimante) sous Printer (Imprimante), puis cliquez sur Open Print Queue (Ouvrir la file d'impression) - Utility (Utilitaire).
Pour Mac OS X 10.4.x
1. Cliquez deux fois sur le disque dur - Applications - Utilities (Utilitaires) - Printer Setup Utility (Configuration d'imprimante).
2. Sélectionnez FAX XXXXXX (votre imprimante) sous Printer (Imprimante), puis cliquez sur Utility (Utilitaire).

Ouvrir Address Book (Carnet d'adresses) ou Contacts

Affiche la liste des contacts du Address Book (Carnet d'adresses) ou des Contacts et permet de spécifier un destinataire dans la liste. Pour Mac OS 10.7.x et versions ultérieures, vous devez synchroniser Address Book (Carnet d'adresses) ou Contacts à l'avance à l'aide de FAX Utility pour afficher le contenu. Pour plus d'informations, consultez l'aide de FAX Utility.
Address Book (Carnet d'adresses) et Contacts sont des applications de répertoire pour Mac OS X. Les applications Address Book (Carnet d'adresses) et Contacts ont leurs propres fonctions d'importation/exportation. Vous pouvez trouver ces applications dans le dossier Applications. Pour plus d'informations, consultez l'aide fournie.
Haut 

+ / -

+ est disponible lors de la saisie des informations relatives au destinataire dans la section Add (Ajouter). Cliquez sur + pour ajouter les données à la section Recipient List (Liste des destinataires).
- est disponible lorsque vous sélectionnez les informations relatives au destinataire sous Recipient List (Liste des destinataires). Cliquez sur - pour supprimer des données de la liste.

Add (Ajouter)

Permet d'ajouter des informations relatives au destinataire, telles que le nom et le numéro de fax.
Saisissez les informations requises et cliquez sur +.
Élément
Explication
Name (Nom)*1
Saisissez le nom du destinataire.
Vous pouvez saisir un maximum de 60 caractères au format UTF-8 (code de caractères).
Company/Corp (Entreprise/gpe)
Saisissez le nom de l'entreprise du destinataire.
Vous pouvez saisir un maximum de 60 caractères au format UTF-8 (code de caractères).
Fax Number (Numéro de fax)*2
Saisissez le numéro de fax du destinataire.
Si vous souhaitez interrompre le fonctionnement de l'imprimante (du fax) pendant quelques instants avant de composer le numéro, insérez un tiret (-) au début du numéro de fax. Les tirets (-) insérés entre les numéros sont automatiquement supprimés.
Ex. -81-123-45-6789 -81123456789
*1 Un message de confirmation s'affiche si aucun Name (Nom) n'est saisi.
*2 Élément requis.
Haut 

External Access Prefix (Préfixe d'accès externe)

Saisissez un numéro d'accès externe si la ligne fax connectée nécessite un préfixe d'accès externe. Ainsi, si la ligne est connectée à un commutateur privé, système utilisé par la plupart des entreprises, et que vous devez composer un préfixe d'accès externe, tel que 0 ou 9, pour obtenir une ligne externe, saisissez le préfixe.
Si votre imprimante dispose d'un paramètre de type de ligne PBX et a été défini pour utiliser # (dièse) au lieu d'entrer le code de préfixe exact, entrez # (dièse).
Lors de l'envoi d'un fax à plusieurs destinataires, l'imprimante (le fax) compose le préfixe avant chaque numéro de fax.
