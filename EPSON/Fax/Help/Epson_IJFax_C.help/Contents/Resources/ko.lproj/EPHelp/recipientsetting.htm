수신처 설정 메뉴에서 팩스 수신처를 설정합니다.
참고:
컴퓨터에서 팩스를 전송하는 더 자세한 정보는 FAX Utility 도움말을 참조하십시오. FAX Utility를 시작하려면 아래 단계를 따르십시오.
Mac OS X 10.6.x 이상
1. 애플 - 시스템 환경설정 - 프린트 & 스캔 혹은 프린트 & 팩스를 클릭합니다.
2. 프린터에서 FAX XXXXXX(사용 중인 프린터)를 선택한 다음 옵션 및 소모품 - 유틸리티 - 프린터 유틸리티 열기를 선택합니다.
Mac OS X 10.5.x
1. 애플 - 시스템 환경설정 - 프린트 & 팩스를 클릭합니다.
2. 프린터에서 FAX XXXXXX (사용 중인 프린터)를 선택한 다음 인쇄 큐 열기 - 유틸리티를 클릭합니다.
Mac OS X 10.4.x
1. 하드 디스크 - 응용 프로그램 - 유틸리티 - 프린터 설정 유틸리티를 더블 클릭합니다.
2. 프린터에서 FAX XXXXXX(사용 중인 프린터)를 선택한 다음 유틸리티를 클릭합니다.

주소록 또는 연락처 열기

주소록 또는 연락처의 연락처 목록이 표시되고 목록에서 수신처를 지정할 수 있습니다. Mac OS 10.7.x 이상에서 FAX Utility를 사용하여 연락처를 표시하려면 먼저 주소록 또는 연락처 동기화가 필요합니다. 더 자세한 사항은 FAX Utility 도움말을 참조하십시오.
주소록과 연락처는 Mac OS X용 주소록 응용프로그램입니다. 주소록 또는 연락처 응용프로그램은 각각의 불러오기/내보내기 기능을 가지고 있습니다. 응용프로그램 폴더에서 이 프로그램들을 찾을 수 있습니다. 더 자세한 사항은 응용프로그램의 도움말을 참조하십시오.
맨위로 

+ / -

+는 추가 부분에 수신처 정보를 입력할 때 사용할 수 있습니다. +를 클릭하여 수신처 목록에 데이터를 추가할 수 있습니다.
-는 수신처 목록에서 수신처 정보를 선택한 경우 사용할 수 있습니다. -를 클릭하여 목록에서 데이터를 삭제할 수 있습니다.
맨위로 

추가

이름, 팩스 번호 등 수신처 정보를 추가합니다.
필수 정보를 입력한 다음 +를 클릭합니다.
항목
설명
이름*1
수신처 이름을 입력합니다.
UTF-8(문자 코드)에서 최대 60문자를 입력할 수 있습니다.
회사/단체
수신처 회사 이름을 입력합니다.
UTF-8(문자 코드)에서 최대 60문자를 입력할 수 있습니다.
팩스 번호*2
수신처 팩스 번호를 입력합니다.
번호를 다이얼링하기 전에 프린터(팩스)를 일시적으로 정지하려면 팩스 번호 시작 부분에 하이픈(-)을 삽입합니다. 번호 사이의 하이픈(-)은 자동으로 삭제됩니다.
예. -81-123-45-6789 -81123456789
*1 이름을 입력하지 않으면 확인 메시지가 표시됩니다.
*2 필수 항목

외부 액세스 국번

연결된 팩스 선에 외부 액세스 국번이 필요한 경우 외부 액세스 번호를 입력합니다. 예를 들어 전화선이 사설교환기(private branch exchange, PBX) 시스템에 연결된 경우(대부분의 회사에서 사용) 외부로 연결하려면 0 또는 9 등의 외부 액세스 국번을 다이얼링해야 합니다.
사용 중인 프린터가 PBX 라인 형식으로 설정되어 있고, 외부 접속 키 대신 #(우물 정자)를 사용 중이면 #을 입력하십시오.
하나 이상의 수신처에 팩스를 보내는 경우 프린터(팩스)는 각 팩스 번호 앞에 국번을 다이얼링합니다.
