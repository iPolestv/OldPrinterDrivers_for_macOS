［送付先設定］メニューでは、ファクスの送付先を設定します。
参考
コンピューターからのファクス送信手順はFAX Utilityのヘルプをご覧ください。
FAX Utilityの起動方法
（Mac OS X v10.6.x 以降の場合）
1. アップルメニューから［システム環境設定］を選択して［プリントとスキャン］（または［プリントとファクス］）の順にクリックします。
2. プリンタリストからFAX XX-XXXX（ご使用のプリンター）を選択し、［オプションとサプライ］-［ユーティリティ］-［プリンタユーティリティを開く］の順にクリックします。
（Mac OS X v10.5.x の場合）
1. アップルメニューから［システム環境設定］を選択して［プリントとファクス］をクリックします。
2. プリンタリストからFAX XX-XXXX（ご使用のプリンター）を選択し、［プリントキューを開く］-［ユーティリティ］の順にクリックします。
（Mac OS X v10.4.x の場合）
1. ［ハードディスク］-［アプリケーション］-［ユーティリティ］-［プリンタ設定ユーティリティ］の順にダブルクリックします。
2. プリンタリストからFAX XX-XXXX（ご使用のプリンター）を選択し、［ユーティリティ］をクリックします。

送付先一覧

これから送付する相手の情報が一覧表示されます。
先頭へ 

連絡先（またはアドレスブック）*を開く

Mac OS X標準添付のアドレスブックを表示します。アドレスブックに登録されているアドレスデータを使ってファクスの送付先を設定します。Mac OS X v10.7.x以降の場合は、あらかじめFax Utilityで連絡先（またはアドレスブック）の同期を行う必要があります。同期の方法は、FAX Utilityのヘルプをご覧ください。
先頭へ 

［+］/［-］

［追加］に送付先の情報を入力すると、［+］が使用できるようになります。［+］をクリックすると、［追加］に入力した送付先の情報が［送付先一覧］に追加されます。
［送付先一覧］で送付先を選択すると、［-］が使用できるようになります。［-］をクリックすると、［送付先一覧］で選択した送付先の情報が一覧から削除されます。
先頭へ 

追加

名前やファクス番号などの送付先を直接入力して追加します。
必要事項を入力して［+］をクリックしてください。
項目
説明
名前 ※1
送付先の名前を入力します。
入力できる文字はUTF-8（文字コード）で表現可能なすべての文字です。入力できる文字数は60文字です。
会社名/団体名
送付先の会社名/団体名を入力します。
入力できる文字はUTF-8（文字コード）で表現可能なすべての文字です。入力できる文字数は60文字です。
ファクス番号 ※2
送付先のファクス番号を入力します。
ファクス番号の先頭に「-」（ハイフン）を入力すると、ポーズとして動作します。区切り記号として入力した「-」（ハイフン）は削除されます。
例：-03-XXXX-VVVV → -03XXXXVVVV
※ 1　［名前］が入力されていないときは、確認メッセージが表示されます。
※ 2　必ず入力してください。
先頭へ 

外線発信番号

ご使用のファクスの接続回線が外線発信を必要とするとき、外線発信の番号を入力します。
企業などの内線電話システムなどで、外線発信するときに電話番号の最初に「0」などの外線発信番号を付けて通話する必要があるときに、［外線発信番号］を入力してください。また、ファクスの接続回線を選択できるプリンターで構内交換機（PBX）に設定している場合は、外線発信番号を「#」で登録することをお勧めします。詳しくはプリンターのマニュアルをご覧ください。
複数の送付先に送るときは、［外線発信番号］で指定した番号がファクス番号の先頭に自動的に付加されます。
用語解説
連絡先（アドレスブック）
Mac OS標準のアドレス帳です。このアドレスデータは、「メール」など他のアプリケーションでも使用されていて、PC-FAXドライバーでもこのデータを利用します。
連絡先（またはアドレスブック）は［ハードディスク］−［アプリケーション］にあり、インポート/エクスポートの機能を持っています。詳しくは連絡先（またはアドレスブック）のヘルプをご覧ください。
