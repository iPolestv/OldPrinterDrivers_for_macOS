Встановлення одержувача факсу в меню Recipient Settings (Параметри одержувачів).
Примітка.
Див. детальнішу інформацію про надсилання факсів із комп’ютера в довідці FAX Utility. Щоб запустити FAX Utility, виконайте наведені нижче дії.
Для Mac OS X 10.6.x або пізнішої версії
1. Відкрийте меню Apple – System Preferences (Системні параметри) – Print & Scan (Друк і сканування) або Print & Fax (Друк і факс).
2. Виберіть FAX XXXXXX (ваш принтер) у меню Printer (Принтер), а потім натисніть Options & Supplies (Параметри та витратні матеріали) – Utility (Утиліта) – Open Printer Utility (Відкрити утиліту принтерів).
Для Mac OS X 10.5.x
1. Відкрийте меню Apple – System Preferences (Системні параметри) – Print & Fax (Друк і факс).
2. Виберіть FAX XXXXXX (ваш принтер) у меню "Printer" (Принтер), а потім натисніть Open Print Queue (Черга на друк) – Utility (Утиліта).
Для Mac OS X 10.4.x
1. Двічі клацніть "Hard disk" (Жорсткий диск) – Applications (Програми) – Utilities (Утиліти) – Printer Setup Utility (Утиліта встановлення принтера).
2. Виберіть FAX XXXXXX (ваш принтер) у пункті "Printer" (Принтер), а потім натисніть Utility (Утиліта).

Угору 

Відкрийте програму Address Book (Адресна книга) або Contacts (Контакти)

Відображення списку контактів із програми Address Book (Адресна книга) або Contacts (Контакти) та можливість вибрати одержувача зі списку. Для Mac OS 10.7.x або пізнішої версії потрібно буде синхронізувати програми Address Book(Адресна книга) або Contacts (Контакти) заздалегідь за допомогою FAX Utility, щоб переглянути їхній вміст. Детальну інформацію див. у довідці FAX Utility.
Address Book (Адресна книга) та Contacts (Контакти) – це програми для Mac OS X, що містять адресні книги. Програми Address Book (Адресна книга) або Contacts (Контакти) мають власні функції імпорту/експорту. Ці програми можна знайти в папці Applications (Програми). Детальнішу інформацію див. у довідці програми.
Угору 

+ / -

Функція + доступна під час введення інформації про одержувача в розділі Add (Додати). Натисніть кнопку +, щоб додати дані в розділ Recipient List (Список одержувачів).
Функція - доступна під час вибору інформації про одержувача в розділі Recipient List (Список одержувачів). Натисніть кнопку -, щоб видалити дані зі списку.
Угору 

Add (Додати)

Додавання інформації про одержувача, як-от номер телефону або факсу.
Введіть потрібну інформацію, а потім натисніть кнопку +.
Пункт
Пояснення
Name (Ім'я)*1
Введіть ім’я одержувача.
Ви можете ввести до 60 символів у форматі UTF-8 (код символу).
Company/Corp. (Компанія/корпорація)
Введіть назву компанії одержувача.
Ви можете ввести до 60 символів у форматі UTF-8 (код символу).
Fax Number (Номер факсу)*2
Введіть номер факсу одержувача.
Щоб призупинити принтер (факс) на час набрання номера, поставте дефіс (-) перед номером факсу. Усі дефіси (-) між номерами буде видалено автоматично.
наприклад, -81-123-45-6789 -81123456789
*1 Якщо Name (Ім'я) не введено, відображається запит підтвердження.
*2 Обов’язковий пункт.
Угору 

External Access Prefix (Префікс зовнішнього доступу)

Введіть номер зовнішнього доступу, якщо підключена лінія факсу вимагає ввести префікс зовнішнього доступу. Наприклад, якщо лінія підключена до системи PBX (приватної АТС), яку використовує більшість компаній, ви повинні набрати префікс зовнішнього доступу, як-от цифра від 0 до 9, щоб підключитися до зовнішньої лінії.
Якщо принтер підтримує налаштування лінії типу PBX та його налаштовано на використання знаку "#" (решітка) замість коду для зовнішнього доступу, введіть знак "#" (решітка).
Під час надсилання факсу кільком одержувачам перед набором кожного номера факсу принтер (факс) набирає код попереднього набору.
