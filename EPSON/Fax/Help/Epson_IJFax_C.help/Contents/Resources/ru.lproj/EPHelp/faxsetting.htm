Fax Settings (Настройки факса)

Color (Цветной)
Image Quality (Кач. изображ.)
В области Fax Settings (Настройки факса) определяются цвет и качество изображения в передаваемых факсимильных сообщениях.
Примечание:
Для получения дополнительной информации об отправке факсимильных сообщений с помощью компьютера см. справку по приложению FAX Utility. Для получения дополнительной информации о запуске приложения FAX Utility см. справку по разделу Recipient Settings (Настройки получателя).

Color (Цветной)

Выберите цвет для передачи факсимильного сообщения.
Пункт
Описание
Black & White (Черно-белый)
Отправляет данные в черно-белом режиме.
Color (Цветной)
Отправляет данные в цвете.
Если факс получателя не поддерживает прием цветных факсимильных сообщений, выберите Black & White (Черно-белый).
Примечание:
В зависимости от принтера, передача цветных факсимильных сообщений может быть недоступна для форматов бумаги крупнее, чем A4, "Letter" или "Legal".
