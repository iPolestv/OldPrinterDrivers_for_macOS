Примечание:
Для получения дополнительной информации об отправке факсимильных сообщений с помощью компьютера см. справку по приложению FAX Utility. Для запуска FAX Utility выполните следующие шаги.
Для Mac OS X 10.6 или более поздней версии
1. Выберите Apple - System Preferences - Print & Scan или Print & Fax.
2. Из списка Printer выберите FAX XXXXXX (ваш принтер), а затем нажмите Options & Supplies - Utility - Open Printer Utility.
Для Mac OS X 10.5.x
1. Щелкните Apple - System Preferences - Print & Fax.
2. Из списка Printer выберите FAX XXXXXX (ваш принтер) и нажмите Open Print Queue - Utility.
Для Mac OS X 10.4.x
1. Дважды щелкните Hard disk - Applications - Utilities - Printer Setup Utility.
2. Из списка Printer выберите FAX XXXXXX (ваш принтер) и нажмите Utility.

Recipient List (Список получателей)

Содержит информацию о получателях.
К началу 

Откройте Address Book или Contacts.

Откроется список контактов для Address Book или Contacts с возможностью определения получателя из списка. Для Mac OS 10.7.x или выше необходимо синхронизировать Address Book или Contacts заранее с использованием FAX Utility для просмотра содержимого. Для получения дополнительной информации см. справку по приложению FAX Utility.
Address Book и Contacts - это приложения адресной книги для Mac OS X. Приложения Address Book и Contacts имеют собственные функции импорта/экспорта. Эти приложения доступны в папке Applications. Для получения дополнительной информации см. справку по соответствующему приложению.
К началу 

+ / -

Кнопка + доступна в разделе Add (Добавить) при добавлении информации о получателе. Нажмите + для добавления данных в Recipient List (Список получателей).
Кнопка - доступна при выборе информации о получателе в области Recipient List (Список получателей). Для удаления данных из списка нажмите -.
К началу 

Add (Добавить)

Добавляет информацию о получателе, например имя и номер факса.
Введите необходимую информацию и нажмите +.
Пункт
Описание
Name (Имя)*1
Введите имя получателя.
Можно ввести до 60 символов в кодировке UTF-8.
Company/Corp (Компания)
Введите название компании получателя.
Можно ввести до 60 символов в кодировке UTF-8.
Fax Number (Номер факса)*2
Введите номер факса получателя.
Если требуется, чтобы перед набором номера принтер (факс) сделал паузу, вставьте в начале номера дефис (-). Дефисы (-) между цифрами удаляются автоматически.
Например, -81-123-45-6789 -81123456789
*1 Если значение в поле Name (Имя) не введено, появляется подтверждающее сообщение.
*2 Обязательный для заполнения пункт.
К началу 

External Access Prefix (Префикс внешнего доступа)

Если телефонная линия требует префикса внешнего доступа, введите его. Например, если линия подключена к частной телефонной станции, которыми пользуется большинство компаний, и необходимо набирать префикс (например, 0 или 9) для подключения к внешней линии, введите этот префикс.
Если принтер подключен к частной телефонной станции и для него задана необходимость ввода символа решетки (#) вместо точного кода префикса, введите этот символ.
При отправке факса нескольким получателям префикс набирается перед каждым из выбранных номеров.
