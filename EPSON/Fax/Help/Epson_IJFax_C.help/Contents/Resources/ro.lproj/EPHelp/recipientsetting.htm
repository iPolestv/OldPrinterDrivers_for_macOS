Setează destinatarul faxului în meniul Recipient Settings (Setări destinatari).
Notă:
Pentru mai multe informaţii despre trimiterea faxurilor de pe un computer, consultaţi ajutorul pentru FAX Utility. Urmaţi paşii de mai jos pentru a lansa FAX Utility.
Pentru Mac OS X 10.6.x sau versiuni ulterioare
1. Faceţi clic pe Apple - System Preferences (Preferinţe sistem) - Print & Scan (Imprimare şi scanare) sau Print & Fax (Imprimare şi fax).
2. Selectaţi FAX XXXXXX (imprimanta dvs.) din Printer (Imprimată) şi apoi faceţi clic pe Options & Supplies (Opţiuni şi consumabile) - Utility (Utilitar) - Open Printer Utility (Deschidere utilitar imprimantă).
Pentru Mac OS X 10.5.x
1. Faceţi clic pe Apple - System Preferences (Preferinţe sistem) - Print & Fax (Imprimare şi fax).
2. Selectaţi FAX XXXXXX (imprimanta dvs.) din Imprimată şi apoi faceţi clic pe Open Print Queue (Deschidere coadă de imprimare) - Utility (Utilitar).
Pentru Mac OS X 10.4.x
1. Faceţi dublu clic pe unitatea de disc - Applications (Aplicaţii) - Utilities (Utilitare) - Printer Setup Utility (Utilitar de configurare imprimantă).
2. Selectaţi FAX XXXXXX (imprimanta dvs.) din Imprimată şi apoi faceţi clic pe Utility (Utilitar).

Deschideţi Address Book (Agendă) sau Contacts (Contacte)

Afişează lista de contacte pentru Address Book (Agendă) sau Contacts (Contacte) şi vă permite să specificaţi un destinatar din listă. Pentru Mac OS 10.7.x sau versiuni ulterioare, trebuie să sincronizaţi în prealabil aplicaţia Address Book (Agendă) sau Contacts (Contacte) folosind FAX Utility pentru a afişa conţinut. Pentru mai multe informaţii, consultaţi ajutorul pentru FAX Utility.
Address Book (Agendă) şi Contacts (Contacte) sunt aplicaţii de tip agendă pentru Mac OS X. Aplicaţia Address Book (Agendă) şi Contacts (Contacte) au propriile lor funcţii de import/export. Aceste aplicaţii sunt disponibile în directorul Applications (Aplicaţii). Pentru mai multe informaţii, consultaţi ajutorul furnizat împreună cu aplicaţia.
Sus 

+ / -

Butonul + este disponibil la introducerea informaţiilor despre destinatar în secţiunea Add (Adăugare). Faceţi clic pe + pentru a adăuga datele în Recipient List (Listă destinatari).
Butonul - este disponibil dacă selectaţi informaţiile unui destinatar din Recipient List (Listă destinatari). Faceţi clic pe - pentru a şterge datele din listă.
Sus 

Add (Adăugare)

Adaugă informaţii despre destinatar, cum ar fi numele şi numărul de fax.
Introduceţi informaţiile necesare şi apoi faceţi clic pe +.
Element
Explicaţie
Name (Nume)*1
Introduceţi numele destinatarului.
Puteţi introduce până la 60 de caractere UTF-8 (codul de caractere).
Company/Corp. (Companie/Corp.)
Introduceţi numele companiei destinatarului.
Puteţi introduce până la 60 de caractere UTF-8 (codul de caractere).
Fax Number (Număr de fax)*2
Introduceţi numărul de fax al destinatarului.
Dacă doriţi să întrerupeţi puţin imprimanta (faxul) înainte să formaţi numărul, introduceţi o liniuţă (-) la începutul numărului de fax. Liniuţele (-) introduse între cifre sunt şterse automat.
de ex. -81-123-45-6789 -81123456789
*1 În cazul în care nu se introduce niciun Name (Nume), se afişează un mesaj de confirmare.
*2 Element obligatoriu.

External Access Prefix (Prefix acces extern)

Introduceţi un număr de acces extern în cazul în care linia de fax conectată necesită un prefix de acces extern. De exemplu, dacă linia este conectată la un sistem PBX (Private Branch Exchange), care este folosit de majoritate companiilor, iar dvs. trebuie să formaţi un prefix de acces extern cum ar fi 0 sau 9 pentru a accesa o linie exterioară, introduceţi numărul prefixului.
Dacă imprimanta dvs. are o setare de tip linie PBX şi a fost setată să utilizeze # (diez) în locul introducerii prefixului exact, introduceţi # (diez).
La trimiterea unui fax către mai mulţi destinatari, imprimanta (faxul) formează numărul de prefix înainte de a forma fiecare număr de fax.
