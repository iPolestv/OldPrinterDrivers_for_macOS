External Access Prefix (קידומת גישה מבחוץ)
הגדרת הנמען לפקס דרך התפריט Recipient Settings (הגדרות הנמענים).
הערה:
למידע נוסף על שליחת פקסים מהמחשב, עיין בעזרה של FAX Utility. פעל על פי השלבים הבאים כדי לפתוח את FAX Utility.
עבור Mac OS X 10.6.x ואילך
1. לחץ על Apple - System Preferences (העדפות מערכת) - Print & Scan (הדפסה וסריקה) או Print & Fax (הדפסה ופקס).
2. בחר את FAX XXXXXX (המדפסת) בשדה Printer (מדפסת), ולחץ על Options & Supplies (אפשרויות ואספקה) - Utility (כלי עזר) - Open Printer Utility (פתח את כלי העזר למדפסת).
עבור Mac OS X 10.5.x
1. לחץ על Apple - System Preferences (העדפות מערכת) - Print & Fax (הדפסה ופקס).
2. בחר את FAX XXXXXX (המדפסת) בשדה Printer (מדפסת), ולחץ על Open Print Queue (פתח תור מדפסת) - Utility (כלי עזר).
עבור Mac OS X 10.4.x
1. לחץ לחיצה כפולה על הכונן הקשיח - Applications (יישומים) - Utilities (כלי עזר) - Printer Setup Utility (כלי העזר להגדרת המדפסת).
2. בחר את FAX XXXXXX (המדפסת) בשדה Printer (מדפסת), ולחץ על Utility (כלי עזר).

Recipient List (רשימת הנמענים)

רשימה של פרטי הנמענים לשליחה.
עליון 

פתח את Address Book (פנקס הכתובות) או את Contacts (אנשי הקשר).

הצגת רשימת אנשי הקשר של Address Book (פנקס הכתובות) או של Contacts (אנשי הקשר) כדי לציין נמען מהרשימה. עבור Mac OS 10.7.x ואילך עליך לסנכרן את Address Book (פנקס הכתובות) או את Contacts (אנשי הקשר) מראש באמצעות FAX Utility כדי להציג את התוכן. למידע נוסף עיין בעזרה של FAX Utility.
Address Book (פנקס הכתובות) וכן Contacts (אנשי הקשר) הם יישומים המשמשים כספר טלפונים של Mac OS X. ליישומים Address Book (פנקס הכתובות) או Contacts (אנשי הקשר) יש פונקציות יבוא/יצוא משלהם. ניתן למצוא יישומים אלה בתיקייה Applications (יישומים). למידע נוסף עיין בעזרה של היישום.
עליון 

+ / -

+ זמין כשמזינים את פרטי הנמען בחלק Add (הוסף). לחץ על + כדי להוסיף את הנתונים אל Recipient List (רשימת הנמענים).
- זמין אם בוחרים את פרטי הנמענים מתוך Recipient List (רשימת הנמענים). לחץ על - כדי למחוק את הנתונים מהרשימה.
עליון 

Add (הוסף)

הוספת פרטי הנמען, כגון שם ומספר פקס.
הזן את המידע הדרוש ולחץ על +.
פריט
הסבר
Name (שם)*1
הזן את השם של הנמען.
ניתן להזין עד 60 תווים ב-UTF-8 (קוד תווים).
Company/Corp. (חברה/ארגון)
הזן את שם החברה של הנמען.
ניתן להזין עד 60 תווים ב-UTF-8 (קוד תווים).
Fax Number (מספר פקס)*2
הזן את מספר הפקס של הנמען.
אם ברצונך להשהות את המדפסת (הפקס) לרגע לפני חיוג המספר, הזן מקף (-) בתחילת מספר הפקס. כל מקף (-) שתוסיף בין מספרים יימחק אוטומטית.
לדוגמה: ‎-81-123-45-6789 -81123456789
*1 אם לא מוזן אף Name (שם) מוצגת הודעת אישור.
*2 פריט חובה.

External Access Prefix (קידומת גישה מבחוץ)

אם כדי לחייג מקו הטלפון של הפקס המחובר צריך לחייג קידומת גישה לקו חוץ הזן כאן את קידומת הגישה. לדוגמה, אם הקו מחובר למרכזיה פרטית (PBX), כמו במרבית החברות, ועליך לחייג קידומת גישה לקו חוץ כגון 0 או 9, הזן את הקידומת.
אם המדפסת מחוברת לקו במרכזיה פרטית (PBX) שהוגדר להשתמש ב-# (סולמית) במקום קידומת חיוג, הזן # (סולמית).
אם אתה שולח פקס ליותר מנמען אחד, המדפסת (הפקס) תחייג את הקידומת לפני חיוג לכל אחד ממספרי הפקס השונים.
