Permite configurar el destinatario del fax en el menú Recipient Settings (Ajustes destinatario).
Nota:
Consulte la ayuda de FAX Utility para obtener más información sobre el envío de faxes desde un ordenador. Siga los pasos indicados a continuación para iniciar FAX Utility.
En Mac OS X 10.6.x o posterior
1. Haga clic en Apple - System Preferences (Preferencias del Sistema) - Print & Scan (Impresión y escáner) o Print & Fax (Impresión y fax).
2. Seleccione FAX XXXXXX (su impresora) en Printer (Impresora) y, a continuación, haga clic en Options & Supplies (Opciones y recambios) - Utility (Utilidad) - Open Printer Utility (Abrir Utilidad de Impresora).
En Mac OS X 10,5.x
1. Haga clic en Apple - System Preferences (Preferencias del Sistema) - Print & Fax (Impresión y fax).
2. Seleccione FAX XXXXXX (su impresora) en Printer (Impresora) y, a continuación, haga clic en Open Print Queue (Abrir cola de impresión) - Utility (Utilidad).
En Mac OS X 10.4.x
1. Haga doble clic en Hard Disk (Disco duro) - Applications (Aplicaciones) - Utilities (Utilidades) - Printer Setup Utility (Utilidad Configuración Impresoras).
2. Seleccione FAX XXXXXX (su impresora) en Printer (Impresora) y, a continuación, haga clic en Utility (Utilidad).

Recipient List (Lista destinatarios)

Indica los datos de los destinatarios del envío.
Principio 

Abrir Address Book (Agenda) o Contacts (Contactos)

Muestra la lista de contactos para Address Book (Agenda) o Contacts (Contactos) y permite especificar un destinatario de la lista. Para Mac OS 10.7.x o posterior, debe sincronizar Address Book (Agenda) o Contacts (Contactos) por adelantado utilizando FAX Utility para mostrar el contenido. Consulte la ayuda de FAX Utility para obtener más información.
Address Book (Agenda) y Contacts (Contactos) son aplicaciones de agenda para Mac OS X. Las aplicaciones Address Book (Agenda) o Contacts (Contactos) tienen sus propias funciones de importación y exportación. Encontrará estas aplicaciones en la carpeta Applications (Aplicaciones). Para más información, consulte la ayuda que se adjunta con la aplicación.
Principio 

+ / -

+ está disponible cuando se escriben datos de destinatarios en la sección Add (Añadir). Haga clic en + para añadir los datos a la Recipient List (Lista destinatarios).
- está disponible si se seleccionan los datos de destinatarios en la Recipient List (Lista destinatarios). Haga clic en - para borrar los datos de la lista.
Principio 

Add (Añadir)

Añade los datos del destinatario (el nombre y el número de fax, por ejemplo).
Escriba los datos deseados y haga clic en +.
Opción
Explicación
Name (Nombre)*1
Escriba el nombre del destinatario.
Se admiten 60 caracteres como máximo en UTF-8 (código de caracteres).
Company/Corp (Empresa/Corp.)
Escriba el nombre de la empresa del destinatario.
Se admiten sesenta caracteres como máximo en UTF-8 (código de caracteres).
Fax Number (Número de fax)*2
Escriba el número de fax del destinatario.
Si desea que la impresora (fax) haga una breve pausa antes de marcar el número, inserte un guión (-) al principio del número de fax. Todos los guiones (-) insertados entre números se borrarán automáticamente.
Por ej.: -81-123-45-6789 -81123456789
*1 Si no se escribe ningún Name (Nombre), aparecerá un mensaje de confirmación.
*2 Dato obligatorio.
Principio 

External Access Prefix (Prefijo de acceso al exterior)

Escriba el número de acceso al exterior si la línea a la que está conectado el fax exige un prefijo para llamar al exterior. Por ejemplo: si la línea está conectada a una centralita, lo que ocurre en muchas empresas, y usted tiene que marcar un prefijo (por ejemplo: 0 o 9) para acceder a una línea exterior, escriba aquí ese prefijo.
Si su impresora tiene un tipo de línea PBX y se ha configurado para utilizar # (almohadilla) en lugar de indicar el prefijo exacto, escriba # (almohadilla).
Si va a enviar un fax a varios destinatarios, la impresora (fax) marcará ese prefijo antes de cada número de fax.
