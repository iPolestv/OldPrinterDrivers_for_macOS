Ορίζει τον παραλήπτη φαξ στο μενού Recipient Settings (Ρυθμίσεις παραλήπτη).
Σημείωση:
Δείτε τη βοήθεια του FAX Utility για περισσότερες πληροφορίες σχετικά με την αποστολή φαξ από υπολογιστή. Ακολουθήστε τα παρακάτω βήματα για να εκκινήσετε το FAX Utility.
Για Mac OS X 10.6.x ή νεότερη έκδοση
1. Κάντε κλικ στο Apple - System Preferences (Προτιμήσεις συστήματος) - Print & Scan (Εκτύπωση & σάρωση) ή Print & Fax (Εκτύπωση & φαξ).
2. Επιλέξτε ΦΑΞ XXXXXX (ο εκτυπωτής σας) από την επιλογή Printer (Εκτυπωτής) και κατόπιν κάντε κλικ στο Options & Supplies (Επιλογές & τροφοδοσία) - Utility (Βοηθητικό πρόγραμμα) - Open Printer Utility (Άνοιγμα βοηθητικού προγράμματος εκτυπωτή).
Για Mac OS X 10.5.x
1. Κάντε κλικ στο Apple - System Preferences (Προτιμήσεις συστήματος) - Print & Fax (Εκτύπωση & φαξ).
2. Επιλέξτε ΦΑΞ XXXXXX (ο εκτυπωτής σας) από τον Εκτυπωτή και κατόπιν κάντε κλικ στο Open Print Queue (Άνοιγμα εργασιών εκτύπωσης σε ουρά) - Utility (Βοηθητικό πρόγραμμα).
Για Mac OS X 10.4.x
1. Κάντε διπλό κλικ σε Σκληρός δίσκος - Applications (Εφαρμογές) - Utilities (Βοηθητικά προγράμματα) - Printer Setup Utility (Βοηθητικό πρόγραμμα ρύθμισης εκτυπωτή).
2. Επιλέξτε ΦΑΞ XXXXXX (ο εκτυπωτής σας) από τον Εκτυπωτή και κατόπιν κάντε κλικ στο Utility (Βοηθητικό πρόγραμμα).

Recipient List (Λίστα παραληπτών)

Περιλαμβάνει τις πληροφορίες παραληπτών προς αποστολή.
Επιστροφή στην αρχή 

Ανοίξτε το Address Book (Βιβλίο διευθύνσεων) ή τις Contacts (Επαφές)

Προβάλλει τον κατάλογο επαφών για το Address Book (Βιβλίο διευθύνσεων) ή τις Contacts (Επαφές) και σας επιτρέπει να καθορίσετε έναν παραλήπτη από τον κατάλογο. Για Mac OS 10.7.x ή νεότερη έκδοση, πρέπει να συγχρονίσετε το Address Book (Βιβλίο διευθύνσεων) ή τις Contacts (Επαφές) εκ των προτέρων, χρησιμοποιώντας το FAX Utility για προβολή των περιεχομένων. Δείτε τη βοήθεια FAX Utility για περισσότερες πληροφορίες.
Τα Address Book (Βιβλίο διευθύνσεων) και Contacts (Επαφές) αποτελούν εφαρμογές βιβλίου διευθύνσεων για το Mac OS X. Οι εφαρμογές Address Book (Βιβλίο διευθύνσεων) ή Contacts (Επαφές) έχουν τις δικές τους λειτουργίες εισαγωγής/εξαγωγής. Μπορείτε να βρείτε αυτές τις εφαρμογές στο φάκελο Applications (Εφαρμογές). Δείτε τη βοήθεια που παρέχεται με την εφαρμογή για περισσότερες πληροφορίες.
Επιστροφή στην αρχή 

+ / -

Το + είναι διαθέσιμο όταν εισέρχεστε στις πληροφορίες παραλήπτη στην ενότητα Add (Προσθήκη). Κάντε κλικ στο + για να προσθέσετε τα δεδομένα στο Recipient List (Λίστα παραληπτών).
Το - είναι διαθέσιμο εάν επιλέξετε τις πληροφορίες παραλήπτη στον Recipient List (Λίστα παραληπτών). Κάντε κλικ στο - για να διαγράψετε τα δεδομένα από τον κατάλογο.
Επιστροφή στην αρχή 

Add (Προσθήκη)

Προσθέτει τις πληροφορίες παραλήπτη, όπως όνομα και αριθμό φαξ.
Εισαγάγετε τις απαραίτητες πληροφορίες και κατόπιν κάντε κλικ στο +.
Στοιχείο
Επεξήγηση
Name (Όνομα)*1
Εισαγάγετε το όνομα παραλήπτη.
Μπορείτε να εισαγάγετε έως 60 χαρακτήρες σε UTF-8 (κώδικας χαρακτήρων).
Company/Corp. (Εταιρεία)
Εισαγάγετε το όνομα εταιρείας του παραλήπτη.
Μπορείτε να εισαγάγετε έως 60 χαρακτήρες σε UTF-8 (κώδικας χαρακτήρων).
Fax Number (Αριθμός φαξ)*2
Εισαγάγετε τον αριθμό φαξ του παραλήπτη.
Εάν επιθυμείτε παύση του εκτυπωτή (φαξ) για μία στιγμή προτού πληκτρολογήσετε τον αριθμό, εισαγάγετε παύλα (-) στην αρχή του αριθμού φαξ. Τυχόν παύλες (-) που έχουν εισαχθεί μεταξύ αριθμών διαγράφονται αυτόματα.
π.χ. -81-123-45-6789 -81123456789
*1 Προβάλλεται μήνυμα επιβεβαίωσης εάν δεν έχει εισαχθεί το Name (Όνομα).
*2 Απαραίτητο στοιχείο.
Επιστροφή στην αρχή 

External Access Prefix (Πρόθεμα εξωτερικής πρόσβασης)

Εισαγάγετε αριθμό εξωτερικής πρόσβασης εάν η συνδεδεμένη γραμμή φαξ απαιτεί πρόθεμα εξωτερικής πρόσβασης. Παραδείγματος χάριν, εάν η γραμμή είναι συνδεδεμένη σε σύστημα PBX (ιδιωτική διακλάδωση ανταλλαγής) το οποίο χρησιμοποιείται από τις περισσότερες εταιρείες, και πρέπει να καλέσετε πρόθεμα εξωτερικής πρόσβασης όπως 0 ή 9 για να έχετε εξωτερική γραμμή, εισαγάγετε τον αριθμό προθέματος.
Εάν ο εκτυπωτής σας διαθέτει ρύθμιση γραμμής τύπου PBX και έχει ρυθμιστεί για χρήση του # (δίεση) αντί του ακριβούς κωδικού προθέματος, εισαγάγετε # (δίεση).
Κατά την αποστολή φαξ σε περισσότερους από έναν παραλήπτες, ο εκτυπωτής (φαξ) καλεί τον αριθμό προθέματος προτού καλέσει κάθε έναν αριθμό φαξ.
