หมายเหตุ:
โปรดดูคำแนะนำ FAX Utility สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการส่งแฟกซ์จากคอมพิวเตอร์ ปฏิบัติตามขั้นตอนด้านล่างเพื่อเริ่ม FAX Utility
สำหรับ Mac OS X 10.6.x หรือใหม่กว่า
1. คลิกที่ Apple - System Preferences (รายการตัวเลือกระบบ) - Print & Scan (พิมพ์ & สแกน) หรือ Print & Fax (พิมพ์ & แฟกซ์)
2. เลือก FAX XXXXXX (เครื่องพิมพ์ของคุณ) จาก Printer (เครื่องพิมพ์) จากนั้นคลิก Options & Supplies (ตัวเลือก & วัสดุการพิมพ์) - Utility (ยูทิลิตี้) - Open Printer Utility (เปิดยูทิลิตี้เครื่องพิมพ์)
สำหรับ Mac OS X 10.5.x
1. คลิกที่ Apple - System Preferences (รายการตัวเลือกระบบ) - Print & Fax (พิมพ์ & แฟกซ์)
2. เลือก FAX XXXXXX (เครื่องพิมพ์ของคุณ) จากเครื่องพิมพ์ จากนั้นคลิกที่ Open Print Queue (เปิดลำดับการพิมพ์) - Utility (ยูทิลิตี้)
สำหรับ Mac OS X 10.4.x
1. ดับเบิลคลิกที่ฮาร์ดดิสก์ - Applications (แอพพลิเคชัน) - Utilities (ยูทิลิตี้) - Printer Setup Utility (ยูทิลิตี้การตั้งค่าเครื่องพิมพ์)
2. เลือก FAX XXXXXX (เครื่องพิมพ์ของคุณ) จากเครื่องพิมพ์ จากนั้นคลิกที่ Utility (ยูทิลิตี้)

เปิด Address Book (สมุดที่อยู่) หรือ Contacts (รายชื่อ)

แสดงรายการรายชื่อสำหรับ Address Book (สมุดที่อยู่) หรือ Contacts (รายชื่อ) และอนุญาตให้คุณระบุผู้รับจากรายการ สำหรับ Mac OS 10.7.x หรือใหม่กว่า คุณจำเป็นต้องซิงโครไนซ์ Address Book (สมุดที่อยู่) หรือ Contacts (รายชื่อ) ล่วงหน้าโดยใช้ FAX Utility เพื่อแสดงเนื้อหา โปรดดทีู่คำแนะนำ FAX Utility สำหรับข้อมูลเพิ่มเติม
Address Book (สมุดที่อยู่) และ Contacts (รายชื่อ) เป็นแอพพลิเคชันสมุดที่อยู่สำหรับ Mac OS X และแอพพลิเคชัน Address Book (สมุดที่อยู่) หรือ Contacts (รายชื่อ) มีฟังก์ชันนำเข้า/ส่งออกของตัวเอง คุณสามารถพบแอพพลิเคชันเหล่านี้ได้ในโฟลเดอร์ Applications (แอพพลิเคชัน) โปรดดูคำแนะนำที่มาพร้อมกับแอพพลิเคชัน สำหรับข้อมูลเพิ่มเติม

External Access Prefix (รหัสนำหน้าใช้งานสายนอก)

ป้อนหมายเลขการใช้งานสายนอกถ้าสายแฟกซ์ที่เชื่อมต่ออยู่จำเป็นต้องมีรหัสนำหน้าใช้งานสายนอก ตัวอย่างเช่น ถ้าสายมีการเชื่อมต่ออยู่กับระบบ PBX (ตู้สาขาส่วนตัว) ซึ่งใช้ในบริษัทส่วนใหญ่ และคุณจำเป็นต้องหมุนรหัสนำหน้าใช้งานสายนอก เช่น 0 หรือ 9 เพื่อโทรออกสายนอกคุณต้องป้อนรหัสนำหน้า
ถ้าเครื่องพิมพ์ของคุณมีการตั้งค่าชนิดสาย PBX และถูกตั้งค่าให้ใช้ # (เครื่องหมายสี่เหลี่ยม) คุณต้องป้อน # (เครื่องหมายสี่เหลี่ยม) แทนการป้อนรหัสนำหน้าที่แท้จริง
ในขณะที่ส่งแฟกซ์ไปยังผู้รับมากกว่าหนึ่งคน เครื่องพิมพ์ (แฟกซ์) จะหมุนรหัสนำหน้าก่อนที่จะโทรออกไปยังหมายเลขแฟกซ์แต่ละหลายเลข
