Zur Einstellung des Faxempfängers im Menü Recipient Settings (Empfängereinstellungen).
Hinweis:
Weitere Informationen zum Senden von Faxen von einem Computer aus finden Sie in der FAX Utility-Hilfe. Folgen Sie den hier beschriebenen Schritten, um die FAX Utility zu starten.
Für Mac OS X 10.6.x oder später
1. Klicken Sie auf Apple - System Preferences (Systemeinstellungen) - Print & Scan (Drucken & Scannen) oder Print & Fax (Drucken & Faxen).
2. Wählen Sie unter Printer (Drucker) FAX XXXXXX (Ihren Drucker), und klicken Sie dann auf Options & Supplies (Optionen & Zubehör) - Utility (Dienstprogramm) - Open Printer Utility (Druckerdienstprogramm öffnen).
Mac OS X 10.5.x
1. Klicken Sie auf Apple - System Preferences (Systemeinstellungen) - Print & Fax (Drucken & Faxen).
2. Wählen Sie unter "Printer" (Drucker) die Option "FAX XXXXXX" (Ihren Drucker) und klicken Sie auf Open Print Queue (Drucker-Warteliste öffnen) - Utility (Dienstprogramm).
Mac OS X 10.4.x
1. Doppelklicken Sie auf "Hard disk (Festplatte)" - Applications (Programme) - Utilities (Dienstprogramme) - Printer Setup Utility (Dienstprogramm zur Druckerkonfiguration).
2. Wählen Sie unter "Printer" (Drucker) die Option "FAX XXXXXX" (Ihren Drucker) und klicken Sie anschließend auf Utility (Dienstprogramm).

Recipient List (Empfängerliste)

Zeigt die Informationen der Empfänger, an die gesendet werden soll.
Oben 

Öffnen Sie Address Book (Adressbuch) oder Contacts (Kontakte)

Zeigt die Kontaktliste für Address Book (Adressbuch) oder Contacts (Kontakte) an und ermöglicht die Auswahl eines Empfängers aus der Liste. Unter Mac OS 10.7.x oder höher müssen Sie Address Book (Adressbuch) oder Contacts (Kontakte) zuvor mit FAX Utility synchronisieren, um die Inhalte anzuzeigen. Weitere Informationen finden Sie in der FAX Utility-Hilfe.
Address Book (Adressbuch) und Contacts (Kontakte) sind Adressbuchanwendungen unter Mac OS X. Die Anwendungen Address Book (Adressbuch) und Contacts (Kontakte) verfügen über eigene Funktionen zum Importieren und Exportieren. Sie finden diese Anwendungen im Ordner Applications (Programme). Weitere Informationen erhalten Sie in der Hilfe zur jeweiligen Anwendung.
Oben 

+ / -

+ kann bei der Eingabe von Empfängerdaten im Abschnitt Add (Hinzufügen) verwendet werden. Klicken Sie auf +, um Daten zur Recipient List (Empfängerliste) hinzuzufügen.
- kann bei der Auswahl von Empfängerdaten in der Recipient List (Empfängerliste) verwendet werden. Klicken Sie auf -, um die Daten aus der Liste zu löschen.
Oben 

Add (Hinzufügen)

Fügt die Empfängerdaten hinzu, wie z. B. Name und Faxnummer.
Geben Sie die erforderlichen Daten ein und klicken Sie dann auf +.
Eintrag
Erklärung
Name*1
Geben Sie den Namen des Empfängers ein.
Sie können bis zu 60 Zeichen im Zeichencode UTF-8 eingeben.
Company/Corp (Firma/Org.)
Geben Sie den Firmennamen des Empfängers ein.
Sie können bis zu 60 Zeichen im Zeichencode UTF-8 eingeben.
Fax Number (Faxnummer)*2
Geben Sie die Faxnummer des Empfängers ein.
Wenn der Drucker (Fax) mit einer kurzen Verzögerung die Nummer wählen soll, fügen Sie am Anfang der Faxnummer einen Bindestrich (-) ein. Alle Bindestriche (-) zwischen den Ziffern werden automatisch gelöscht.
Beispiel: -81-123-45-6789 -81123456789
*1 Wird kein Name eingegeben, erscheint eine Bestätigungsmeldung.
*2 Eintrag erforderlich.
Oben 

External Access Prefix (Vorwahl für externen Zugriff)

Geben Sie eine externe Zugriffsnummer ein, wenn für die verbundene Faxleitung eine Vorwahl für externen Zugriff (Amtsholung) benötigt wird. Wenn die Leitung z. B. mit einer Nebenstellenanlage verbunden ist, wie es in den meisten Unternehmen der Fall ist, und für eine Amtsleitung muss zuerst eine Vorwahl (0 oder 9) gewählt werden, geben Sie hier die Vorwahl für externen Zugriff ein.
Wenn Ihr Drucker über eine Sammelanschlusseinstellung verfügt und zur Verwendung von # (Raute) statt einer bestimmten Vorwahl eingerichtet ist, geben Sie # (Raute) ein.
Wenn ein Fax an mehr als einen Empfänger gesendet wird, wählt der Drucker (Fax) die Vorwahl für externen Zugriff (Amtsholung) und dann erst die einzelnen Faxnummern.
