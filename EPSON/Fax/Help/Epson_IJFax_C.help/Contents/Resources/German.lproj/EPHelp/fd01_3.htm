Verhindern, dass Faxe an den falschen Empfänger gesendet werden

Um zu verhindern, dass Faxe an einen falschen Empfänger gesendet werden, können Sie festlegen, dass ein Benutzer eine manuell eingegebene Faxnummer zweimal eingeben muss.
Klicken Sie auf Apple - System Preferences (Systemeinstellungen) - Print & Scan (Drucken & Scannen) oder Print & Fax (Drucken & Faxen).
Wählen Sie unter Printer (Drucker) die Option "FAX XXXXXX" (Ihren Drucker) und klicken Sie auf Options & Supplies (Optionen & Zubehör) - Driver (Treiber).
Wählen Sie Enter fax number twice (Geben Sie die Faxnummer zweimal ein).
