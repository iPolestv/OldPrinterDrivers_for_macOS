Uwaga:
Sprawdź pomoc FAX Utility w celu uzyskania dalszych informacji o wysyłaniu faksów z komputera. Wykonaj czynności poniżej w celu uruchomienia FAX Utility.
Dla Mac OS X 10.6.x lub wersji późniejszej
1. Kliknij Apple - System Preferences (Preferencje systemu) - Print & Scan (Drukowanie i skanowanie) lub Print & Fax (Drukowanie i faksowanie).
2. Wybierz FAX XXXXXX (twoja drukarka) z Printer (Drukarki), a następnie kliknij Options & Supplies (Opcje i zaopatrzenie) - Utility (Narzędzie) - Open Printer Utility (Otwórz program drukarki).
Dla Mac OS X 10.5.x
1. Kliknij Apple - System Preferences (Preferencje systemu) - Print & Fax (Drukowanie i faksowanie).
2. Wybierz FAX XXXXXX (twoja drukarka) z opcji Printer (Drukarki), a następnie kliknij Open Print Queue (Otwórz kolejkę drukowania) - Utility (Narzędzie).
Dla Mac OS X 10.4.x
1. Kliknij dwukrotnie dysk twardy - Applications (Aplikacje) - Utilities (Narzędzia) - Printer Setup Utility (Narzędzie konfiguracji drukarki).
2. Wybierz FAX XXXXXX (twoja drukarka) z opcji Printer (Drukarki), a następnie kliknij Utility (Narzędzie).

Otwórz Address Book (Książka adresowa) lub Contacts (Kontakty)

Wyświetlanie listy kontaktów dla Address Book (Książka adresowa) lub Contacts (Kontakty) i umożliwienie określenia odbiorcy z listy. Dla Mac OS 10.7.x lub wersji późniejszej, należy wcześniej zsynchronizować Address Book (Książka adresowa) lub Contacts (Kontakty) z użyciem FAX Utility w celu wyświetlenia zawartości. Sprawdź pomoc FAX Utility w celu uzyskania dalszych informacji.
Address Book (Książka adresowa) i Contacts (Kontakty) to aplikacje książki adresowej dla Mac OS X. Aplikacje Address Book (Książka adresowa) lub Contacts (Kontakty) mają swoje własne funkcje importu/eksportu. Aplikacje te znajdują się w folderze Applications (Aplikacje). W celu uzyskania dalszych informacji, sprawdź pomoc dostarczoną z aplikacją.

+ / -

+ Jest dostępne, po przejściu do informacji o odbiorcy w części Add (Dodaj). Kliknij +, aby dodać dane do Recipient List (Lista adresatów).
- jest dostępna po wybraniu informacji o odbiorcy w Recipient List (Lista adresatów). Kliknij -, aby usunąć dane z listy.

Add (Dodaj)

Dodanie informacji o odbiorcy, takich jak nazwa i numer faksu.
Wprowadź niezbędne informacje, a następnie kliknij +.
Pozycja
Objaśnienie
Name (Nazwa)*1
Wprowadź nazwę odbiorcy.
Wprowadzić można do 60 znaków w UTF-8 (kodowanie znaków).
Company/Corp. (Firma)
Wprowadź nazwę firmy.
Wprowadzić można do 60 znaków w UTF-8 (kodowanie znaków).
Fax Number (Numer faksu)*2
Wprowadź numer faksu odbiorcy.
Aby wstrzymać drukarkę (faks) na chwilę, przed wybraniem numeru, wstaw myślnik (-) na początku numeru faksu. Wszelkie myślniki (-) wstawione pomiędzy numerami są automatycznie usuwane.
np. -81-123-45-6789 -81123456789
*1 Wyświetlany jest komunikat potwierdzenia, jeśli nie została wprowadzona żadna Name (Nazwa).
*2 Wymagana pozycja.

External Access Prefix (Prefiks zewnętrznego dostępu)

Wprowadź numer zewnętrznego dostępu, jeśli podłączona linia faksu wymaga prefiksu zewnętrznego dostępu. Na przykład, jeśli linia jest podłączona do systemu PBX (centrala abonencka), który jest używany przez większość firm i do uzyskania dostępu do zewnętrznej linii wymagane jest wybranie prefiksu zewnętrznego dostępu, takiego jak 0 lub 9, wprowadź numer prefiksu.
Jeśli drukarka ma ustawienie typu linii PBX i została ustawiona na używanie # (krzyżyk) zamiast wprowadzania dokładnego kodu prefiksu, wprowadź # (krzyżyk).
Podczas wysyłania faksu do więcej niż jednego odbiorcy, drukarka (faks) przed każdym numerem faksu wybiera numer prefiksu.
