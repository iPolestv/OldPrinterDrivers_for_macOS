最前面 

開啟通訊錄或聯絡人

顯示〝通訊錄〞或〝聯絡人〞的聯絡人清單，可讓您從清單中指定收件人。若為 Mac OS 10.7.x 以上版本，您必須先使用 FAX Utility 同步〝通訊錄〞或〝聯絡人〞，才能顯示內容。如需詳細資訊，請參考 FAX Utility 輔助說明。
〝通訊錄〞和〝聯絡人〞為 Mac OS X 的通訊錄應用程式。〝通訊錄〞或〝聯絡人〞應用程式具有各自的匯入 / 匯出功能。您可在〝應用程式〞資料夾找到這些應用程式。如需詳細資訊，請參考應用程式提供的輔助說明。
最前面 

+ / -

當在〝新增〞欄位中輸入收件人資訊時，可使用『+』。按下『+』，將資料新增至〝收件人清單〞中。
當在〝收件人清單〞中選取收件人資訊時，則可使用『-』。按下『-』，自清單中刪除資料。
最前面 

新增

新增收件人資訊，如姓名及傳真號碼。
輸入必要資訊，然後按下『+』。
項目
說明
姓名*1
輸入收件人姓名。
您最多可以 UTF-8 (字元碼) 輸入 60 個字元。
公司名稱
輸入收件人的公司名稱。
您最多可以 UTF-8 (字元碼) 輸入 60 個字元。
傳真號碼*2
輸入收件人的傳真號碼。
如果在撥號前，想要先暫停印表機 (傳真) ，請在傳真號碼的開頭前，插入一個連字號 (-)。任何插入於兩個號碼間的連字號 (-) 都將自動刪除。
範例： -81-123-45-6789 -81123456789
*1 若未輸入姓名，則會顯示確認訊息。
*2 必填項目。

外接存取首碼

若連接的傳真電話線需要外接存取首碼，請輸入外接存取首碼。例如，若傳真線路是連接至多數公司所使用的 PBX (企業用戶專用交換機) 系統，而您需要輸入外接存取首碼，如 0 或 9 才可撥出外線電話，則請輸入該首碼。
若您的印表機具有 PBX 線路類型設定並設為使用 # (井字鍵)，而不是使用首碼，則請輸入 # (井字鍵)。
若要傳真給多名收件人，印表機 (傳真) 在撥出每一個傳真號碼前，會加上首碼。
