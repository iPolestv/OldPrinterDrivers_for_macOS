External Access Prefix (Ulkoisen käytön etuliite)
Asettaa faksin vastaanottajat Recipient Settings (Vastaanottaja-asetukset)-valikossa.
Huomautus:
Katso FAX Utility-ohjeesta lisätietoja faksien lähettämisestä tietokoneesta. Toimi seuraavien ohjeiden mukaisesti käynnistääksesi FAX Utility-ohjelman.
Mac OS X 10.6.x tai uudempi
1. Napsauta Apple - System Preferences (Järjestelmäasetukset) - Print & Scan (Tulosta ja skannaa) tai Print & Fax (Tulosta ja faksaa).
2. Valitse FAX XXXXXX (tulostimesi) kohdasta Printer (Tulostin) ja napsauta sitten Options & Supplies (Valinnat ja varusteet) - Utility (Apuohjelma) - Open Printer Utility (Avaa tulostinapuohjelma).
Mac OS X 10.5.x
1. Napsauta Apple - System Preferences (Järjestelmäasetukset) - Print & Fax (Tulosta ja faksaa).
2. Valitse FAX XXXXXX (tulostimesi) kohdasta Printer ja napsauta sitten Open Print Queue (Avaa tulostusjono) - Utility (Apuohjelma).
Mac OS X 10.4.x
1. Kaksoisnapsauta Hard disk - Applications (Sovellukset) - Utilities (Apuohjelmat) - Printer Setup Utility (Tulostinasetusapuohjelma).
2. Valitse FAX XXXXXX (tulostimesi) kohdasta Printer ja napsauta sitten Utility (Apuohjelma).

Avaa Address Book (Osoitekirja) tai Contacts (Yhteystiedot)

Näyttää kohteen Address Book (Osoitekirja) tai Contacts (Yhteystiedot) yhteystietoluettelon ja antaa määrittää luettelosta vastaanottajan. Mac OS 10.7.x tai uudemmassa käyttöjärjestelmässä sinun on synkronoitava Address Book (Osoitekirja) tai Contacts (Yhteystiedot) etukäteen käyttämällä FAX Utility-sovellusta sisältöjen näyttämiseen. Katso FAX Utility-ohjeesta lisätietoja.
Address Book (Osoitekirja) ja Contacts (Yhteystiedot) ovat Mac OS X -käyttöjärjestelmän osoitekirjasovelluksia. Address Book (Osoitekirja) - tai Contacts (Yhteystiedot)-sovelluksilla on omat tuonti-/vientitoimintonsa. Nämä sovellukset sijaitsevat Applications (Sovellukset)-kansiossa. Katso sovelluksen ohjeesta lisätietoja.

+ / -

+ on käytettävissä annettaessa vastaanottajan tietoja Add (Lisää)-osassa. Napsauta + lisätäksesi tietoja Recipient List (Vastaanottajaluettelo).
- on käytettävissä, kun valitset vastaanottajan tietoja Recipient List (Vastaanottajaluettelo). Napsauta - poistaaksesi tietoja luettelosta.
Ylös 

Add (Lisää)

Lisää vastaanottajan tietoja, kuten nimen ja faksinumeron.
Syötä tarvittavat tiedot ja napsauta sitten +.
Kohde
Selitys
Name (Nimi)*1
Syötä vastaanottajan nimi.
Voit syöttää enintään 60 UTF-8-koodin merkkiä (merkkikoodi).
Company/Corp. (Yritys/yhtiö)
Syötä vastaanottajan yrityksen nimi.
Voit syöttää enintään 60 UTF-8-koodin merkkiä (merkkikoodi).
Fax Number (Faksinumero)*2
Syötä vastaanottajan faksinumero.
Jos haluat keskeyttää tulostimen (faksin) hetkeksi ennen numeron valitsemista, liitä yhdysviiva (-) faksinumeron alkuun. Kaikki yhdysviivat (-), jotka on liitetty numeroiden väliin, poistetaan automaattisesti.
esim. -81-123-45-6789 -81123456789
*1 Vahvistusviesti näytetään, jos mitään Name (Nimi) ei ole syötetty.
*2 Vaadittu kohde.

External Access Prefix (Ulkoisen käytön etuliite)

Syötä ulkoisen käytön numero, jos yhdistetty faksilinja vaatii ulkoisen käytön etuliitteen. Jos linja on esimerkiksi liitetty useimpien yritysten käyttämään PBX (private branch exchange) -järjestelmään, ja sinun on valittava ulkoisen käytön etuliite, kuten 0 tai 9, päästäksesi ulkolinjalle, syötä etuliitenumero.
Jos tulostimellasi on PBX-linjatyyppiasetus ja se on asetettu käyttämään #-merkkiä (hashtagia) tarkan etuliitteen asemesta, syötä # (hashtag).
Kun lähetät faksin useammalle kuin yhdelle vastaanottajalle, tulostin (faksi) valitsee ensin etuliitenumeron ennen kunkin faksinumeron valitsemista.
