Setkan penerima faks di dalam menu Recipient Settings (Tetapan Penerima).
Nota:
Lihat bantuan FAX Utility untuk maklumat lanjut tentang faks dihantar dari komputer. Ikuti langkah di bawah untuk memulakan FAX Utility.
Untuk Mac OS X 10.6.x atau kemudian
1. Klik Apple - System Preferences (Keutamaan Sistem) - Print & Scan (Cetak & Imbas) atau Print & Fax (Cetak & Faks).
2. Pilih FAX XXXXXX (pencetak anda) daripada Printer (Pencetak) dan kemudian klik Options & Supplies (Pilihan & Bekalan) - Utility (Utiliti) - Open Printer Utility (Buka Utiliti Pencetak).
Untuk Mac OS X 10.5.x
1. Klik Apple - System Preferences (Keutamaan Sistem) - Print & Fax (Cetak & Faks).
2. Pilih FAX XXXXXX (pencetak anda) daripada Pencetak dan kemudian klik Open Print Queue (Buka Gilir Cetakan) - Utility (Utiliti).
Untuk Mac OS X 10.4.x
1. Klik cakera Keras dua kali - Applications (Aplikasi) - Utilities (Utiliti) - Printer Setup Utility (Utiliti Persediaan Pencetak).
2. Pilih FAX XXXXXX (pencetak anda) daripada Pencetak dan kemudian klik Utility (Utiliti).

Buka Address Book (Buku Alamat) atau Contacts (Kenalan)

Paparkan senarai kenalan untuk Address Book (Buku Alamat) atau Contacts (Kenalan) dan membenarkan anda untuk menentukan penerima daripada senarai. Untuk Mac OS 10.7.x atau kemudian, anda perlu segerakkan dahulu Address Book (Buku Alamat) atau Contacts (Kenalan) menggunakan FAX Utility untuk memaparkan kandungan. Lihat bantuan FAX Utility untuk maklumat lanjut.
Address Book (Buku Alamat) dan Contacts (Kenalan) adalah aplikasi buku alamat untuk Mac OS X. Aplikasi Address Book (Buku Alamat) atau Contacts (Kenalan) mempunyai fungsi import/eksport yang tersendiri. Aplikasi ini dapat didapati di dalam folder Applications (Aplikasi). Lihat bantuan aplikasi yang disediakan untuk maklumat lanjut.

+ / -

+ tersedia semasa maklumat penerima dimasukkan pada seksyen Add (Tambah). Klik + untuk menambahkan data kepada Recipient List (Senarai Penerima).
- tersedia jika maklumat penerima terpilih dalam Recipient List (Senarai Penerima). Klik - untuk padamkan data dari senarai.
Atas 

Add (Tambah)

Tambahkan maklumat penerima seperti nama dan nombor faks.
Masukkan maklumat yang perlu dan kemudian klik +.
Item
Penerangan
Name (Nama)*1
Masukkan nama penerima.
Sehingga 60 aksara boleh anda masukkan dalam UTF-8 (kod aksara).
Company/Corp. (Syarikat/Perb.)
Masukkan nama syarikat penerima.
Sehingga 60 aksara boleh anda masukkan dalam UTF-8 (kod aksara).
Fax Number (Nombor Faks)*2
Masukkan nombor faks penerima.
Jika anda ingin menjeda pencetak (faks) seketika sebelum nombor didail, selitkan tanda sempang (-) di permulaan nombor faks. Sebarang selitan tanda sempang (-) di antara nombor akan dipadamkan secara automatik.
cth. -81-123-45-6789 -81123456789
*1 Mesej pengesahan dipaparkan jika tiada Name (Nama) dimasukkan.
*2 Item Diperlukan.

External Access Prefix (Awalan Akses Luaran)

Masukkan nombor akses luaran jika talian faks memerlukan awalan akses luaran. Sebagai contoh, jika talian disambungkan ke PBX sistem (pertukaran cabang peribadi), yang digunakan oleh kebanyakan syarikat dan awalan akses luaran perlu didail seperti 0 atau 9 untuk mendapatkan talian luar, masukkan nombor awalan.
Jika pencetak mempunyai tetapan talian PBX dan telah diset untuk guna # (cincangan) dan bukannya memasukkan kod awalan yang tepat, masukkan # (cincangan).
Semasa menghantar faks kepada lebih daripada satu penerima, pencetak (faks) mendail nombor awalan sebelum mendail setiap nombor faks.
