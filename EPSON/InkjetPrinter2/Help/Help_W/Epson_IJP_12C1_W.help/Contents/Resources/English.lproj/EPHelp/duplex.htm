Available options differ depending on the product's model.

Two-sided Printing

Select to print on both sides of the paper. When printing on both sides, select a binding position for the printed paper.
Off
Select this if you are not printing on both sides.
Long-Edge binding
Select this if you are printing on both sides in portrait format (vertical).
Short-Edge binding
Select this if you are printing on both sides in landscape format (horizontal).

Face Down Printing

Outputs the printouts face down. Printouts are stacked in the same order as the document.
Top 

Document Type

Select the document type to set the appropriate print density and additional ink drying time depending on the content of the printouts.
If you want to adjust the print density or additional ink drying time manually, click the arrow mark next to Adjustments.
Top 

Adjustments

Display the following options by clicking the arrow mark.

Print Density

Adjusts the print density for finer printing when printing photos with dense data.

Increased Ink Drying Time

Adjusts the ink drying time when printing photos with dense data to make sure the fine printing is thoroughly dried.
