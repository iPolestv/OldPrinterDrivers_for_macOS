Paper Source

Select a paper source. Available settings differ depending on the product's model.
To use Auto Select, first select the paper size from this product's control panel or in Printer Settings from the Epson Printer Utility. Accessing the Epson Printer Utility:
  • For Mac OS X 10.6 and 10.7, click System Preferences on the Apple menu and then click Print & Fax or Print & Scan. Select your printer from the Printers list box, click Options & Supplies, Utility, and then click Open Printer Utility.
  • For Mac OS X 10.5, click System Preferences on the Apple menu and then click Print & Fax. Select your printer from the Printers list box, click Open Print Queue, and then click Utility.
When you are using both cassettes as one large capacity cassette, select Cassette 1 and set the following conditions.
  • Make sure the same type and size of paper is loaded in both cassettes.
  • Select Automatic Cassette Switching. Click System Preferences on the Apple menu, click Print & Fax or Print & Scan, Options & Supplies, and then select the Driver tab.

Print Quality

Select the desired Print Quality setting from the pop-up menu. The availability of Print Quality setting depends on the Media Type setting you select. Select the Media Type before you set the Print Quality.

Borderless

Select for borderless printing. This feature is only available when the selected paper type, size, and print quality settings support borderless printing.

Expansion

To reduce the amount of cropping around the edges of your image, select Min or Mid. If you see a white line on the edge of the print, select Max.

Grayscale

Select to print in monochrome. This setting may not be available depending on other settings.

Mirror Image

This setting prints an image as it would appear in a mirror; text and graphics appear backwards.
