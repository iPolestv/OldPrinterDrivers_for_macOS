Source papier

Sélectionnez une source papier. Les paramètres disponibles varient en fonction du modèle.
Pour utiliser Auto Select, sélectionnez d'abord la taille papier à partir du panneau de commande de l'appareil ou dans les paramètres de l'imprimante à partir de l'Utilitaire de l'imprimante Epson. Pour accéder à l'Utilitaire de l'imprimante Epson :
  • Pour Mac OS X 10.6 et 10.7, cliquez sur Préférences Système dans le menu Pomme, puis cliquez sur Imprimantes et fax ou Imprimantes et scanners. Sélectionnez votre imprimante dans la liste d'imprimantes, cliquez sur Options et fournitures, Utilitaire, puis cliquez sur Ouvrir l'utilitaire de l'imprimante.
  • Pour Mac OS X 10.5, cliquez sur Préférences Système dans le menu Pomme, puis cliquez sur Imprimantes et fax. Sélectionnez votre imprimante dans la liste d'imprimantes, cliquez sur Ouvrir la liste d'attente d'impression, puis cliquez sur Utilitaire.
Lorsque vous utilisez les deux bacs comme un seul bac à grande capacité, sélectionnez Bac 1 et faites la configuration suivante.
  • Assurez-vous que le papier chargé dans les deux bacs est du même type et de la même taille.
  • Sélectionnez Commutation auto. des bacs. Cliquez sur Préférences Système dans le menu Pomme, puis cliquez sur Imprimantes et fax ou Imprimantes et scanners, Options et fournitures et sélectionnez l'onglet Gestionnaire.

Qualité

Sélectionnez le paramètre Qualité souhaité dans le menu contextuel. Les valeurs disponibles pour le paramètre Qualité dépendent du paramètre Support sélectionné. Définissez le paramètre Support avant de sélectionner la Qualité.

Sans marges

Sélectionnez ce paramètre pour une impression sans marges. Cette fonctionnalité est disponible seulement si les paramètres du type de papier, de la taille et de la qualité d'impression prennent en charge l'impression sans marges.

Agrandissement

Pour réduire la taille de la section rognée sur les côtés de l'image, sélectionnez Min. ou Interm. Si une ligne blanche apparaît sur le bord de l'impression, sélectionnez Max.

Niveaux de gris

Sélectionnez cette option pour imprimer en monochrome. Selon les paramètres selectionnés, il est possible que cette option ne soit pas disponible.

Image miroir

Ce paramètre permet d'imprimer une image inversée de l'original. Le texte et les graphiques s'affichent à l'envers.
